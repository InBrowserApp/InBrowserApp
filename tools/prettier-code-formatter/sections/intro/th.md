## Prettier ตัวจัดรูปแบบโค้ดคืออะไร?

Prettier ตัวจัดรูปแบบโค้ดรันสายการทำงานแบบ standalone ของ Prettier โดยตรง
ในเบราว์เซอร์ เพื่อให้คุณปรับโครงสร้างไฟล์ซอร์สได้โดยไม่ต้องส่งโค้ดไปยัง
เซิร์ฟเวอร์ เหมาะเมื่อคุณต้องการจัดรูปแบบอย่างรวดเร็ว เปรียบเทียบการตั้งค่า
การพิมพ์ที่ต่างกัน หรือได้ไฟล์ที่สะอาดพร้อมคัดลอกหรือดาวน์โหลดทันที

## รูปแบบที่รองรับ

การ rewrite นี้ยังคงโฟกัสกับรูปแบบที่ Prettier รองรับได้ดีอยู่แล้วใน
เบราว์เซอร์ ได้แก่ JavaScript, TypeScript, Flow, JSON, HTML, CSS, SCSS,
Less, Markdown, MDX, YAML, GraphQL และรูปแบบแม่แบบที่เกี่ยวข้อง เช่น Vue และ
Handlebars ตัวเลือกภาษาใช้ควบคุม parser ที่ทำงาน และการนำเข้าไฟล์จะตรวจจับ
parser ให้อัตโนมัติเมื่อรู้จักนามสกุลไฟล์

## การทำงานของการ rewrite นี้

การ rewrite นี้แยกตรรกะการจัดรูปแบบหนัก ๆ ออกจากเส้นทาง UI หลัก คำขอ
จัดรูปแบบถูกสร้างจาก config ภายใน tool ที่เป็น pure จากนั้นจึงรันผ่าน
Prettier pipeline แบบ lazy ที่ใช้ worker เพื่อให้การพิมพ์ปกติยังตอบสนองได้
ดี อินพุตขนาดใหญ่จะพักการจัดรูปแบบอัตโนมัติและสลับไปใช้การสั่ง `จัดรูปแบบตอนนี้`
แบบชัดเจน ซึ่งคาดเดาได้ง่ายกว่าการพยายามจัดรูปแบบไฟล์ใหญ่ทุกครั้งที่กดปุ่ม
