## מהו מפרמט קוד Prettier?

מפרמט קוד Prettier מריץ את צינור העבודה הרשמי של Prettier ישירות
בדפדפן, כך שאפשר לנרמל קבצי מקור בלי לשלוח קוד לשרת. הוא שימושי כשצריך
מעבר עיצוב מהיר, כשמשווים הגדרות הדפסה שונות, או כשצריך קובץ נקי שאפשר
להעתיק או להוריד מיד.

## פורמטים נתמכים

השכתוב הזה שומר את הכלי ממוקד בפורמטים ש-Prettier כבר מטפל בהם היטב
בדפדפן: JavaScript, TypeScript, Flow, JSON, HTML, CSS, SCSS, Less,
Markdown, MDX, YAML, GraphQL, ופורמטי תבנית קרובים כמו Vue ו-Handlebars.
בורר השפה קובע איזה מנתח ירוץ, וייבוא קובץ יזהה את המנתח אוטומטית כאשר
הסיומת מוכרת.

## איך השכתוב עובד

השכתוב שומר את הלוגיקה הכבדה מחוץ למסלול ה-UI הראשי. בקשות העיצוב נבנות
מתצורה מקומית ופשוטה של הכלי, ואז רצות דרך צינור Prettier מבוסס worker
טעינה עצלה כדי שההקלדה הרגילה תישאר מגיבה. קלטים גדולים משהים עיצוב
אוטומטי ועוברים לפעולת `עצבו עכשיו` מפורשת, שהיא צפויה יותר מאשר
לעצב מחדש קובץ ענק בכל לחיצה.
