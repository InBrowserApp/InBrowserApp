## Что такое форматировщик кода Prettier?

Форматировщик кода Prettier запускает официальный standalone-пайплайн Prettier
прямо в браузере, чтобы вы могли нормализовать исходные файлы, не отправляя
код на сервер. Это удобно, когда нужен быстрый прогон форматирования, хочется
сравнить разные параметры длины строки или сразу получить чистый файл, который
можно скопировать или скачать.

## Поддерживаемые форматы

Этот rewrite сохраняет фокус на форматах, которые Prettier уже хорошо
обрабатывает в браузере: JavaScript, TypeScript, Flow, JSON, HTML, CSS, SCSS,
Less, Markdown, MDX, YAML, GraphQL и связанные шаблонные форматы вроде Vue и
Handlebars. Выбор языка управляет тем, какой парсер запускается, а при импорте
файла парсер будет определяться автоматически, если расширение распознано.

## Как работает этот rewrite

В rewrite тяжёлая логика форматирования вынесена из основного UI-потока.
Запросы на форматирование собираются из чистой локальной конфигурации инструмента
и затем выполняются через ленивый worker-backed пайплайн Prettier, чтобы обычный
ввод оставался отзывчивым. Для больших входных данных автоформатирование
приостанавливается и появляется явное действие `Форматировать сейчас`, что предсказуемее,
чем попытка переформатировать огромный файл на каждом нажатии клавиши.
