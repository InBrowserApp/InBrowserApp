## Qu'est-ce qu'un jour ouvré ?

Un jour ouvré est tout jour que votre équipe, votre entreprise ou votre client considère comme travaillé. Dans beaucoup de cas, cela correspond au lundi au vendredi hors jours fériés, mais cet outil prend aussi en charge les calendriers régionaux, les week-ends personnalisés et les semaines de travail alternatives.

## Règles des jours ouvrés

Commencez par choisir si vous voulez définir directement les jours de week-end exclus ou les jours ouvrés inclus. Ajoutez ensuite des jours fériés facultatifs au format YYYY-MM-DD pour que le calcul reflète votre calendrier réel.

- Sélectionner par: Week-ends / Jours ouvrés
- Sélectionnez les jours qui sont des week-ends.
- Sélectionnez les jours ouvrés.
- Jours fériés: Optionnel. Les jours fériés en semaine sont exclus.
- Les jours fériés le week-end sont déjà exclus par les réglages des week-ends.

## Compter les jours ouvrés

Utilisez ce panneau lorsque vous avez besoin du total de jours ouvrés sur une période. Le détail sépare les jours ouvrés, les jours totaux, les week-ends et les jours fériés tombant sur des jours ouvrés pour expliquer le résultat.

- Date de début / Date de fin
- Inclure les dates de début/fin
- Jours ouvrés
- Jours totaux
- Week-ends
- Jours fériés (en semaine)

## Ajouter/Soustraire des jours ouvrés

Utilisez ce panneau lorsque vous connaissez déjà la date de départ et que vous avez besoin d'une échéance, d'une date de suivi ou d'un objectif SLA. Il applique les mêmes règles de week-end et de jours fériés que le calcul par plage, afin que les deux résultats restent cohérents.

- Date de base
- Jours ouvrés
- Compter la date de départ comme jour 1
- Ajouter / Soustraire Date
