## מהו קוד סטטוס HTTP?

קודי סטטוס HTTP הם מספרים תלת-ספרתיים המוחזרים על ידי שרת כדי לציין את תוצאת בקשת הלקוח. הם מקובצים לחמש קטגוריות:

- **1xx מידע:** מציין שהבקשה התקבלה והתהליך נמשך.
- **2xx הצלחה:** מציין שהבקשה התקבלה, הובנה ואושרה בהצלחה.
- **3xx הפניה מחדש:** מציין שיש לנקוט בפעולה נוספת כדי להשלים את הבקשה.
- **4xx שגיאת לקוח:** מציין שהלקוח ביצע שגיאה בבקשה (למשל, 404 לא נמצא, 401 לא מורשה).
- **5xx שגיאת שרת:** מציין שהשרת נתקל בשגיאה בעת עיבוד הבקשה (למשל, 500 שגיאת שרת פנימית, 503 השירות אינו זמין).
