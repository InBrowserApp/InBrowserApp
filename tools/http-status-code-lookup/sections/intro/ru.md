## Что такое HTTP Код Состояния?

HTTP коды состояния - это трехзначные числа, возвращаемые сервером для указания результата запроса клиента. Они сгруппированы в пять категорий:

- **1xx Информационные:** Указывает, что запрос был получен и процесс продолжается.
- **2xx Успех:** Указывает, что запрос был успешно получен, понят и принят.
- **3xx Перенаправление:** Указывает, что необходимо предпринять дальнейшие действия для завершения запроса.
- **4xx Ошибка Клиента:** Указывает, что клиент допустил ошибку в запросе (например, 404 Не Найдено, 401 Не Авторизован).
- **5xx Ошибка Сервера:** Указывает, что сервер столкнулся с ошибкой при обработке запроса (например, 500 Внутренняя Ошибка Сервера, 503 Служба Недоступна).
