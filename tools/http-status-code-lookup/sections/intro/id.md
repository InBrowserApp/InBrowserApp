## Apa itu Kode Status HTTP?

Kode status HTTP adalah angka tiga digit yang dikembalikan oleh server untuk menunjukkan hasil permintaan klien. Mereka dikelompokkan menjadi lima kategori:

- **1xx Informasional:** Menunjukkan bahwa permintaan telah diterima dan proses sedang berlanjut.
- **2xx Berhasil:** Menunjukkan bahwa permintaan telah berhasil diterima, dipahami, dan diterima.
- **3xx Pengalihan:** Menunjukkan bahwa tindakan lebih lanjut perlu diambil untuk menyelesaikan permintaan.
- **4xx Kesalahan Klien:** Menunjukkan bahwa klien membuat kesalahan dalam permintaan (misalnya, 404 Tidak Ditemukan, 401 Tidak Diizinkan).
- **5xx Kesalahan Server:** Menunjukkan bahwa server mengalami kesalahan saat memproses permintaan (misalnya, 500 Kesalahan Internal Server, 503 Layanan Tidak Tersedia).
