## Wat is een HTTP Statuscode?

HTTP statuscodes zijn driecijferige nummers die door een server worden teruggestuurd om het resultaat van een clientverzoek aan te geven. Ze zijn gegroepeerd in vijf categorieen:

- **1xx Informatief:** Geeft aan dat het verzoek is ontvangen en dat het proces wordt voortgezet.
- **2xx Succesvol:** Geeft aan dat het verzoek succesvol is ontvangen, begrepen en geaccepteerd.
- **3xx Omleiding:** Geeft aan dat verdere actie moet worden ondernomen om het verzoek te voltooien.
- **4xx Clientfout:** Geeft aan dat de client een fout heeft gemaakt in het verzoek (bijvoorbeeld, 404 Niet Gevonden, 401 Niet Geautoriseerd).
- **5xx Serverfout:** Geeft aan dat de server een fout heeft aangetroffen bij het verwerken van het verzoek (bijvoorbeeld, 500 Interne Serverfout, 503 Service Niet Beschikbaar).
