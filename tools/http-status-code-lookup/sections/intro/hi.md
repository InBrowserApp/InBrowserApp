## HTTP स्थिति कोड क्या है?

HTTP स्थिति कोड तीन अंकों वाली संख्याएं हैं जो क्लाइंट के अनुरोध के परिणाम को इंगित करने के लिए सर्वर द्वारा वापस की जाती हैं। उन्हें पांच श्रेणियों में समूहीकृत किया गया है:

- **1xx सूचनात्मक:** इंगित करता है कि अनुरोध प्राप्त किया गया था और प्रक्रिया जारी है।
- **2xx सफलता:** इंगित करता है कि अनुरोध सफलतापूर्वक प्राप्त, समझा और स्वीकार किया गया था।
- **3xx पुनर्निर्देशन:** इंगित करता है कि अनुरोध को पूरा करने के लिए आगे की कार्रवाई करने की आवश्यकता है।
- **4xx क्लाइंट त्रुटि:** इंगित करता है कि क्लाइंट ने अनुरोध में त्रुटि की है (उदाहरण के लिए, 404 नहीं मिला, 401 अधिकृत नहीं)।
- **5xx सर्वर त्रुटि:** इंगित करता है कि सर्वर ने अनुरोध को संसाधित करते समय त्रुटि का सामना किया है (उदाहरण के लिए, 500 आंतरिक सर्वर त्रुटि, 503 सेवा उपलब्ध नहीं)।
