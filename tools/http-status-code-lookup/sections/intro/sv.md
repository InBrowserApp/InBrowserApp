## Vad ar en HTTP Statuskod?

HTTP statuskoder ar tresiffriga nummer som returneras av en server for att ange resultatet av en klientbegaran. De ar grupperade i fem kategorier:

- **1xx Informativ:** Anger att begaran har mottagits och att processen fortsatter.
- **2xx Framgangsrik:** Anger att begaran har mottagits, forstatts och accepterats framgangsrikt.
- **3xx Omdirigering:** Anger att ytterligare atgarder maste vidtas for att slutföra begaran.
- **4xx Klientfel:** Anger att klienten har gjort ett fel i begaran (t.ex. 404 Hittades Inte, 401 Obehorig).
- **5xx Serverfel:** Anger att servern stotte pa ett fel vid bearbetning av begaran (t.ex. 500 Internt Serverfel, 503 Tjanst Ej Tillganglig).
