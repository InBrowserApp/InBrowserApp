## HTTP ステータスコードとは？

HTTP ステータスコードは、クライアントのリクエストの結果を示すためにサーバーが返す3桁の数字です。5つのカテゴリに分類されます：

- **1xx 情報：** リクエストが受信され、処理が継続中であることを示します。
- **2xx 成功：** リクエストが正常に受信、理解、承認されたことを示します。
- **3xx リダイレクト：** リクエストを完了するために追加のアクションが必要であることを示します。
- **4xx クライアントエラー：** クライアントがリクエストでエラーを起こしたことを示します（例：404 見つかりません、401 認証が必要）。
- **5xx サーバーエラー：** サーバーがリクエストの処理中にエラーに遭遇したことを示します（例：500 内部サーバーエラー、503 サービス利用不可）。
