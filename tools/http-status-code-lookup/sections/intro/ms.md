## Apakah Kod Status HTTP?

Kod status HTTP ialah nombor tiga digit yang dikembalikan oleh pelayan untuk menunjukkan hasil permintaan pelanggan. Ia dikumpulkan kepada lima kategori:

- **1xx Maklumat:** Menunjukkan bahawa permintaan telah diterima dan proses sedang diteruskan.
- **2xx Berjaya:** Menunjukkan bahawa permintaan telah berjaya diterima, difahami dan diterima.
- **3xx Ubah hala:** Menunjukkan bahawa tindakan lanjut perlu diambil untuk melengkapkan permintaan.
- **4xx Ralat Pelanggan:** Menunjukkan bahawa pelanggan membuat kesilapan dalam permintaan (contohnya, 404 Tidak Dijumpai, 401 Tidak Dibenarkan).
- **5xx Ralat Pelayan:** Menunjukkan bahawa pelayan menghadapi ralat semasa memproses permintaan (contohnya, 500 Ralat Pelayan Dalaman, 503 Perkhidmatan Tidak Tersedia).
