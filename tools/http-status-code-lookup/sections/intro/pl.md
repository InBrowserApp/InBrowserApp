## Czym jest Kod Statusu HTTP?

Kody statusu HTTP to trzycyfrowe numery zwracane przez serwer w celu wskazania wyniku zadania klienta. Sa pogrupowane w piec kategorii:

- **1xx Informacyjny:** Oznacza, ze zadanie zostalo odebrane i proces jest kontynuowany.
- **2xx Sukces:** Oznacza, ze zadanie zostalo pomyslnie odebrane, zrozumiane i zaakceptowane.
- **3xx Przekierowanie:** Oznacza, ze nalezy podjac dalsze dzialania w celu ukonczenia zadania.
- **4xx Blad Klienta:** Oznacza, ze klient popelnil blad w zadaniu (np. 404 Nie Znaleziono, 401 Nieautoryzowany).
- **5xx Blad Serwera:** Oznacza, ze serwer napotkal blad podczas przetwarzania zadania (np. 500 Wewnetrzny Blad Serwera, 503 Usluga Niedostepna).
