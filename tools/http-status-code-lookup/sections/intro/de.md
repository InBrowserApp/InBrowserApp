## Was ist ein HTTP-Statuscode?

HTTP-Statuscodes sind dreistellige Zahlen, die von einem Server zuruckgegeben werden, um das Ergebnis einer Client-Anfrage anzugeben. Sie sind in funf Kategorien gruppiert:

- **1xx Informationell:** Zeigt an, dass die Anfrage empfangen wurde und der Prozess fortgesetzt wird.
- **2xx Erfolg:** Zeigt an, dass die Anfrage erfolgreich empfangen, verstanden und akzeptiert wurde.
- **3xx Weiterleitung:** Zeigt an, dass weitere Maanahmen erforderlich sind, um die Anfrage abzuschliessen.
- **4xx Client-Fehler:** Zeigt an, dass der Client einen Fehler in der Anfrage gemacht hat (z.B. 404 Nicht Gefunden, 401 Nicht Autorisiert).
- **5xx Server-Fehler:** Zeigt an, dass der Server beim Verarbeiten der Anfrage einen Fehler festgestellt hat (z.B. 500 Interner Serverfehler, 503 Dienst Nicht Verfugbar).
