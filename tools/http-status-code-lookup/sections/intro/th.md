## รหัสสถานะ HTTP คืออะไร?

รหัสสถานะ HTTP เป็นตัวเลขสามหลักที่ส่งกลับโดยเซิร์ฟเวอร์เพื่อระบุผลลัพธ์ของคำขอของไคลเอนต์ ถูกจัดกลุ่มเป็นห้าประเภท:

- **1xx ข้อมูล:** ระบุว่าคำขอได้รับการรับและกระบวนการกำลังดำเนินการต่อ
- **2xx สำเร็จ:** ระบุว่าคำขอได้รับการรับ เข้าใจ และยอมรับสำเร็จ
- **3xx เปลี่ยนเส้นทาง:** ระบุว่าจำเป็นต้องดำเนินการเพิ่มเติมเพื่อให้คำขอเสร็จสมบูรณ์
- **4xx ข้อผิดพลาดของไคลเอนต์:** ระบุว่าไคลเอนต์ทำผิดพลาดในคำขอ (เช่น 404 ไม่พบ 401 ไม่ได้รับอนุญาต)
- **5xx ข้อผิดพลาดของเซิร์ฟเวอร์:** ระบุว่าเซิร์ฟเวอร์พบข้อผิดพลาดขณะประมวลผลคำขอ (เช่น 500 ข้อผิดพลาดภายในเซิร์ฟเวอร์ 503 บริการไม่พร้อมใช้งาน)
