## HTTP Durum Kodu Nedir?

HTTP durum kodlari, bir istemcinin isteginin sonucunu belirtmek icin bir sunucu tarafindan dondurulen uc haneli sayilardir. Bes kategoriye ayrilirlar:

- **1xx Bilgilendirici:** Istegin alindigini ve surecin devam ettigini gosterir.
- **2xx Basarili:** Istegin basariyla alindigini, anlasildigini ve kabul edildigini gosterir.
- **3xx Yonlendirme:** Istegi tamamlamak icin daha fazla eylem yapilmasi gerektigini gosterir.
- **4xx Istekci Hatasi:** Istekcinin istekte bir hata yaptigini gosterir (ornegin, 404 Bulunamadi, 401 Yetkisiz).
- **5xx Sunucu Hatasi:** Sunucunun istegi islerken bir hatayla karsilastigini gosterir (ornegin, 500 Ic Sunucu Hatasi, 503 Servis Kullanilamiyor).
