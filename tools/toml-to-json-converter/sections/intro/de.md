## Wofür dieses Tool gedacht ist

Verwende dieses Tool, wenn du gültiges TOML in JSON umwandeln musst, das sich leichter lesen, teilen oder committen lässt. Alles läuft lokal im Browser.

## Wo es hilft

- API-Beispiele in JSON für Doku oder Konfigurationsdateien umwandeln.
- Kopierte TOML-Payloads in ein Format überführen, das sich visuell leichter vergleichen lässt.
- Strukturierte Inhalte für Tools und Workflows vorbereiten, die JSON erwarten.

## Worauf du achten solltest

- Die Eingabe muss in diesem Tool gültiges TOML sein.
- Der Konverter erhält die Datenstruktur, aber die JSON-Darstellung kann sich von handgeschriebenem Stil unterscheiden.
- Wenn du in die andere Richtung konvertieren willst, nutze stattdessen den JSON-zu-TOML-Konverter.
