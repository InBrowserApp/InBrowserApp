## เครื่องมือนี้ใช้ทำอะไร

ใช้เครื่องมือนี้เมื่อคุณต้องการแปลง TOML ที่ถูกต้องให้เป็น JSON ที่อ่าน แชร์ หรือ commit ได้ง่ายกว่า ทุกอย่างทำงานภายในเบราว์เซอร์แบบ local

## เหมาะกับสถานการณ์ไหน

- แปลงตัวอย่าง API เป็น JSON สำหรับเอกสารหรือไฟล์คอนฟิก
- เขียน payload TOML ที่คัดลอกมาใหม่ให้อยู่ในรูปแบบที่เทียบด้วยสายตาได้ง่ายกว่า
- เตรียมเนื้อหาแบบมีโครงสร้างสำหรับเครื่องมือและ workflow ที่ต้องการ JSON

## สิ่งที่ควรระวัง

- อินพุตในเครื่องมือนี้ต้องเป็น TOML ที่ถูกต้อง
- ตัวแปลงจะคงโครงสร้างข้อมูลไว้ แต่รูปแบบการแสดงผลของ JSON อาจต่างจากสไตล์ที่เขียนด้วยมือ
- หากต้องการแปลงย้อนกลับ ให้ใช้เครื่องมือ JSON เป็น TOML แทน
