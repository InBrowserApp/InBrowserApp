## このツールでできること

画面や画像から色を取得。ピクセルをサンプルして HEX、RGB、HSL、HSV、CMYK の値をすぐにコピー。

## 色を拾うときのポイント

- ブラウザが EyeDropper をサポートしていて、他のアプリやタブ、デザイン資料から色を取りたいときは画面取色を使います。
- スクリーンショット、ロゴ、商品写真、UI モックアップから正確な 1 ピクセルを拾いたいときは画像取色を使います。

## 出力について

- `#RRGGBBAA`、`rgba(...)`、`hsla(...)`、`hsva(...)` のような透過付き形式が必要な場合は Alpha を有効にします。
- Web や UI 作業では HEX と RGB が最も使いやすく、印刷向けの受け渡しでは CMYK が役立ちます。
