# Unisci file PDF nel tuo browser

Usa questo strumento di unione PDF quando ti serve un unico documento da più PDF di origine, per esempio per combinare pagine scansionate, unire moduli firmati o raccogliere report da condividere. Aggiungi due o più file, controlla il numero di pagine, poi organizza la coda prima di creare il PDF finale.

## Come funziona l'ordine di unione

Lo strumento aggiunge ogni pagina del primo PDF, poi ogni pagina del PDF successivo, continuando lungo la coda. Puoi riordinare i file con i controlli freccia, trascinare le righe su desktop, rimuovere gli errori e visualizzare l'anteprima di ogni file sorgente prima dell'unione.

## Privacy e gestione dei file

Tutta l'analisi e l'unione vengono eseguite localmente nel browser con `pdf-lib` e un worker in background. I tuoi file non vengono caricati su InBrowser.App e il link di download generato esiste solo nella sessione corrente del browser.

## Limiti da conoscere

I PDF cifrati o danneggiati non possono essere uniti in modo affidabile. Se un file è protetto da una password del proprietario, rimuovi prima quella restrizione e aggiungi di nuovo il PDF sbloccato. I file molto grandi possono richiedere più tempo perché il browser deve copiare ogni pagina in un nuovo documento.
