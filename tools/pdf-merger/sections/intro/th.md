# รวมไฟล์ PDF ในเบราว์เซอร์ของคุณ

ใช้เครื่องมือรวม PDF นี้เมื่อคุณต้องการเอกสารเดียวจาก PDF ต้นทางหลายไฟล์ เช่น รวมหน้าที่สแกน ต่อแบบฟอร์มที่ลงนามแล้ว หรือจัดชุดรายงานเพื่อแชร์ เพิ่มไฟล์สองไฟล์ขึ้นไป ตรวจสอบจำนวนหน้า แล้วจัดลำดับคิวก่อนสร้าง PDF สุดท้าย

## ลำดับการรวมทำงานอย่างไร

เครื่องมือจะต่อทุกหน้าจาก PDF แรก ตามด้วยทุกหน้าจาก PDF ถัดไป และดำเนินต่อไปตามลำดับในคิว คุณสามารถจัดลำดับไฟล์ใหม่ด้วยปุ่มลูกศร ลากแถวบนเดสก์ท็อป นำรายการที่ผิดออก และดูตัวอย่างไฟล์ต้นทางแต่ละไฟล์ก่อนรวม

## ความเป็นส่วนตัวและการจัดการไฟล์

การแยกอ่านและการรวมทั้งหมดทำงานในเบราว์เซอร์ของคุณด้วย `pdf-lib` และ worker เบื้องหลัง ไฟล์ของคุณจะไม่ถูกอัปโหลดไปยัง InBrowser.App และลิงก์ดาวน์โหลดที่สร้างขึ้นจะมีอยู่เฉพาะในเซสชันเบราว์เซอร์ปัจจุบัน

## ข้อจำกัดที่ควรรู้

PDF ที่เข้ารหัสหรือเสียหายอาจรวมได้ไม่เสถียร หากไฟล์ถูกป้องกันด้วยรหัสผ่านเจ้าของ ให้นำข้อจำกัดนั้นออกก่อนแล้วเพิ่ม PDF ที่ปลดล็อกแล้วอีกครั้ง ไฟล์ขนาดใหญ่มากอาจใช้เวลานานขึ้น เพราะเบราว์เซอร์ต้องคัดลอกทุกหน้าไปยังเอกสารใหม่
