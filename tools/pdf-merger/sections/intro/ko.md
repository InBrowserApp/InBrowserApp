# 브라우저에서 PDF 파일 병합하기

스캔한 페이지를 합치거나, 서명된 양식을 이어 붙이거나, 공유할 보고서를 하나로 묶는 등 여러 원본 PDF에서 하나의 문서가 필요할 때 이 PDF 병합 도구를 사용하세요. 파일을 두 개 이상 추가하고 페이지 수를 확인한 다음, 최종 PDF를 만들기 전에 대기열 순서를 정렬하세요.

## 병합 순서 작동 방식

이 도구는 첫 번째 PDF의 모든 페이지를 추가한 다음 다음 PDF의 모든 페이지를 추가하며, 대기열 아래쪽으로 계속 진행합니다. 화살표 컨트롤로 파일 순서를 바꾸고, 데스크톱에서는 행을 드래그하며, 잘못 추가한 파일을 제거하고, 병합 전에 각 원본 파일을 미리 볼 수 있습니다.

## 개인정보 보호 및 파일 처리

모든 파싱과 병합은 `pdf-lib` 및 백그라운드 워커를 사용하여 브라우저 안에서 로컬로 실행됩니다. 파일은 InBrowser.App에 업로드되지 않으며, 생성된 다운로드 링크는 현재 브라우저 세션 안에서만 존재합니다.

## 알아둘 제한 사항

암호화되었거나 손상된 PDF는 안정적으로 병합할 수 없습니다. 파일이 소유자 암호로 보호되어 있다면 먼저 해당 제한을 제거한 뒤 잠금 해제된 PDF를 다시 추가하세요. 매우 큰 파일은 브라우저가 모든 페이지를 새 문서로 복사해야 하므로 시간이 더 오래 걸릴 수 있습니다.
