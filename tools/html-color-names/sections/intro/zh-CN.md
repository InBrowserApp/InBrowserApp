## 什么是 HTML 颜色名称？

HTML 颜色名称是预定义的颜色关键字，可在 CSS 和 HTML 中使用，无需十六进制或 RGB 值即可指定颜色。所有现代浏览器都支持这些名称。

### 基本颜色：

最初的 16 种 HTML 颜色（黑色、白色、红色、绿色、蓝色等）从 HTML 早期版本起就已支持。

### 扩展颜色：

CSS3 引入了 140 多种命名颜色，包括 "aliceblue"、"coral"、"darkslategray" 等变体。

### 用法：

在 CSS 中直接使用颜色名称即可，例如 "color: tomato;" 或 "background-color: steelblue;"。
