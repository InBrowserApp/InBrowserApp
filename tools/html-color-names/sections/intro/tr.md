## HTML renk adları nedir?

HTML renk adları, CSS ve HTML'de onaltılık veya RGB değerleri kullanmadan renkleri belirtmek için kullanılabilen, önceden tanımlı renk anahtar sözcükleridir. Bu adlar tüm modern tarayıcılar tarafından desteklenir.

### Temel renkler:

Orijinal 16 HTML rengi (siyah, beyaz, kırmızı, yeşil, mavi vb.) HTML'nin ilk sürümlerinden beri desteklenir.

### Genişletilmiş renkler:

CSS3, "aliceblue", "coral", "darkslategray" gibi varyasyonlar dahil 140'tan fazla adlandırılmış renk getirdi.

### Kullanım:

Renk adını CSS'de doğrudan kullanın, örn. "color: tomato;" veya "background-color: steelblue;".
