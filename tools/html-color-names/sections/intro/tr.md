## HTML renk adları nedir?

HTML renk adları, CSS ve HTML'de onaltılık veya RGB değerleri kullanmadan renkleri belirtmek için kullanılabilen, önceden tanımlı renk anahtar sözcükleridir. Bu adlar tüm modern tarayıcılar tarafından desteklenir.

**Temel renkler:** Orijinal 16 HTML rengi (siyah, beyaz, kırmızı, yeşil, mavi vb.) HTML'nin ilk sürümlerinden beri desteklenir.

**Genişletilmiş renkler:** CSS3, "aliceblue", "coral", "darkslategray" gibi varyasyonlar dahil 140'tan fazla adlandırılmış renk getirdi.

## Bu liste size nasıl yardımcı olur

Bir CSS rengini ada veya HEX değerine göre aramak, kataloğu renk ailesine göre daraltmak ve adlı bir rengi stil sayfasına, tasarım token’ına ya da demoya eklemeden önce renk örneklerini karşılaştırmak için aramayı kullanın.

## Adlandırılmış renkler ne zaman kullanışlıdır

Adlandırılmış renkler, okunabilir CSS, hızlı prototipler veya kolay hatırlanan örnekler istediğinizde kullanışlıdır. Renk adını CSS'de doğrudan kullanın, örn. "color: tomato;" veya "background-color: steelblue;".
