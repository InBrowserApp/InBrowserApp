## Apakah Nama Warna HTML?

Nama warna HTML ialah kata kunci warna yang ditakrifkan terlebih dahulu dan boleh digunakan dalam CSS dan HTML untuk menentukan warna tanpa nilai heksadesimal atau RGB. Nama-nama ini disokong oleh semua pelayar moden.

**Warna asas:** 16 warna HTML asal (hitam, putih, merah, hijau, biru, dll.) telah disokong sejak versi HTML terawal.

**Warna lanjutan:** CSS3 memperkenalkan lebih 140 warna bernama, termasuk variasi seperti "aliceblue", "coral", "darkslategray" dan banyak lagi.

## Bagaimana senarai ini membantu

Gunakan carian untuk mencari warna CSS mengikut nama atau nilai HEX, tapis katalog mengikut keluarga warna, dan bandingkan swatch sebelum meletakkan warna bernama ke dalam helaian gaya, token reka bentuk atau demo.

## Bila warna bernama berguna

Warna bernama berguna apabila anda mahukan CSS yang mudah dibaca, prototaip pantas, atau contoh yang mudah diingati. Gunakan sahaja nama warna dalam CSS, contohnya "color: tomato;" atau "background-color: steelblue;".
