## Apakah Nama Warna HTML?

Nama warna HTML ialah kata kunci warna yang ditakrifkan terlebih dahulu dan boleh digunakan dalam CSS dan HTML untuk menentukan warna tanpa nilai heksadesimal atau RGB. Nama-nama ini disokong oleh semua pelayar moden.

### Warna asas:

16 warna HTML asal (hitam, putih, merah, hijau, biru, dll.) telah disokong sejak versi HTML terawal.

### Warna lanjutan:

CSS3 memperkenalkan lebih 140 warna bernama, termasuk variasi seperti "aliceblue", "coral", "darkslategray" dan banyak lagi.

### Penggunaan:

Gunakan sahaja nama warna dalam CSS, contohnya "color: tomato;" atau "background-color: steelblue;".
