## מהם שמות צבעי HTML?

שמות צבעי HTML הם מילות מפתח צבע מוגדרות מראש שאפשר להשתמש בהן ב-CSS וב-HTML כדי לציין צבעים בלי להשתמש בערכי הקסדצימלי או RGB. שמות אלו נתמכים בכל הדפדפנים המודרניים.

**צבעים בסיסיים:** 16 צבעי ה-HTML המקוריים (שחור, לבן, אדום, ירוק, כחול וכו') נתמכים מאז הגרסאות המוקדמות ביותר של HTML.

**צבעים מורחבים:** CSS3 הציג יותר מ-140 צבעים בעלי שם, כולל וריאציות כמו "aliceblue", "coral", "darkslategray" ועוד רבים.

## איך הכלי הזה עוזר

השתמשו בחיפוש כדי למצוא צבע CSS לפי שם או ערך HEX, לצמצם את הרשימה לפי משפחת צבע ולהשוות דוגמיות לפני שמדביקים צבע בשם לתוך stylesheet, design token או demo.

## מתי צבעים בעלי שם שימושיים

צבעים בעלי שם שימושיים כשצריך CSS קריא, אבות־טיפוס מהירים או דוגמאות שקל לזכור. פשוט השתמשו בשם הצבע ב-CSS, למשל "color: tomato;" או "background-color: steelblue;".
