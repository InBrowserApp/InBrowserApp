## O que são nomes de cores HTML?

Nomes de cores HTML são palavras-chave de cor predefinidas que podem ser usadas em CSS e HTML para especificar cores sem usar valores hexadecimais ou RGB. Esses nomes são suportados por todos os navegadores modernos.

**Cores básicas:** As 16 cores HTML originais (preto, branco, vermelho, verde, azul etc.) são suportadas desde as primeiras versões do HTML.

**Cores estendidas:** O CSS3 introduziu mais de 140 cores nomeadas, incluindo variações como "aliceblue", "coral", "darkslategray" e muitas outras.

## Quando cores nomeadas são úteis

Cores nomeadas são úteis quando você quer um CSS legível, protótipos rápidos ou exemplos fáceis de lembrar. Basta usar o nome da cor no seu CSS, por exemplo, "color: tomato;" ou "background-color: steelblue;".
