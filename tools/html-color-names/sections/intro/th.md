## ชื่อสี HTML คืออะไร?

ชื่อสี HTML คือคีย์เวิร์ดสีที่กำหนดไว้ล่วงหน้า ซึ่งสามารถใช้ใน CSS และ HTML เพื่อระบุสีโดยไม่ต้องใช้ค่าเลขฐานสิบหกหรือค่า RGB ชื่อเหล่านี้รองรับโดยเบราว์เซอร์สมัยใหม่ทั้งหมด

**สีพื้นฐาน:** สี HTML ดั้งเดิม 16 สี (ดำ ขาว แดง เขียว น้ำเงิน ฯลฯ) ได้รับการรองรับมาตั้งแต่ HTML รุ่นแรกๆ

**สีเพิ่มเติม:** CSS3 เพิ่มสีที่มีชื่อมากกว่า 140 สี รวมถึงรูปแบบอย่าง "aliceblue", "coral", "darkslategray" และอีกมากมาย

## เครื่องมือนี้ช่วยอะไรได้บ้าง

ใช้การค้นหาเพื่อหาสี CSS ตามชื่อหรือค่า HEX กรองรายการตามกลุ่มสี และเปรียบเทียบตัวอย่างสีก่อนนำชื่อสีไปใช้ใน stylesheet, design token หรือเดโม

## เมื่อใดที่สีแบบมีชื่อมีประโยชน์

สีแบบมีชื่อมีประโยชน์เมื่อคุณต้องการ CSS ที่อ่านง่าย ทำต้นแบบได้เร็ว หรือมีตัวอย่างที่จำได้ง่าย เพียงใช้ชื่อสีใน CSS เช่น "color: tomato;" หรือ "background-color: steelblue;".
