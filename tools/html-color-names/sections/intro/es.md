## ¿Qué son los nombres de colores HTML?

Los nombres de colores HTML son palabras clave de color predefinidas que pueden usarse en CSS y HTML para especificar colores sin usar valores hexadecimales o RGB. Estos nombres son compatibles con todos los navegadores modernos.

### Colores básicos:

Los 16 colores HTML originales (negro, blanco, rojo, verde, azul, etc.) se admiten desde las primeras versiones de HTML.

### Colores extendidos:

CSS3 introdujo más de 140 colores con nombre, incluidas variaciones como "aliceblue", "coral", "darkslategray" y muchos más.

### Uso:

Solo tienes que usar el nombre del color en tu CSS, por ejemplo, "color: tomato;" o "background-color: steelblue;".
