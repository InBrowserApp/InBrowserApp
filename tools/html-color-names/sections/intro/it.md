## Cosa sono i nomi dei colori HTML?

I nomi dei colori HTML sono parole chiave di colore predefinite che possono essere usate in CSS e HTML per specificare colori senza usare valori esadecimali o RGB. Questi nomi sono supportati da tutti i browser moderni.

**Colori di base:** I 16 colori HTML originali (nero, bianco, rosso, verde, blu, ecc.) sono supportati sin dalle prime versioni di HTML.

**Colori estesi:** CSS3 ha introdotto oltre 140 colori con nome, incluse varianti come "aliceblue", "coral", "darkslategray" e molti altri.

## Come aiuta questa ricerca

Usa la ricerca per trovare un colore CSS per nome o valore HEX, restringi il catalogo per famiglia cromatica e confronta i campioni prima di inserire un colore con nome in un foglio di stile, in un design token o in una demo.

## Quando i colori con nome sono utili

I colori con nome sono utili quando vuoi un CSS leggibile, prototipi rapidi o esempi facili da ricordare. Basta usare il nome del colore nel CSS, ad esempio "color: tomato;" o "background-color: steelblue;".
