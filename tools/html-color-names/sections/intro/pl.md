## Czym są nazwy kolorów HTML?

Nazwy kolorów HTML to predefiniowane słowa kluczowe kolorów, których można używać w CSS i HTML do określania kolorów bez wartości szesnastkowych lub RGB. Te nazwy są obsługiwane przez wszystkie nowoczesne przeglądarki.

### Podstawowe kolory:

Oryginalne 16 kolorów HTML (czarny, biały, czerwony, zielony, niebieski itd.) są obsługiwane od najwcześniejszych wersji HTML.

### Rozszerzone kolory:

CSS3 wprowadził ponad 140 nazwanych kolorów, w tym warianty takie jak "aliceblue", "coral", "darkslategray" i wiele innych.

### Użycie:

Wystarczy użyć nazwy koloru w CSS, np. "color: tomato;" lub "background-color: steelblue;".
