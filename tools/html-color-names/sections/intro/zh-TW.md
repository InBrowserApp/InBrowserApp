## 什麼是 HTML 顏色名稱？

HTML 顏色名稱是預先定義的顏色關鍵字，可在 CSS 和 HTML 中使用，無需十六進位或 RGB 值即可指定顏色。所有現代瀏覽器都支援這些名稱。

**基本顏色：** 最初的 16 種 HTML 顏色（黑色、白色、紅色、綠色、藍色等）自 HTML 早期版本起就已支援。

**擴展顏色：** CSS3 引入了 140 多種命名顏色，包括 "aliceblue"、"coral"、"darkslategray" 等變體。

## 這個查詢工具能幫你什麼

## 什麼時候適合使用命名顏色

當你需要更易讀的 CSS、快速原型或更容易記住的範例時，命名顏色會很有用。 在 CSS 中直接使用顏色名稱，例如 "color: tomato;" 或 "background-color: steelblue;"。
