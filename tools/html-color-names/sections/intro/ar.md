## ما هي أسماء ألوان HTML؟

أسماء ألوان HTML هي كلمات مفتاحية لونية محددة مسبقًا يمكن استخدامها في CSS وHTML لتحديد الألوان دون استخدام قيم سداسية عشرية أو قيم RGB. هذه الأسماء مدعومة في جميع المتصفحات الحديثة.

**الألوان الأساسية:** الألوان الستة عشر الأصلية في HTML (الأسود، الأبيض، الأحمر، الأخضر، الأزرق، إلخ) مدعومة منذ الإصدارات الأولى من HTML.

**الألوان الموسعة:** قدّم CSS3 أكثر من 140 لونًا مسمى، بما في ذلك تنويعات مثل "aliceblue" و"coral" و"darkslategray" وغيرها الكثير.

## كيف يفيدك هذا الدليل

استخدم البحث للعثور على لون CSS بالاسم أو بقيمة HEX، وضيّق القائمة حسب عائلة اللون، وقارن العينات قبل لصق لون مسمّى في ملف أنماط أو رمز تصميم أو عرض توضيحي.

## متى تكون الألوان المسمّاة مفيدة

تكون الألوان المسمّاة مفيدة عندما تريد CSS واضحًا، ونماذج أولية سريعة، وأمثلة يسهل تذكرها. استخدم اسم اللون مباشرة في CSS، مثل "color: tomato;" أو "background-color: steelblue;".
