## Vad är HTML-färgnamn?

HTML-färgnamn är fördefinierade färgnyckelord som kan användas i CSS och HTML för att ange färger utan att använda hexadecimala eller RGB-värden. Dessa namn stöds av alla moderna webbläsare.

### Grundfärger:

De ursprungliga 16 HTML-färgerna (svart, vit, röd, grön, blå etc.) har stödts sedan de tidigaste HTML-versionerna.

### Utökade färger:

CSS3 introducerade 140+ namngivna färger, inklusive varianter som "aliceblue", "coral", "darkslategray" med flera.

### Användning:

Använd bara färgnamnet i din CSS, t.ex. "color: tomato;" eller "background-color: steelblue;".
