## HTML 색상 이름이란?

HTML 색상 이름은 CSS와 HTML에서 16진수나 RGB 값 없이 색상을 지정할 수 있는 미리 정의된 색상 키워드입니다. 이 이름들은 모든 최신 브라우저에서 지원됩니다.

### 기본 색상:

HTML 초기부터 지원된 16가지 기본 색상(검정, 흰색, 빨강, 초록, 파랑 등)입니다.

### 확장 색상:

CSS3는 "aliceblue", "coral", "darkslategray" 등 다양한 변형을 포함해 140개 이상의 이름 있는 색상을 도입했습니다.

### 사용법:

CSS에서 색상 이름을 그대로 사용하면 됩니다. 예: "color: tomato;" 또는 "background-color: steelblue;".
