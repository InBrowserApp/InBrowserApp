## Was sind HTML-Farbnamen?

HTML-Farbnamen sind vordefinierte Farb-Schlüsselwörter, die in CSS und HTML verwendet werden können, um Farben ohne Hexadezimal- oder RGB-Werte anzugeben. Diese Namen werden von allen modernen Browsern unterstützt.

### Grundfarben:

Die ursprünglichen 16 HTML-Farben (Schwarz, Weiß, Rot, Grün, Blau usw.) werden seit den frühesten HTML-Versionen unterstützt.

### Erweiterte Farben:

CSS3 führte über 140 benannte Farben ein, darunter Varianten wie "aliceblue", "coral", "darkslategray" und viele mehr.

### Verwendung:

Verwenden Sie einfach den Farbnamen in Ihrem CSS, z. B. "color: tomato;" oder "background-color: steelblue;".
