## Hva er HTML-fargenavn?

HTML-fargenavn er forhåndsdefinerte fargenøkkelord som kan brukes i CSS og HTML for å angi farger uten å bruke heksadesimale eller RGB-verdier. Disse navnene støttes av alle moderne nettlesere.

### Grunnfarger:

De opprinnelige 16 HTML-fargene (svart, hvit, rød, grønn, blå osv.) har vært støttet siden de tidligste HTML-versjonene.

### Utvidede farger:

CSS3 introduserte mer enn 140 navngitte farger, inkludert varianter som "aliceblue", "coral", "darkslategray" og mange flere.

### Bruk:

Bare bruk fargenavnet i CSS, for eksempel "color: tomato;" eller "background-color: steelblue;".
