## Tên màu HTML là gì?

Tên màu HTML là các từ khóa màu được định nghĩa sẵn, có thể dùng trong CSS và HTML để chỉ định màu mà không cần giá trị thập lục phân hoặc RGB. Các tên này được hỗ trợ bởi mọi trình duyệt hiện đại.

**Màu cơ bản:** 16 màu HTML gốc (đen, trắng, đỏ, xanh lá, xanh dương, v.v.) đã được hỗ trợ từ những phiên bản HTML đầu tiên.

**Màu mở rộng:** CSS3 giới thiệu hơn 140 màu có tên, bao gồm các biến thể như "aliceblue", "coral", "darkslategray" và nhiều màu khác.

## Công cụ tra cứu này giúp gì

Dùng ô tìm kiếm để tìm màu CSS theo tên hoặc giá trị HEX, lọc danh mục theo nhóm màu và so sánh ô mẫu trước khi chèn màu có tên vào stylesheet, design token hoặc bản demo.

## Khi nào màu có tên hữu ích

Màu có tên hữu ích khi bạn cần CSS dễ đọc, nguyên mẫu nhanh hoặc ví dụ dễ nhớ. Chỉ cần dùng tên màu trong CSS, ví dụ "color: tomato;" hoặc "background-color: steelblue;".
