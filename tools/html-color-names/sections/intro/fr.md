## Que sont les noms de couleurs HTML ?

Les noms de couleurs HTML sont des mots-clés de couleur prédéfinis utilisables en CSS et HTML pour spécifier des couleurs sans utiliser de valeurs hexadécimales ou RVB. Ces noms sont pris en charge par tous les navigateurs modernes.

**Couleurs de base :** Les 16 couleurs HTML originales (noir, blanc, rouge, vert, bleu, etc.) sont prises en charge depuis les premières versions de HTML.

**Couleurs étendues :** CSS3 a introduit plus de 140 couleurs nommées, dont des variantes comme "aliceblue", "coral", "darkslategray", et bien d'autres.

## Comment cette recherche vous aide

Utilisez la recherche pour trouver une couleur CSS par son nom ou sa valeur HEX, filtrez le catalogue par famille de couleur et comparez les échantillons avant d’insérer une couleur nommée dans une feuille de style, un design token ou une démo.

## Quand les couleurs nommées sont utiles

Les couleurs nommées sont utiles quand vous voulez un CSS lisible, des prototypes rapides ou des exemples faciles à retenir. Utilisez simplement le nom de la couleur dans votre CSS, par exemple "color: tomato;" ou "background-color: steelblue;".
