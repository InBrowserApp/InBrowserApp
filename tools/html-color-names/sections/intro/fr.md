## Que sont les noms de couleurs HTML ?

Les noms de couleurs HTML sont des mots-clés de couleur prédéfinis utilisables en CSS et HTML pour spécifier des couleurs sans utiliser de valeurs hexadécimales ou RVB. Ces noms sont pris en charge par tous les navigateurs modernes.

### Couleurs de base :

Les 16 couleurs HTML originales (noir, blanc, rouge, vert, bleu, etc.) sont prises en charge depuis les premières versions de HTML.

### Couleurs étendues :

CSS3 a introduit plus de 140 couleurs nommées, dont des variantes comme "aliceblue", "coral", "darkslategray", et bien d'autres.

### Utilisation :

Utilisez simplement le nom de la couleur dans votre CSS, par exemple "color: tomato;" ou "background-color: steelblue;".
