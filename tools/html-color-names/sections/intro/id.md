## Apa itu Nama Warna HTML?

Nama warna HTML adalah kata kunci warna yang telah ditentukan sebelumnya dan dapat digunakan di CSS serta HTML untuk menentukan warna tanpa nilai heksadesimal atau RGB. Nama-nama ini didukung oleh semua browser modern.

**Warna dasar:** 16 warna HTML asli (hitam, putih, merah, hijau, biru, dll.) didukung sejak versi awal HTML.

**Warna diperluas:** CSS3 memperkenalkan lebih dari 140 warna bernama, termasuk variasi seperti "aliceblue", "coral", "darkslategray", dan banyak lagi.

## Bagaimana daftar ini membantu

Gunakan pencarian untuk menemukan warna CSS berdasarkan nama atau nilai HEX, persempit katalog menurut keluarga warna, dan bandingkan swatch sebelum menempelkan warna bernama ke stylesheet, design token, atau demo.

## Kapan warna bernama berguna

Warna bernama berguna saat Anda membutuhkan CSS yang mudah dibaca, prototipe cepat, atau contoh yang mudah diingat. Cukup gunakan nama warna di CSS, misalnya "color: tomato;" atau "background-color: steelblue;".
