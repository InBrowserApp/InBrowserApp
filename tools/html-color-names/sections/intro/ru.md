## Что такое названия цветов HTML?

Названия цветов HTML — это заранее определенные ключевые слова цвета, которые можно использовать в CSS и HTML для задания цвета без шестнадцатеричных или RGB значений. Эти имена поддерживаются всеми современными браузерами.

### Базовые цвета:

Исходные 16 цветов HTML (черный, белый, красный, зеленый, синий и т. д.) поддерживаются с самых ранних версий HTML.

### Расширенные цвета:

CSS3 ввел более 140 именованных цветов, включая варианты "aliceblue", "coral", "darkslategray" и многие другие.

### Использование:

Просто используйте название цвета в CSS, например, "color: tomato;" или "background-color: steelblue;".
