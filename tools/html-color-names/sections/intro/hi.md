## HTML रंग नाम क्या हैं?

HTML रंग नाम पूर्वनिर्धारित रंग कीवर्ड हैं जिन्हें CSS और HTML में हेक्साडेसिमल या RGB मानों के बिना रंग निर्दिष्ट करने के लिए उपयोग किया जा सकता है। ये नाम सभी आधुनिक ब्राउज़र द्वारा समर्थित हैं।

### मूल रंग:

HTML के मूल 16 रंग (काला, सफेद, लाल, हरा, नीला आदि) शुरुआती संस्करणों से समर्थित हैं।

### विस्तारित रंग:

CSS3 ने "aliceblue", "coral", "darkslategray" जैसी विविधताओं सहित 140+ नामित रंग पेश किए।

### उपयोग:

CSS में रंग नाम सीधे उपयोग करें, जैसे "color: tomato;" या "background-color: steelblue;".
