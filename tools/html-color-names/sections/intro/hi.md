## HTML रंग नाम क्या हैं?

HTML रंग नाम पूर्वनिर्धारित रंग कीवर्ड हैं जिन्हें CSS और HTML में हेक्साडेसिमल या RGB मानों के बिना रंग निर्दिष्ट करने के लिए उपयोग किया जा सकता है। ये नाम सभी आधुनिक ब्राउज़र द्वारा समर्थित हैं।

**मूल रंग:** HTML के मूल 16 रंग (काला, सफेद, लाल, हरा, नीला आदि) शुरुआती संस्करणों से समर्थित हैं।

**विस्तारित रंग:** CSS3 ने "aliceblue", "coral", "darkslategray" जैसी विविधताओं सहित 140+ नामित रंग पेश किए।

## यह सूची आपकी कैसे मदद करती है

खोज का उपयोग करके नाम या HEX मान से CSS रंग ढूंढें, रंग परिवार के अनुसार सूची को सीमित करें, और किसी नामित रंग को stylesheet, design token या demo में लगाने से पहले swatch की तुलना करें।

## नामित रंग कब उपयोगी होते हैं

नामित रंग तब उपयोगी होते हैं जब आपको पढ़ने में आसान CSS, तेज़ prototype या आसानी से याद रहने वाले उदाहरण चाहिए। CSS में रंग नाम सीधे उपयोग करें, जैसे "color: tomato;" या "background-color: steelblue;".
