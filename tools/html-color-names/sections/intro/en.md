## What are HTML Color Names?

HTML color names are predefined color keywords that can be used in CSS and HTML to specify colors without using hexadecimal or RGB values. These names are supported by all modern browsers.

### Basic Colors:

The original 16 HTML colors (black, white, red, green, blue, etc.) have been supported since the earliest versions of HTML.

### Extended Colors:

CSS3 introduced 140+ named colors, including variations like "aliceblue", "coral", "darkslategray", and many more.

### Usage:

Simply use the color name in your CSS, e.g., "color: tomato;" or "background-color: steelblue;".
