## HTMLの色名とは？

HTMLの色名は、CSSやHTMLで16進値やRGB値を使わずに色を指定できる、あらかじめ定義された色のキーワードです。これらの名前はすべての最新ブラウザでサポートされています。

**基本色:** 元の16色のHTMLカラー（黒、白、赤、緑、青など）は、HTMLの初期バージョンからサポートされています。

**拡張色:** CSS3では "aliceblue"、"coral"、"darkslategray" などのバリエーションを含む140以上の名前付きカラーが導入されました。

## この一覧でできること

色名または HEX 値で CSS カラーを検索し、色系統で絞り込み、名前付きカラーをスタイルシートやデザイントークン、デモに貼り付ける前にスウォッチを比較できます。

## 名前付きカラーが役立つ場面

名前付きカラーは、読みやすい CSS や素早いプロトタイプ、覚えやすいサンプルが欲しいときに便利です。 CSSで色名をそのまま使います。例: "color: tomato;" または "background-color: steelblue;"。
