## Ce que fait cet outil

Ce convertisseur transforme le HTML collé ou importé en Markdown directement dans votre navigateur. Il est utile lorsque le contenu provient d’un CMS, d’un éditeur, d’un extrait exporté ou d’un modèle d’e-mail copié, mais doit finir dans un README, une page de documentation ou une note.

## Quand il est utile

- Nettoyer du HTML exporté avant de le publier en Markdown.
- Transformer de petits fragments de page en brouillons de documentation.
- Normaliser du balisage copié pour que les diffs restent lisibles dans le contrôle de version.

## Ce qu’il faut vérifier après conversion

- Les tableaux complexes, intégrations et styles en ligne peuvent encore demander un nettoyage manuel.
- Choisissez les styles de titres, de listes et de blocs de code adaptés à votre variante Markdown.
- Prévisualisez une dernière fois les liens, images et clôtures de code avant publication.
