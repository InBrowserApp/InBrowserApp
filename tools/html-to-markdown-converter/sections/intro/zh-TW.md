## 這個工具能做什麼

這個轉換器會直接在瀏覽器中把你貼上或匯入的 HTML 轉成 Markdown。當內容原本來自 CMS、編輯器、匯出片段或複製來的郵件範本，但最後要放進 README、文件頁面或筆記時，它就很有幫助。

## 什麼時候適合使用

- 在把匯出的 HTML 發佈成 Markdown 之前先整理乾淨。
- 把小型頁面片段轉成文件草稿。
- 正規化複製來的標記內容，讓版本控制裡的 diff 更容易閱讀。

## 轉換後要檢查什麼

- 複雜表格、嵌入內容與行內樣式通常仍需要手動整理。
- 選擇與目標 Markdown 風格一致的標題、清單與程式碼區塊樣式。
- 發佈前再預覽一次連結、圖片與程式碼圍欄。
