## ما الذي تفعله هذه الأداة

يحوّل هذا المحول HTML الذي تلصقه أو تستورده إلى Markdown مباشرة داخل المتصفح. وهو مفيد عندما يبدأ المحتوى في CMS أو محرر أو مقتطف مُصدَّر أو قالب بريد منسوخ، لكنه يحتاج في النهاية إلى الظهور داخل README أو صفحة توثيق أو ملاحظة.

## متى يكون مفيدًا

- تنظيف HTML المُصدَّر قبل نشره بصيغة Markdown.
- تحويل أجزاء الصفحات الصغيرة إلى مسودات توثيق.
- توحيد الوسوم المنسوخة بحيث تبقى فروق الإصدارات مقروءة في التحكم بالإصدارات.

## ما الذي يجب التحقق منه بعد التحويل

- قد تظل الجداول المعقدة والعناصر المضمّنة والأنماط المضمنة بحاجة إلى تنظيف يدوي.
- اختر أنماط العناوين والقوائم وكتل الشيفرة التي تطابق أسلوب Markdown المستهدف.
- راجع الروابط والصور وأسوار الشيفرة مرة واحدة قبل النشر.
