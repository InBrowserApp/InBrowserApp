## Wat deze tool doet

Deze converter zet geplakte of geïmporteerde HTML direct in de browser om naar Markdown. Dat is handig wanneer inhoud begint in een CMS, editor, geëxporteerd fragment of gekopieerd e-mailsjabloon, maar uiteindelijk in een README, documentatiepagina of notitie terecht moet komen.

## Wanneer het helpt

- Geëxporteerde HTML opschonen voordat je die als Markdown publiceert.
- Kleine paginastukken omzetten in documentatieconcepten.
- Gekopieerde markup normaliseren zodat diffs leesbaar blijven in versiebeheer.

## Wat je na het converteren moet controleren

- Complexe tabellen, embeds en inline stijlen kunnen nog steeds handmatig werk vragen.
- Kies kop-, lijst- en codeblokstijlen die passen bij je doelvariant van Markdown.
- Controleer links, afbeeldingen en code fences nog één keer voordat je publiceert.
