## Apa yang dilakukan alat ini

Konverter ini mengubah HTML yang ditempel atau diimpor menjadi Markdown langsung di browser Anda. Ini membantu saat konten berasal dari CMS, editor, cuplikan ekspor, atau templat email yang disalin, tetapi akhirnya harus masuk ke README, halaman dokumentasi, atau catatan.

## Kapan alat ini membantu

- Membersihkan HTML hasil ekspor sebelum dipublikasikan sebagai Markdown.
- Mengubah fragmen halaman kecil menjadi draf dokumentasi.
- Menormalkan markup yang disalin agar diff tetap mudah dibaca di version control.

## Apa yang perlu diperiksa setelah konversi

- Tabel kompleks, embed, dan gaya inline mungkin masih memerlukan pembersihan manual.
- Pilih gaya heading, daftar, dan blok kode yang cocok dengan varian Markdown tujuan Anda.
- Tinjau tautan, gambar, dan code fence sekali lagi sebelum menerbitkan.
