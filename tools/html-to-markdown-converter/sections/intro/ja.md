## このツールでできること

この変換ツールは、貼り付けた HTML や読み込んだ HTML をブラウザ内で直接 Markdown に変換します。CMS、エディタ、書き出した断片、コピーしたメールテンプレートから始まった内容を、README、ドキュメントページ、メモに落とし込みたいときに便利です。

## 役立つ場面

- 書き出した HTML を Markdown として公開する前に整えるとき。
- 小さなページ断片をドキュメント下書きに変えるとき。
- コピーしたマークアップをそろえて、バージョン管理の diff を読みやすくするとき。

## 変換後に確認したいこと

- 複雑な表、埋め込み要素、インラインスタイルは手作業の調整が必要なことがあります。
- 目的の Markdown 形式に合う見出し、リスト、コードブロックのスタイルを選んでください。
- 公開前にリンク、画像、コードフェンスを一度確認してください。
