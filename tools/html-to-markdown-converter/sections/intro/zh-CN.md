## 这个工具能做什么

这个转换器会直接在浏览器中把你粘贴或导入的 HTML 转成 Markdown。当内容最初来自 CMS、编辑器、导出片段或复制来的邮件模板，但最终要放进 README、文档页面或笔记时，它会很有帮助。

## 什么时候适合使用

- 在把导出的 HTML 发布成 Markdown 之前先清理一遍。
- 把小型页面片段转换成文档草稿。
- 规范复制来的标记内容，让版本控制中的 diff 更容易阅读。

## 转换后要检查什么

- 复杂表格、嵌入内容和内联样式通常仍然需要手动清理。
- 选择与目标 Markdown 风格一致的标题、列表和代码块样式。
- 发布前再预览一次链接、图片和代码围栏。
