## เครื่องมือนี้ทำอะไรได้บ้าง

ตัวแปลงนี้จะเปลี่ยน HTML ที่คุณวางหรือนำเข้าให้เป็น Markdown ได้โดยตรงในเบราว์เซอร์ เหมาะเมื่อเนื้อหาเริ่มจาก CMS, ตัวแก้ไข, ชิ้นส่วนที่ส่งออก หรือเทมเพลตอีเมลที่คัดลอกมา แต่สุดท้ายต้องไปอยู่ใน README, หน้าเอกสาร หรือโน้ต

## ใช้เมื่อไรจึงเหมาะ

- ทำความสะอาด HTML ที่ส่งออกมาก่อนเผยแพร่เป็น Markdown
- เปลี่ยนชิ้นส่วนหน้าเล็ก ๆ ให้เป็นร่างเอกสาร
- ทำให้มาร์กอัปที่คัดลอกมามีรูปแบบสม่ำเสมอ เพื่อให้ diff ในระบบควบคุมเวอร์ชันยังอ่านง่าย

## สิ่งที่ควรตรวจหลังการแปลง

- ตารางที่ซับซ้อน เนื้อหาฝัง และสไตล์แบบ inline อาจยังต้องจัดการด้วยมือ
- เลือกรูปแบบหัวข้อ รายการ และบล็อกโค้ดให้ตรงกับ Markdown เป้าหมายของคุณ
- ตรวจลิงก์ รูปภาพ และ code fence อีกครั้งก่อนเผยแพร่
