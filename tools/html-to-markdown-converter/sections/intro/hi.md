## यह टूल क्या करता है

यह कन्वर्टर पेस्ट किए गए या आयात किए गए HTML को सीधे आपके ब्राउज़र में Markdown में बदल देता है। यह तब उपयोगी होता है जब सामग्री की शुरुआत CMS, एडिटर, एक्सपोर्ट किए गए स्निपेट या कॉपी किए गए ईमेल टेम्पलेट से होती है, लेकिन अंत में उसे README, दस्तावेज़ पेज या नोट में जाना होता है।

## यह कब मदद करता है

- एक्सपोर्ट किए गए HTML को Markdown के रूप में प्रकाशित करने से पहले साफ़ करना।
- छोटे पेज फ्रैगमेंट को दस्तावेज़ ड्राफ्ट में बदलना।
- कॉपी किए गए मार्कअप को सामान्य बनाना ताकि version control में diff पढ़ने योग्य रहे।

## रूपांतरण के बाद क्या जाँचें

- जटिल तालिकाएँ, एम्बेड और inline styles अभी भी मैनुअल सफ़ाई मांग सकते हैं।
- अपने लक्ष्य Markdown शैली से मेल खाने वाले heading, list और code block फ़ॉर्मेट चुनें।
- प्रकाशित करने से पहले links, images और code fences एक बार देख लें।
