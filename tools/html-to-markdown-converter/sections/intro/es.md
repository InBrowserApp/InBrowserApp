## Qué hace esta herramienta

Este conversor transforma el HTML que pegas o importas en Markdown directamente en tu navegador. Resulta útil cuando el contenido empieza en un CMS, editor, fragmento exportado o plantilla de correo copiada, pero debe terminar en un README, una página de documentación o una nota.

## Cuándo ayuda

- Limpiar HTML exportado antes de publicarlo como Markdown.
- Convertir pequeños fragmentos de página en borradores de documentación.
- Normalizar marcado copiado para que los diffs sigan siendo legibles en control de versiones.

## Qué revisar después de convertir

- Las tablas complejas, los elementos incrustados y los estilos en línea todavía pueden requerir limpieza manual.
- Elige los estilos de encabezados, listas y bloques de código que coincidan con tu variante de Markdown.
- Revisa enlaces, imágenes y cercas de código una vez antes de publicar.
