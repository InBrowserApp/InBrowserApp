## 이 도구가 하는 일

이 변환기는 붙여넣거나 가져온 HTML을 브라우저 안에서 바로 Markdown으로 바꿉니다. 콘텐츠가 CMS, 편집기, 내보낸 조각, 복사한 이메일 템플릿에서 시작했지만 최종적으로 README, 문서 페이지, 메모에 들어가야 할 때 유용합니다.

## 도움이 되는 경우

- 내보낸 HTML을 Markdown으로 게시하기 전에 정리할 때.
- 작은 페이지 조각을 문서 초안으로 바꿀 때.
- 복사한 마크업을 정규화해서 버전 관리 diff를 읽기 쉽게 만들 때.

## 변환 후 확인할 점

- 복잡한 표, 임베드, 인라인 스타일은 여전히 수동 정리가 필요할 수 있습니다.
- 대상 Markdown 형식에 맞는 제목, 목록, 코드 블록 스타일을 선택하세요.
- 게시하기 전에 링크, 이미지, 코드 펜스를 한 번 미리 확인하세요.
