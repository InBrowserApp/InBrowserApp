## Apakah ID penduduk PRC?

Nombor ID penduduk PRC 18 digit merangkumi kod alamat, tarikh lahir, kod urutan dan checksum. Alat ini mengesahkan bahagian tersebut serta mengekstrak jantina dan umur.

- Nombor mesti 18 digit
- Format tidak sah (digit atau X)
- Kod wilayah pentadbiran tidak diketahui
- Tarikh lahir tidak sah
- Checksum tidak sah

Kod wilayah berdasarkan set data pembahagian pentadbiran 2023.
