## 中国居民身份证番号とは？

18 桁の中国居民身份证番号には、住所コード、生年月日、順序コード、チェックディジットが含まれます。このバリデーターはそれらをオフラインで検証し、番号の構造を確認できるようにします。

### このバリデーターの検証内容

- 空白やハイフンを除去し、最後の文字を大文字の `X` に正規化します
- 17 桁の数字と最後の数字または `X` を含む、合計 18 文字であることを要求します
- 先頭 6 桁を 2023 年の行政区画データセットと照合し、8 桁の生年月日を解析します
- 先頭 17 桁からチェックディジットを再計算し、最後の文字と比較します

### 結果に表示される内容

- 地域の内訳: 省、市、区/県、および元の地域コード
- 生年月日、現在の年齢、順序コード、および順序コードから導かれる性別
- 正規化された ID と、デバッグ用の期待値/実際のチェックディジット

### 例

`110101199001010015` は次のように読めます。

- `110101` -> 北京市東城区
- `19900101` -> 生年月日 `1990-01-01`
- `001` -> 順序コード
- `5` -> チェックディジット

### 重要な注意

このツールは、構造とチェックサムのオフライン検証のみを行います。これらの検証に通っても、実在する人物や現在有効な証明書であることを保証するものではありません。

地域名は 2023 年の行政区画データセットに基づいています。
