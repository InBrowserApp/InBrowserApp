## Czym jest ID mieszkańca ChRL?

18-cyfrowy numer ID mieszkańca ChRL zawiera kod adresu, datę urodzenia, kod sekwencji i sumę kontrolną. To narzędzie weryfikuje te elementy i określa płeć oraz wiek.

- Numer musi mieć 18 cyfr
- Nieprawidłowy format (cyfry lub X)
- Nieznany kod regionu administracyjnego
- Nieprawidłowa data urodzenia
- Nieprawidłowa suma kontrolna

Kody regionów oparto na zestawie danych podziału administracyjnego z 2023 roku.
