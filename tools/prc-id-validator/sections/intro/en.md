## What is a PRC Resident ID?

The 18-digit PRC Resident ID number includes the address code, birthdate, sequence code, and checksum. This tool validates those parts and extracts gender and age.

- ID number must be 18 digits
- Invalid format (digits or X)
- Unknown administrative region code
- Invalid birthdate
- Invalid checksum

Region codes are based on the 2023 administrative division dataset.
