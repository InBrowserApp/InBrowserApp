## Apa itu ID penduduk Tiongkok?

Nomor ID penduduk 18 digit mencakup kode alamat, tanggal lahir, kode urutan, dan checksum. Alat ini memvalidasi bagian tersebut serta menurunkan jenis kelamin dan usia.

- Nomor harus 18 digit
- Format tidak valid (digit atau X)
- Kode wilayah administratif tidak dikenal
- Tanggal lahir tidak valid
- Checksum tidak valid

Kode wilayah berdasarkan dataset pembagian administratif tahun 2023.
