## 중국 주민 신분증 번호란?

18자리 중국 주민 신분증 번호에는 주소 코드, 생년월일, 순번 코드, 검사 숫자가 포함됩니다. 이 검증기는 이러한 요소를 오프라인에서 확인하고 번호 구조를 이해할 수 있게 도와줍니다.

### 검증 방식

- 공백과 하이픈을 제거하고 마지막 문자를 대문자 `X` 로 정규화합니다
- 앞의 17자리는 숫자이고 마지막 1자리는 숫자 또는 `X` 인 총 18자리 형식을 요구합니다
- 앞 6자리를 2023년 행정구역 데이터와 대조하고 8자리 생년월일을 해석합니다
- 앞 17자리로 검사 숫자를 다시 계산한 뒤 마지막 문자와 비교합니다

### 결과에 표시되는 정보

- 지역 분해: 성, 시, 구/현 및 원본 지역 코드
- 생년월일, 현재 나이, 순번 코드, 그리고 순번 코드에서 유도한 성별
- 디버깅을 위한 정규화된 ID와 예상/실제 검사 숫자

### 예시

`110101199001010015` 는 다음과 같이 읽을 수 있습니다:

- `110101` -> 베이징시 둥청구
- `19900101` -> 생년월일 `1990-01-01`
- `001` -> 순번 코드
- `5` -> 검사 숫자

### 중요한 참고

이 도구는 오프라인에서 구조와 체크섬만 검증합니다. 이러한 검사를 통과했다고 해서 실제 신원이나 현재 유효한 신분증임을 증명하는 것은 아닙니다.

지역명은 2023년 행정구역 데이터셋을 기준으로 합니다.
