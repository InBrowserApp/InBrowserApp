## Cos'è l'ID residente della RPC?

Il numero di ID residente della RPC ha 18 cifre e include codice di indirizzo, data di nascita, codice di sequenza e checksum. Questo strumento valida queste parti ed estrae genere ed età.

- Il numero deve avere 18 cifre
- Formato non valido (solo cifre o X)
- Codice regione amministrativa sconosciuto
- Data di nascita non valida
- Checksum non valido

I codici di regione si basano sul dataset delle divisioni amministrative 2023.
