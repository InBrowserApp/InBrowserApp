## O que é o ID de residente da RPC?

O número de ID de residente da RPC tem 18 dígitos e inclui código de endereço, data de nascimento, código de sequência e checksum. Esta ferramenta valida essas partes e extrai gênero e idade.

- O número deve ter 18 dígitos
- Formato inválido (somente dígitos ou X)
- Código de região administrativa desconhecido
- Data de nascimento inválida
- Checksum inválido

Os códigos de região são baseados no conjunto de dados de divisões administrativas de 2023.
