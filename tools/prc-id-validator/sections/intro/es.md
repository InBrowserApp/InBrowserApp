## ¿Qué es el ID de residente de la RPC?

El número de ID de residente de la RPC tiene 18 dígitos e incluye código de dirección, fecha de nacimiento, código de secuencia y checksum. Esta herramienta valida esas partes y extrae género y edad.

- El número debe tener 18 dígitos
- Formato inválido (solo dígitos o X)
- Código de región administrativa desconocido
- Fecha de nacimiento inválida
- Checksum inválido

Los códigos de región se basan en el conjunto de datos de divisiones administrativas de 2023.
