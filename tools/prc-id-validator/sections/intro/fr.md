## Qu'est-ce que l'ID de résident de la RPC ?

Le numéro d'ID de résident de la RPC comporte 18 chiffres et inclut le code d'adresse, la date de naissance, le code de séquence et la somme de contrôle. Cet outil valide ces éléments et déduit le genre et l'âge.

- Le numéro doit contenir 18 chiffres
- Format invalide (chiffres ou X)
- Code de région administrative inconnu
- Date de naissance invalide
- Somme de contrôle invalide

Les codes de région sont basés sur le jeu de données 2023 des divisions administratives.
