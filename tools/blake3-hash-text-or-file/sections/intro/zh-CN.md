## 什么是 BLAKE3？

BLAKE3 是一种现代加密散列函数，源自 BLAKE2。它在保持强安全性的同时，强调极高性能与并行处理。它默认输出 256 位哈希，并支持可扩展输出长度（XOF）。

**主要特征：**

- **可扩展输出长度**：可产生任意长度的哈希
- **高性能**：在现代 CPU 上速度快且可并行
- **确定性**：相同输入总是产生相同哈希
- **雪崩效应**：输入的微小变化会产生截然不同的输出
- **不可逆**：从哈希反推原始输入在计算上不可行
- **抗碰撞**：很难找到产生相同哈希的两个不同输入
- **密钥散列**：支持可选的 32 字节密钥用于 MAC 功能
- **密钥派生**：可从密钥材料与上下文派生子密钥

**常见用途：**

- 文件完整性验证
- 内容寻址存储与去重
- 数字签名和证书
- 密码存储和身份验证
- 加密协议和系统
