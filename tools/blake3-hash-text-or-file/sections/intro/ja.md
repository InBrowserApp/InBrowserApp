## BLAKE3とは？

BLAKE3はBLAKE2から派生した最新の暗号学的ハッシュ関数です。高い安全性を保ちながら、非常に高い性能と並列処理を重視しています。デフォルトで256ビットのハッシュを生成し、可拡張出力長（XOF）をサポートします。

**主な特徴：**

- **可拡張出力長**：任意の長さのハッシュを生成可能
- **高性能**：最新CPUで高速かつ並列化可能
- **決定論的**：同じ入力は常に同じハッシュを生成
- **雪崩効果**：入力の小さな変化が大きく異なる出力を生む
- **不可逆**：ハッシュを逆算して元の入力を見つけることは計算上不可能
- **衝突耐性**：同じハッシュを生成する2つの異なる入力を見つけることは非常に困難
- **キー付きハッシュ**：MAC機能のためのオプションの32バイトキーをサポート
- **キー導出**：鍵材料とコンテキストからサブキーを導出可能

**一般的な用途：**

- ファイル整合性検証
- コンテンツアドレス型ストレージと重複排除
- デジタル署名と証明書
- パスワード保存と認証
- 暗号プロトコルとシステム
