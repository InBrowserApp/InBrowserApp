## 什麼是 BLAKE3？

BLAKE3 是一種現代加密雜湊函數，源自 BLAKE2。它在維持高安全性的同時，強調極高效能與平行處理。它預設輸出 256 位元雜湊，並支援可延展輸出長度（XOF）。

**主要特徵：**

- **可延展輸出長度**：可產生任意長度的雜湊
- **高效能**：在現代 CPU 上速度快且可平行化
- **確定性**：相同輸入總是產生相同雜湊
- **雪崩效應**：輸入的微小變化會產生截然不同的輸出
- **不可逆**：從雜湊反推原始輸入在計算上不可行
- **抗碰撞**：很難找到產生相同雜湊的兩個不同輸入
- **金鑰雜湊**：支援可選的 32 位元組金鑰以提供 MAC 功能
- **金鑰派生**：可從金鑰材料與上下文派生子金鑰

**常見用途：**

- 檔案完整性驗證
- 內容尋址儲存與去重
- 數位簽章和憑證
- 密碼儲存和身份驗證
- 加密協定和系統
