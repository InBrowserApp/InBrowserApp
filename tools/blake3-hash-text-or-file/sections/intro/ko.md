## BLAKE3란?

BLAKE3는 BLAKE2에서 파생된 최신 암호화 해시 함수입니다. 높은 보안을 유지하면서 매우 높은 성능과 병렬 처리를 목표로 설계되었습니다. 기본적으로 256비트 해시를 생성하며 확장 가능한 출력 길이(XOF)를 지원합니다.

**주요 특징:**

- **확장 가능한 출력 길이**: 임의 길이의 해시를 생성할 수 있음
- **고성능**: 최신 CPU에서 빠르고 병렬화 가능
- **결정론적**: 동일한 입력은 항상 동일한 해시를 생성
- **눈사태 효과**: 입력의 작은 변화가 크게 다른 출력을 생성
- **비가역적**: 해시를 역산해 원본 입력을 찾는 것은 계산적으로 불가능
- **충돌 저항성**: 동일한 해시를 생성하는 두 개의 다른 입력을 찾기 매우 어려움
- **키 해싱**: MAC 기능을 위한 선택적 32바이트 키 지원
- **키 파생**: 키 재료와 컨텍스트에서 하위 키를 파생 가능

**일반적인 용도:**

- 파일 무결성 검증
- 콘텐츠 주소 지정 저장 및 중복 제거
- 디지털 서명 및 인증서
- 비밀번호 저장 및 인증
- 암호화 프로토콜 및 시스템
