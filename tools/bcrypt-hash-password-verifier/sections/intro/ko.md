## 기능

평문 비밀번호가 bcrypt 비밀번호 해시와 일치하는지 검증합니다. 로그인 코드를 디버깅하거나, 가져온 사용자 레코드를 확인하거나, 비밀번호 마이그레이션 후 해시 호환성이 유지되었는지 확인할 때 유용합니다.

## 허용되는 입력

`$2b$10$...` 같은 표준 bcrypt 해시를 붙여넣고 테스트할 비밀번호 후보를 입력하세요. 이 검증기는 비용 값 `04`부터 `31`까지의 `$2a$`, `$2b$`, `$2y$` 접두사를 허용합니다.

## 결과 읽기

일치 결과는 해시 문자열에 포함된 솔트와 비용까지 포함해 bcrypt가 해당 해시에 대해 비밀번호를 받아들였다는 뜻입니다. 불일치는 비밀번호가 검증되지 않았다는 뜻이며, 해시 자체가 안전하지 않다는 증거는 아닙니다. 잘못된 해시 오류는 보통 접두사, 비용, 길이 또는 bcrypt base64 문자가 올바르지 않다는 의미입니다.

## 개인정보 보호 및 보안 참고

- 검증은 브라우저에서 로컬로 실행됩니다.
- 비밀번호와 해시는 로컬 저장소에 저장되지 않습니다.
- bcrypt는 범용 파일 체크섬이 아니라 비밀번호 저장을 위해 설계되었습니다.
- 이 도구는 디버깅과 검증에 사용하고, 프로덕션 인증 시스템의 유일한 감사 수단으로 사용하지 마세요.
