## למה הכלי הזה מיועד

השתמשו בכלי הזה כשצריך להפוך JSON תקין ל-XML שקל יותר לקרוא, לשתף או לעשות לו commit. הכל רץ מקומית בדפדפן.

## איפה הוא עוזר

- להמיר דוגמאות API ל-XML עבור תיעוד או קובצי הגדרה.
- לכתוב מחדש payloads של JSON שהועתקו לפורמט שקל יותר להשוות בעין.
- להכין תוכן מובנה לכלים ולזרימות עבודה שמצפים ל-XML.

## למה לשים לב

- בכלי הזה הקלט חייב להיות JSON תקין.
- הממיר שומר על מבנה הנתונים, אבל הצגת ה-XML עדיין יכולה להיות שונה מסגנון שנכתב ידנית.
- אם צריך את הכיוון ההפוך, השתמשו בממיר XML ל-JSON.
