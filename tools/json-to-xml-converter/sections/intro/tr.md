## Bu araç ne için kullanılır

Geçerli JSON'u okuması, paylaşması veya commit etmesi daha kolay XML'a dönüştürmeniz gerektiğinde bu aracı kullanın. Her şey tarayıcıda yerel olarak çalışır.

## Nerede işe yarar

- API örneklerini dokümantasyon veya yapılandırma dosyaları için XML'a dönüştürmek.
- Kopyalanmış JSON payload'larını gözle karşılaştırması daha kolay bir biçime çevirmek.
- XML bekleyen araçlar ve iş akışları için yapılandırılmış içerik hazırlamak.

## Nelere dikkat edilmeli

- Bu araçta girdi geçerli JSON olmalıdır.
- Dönüştürücü veri yapısını korur, ancak XML sunumu elle yazılmış bir stilden farklı olabilir.
- Ters yöne dönmeniz gerekiyorsa bunun yerine XML'dan JSON'a dönüştürücüyü kullanın.
