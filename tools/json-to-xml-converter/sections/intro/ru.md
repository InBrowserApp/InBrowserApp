## Для чего нужен этот инструмент

Используйте этот инструмент, когда нужно превратить корректный JSON в XML, который легче читать, делиться им или коммитить. Все выполняется локально в браузере.

## Где он полезен

- Преобразовывать примеры API в XML для документации или конфигурационных файлов.
- Переписывать скопированные JSON-payload'ы в формат, который проще сравнивать глазами.
- Подготавливать структурированный контент для инструментов и процессов, которые ожидают XML.

## Что важно учитывать

- В этом инструменте входные данные должны быть корректным JSON.
- Конвертер сохраняет структуру данных, но представление XML может отличаться от написанного вручную стиля.
- Если нужно преобразование в обратную сторону, используйте конвертер XML в JSON.
