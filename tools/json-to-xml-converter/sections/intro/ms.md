## Kegunaan alat ini

Gunakan alat ini apabila anda perlu menukar JSON yang sah kepada XML yang lebih mudah dibaca, dikongsi atau di-commit. Semuanya berjalan secara setempat dalam pelayar.

## Bila ia membantu

- Menukar contoh API kepada XML untuk dokumentasi atau fail konfigurasi.
- Menulis semula payload JSON yang disalin kepada format yang lebih mudah dibandingkan dengan mata.
- Menyediakan kandungan berstruktur untuk alat dan aliran kerja yang menjangkakan XML.

## Perkara yang perlu diberi perhatian

- Dalam alat ini, input mestilah JSON yang sah.
- Penukar mengekalkan struktur data, tetapi persembahan XML masih boleh berbeza daripada gaya yang ditulis tangan.
- Jika anda perlu pergi ke arah sebaliknya, gunakan penukar XML ke JSON.
