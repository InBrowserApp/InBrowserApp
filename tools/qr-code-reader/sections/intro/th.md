## QR Code Reader ทำอะไร

QR Code Reader ดึงข้อความจาก QR Code ในรูปภาพภายในเครื่องหรือจากกล้อง ใช้ได้กับลิงก์ อีเมล โทรศัพท์ Wi-Fi vCard ปฏิทิน ตำแหน่ง และข้อความธรรมดา

## ควรใช้เมื่อใด

ใช้กับภาพหน้าจอ ป้ายที่ดาวน์โหลด รูปบรรจุภัณฑ์ หรือ QR Code จริง เมื่อต้องการตรวจสอบเนื้อหาก่อนเปิดหรือคัดลอก

## ความเป็นส่วนตัวและกล้อง

รูปภาพจะถอดรหัสในเบราว์เซอร์และไม่ถูกอัปโหลด กล้องใช้สิทธิ์จากเบราว์เซอร์และหยุดอัตโนมัติหลังอ่านรหัสได้

## เคล็ดลับให้สแกนได้ดีขึ้น

- ใช้รูปที่ชัดและเห็น QR Code ทั้งหมด
- เพิ่มแสงและหลีกเลี่ยงแสงสะท้อนเมื่อใช้กล้อง
- ถือกล้องให้นิ่งและให้รหัสอยู่ในกรอบสี่เหลี่ยม
- หาก SVG หรือรูปที่แก้ไขแล้วอ่านไม่ได้ ให้ส่งออกเป็น PNG หรือ JPEG
