## Ce que fait QR Code Reader

QR Code Reader extrait le texte stocké dans un QR code depuis une image locale ou votre caméra. Il fonctionne pour les liens, adresses e-mail, liens téléphoniques, chaînes Wi-Fi, vCards, données de calendrier, positions et texte brut.

## Quand l'utiliser

Utilisez cet outil avec une capture d'écran, une étiquette téléchargée, une photo d'emballage ou un QR code physique lorsque vous voulez inspecter le contenu exact avant d'agir. La valeur décodée reste visible pour la copier, ouvrir les liens pris en charge ou la comparer à la source.

## Confidentialité et caméra

Les images sont décodées localement dans le navigateur et ne sont pas envoyées. Le scan par caméra utilise la demande d'autorisation du navigateur et s'arrête automatiquement après décodage. Vous pouvez aussi arrêter la caméra manuellement à tout moment.

## Conseils pour de meilleurs scans

- Utilisez une image nette, non recadrée, où tout le QR code est visible.
- Augmentez la luminosité et évitez les reflets avec la caméra.
- Gardez la caméra stable et le code dans le cadre carré.
- Si un SVG ou une image modifiée ne se scanne pas, exportez-la en PNG ou JPEG puis réessayez.
