## QR Code 讀取器的作用

QR Code 讀取器可以從本機圖片或攝影機中擷取 QR Code 內的文字。它適用於連結、電子郵件地址、電話連結、Wi-Fi 設定字串、vCard、日曆資料、位置和純文字等常見內容。

## 何時使用它

當你有截圖、下載的標籤、包裝照片或實體 QR Code，並想在操作前查看確切內容時，可以使用此工具。解碼值會保持可見，方便複製、開啟支援的連結或與來源核對。

## 隱私和攝影機行為

圖片會在瀏覽器中本機解碼，不會上傳。攝影機掃描使用瀏覽器的權限提示，並會在解碼出內容後自動停止。你也可以隨時手動停止攝影機。

## 獲得更好掃描結果的提示

- 使用清晰、未裁切且完整顯示 QR Code 的圖片。
- 用攝影機掃描時增加亮度並避免反光。
- 保持攝影機穩定，並讓代碼位於方形框內。
- 如果 SVG 或編輯後的圖片無法掃描，請匯出為 PNG 或 JPEG 後重試。
