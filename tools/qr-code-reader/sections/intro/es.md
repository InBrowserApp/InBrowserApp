## Qué hace QR Code Reader

QR Code Reader extrae el texto almacenado dentro de un código QR desde una imagen local o desde tu cámara. Funciona con enlaces, direcciones de email, enlaces telefónicos, cadenas de configuración Wi-Fi, vCards, datos de calendario, ubicaciones y texto plano.

## Cuándo usarlo

Úsalo cuando tengas una captura, una etiqueta descargada, una foto de un paquete o un código QR físico y quieras inspeccionar la carga exacta antes de actuar. El valor decodificado queda visible para copiarlo, abrir enlaces compatibles o compararlo con la fuente.

## Privacidad y comportamiento de la cámara

Las imágenes se decodifican localmente en el navegador y no se suben. El escaneo con cámara usa el permiso del navegador y se detiene automáticamente después de decodificar un código. También puedes detener la cámara manualmente en cualquier momento.

## Consejos para mejores escaneos

- Usa una imagen clara y sin recortes donde se vea todo el código QR.
- Aumenta la luz y evita reflejos al escanear con cámara.
- Mantén la cámara estable y el código dentro del marco cuadrado.
- Si un SVG o una imagen editada no se escanea, expórtala como PNG o JPEG e inténtalo de nuevo.
