## Что делает этот инструмент

Преобразование цветов между форматами HEX, RGB, HSL, HSV, HWB, LAB, LCH, CMYK и ключевыми словами CSS. Двунаправленное преобразование в реальном времени с поддержкой альфа-канала.

## Примечания по форматам

- `HEX`: Шестнадцатеричная запись цвета, наиболее распространена в веб-разработке
- `RGB`: Красный, Зелёный, Синий - аддитивная цветовая модель для экранов
- `HSL`: Оттенок, Насыщенность, Светлота - интуитивная настройка цвета
- `HSV`: Оттенок, Насыщенность, Значение - также известен как HSB, распространён в палитрах
- `HWB`: Оттенок, Белизна, Чернота - формат CSS4, без альфа
- `LAB`: CIE LAB - перцептивно однородное цветовое пространство, без альфа
- `LCH`: Светлота, Цветность, Оттенок - цилиндрический LAB, легче настраивать оттенок, без альфа
- `CMYK`: Голубой, Пурпурный, Жёлтый, Чёрный - стандарт печати, без альфа
- `CSS-ключевое слово`: Именованные CSS-цвета как red, blue, coral - 140+ предопределённых цветов, без альфа
