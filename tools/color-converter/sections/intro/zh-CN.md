## 这个工具能做什么

在 HEX、RGB、HSL、HSV、HWB、LAB、LCH、CMYK 和 CSS 关键字格式之间转换颜色。支持 Alpha 通道的实时双向转换。

## 格式说明

- `HEX`: 十六进制颜色表示法，网页开发最常用
- `RGB`: 红、绿、蓝三原色模型，屏幕显示标准
- `HSL`: 色相、饱和度、亮度 - 更直观的颜色调整
- `HSV`: 色相、饱和度、明度 - 也称 HSB，常用于取色器
- `HWB`: 色相、白度、黑度 - CSS4 颜色格式，不支持透明度
- `LAB`: CIE LAB - 感知均匀色彩空间，不支持透明度
- `LCH`: 明度、色度、色相 - LAB 的圆柱坐标形式，更易调整色相，不支持透明度
- `CMYK`: 青、品红、黄、黑 - 印刷标准，不支持透明度
- `CSS 关键字`: CSS 命名颜色如 red、blue、coral - 140+ 预定义颜色，不支持透明度
