## Apa yang alat ini lakukan

Tukar warna antara format HEX, RGB, HSL, HSV, HWB, LAB, LCH, CMYK dan kata kunci CSS. Penukaran dua hala masa nyata dengan sokongan saluran alfa.

## Nota format

- `HEX`: Notasi warna heksadesimal, paling biasa dalam pembangunan web
- `RGB`: Merah, Hijau, Biru - model warna aditif untuk skrin
- `HSL`: Hue, Ketepuan, Kecerahan - pelarasan warna intuitif
- `HSV`: Hue, Ketepuan, Nilai - juga dikenali sebagai HSB, biasa dalam pemilih warna
- `HWB`: Hue, Keputihan, Kehitaman - format CSS4, tiada alfa
- `LAB`: CIE LAB - ruang warna seragam persepsi, tiada alfa
- `LCH`: Kecerahan, Kroma, Hue - LAB silinder, pelarasan hue lebih mudah, tiada alfa
- `CMYK`: Sian, Magenta, Kuning, Hitam - standard cetakan, tiada alfa
- `Kata kunci CSS`: Nama warna CSS seperti red, blue, coral - 140+ warna pratakrif, tiada alfa
