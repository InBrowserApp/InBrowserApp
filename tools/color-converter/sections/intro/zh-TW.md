## 這個工具能做什麼

在 HEX、RGB、HSL、HSV、HWB、LAB、LCH、CMYK 和 CSS 關鍵字格式之間轉換顏色。支援 Alpha 通道的即時雙向轉換。

## 格式說明

- `HEX`: 十六進制顏色表示法，網頁開發最常用
- `RGB`: 紅、綠、藍三原色模型，螢幕顯示標準
- `HSL`: 色相、飽和度、亮度 - 更直觀的顏色調整
- `HSV`: 色相、飽和度、明度 - 也稱 HSB，常用於取色器
- `HWB`: 色相、白度、黑度 - CSS4 顏色格式，不支援透明度
- `LAB`: CIE LAB - 感知均勻色彩空間，不支援透明度
- `LCH`: 明度、色度、色相 - LAB 的圓柱坐標形式，更易調整色相，不支援透明度
- `CMYK`: 青、洋紅、黃、黑 - 印刷標準，不支援透明度
- `CSS 關鍵字`: CSS 命名顏色如 red、blue、coral - 140+ 預定義顏色，不支援透明度
