## Fungsi alat ini

Konversi warna antara format HEX, RGB, HSL, HSV, HWB, LAB, LCH, CMYK, dan kata kunci CSS. Konversi dua arah waktu nyata dengan dukungan saluran alfa.

## Catatan format

- `HEX`: Notasi warna heksadesimal, paling umum dalam pengembangan web
- `RGB`: Merah, Hijau, Biru - model warna aditif untuk layar
- `HSL`: Hue, Saturasi, Kecerahan - penyesuaian warna intuitif
- `HSV`: Hue, Saturasi, Value - juga dikenal sebagai HSB, umum di pemilih warna
- `HWB`: Hue, Putih, Hitam - format CSS4, tanpa alfa
- `LAB`: CIE LAB - ruang warna seragam perseptual, tanpa alfa
- `LCH`: Kecerahan, Kroma, Hue - LAB silinder, penyesuaian hue lebih mudah, tanpa alfa
- `CMYK`: Sian, Magenta, Kuning, Hitam - standar cetak, tanpa alfa
- `Kata kunci CSS`: Nama warna CSS seperti red, blue, coral - 140+ warna terdefinisi, tanpa alfa
