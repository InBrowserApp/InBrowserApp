## Ce que fait cet outil

Convertir les couleurs entre les formats HEX, RGB, HSL, HSV, HWB, LAB, LCH, CMYK et mots-clés CSS. Conversion bidirectionnelle en temps réel avec support du canal alpha.

## Notes sur les formats

- `HEX`: Notation hexadécimale, la plus courante en développement web
- `RGB`: Rouge, Vert, Bleu - modèle de couleur additif pour écrans
- `HSL`: Teinte, Saturation, Luminosité - ajustements de couleur intuitifs
- `HSV`: Teinte, Saturation, Valeur - aussi appelé HSB, courant dans les sélecteurs de couleur
- `HWB`: Teinte, Blancheur, Noirceur - format CSS4, sans alpha
- `LAB`: CIE LAB - espace colorimétrique perceptuellement uniforme, sans alpha
- `LCH`: Luminosité, Chroma, Teinte - LAB cylindrique, ajustement de teinte plus facile, sans alpha
- `CMYK`: Cyan, Magenta, Jaune, Noir - standard d'impression, sans alpha
- `Mot-clé CSS`: Couleurs CSS nommées comme red, blue, coral - 140+ couleurs prédéfinies, sans alpha
