## เครื่องมือนี้ทำอะไรได้บ้าง

แปลงสีระหว่างรูปแบบ HEX, RGB, HSL, HSV, HWB, LAB, LCH, CMYK และคำสำคัญ CSS การแปลงแบบสองทิศทางในเวลาจริงพร้อมรองรับช่องอัลฟา

## หมายเหตุเกี่ยวกับรูปแบบ

- `HEX`: สัญกรณ์สีฐานสิบหก ใช้มากที่สุดในการพัฒนาเว็บ
- `RGB`: แดง เขียว น้ำเงิน - โมเดลสีบวกสำหรับหน้าจอ
- `HSL`: เฉดสี ความอิ่มตัว ความสว่าง - ปรับสีได้ง่าย
- `HSV`: เฉดสี ความอิ่มตัว ค่า - เรียกอีกอย่างว่า HSB ใช้ทั่วไปในตัวเลือกสี
- `HWB`: เฉดสี ความขาว ความดำ - รูปแบบ CSS4 ไม่มีอัลฟา
- `LAB`: CIE LAB - พื้นที่สีสม่ำเสมอทางการรับรู้ ไม่มีอัลฟา
- `LCH`: ความสว่าง โครมา เฉดสี - LAB ทรงกระบอก ปรับเฉดสีง่ายขึ้น ไม่มีอัลฟา
- `CMYK`: ฟ้า แดงม่วง เหลือง ดำ - มาตรฐานการพิมพ์ ไม่มีอัลฟา
- `คีย์เวิร์ด CSS`: ชื่อสี CSS เช่น red, blue, coral - 140+ สีที่กำหนดไว้ ไม่มีอัลฟา
