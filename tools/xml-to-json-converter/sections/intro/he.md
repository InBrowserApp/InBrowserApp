## למה הכלי הזה מיועד

השתמשו בכלי הזה כשצריך להפוך XML תקין ל-JSON שקל יותר לקרוא, לשתף או לעשות לו commit. הכל רץ מקומית בדפדפן.

## איפה הוא עוזר

- להמיר דוגמאות API ל-JSON עבור תיעוד או קובצי הגדרה.
- לכתוב מחדש payloads של XML שהועתקו לפורמט שקל יותר להשוות בעין.
- להכין תוכן מובנה לכלים ולזרימות עבודה שמצפים ל-JSON.

## למה לשים לב

- בכלי הזה הקלט חייב להיות XML תקין.
- הממיר שומר על מבנה הנתונים, אבל הצגת ה-JSON עדיין יכולה להיות שונה מסגנון שנכתב ידנית.
- אם צריך את הכיוון ההפוך, השתמשו בממיר JSON ל-XML.
