## Waar deze tool voor is

Gebruik deze tool wanneer je geldige XML wilt omzetten naar JSON dat makkelijker te lezen, delen of committen is. Alles draait lokaal in de browser.

## Waar het helpt

- API-voorbeelden omzetten naar JSON voor documentatie of configuratiebestanden.
- Gekopieerde XML-payloads herschrijven naar een formaat dat visueel makkelijker te vergelijken is.
- Gestructureerde inhoud voorbereiden voor tools en workflows die JSON verwachten.

## Waar je op moet letten

- In deze tool moet de invoer geldige XML zijn.
- De converter bewaart de datastructuur, maar de JSON-weergave kan nog steeds verschillen van handgeschreven stijl.
- Als je de andere kant op wilt, gebruik dan de JSON-naar-XML-converter.
