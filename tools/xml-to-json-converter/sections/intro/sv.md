## Vad verktyget är till för

Använd verktyget när du behöver göra om giltig XML till JSON som är lättare att läsa, dela eller committa. Allt körs lokalt i webbläsaren.

## När det hjälper

- Konvertera API-exempel till JSON för dokumentation eller konfigurationsfiler.
- Skriva om kopierade XML-payloads till ett format som är lättare att jämföra med ögat.
- Förbereda strukturerat innehåll för verktyg och arbetsflöden som förväntar sig JSON.

## Vad du ska tänka på

- Inmatningen måste vara giltig XML i det här verktyget.
- Konverteraren bevarar datastrukturen, men JSON-presentationen kan ändå skilja sig från handskriven stil.
- Om du behöver gå åt andra hållet ska du använda JSON till XML-konverteraren i stället.
