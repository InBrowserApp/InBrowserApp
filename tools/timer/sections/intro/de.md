## Was dieses Tool macht

Verwenden Sie diesen Timer, um in Ihrem Browser einen Countdown mit Start-, Pause-, Reset- und Vollbild-Steuerung auszuführen. Der Timer speichert die eingestellte Dauer und den aktuellen Zustand im lokalen Speicher, sodass ein Neuladen den Countdown nach Möglichkeit wiederherstellen kann.

## So funktionieren die Hinweise

- Aktivieren Sie den Ton, um beim Erreichen von null einen kurzen Signalton abzuspielen.
- Aktivieren Sie die Vibration, wenn Ihr Gerät sie unterstützt.
- Aktivieren Sie Benachrichtigungen, wenn Sie auch dann einen Browser-Hinweis möchten, wenn der Tab nicht im Vordergrund ist.

## Gut zu wissen

- Die Anzeige zeigt Stunden, Minuten, Sekunden und Hundertstelsekunden.
- Reset setzt den Timer auf die aktuell eingestellte Dauer zurück.
- Wenn Sie während eines laufenden Countdowns neu laden, versucht das Tool, anhand der gespeicherten Endzeit fortzusetzen.
