## Para que esta ferramenta serve

Use este temporizador para executar uma contagem regressiva no navegador com controles de iniciar, pausar, redefinir e tela cheia. O temporizador mantém a duração configurada e o estado atual no armazenamento local, então uma atualização pode restaurar a contagem regressiva quando possível.

## Como os alertas funcionam

- Ative o som para reproduzir um bipe curto quando a contagem regressiva chegar a zero.
- Ative a vibração se o seu dispositivo oferecer suporte.
- Ative as notificações se quiser um alerta do navegador mesmo quando a aba não estiver em primeiro plano.

## Bom saber

- O visor mostra horas, minutos, segundos e centésimos de segundo.
- Redefinir retorna o temporizador para a duração configurada no momento.
- Se você recarregar enquanto uma contagem regressiva estiver ativa, a ferramenta retoma a partir do horário final salvo quando possível.
