## What this tool does

Use this timer to run a countdown in your browser with start, pause, reset, and fullscreen controls. The timer keeps its configured duration and current state in local storage, so a refresh can restore the countdown when possible.

## How alerts work

- Enable sound to play a short beep when the countdown reaches zero.
- Enable vibration if your device supports it.
- Enable notifications if you want a browser alert even when the tab is not frontmost.

## Good to know

- The display shows hours, minutes, seconds, and hundredths of a second.
- Reset returns the timer to the currently configured duration.
- If you reload while a countdown is active, the tool resumes from the saved end time when possible.
