## UUID v3란 무엇인가요?

UUID v3는 이름 기반 UUID 형식입니다. 네임스페이스 UUID와 이름을 받아
MD5로 해시하고, 결과를 표준 UUID 형식으로 변환합니다. 중요한 동작은
결정성입니다. 같은 네임스페이스와 이름은 항상 같은 UUID를 생성합니다.

이 도구는 전적으로 브라우저 안에서 실행됩니다. 네임스페이스, 이름,
생성된 UUID는 결과를 다른 곳에 복사하지 않는 한 기기에 남아 있습니다.

## 언제 사용하나요

- DNS 이름, URL, 객체 경로, 사용자 이름처럼 알려진 이름에 대해 안정적인
  식별자가 필요할 때 UUID v3를 사용하세요.
- 식별하려는 이름의 종류와 맞는 네임스페이스를 선택하세요. DNS와 URL이
  가장 흔한 프리셋입니다.
- 같은 네임스페이스를 일관되게 재사용하세요. 네임스페이스를 바꾸면 이름이
  같아도 생성되는 모든 UUID가 달라집니다.
- 선택할 수 있고 더 강한 해시가 있는 이름 기반 UUID가 필요하다면 UUID v5
  또는 다른 최신 식별자를 선호하세요. UUID v3는 MD5 기반 UUID를 명시적으로
  기대하는 시스템과의 호환성을 위해 존재합니다.

## 안전 관련 참고 사항

UUID v3는 임의 ID가 아니며 비밀도 아닙니다. 네임스페이스와 이름을 아는
사람은 누구나 같은 UUID를 다시 생성할 수 있습니다. 비밀번호, 세션 토큰,
API 키 또는 예측할 수 없어야 하는 다른 값에 사용하지 마세요.
