## What this tool does

PDF Page Number Adder creates a new PDF copy with page numbers drawn onto the
pages you choose. Upload a document, leave the page range empty to number every
page, or target only selected pages with ranges such as `1-3,5,8-10`. You can
control the starting number, number format, page position, font family, text
size, and margins before downloading the result.

## Good use cases

- Adding simple page numbers before sending a draft, packet, contract, or
  worksheet for review.
- Numbering only appendix pages, inserted pages, or a custom page subset
  without changing the rest of the document.
- Creating a privacy-friendly numbered copy when the source PDF should not be
  uploaded to an external service.
- Standardizing placement and margins before printing or combining documents in
  another workflow.

## Privacy notes

The PDF is read and rewritten locally in your browser. This tool does not upload
the file, but the numbered copy still contains the original document content and
most existing PDF metadata. Review the downloaded file before sharing it outside
your team or attaching it to a public issue.

## Limitations

This tool writes plain black page-number text using standard PDF fonts. It does
not remove passwords, repair damaged PDFs, redact content, or change existing
page artwork. The preview is a placement aid and may differ slightly from the
final PDF in other viewers. Encrypted PDFs need to be unlocked before page
numbers can be added.
