## 이 도구의 기능

PDF 페이지 번호 추가 도구는 선택한 페이지에 페이지 번호를 그려 넣은 새
PDF 사본을 만듭니다. 문서를 업로드한 뒤 페이지 범위를 비워 두면 모든
페이지에 번호를 매기고, `1-3,5,8-10` 같은 범위로 선택한 페이지만 지정할
수 있습니다. 결과를 다운로드하기 전에 시작 번호, 번호 형식, 페이지 위치,
글꼴 계열, 텍스트 크기, 여백을 조절할 수 있습니다.

## 좋은 사용 사례

- 검토용 초안, 묶음 자료, 계약서, 워크시트를 보내기 전에 간단한 페이지
  번호를 추가할 때.
- 문서의 나머지 부분은 바꾸지 않고 부록 페이지, 삽입된 페이지, 사용자 지정
  페이지 일부에만 번호를 매길 때.
- 원본 PDF를 외부 서비스에 업로드하면 안 되는 경우 개인정보 보호에 유리한
  번호가 매겨진 사본을 만들 때.
- 다른 워크플로에서 문서를 인쇄하거나 결합하기 전에 배치와 여백을
  표준화할 때.

## 개인정보 보호 참고 사항

PDF는 브라우저에서 로컬로 읽고 다시 작성됩니다. 이 도구는 파일을
업로드하지 않지만, 페이지 번호가 추가된 사본에는 여전히 원본 문서 내용과
대부분의 기존 PDF 메타데이터가 포함됩니다. 팀 외부로 공유하거나 공개
이슈에 첨부하기 전에 다운로드한 파일을 검토하세요.

## 제한 사항

이 도구는 표준 PDF 글꼴을 사용해 일반 검은색 페이지 번호 텍스트를
씁니다. 암호를 제거하거나, 손상된 PDF를 복구하거나, 내용을 가리거나, 원본
페이지 아트워크를 시각적으로 미리 보여 주지는 않습니다. 암호화된 PDF는
페이지 번호를 추가하기 전에 잠금을 해제해야 합니다.
