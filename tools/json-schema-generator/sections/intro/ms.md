## Apakah itu JSON Schema?

JSON Schema ialah standard untuk menerangkan struktur data JSON. Ia membolehkan anda menetapkan medan wajib, jenis data, julat dan peraturan format untuk pengesahan yang konsisten.

**Kegunaan biasa**

- Mengesahkan payload permintaan dan respons API
- Memaksa struktur fail konfigurasi
- Mendokumentasikan kontrak data antara pasukan
