## Wat is JSON Schema?

JSON Schema is een standaard om de structuur van JSON-gegevens te beschrijven. Hiermee kun je verplichte velden, datatypen, bereiken en formatregels definiëren voor consistente validatie.

**Veelvoorkomende toepassingen**
- API-request- en responsepayloads valideren
- Structuur van configuratiebestanden afdwingen
- Datacontracten tussen teams documenteren
