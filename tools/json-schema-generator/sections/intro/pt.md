## O que é JSON Schema?

JSON Schema é um padrão para descrever a estrutura de dados JSON. Ele permite definir campos obrigatórios, tipos de dados, intervalos e regras de formato para validar os dados de forma consistente.

**Usos comuns**

- Validar payloads de requisição e resposta de APIs
- Impor a estrutura de arquivos de configuração
- Documentar contratos de dados entre equipes
