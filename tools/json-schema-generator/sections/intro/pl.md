## Czym jest JSON Schema?

JSON Schema to standard opisu struktury danych JSON. Pozwala definiować pola wymagane, typy danych, zakresy i reguły formatu, aby konsekwentnie walidować dane.

**Typowe zastosowania**

- Walidacja payloadów żądań i odpowiedzi API
- Wymuszanie struktury plików konfiguracyjnych
- Dokumentowanie kontraktów danych między zespołami
