## Che cos'è JSON Schema?

JSON Schema è uno standard per descrivere la struttura dei dati JSON. Consente di definire campi obbligatori, tipi di dati, intervalli e regole di formato per validare i dati in modo coerente.

**Usi comuni**

- Validare i payload di richiesta e risposta delle API
- Imporre la struttura dei file di configurazione
- Documentare i contratti dati tra team
