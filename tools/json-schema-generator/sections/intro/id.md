## Apa itu JSON Schema?

JSON Schema adalah standar untuk menggambarkan struktur data JSON. Ini memungkinkan Anda mendefinisikan bidang wajib, tipe data, rentang, dan aturan format agar data tervalidasi secara konsisten.

**Penggunaan umum**
- Memvalidasi payload permintaan dan respons API
- Menegakkan struktur file konfigurasi
- Mendokumentasikan kontrak data antar tim
