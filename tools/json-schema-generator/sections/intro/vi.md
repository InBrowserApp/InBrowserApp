## JSON Schema là gì?

JSON Schema là tiêu chuẩn mô tả cấu trúc dữ liệu JSON. Nó cho phép xác định các trường bắt buộc, kiểu dữ liệu, phạm vi và quy tắc định dạng để xác thực dữ liệu nhất quán.

**Trường hợp sử dụng phổ biến**

- Xác thực payload yêu cầu và phản hồi API
- Áp đặt cấu trúc tệp cấu hình
- Tài liệu hóa hợp đồng dữ liệu giữa các nhóm
