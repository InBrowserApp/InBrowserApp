## Vad är JSON Schema?

JSON Schema är en standard för att beskriva strukturen på JSON-data. Den låter dig definiera obligatoriska fält, datatyper, intervall och formatregler för konsekvent validering.

**Vanliga användningsområden**

- Validera API-förfrågningar och svarspayloads
- Tvinga struktur för konfigurationsfiler
- Dokumentera datakontrakt mellan team
