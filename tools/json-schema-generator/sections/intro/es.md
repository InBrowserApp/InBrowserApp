## ¿Qué es JSON Schema?

JSON Schema es un estándar para describir la estructura de los datos JSON. Permite definir campos obligatorios, tipos de datos, rangos y reglas de formato para validar los datos de forma consistente.

**Usos comunes**
- Validar cargas de solicitud y respuesta de API
- Forzar la estructura de archivos de configuración
- Documentar contratos de datos entre equipos
