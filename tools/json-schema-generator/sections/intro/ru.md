## Что такое JSON Schema?

JSON Schema — это стандарт для описания структуры JSON-данных. Он позволяет задавать обязательные поля, типы данных, диапазоны и правила форматов для согласованной проверки.

**Частые сценарии**

- Проверка запросов и ответов API
- Контроль структуры файлов конфигурации
- Документирование контрактов данных между командами
