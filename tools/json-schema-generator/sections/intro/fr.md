## Qu'est-ce que JSON Schema?

JSON Schema est une norme qui décrit la structure des données JSON. Elle permet de définir les champs requis, les types de données, les plages et les règles de format afin de valider les données de manière cohérente.

**Usages courants**
- Valider les charges utiles des requêtes et réponses d'API
- Imposer la structure des fichiers de configuration
- Documenter les contrats de données entre équipes
