## Was ist JSON Schema?

JSON Schema ist ein Standard zur Beschreibung der Struktur von JSON-Daten. Damit lassen sich Pflichtfelder, Datentypen, Bereiche und Formatregeln definieren, um Daten konsistent zu validieren.

**Häufige Anwendungen**

- API-Anfrage- und Antwortdaten validieren
- Struktur von Konfigurationsdateien erzwingen
- Datenverträge zwischen Teams dokumentieren
