## Hva er JSON Schema?

JSON Schema er en standard for å beskrive strukturen til JSON-data. Den lar deg definere obligatoriske felt, datatyper, intervaller og formatregler for konsekvent validering.

**Vanlige bruksområder**

- Validere API-forespørsler og responsdata
- Håndheve struktur for konfigurasjonsfiler
- Dokumentere datakontrakter mellom team
