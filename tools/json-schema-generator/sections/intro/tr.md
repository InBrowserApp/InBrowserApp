## JSON Schema nedir?

JSON Schema, JSON verilerinin yapısını tanımlamak için kullanılan bir standarttır. Zorunlu alanları, veri türlerini, aralıkları ve biçim kurallarını tanımlayarak verileri tutarlı biçimde doğrulamayı sağlar.

**Yaygın kullanım alanları**

- API istek ve yanıt yüklerini doğrulama
- Yapılandırma dosyası yapısını zorunlu kılma
- Takımlar arası veri sözleşmelerini belgeleme
