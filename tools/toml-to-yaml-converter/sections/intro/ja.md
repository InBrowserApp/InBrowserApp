## このツールの用途

有効な TOML を、読みやすく共有しやすくコミットにも向いた YAML に変換したいときに使うツールです。すべての処理はブラウザ内でローカルに実行されます。

## 向いている場面

- API のサンプルをドキュメントや設定ファイル向けの YAML に変換する。
- コピーした TOML ペイロードを目視で比較しやすい形式に整える。
- YAML を前提とするツールやワークフロー向けに構造化データを準備する。

## 注意したいこと

- このツールでは入力は有効な TOML である必要があります。
- コンバーターはデータ構造を保ちますが、YAML の見た目は手書きのスタイルと異なることがあります。
- 逆方向に変換したい場合は、YAML から TOML へのコンバーターを使ってください。
