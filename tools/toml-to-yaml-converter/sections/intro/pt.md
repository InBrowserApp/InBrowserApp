## Para que serve esta ferramenta

Use esta ferramenta quando precisar transformar TOML válido em YAML mais fácil de ler, compartilhar ou versionar. Tudo roda localmente no navegador.

## Onde ela ajuda

- Converter exemplos de API em YAML para documentação ou arquivos de configuração.
- Reescrever payloads TOML copiados em um formato mais fácil de comparar visualmente.
- Preparar conteúdo estruturado para ferramentas e fluxos que esperam YAML.

## O que observar

- Nesta ferramenta a entrada precisa ser TOML válido.
- O conversor preserva a estrutura dos dados, mas a apresentação em YAML ainda pode diferir de um estilo escrito à mão.
- Se você precisar do caminho inverso, use o conversor de YAML para TOML.
