## Для чего нужен этот инструмент

Используйте этот инструмент, когда нужно превратить корректный TOML в YAML, который легче читать, делиться им или коммитить. Все выполняется локально в браузере.

## Где он полезен

- Преобразовывать примеры API в YAML для документации или конфигурационных файлов.
- Переписывать скопированные TOML-payload'ы в формат, который проще сравнивать глазами.
- Подготавливать структурированный контент для инструментов и процессов, которые ожидают YAML.

## Что важно учитывать

- В этом инструменте входные данные должны быть корректным TOML.
- Конвертер сохраняет структуру данных, но представление YAML может отличаться от написанного вручную стиля.
- Если нужно преобразование в обратную сторону, используйте конвертер YAML в TOML.
