## Qu'est-ce qu'un convertisseur cURL ?

Un convertisseur cURL transforme une commande cURL en code pour de nombreux langages et clients HTTP. Il permet de réutiliser une requête d'API entre différents stacks sans réécrire les en-têtes, le corps ou l'auth.
