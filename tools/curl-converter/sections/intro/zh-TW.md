## 什麼是 cURL 轉換器?

cURL 轉換器會將 cURL 命令轉換為多種語言與 HTTP 客戶端可直接使用的程式碼。當你從 API 文件、瀏覽器開發者工具或終端機歷史裡拿到一條可用請求，想把它搬進應用程式碼時，它可以幫你省去手動重建請求方法、URL、標頭、Cookie 和請求本文的過程。

## 這個工具適合在什麼時候使用

- 你已經在 API 文件或終端機歷史裡有一條可運作的 cURL 範例。
- 你想在 `fetch`、Python `requests`、Go、Java、PHP 等目標之間比較同一個請求的寫法，再決定要用哪一種。
- 你想先產生一份可用的基礎程式碼，再補上專案自己的錯誤處理、重試、驗證刷新與設定封裝。

## 轉換後建議檢查什麼

- 確認所選目標與專案實際使用的 HTTP 函式庫和執行環境一致。
- 仔細閱讀警告訊息。有些 shell 引號規則、環境變數或不受支援的 cURL 參數仍然需要手動調整。
- 在提交產生的程式碼前，替換掉範例 token、敏感資訊或範例 URL。
