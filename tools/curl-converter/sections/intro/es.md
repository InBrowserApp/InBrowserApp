## ¿Qué es un conversor de cURL?

Un conversor de cURL transforma un comando cURL en código para muchos lenguajes y clientes HTTP. Te ayuda a reutilizar una solicitud de API en distintos stacks sin reescribir encabezados, cuerpo ni autenticación.
