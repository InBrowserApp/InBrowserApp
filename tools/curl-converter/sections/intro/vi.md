## Trình chuyển đổi cURL là gì?

Trình chuyển đổi cURL biến một lệnh cURL thành mã cho nhiều ngôn ngữ và client HTTP. Nó giúp bạn tái sử dụng yêu cầu API giữa các stack khác nhau mà không phải viết lại header, body hay xác thực.
