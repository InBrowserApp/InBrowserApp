## 什么是 cURL 转换器?

cURL 转换器会将 cURL 命令转换为多种语言与 HTTP 客户端可直接使用的代码。当你从 API 文档、浏览器开发者工具或终端历史里拿到一条可用请求，想把它搬进应用代码时，它可以帮你省掉手动重建请求方法、URL、请求头、Cookie 和请求体的过程。

**致谢**
由 [curlconverter](https://curlconverter.com)（Nick Carneiro）提供支持。

## 这个工具适合在什么时候使用

- 你已经在 API 文档或终端历史里有一条可工作的 cURL 示例。
- 你想在 `fetch`、Python `requests`、Go、Java、PHP 等目标之间对比同一请求的写法，再决定用哪一种。
- 你想先生成一份可用基础代码，再补上项目自己的错误处理、重试、鉴权刷新和配置封装。

## 转换后建议检查什么

- 确认所选目标与项目实际使用的 HTTP 库和运行时一致。
- 仔细阅读警告信息。有些 shell 引号规则、环境变量或不受支持的 cURL 参数仍然需要手动调整。
- 在提交生成代码前，替换掉示例 token、敏感信息或示例 URL。
