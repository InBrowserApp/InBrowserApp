## ตัวแปลง cURL คืออะไร?

ตัวแปลง cURL จะแปลงคำสั่ง cURL ให้เป็นโค้ดพร้อมใช้งานสำหรับหลายภาษาและไคลเอนต์ HTTP เหมาะเมื่อเอกสาร API, เครื่องมือนักพัฒนาในเบราว์เซอร์ หรือประวัติในเทอร์มินัลมีคำขอที่ใช้งานได้อยู่แล้ว และคุณต้องการย้ายมันไปเป็นโค้ดในแอปโดยไม่ต้องประกอบ method, URL, headers, cookies หรือ body ใหม่ด้วยมือ

## เครื่องมือนี้มีประโยชน์เมื่อใด

- เมื่อคุณเริ่มจากตัวอย่าง cURL ที่ใช้งานได้ในเอกสาร API หรือประวัติเทอร์มินัล
- เมื่อคุณต้องการเปรียบเทียบคำขอเดียวกันระหว่าง `fetch`, Python `requests`, Go, Java, PHP และเป้าหมายอื่น ๆ ก่อนตัดสินใจเลือก
- เมื่อคุณต้องการสร้างโค้ดตั้งต้นอย่างรวดเร็ว แล้วค่อยเพิ่มการจัดการข้อผิดพลาด การ retry การรีเฟรชการยืนยันตัวตน และการตั้งค่าของโปรเจ็กต์

## ควรตรวจอะไรหลังการแปลง

- ตรวจให้แน่ใจว่าเป้าหมายที่เลือกตรงกับไลบรารี HTTP และ runtime ที่โปรเจ็กต์ของคุณใช้งานจริง
- อ่านคำเตือนอย่างละเอียด กฎการใส่เครื่องหมายคำพูดของ shell, ตัวแปรสภาพแวดล้อม หรือแฟล็ก cURL ที่ยังไม่รองรับบางอย่างอาจต้องแก้ด้วยมือ
- เปลี่ยน token ตัวอย่าง ความลับ หรือ URL ตัวอย่างก่อน commit โค้ดที่สร้างแล้ว
