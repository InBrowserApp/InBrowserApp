## Czym jest konwerter cURL?

Konwerter cURL zamienia polecenie cURL na kod dla wielu języków i klientów HTTP. Pozwala ponownie używać żądań API w różnych stosach bez przepisywania nagłówków, treści lub uwierzytelniania.
