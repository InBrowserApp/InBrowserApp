## מהו ממיר cURL?

ממיר cURL הופך פקודת cURL לקוד מוכן לשימוש עבור שפות רבות ולקוחות HTTP. זה שימושי כאשר תיעוד API, כלי המפתחים בדפדפן או היסטוריית הטרמינל כבר נותנים בקשה עובדת, ואתם רוצים להעביר אותה לקוד האפליקציה בלי לבנות מחדש ידנית את המתודה, ה-URL, הכותרות, העוגיות או הגוף.

**קרדיט**
מופעל על־ידי [curlconverter](https://curlconverter.com) מאת Nick Carneiro.

## מתי הכלי הזה שימושי

- כאשר מתחילים מדוגמת cURL שעובדת כבר בתיעוד API או בהיסטוריית הטרמינל.
- כאשר רוצים להשוות את אותה בקשה בין `fetch`, Python `requests`, Go, Java, PHP ויעדים אחרים לפני שבוחרים.
- כאשר רוצים לייצר בסיס מהיר ואז להוסיף טיפול בשגיאות, ניסיונות חוזרים, רענון אימות והגדרות של הפרויקט.

## מה כדאי לבדוק אחרי ההמרה

- ודאו שהיעד שנבחר תואם לספריית ה-HTTP ול-runtime שהפרויקט באמת משתמש בהם.
- קראו את האזהרות היטב. חלק מכללי הציטוט של shell, משתני הסביבה או דגלי cURL שלא נתמכים עלולים לדרוש תיקון ידני.
- החליפו טוקנים לדוגמה, סודות או כתובות URL לדוגמה לפני שמבצעים commit לקוד שנוצר.
