## Was ist ein cURL-Konverter?

Ein cURL-Konverter wandelt einen cURL-Befehl in Code für viele Sprachen und HTTP-Clients um. So kannst du eine API-Anfrage über verschiedene Stacks hinweg wiederverwenden, ohne Header, Body oder Auth neu zu schreiben.
