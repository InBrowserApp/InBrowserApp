## ما هو محول cURL؟

محول cURL يحول أمر cURL إلى كود جاهز للاستخدام لعدة لغات وعملاء HTTP. يكون مفيدًا عندما تعطيك وثائق API أو أدوات المطور في المتصفح أو سجل الطرفية طلبًا يعمل بالفعل وتريد نقله إلى كود التطبيق بدون إعادة بناء الطريقة أو عنوان URL أو الرؤوس أو ملفات تعريف الارتباط أو الجسم يدويًا.

**إشادة**
مدعوم بواسطة [curlconverter](https://curlconverter.com) من Nick Carneiro.

## متى يكون هذا مفيدًا؟

- عندما تبدأ من مثال cURL يعمل في وثائق API أو في سجل الطرفية.
- عندما تريد مقارنة الطلب نفسه بين `fetch` وPython `requests` وGo وJava وPHP وغيرها قبل اختيار الهدف المناسب.
- عندما تريد إنشاء نسخة أولية سريعة ثم إضافة معالجة الأخطاء وإعادة المحاولة وتجديد المصادقة والإعدادات الخاصة بمشروعك.

## ما الذي يجب مراجعته بعد التحويل؟

- تأكد من أن الهدف المحدد يطابق مكتبة HTTP وبيئة التشغيل المستخدمة فعليًا في مشروعك.
- اقرأ التحذيرات بعناية. قد تحتاج بعض قواعد الاقتباس في shell أو متغيرات البيئة أو خيارات cURL غير المدعومة إلى تعديل يدوي.
- استبدل الرموز المميزة أو الأسرار أو عناوين URL التجريبية قبل اعتماد الكود المولد.
