## cURL コンバーターとは?

cURL コンバーターは cURL コマンドを多言語・多様な HTTP クライアント向けのそのまま使えるコードに変換します。API ドキュメント、ブラウザーの開発者ツール、またはターミナル履歴にある動作済みリクエストをアプリのコードへ移したいときに、メソッド、URL、ヘッダー、Cookie、本文を手で組み直す手間を減らせます。

**クレジット**
[curlconverter](https://curlconverter.com)（Nick Carneiro）提供。

## このツールが役立つ場面

- API ドキュメントやターミナル履歴にある動作する cURL 例から始めたいとき。
- 同じリクエストを `fetch`、Python `requests`、Go、Java、PHP などの各ターゲットで比較してから選びたいとき。
- まずたたき台のコードを作り、その後に自分のプロジェクト向けのエラー処理、リトライ、認証更新、設定処理を加えたいとき。

## 変換後に確認したいこと

- 選んだターゲットが、プロジェクトで実際に使っている HTTP ライブラリとランタイムに合っているか確認してください。
- 警告はよく確認してください。シェルのクォート規則、環境変数、未対応の cURL フラグは手作業で調整が必要な場合があります。
- 生成コードをコミットする前に、ダミーのトークン、秘密情報、サンプル URL を置き換えてください。
