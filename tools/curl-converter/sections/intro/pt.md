## O que é um conversor de cURL?

Um conversor de cURL transforma um comando cURL em código para muitas linguagens e clientes HTTP. Ele ajuda a reutilizar uma requisição de API em diferentes stacks sem reescrever cabeçalhos, corpo ou autenticação.
