## cURL 변환기란?

cURL 변환기는 cURL 명령을 여러 언어와 HTTP 클라이언트에서 바로 사용할 수 있는 코드로 바꿉니다. API 문서, 브라우저 개발자 도구, 터미널 기록에서 이미 동작하는 요청을 얻었고, 메서드, URL, 헤더, 쿠키, 본문을 손으로 다시 만들지 않고 앱 코드로 옮기고 싶을 때 유용합니다.

## 이런 때 유용합니다

- API 문서나 터미널 기록에 있는 동작하는 cURL 예제로 시작할 때.
- 같은 요청을 `fetch`, Python `requests`, Go, Java, PHP 등 여러 대상에서 비교한 뒤 선택하고 싶을 때.
- 먼저 빠르게 기본 코드를 만든 다음, 프로젝트에 맞는 오류 처리, 재시도, 인증 갱신, 설정 로직을 추가하고 싶을 때.

## 변환 후 확인할 점

- 선택한 대상이 프로젝트에서 실제로 사용하는 HTTP 라이브러리와 런타임에 맞는지 확인하세요.
- 경고를 주의해서 읽으세요. 일부 shell quoting 규칙, 환경 변수, 지원되지 않는 cURL 플래그는 수동 정리가 필요할 수 있습니다.
- 생성된 코드를 커밋하기 전에 예시 토큰, 비밀값, 예시 URL을 실제 값으로 바꾸세요.
