## Wofür dieses Tool gedacht ist

Verwende dieses Tool, wenn du gültiges JSON in YAML umwandeln musst, das sich leichter lesen, teilen oder committen lässt. Alles läuft lokal im Browser.

## Wo es hilft

- API-Beispiele in YAML für Doku oder Konfigurationsdateien umwandeln.
- Kopierte JSON-Payloads in ein Format überführen, das sich visuell leichter vergleichen lässt.
- Strukturierte Inhalte für Tools und Workflows vorbereiten, die YAML erwarten.

## Worauf du achten solltest

- Die Eingabe muss in diesem Tool gültiges JSON sein.
- Der Konverter erhält die Datenstruktur, aber die YAML-Darstellung kann sich von handgeschriebenem Stil unterscheiden.
- Wenn du in die andere Richtung konvertieren willst, nutze stattdessen den YAML-zu-JSON-Konverter.
