## 이 도구는 언제 쓰나요

유효한 JSON을 더 읽기 쉽고 공유하거나 커밋하기 좋은 YAML로 바꾸고 싶을 때 이 도구를 사용합니다. 모든 처리는 브라우저 안에서 로컬로 실행됩니다.

## 잘 맞는 상황

- API 예제를 문서나 설정 파일용 YAML로 변환할 때.
- 복사한 JSON payload를 눈으로 비교하기 쉬운 형식으로 다시 정리할 때.
- YAML을 기대하는 도구와 워크플로를 위해 구조화된 내용을 준비할 때.

## 주의할 점

- 이 도구에서는 입력이 유효한 JSON이어야 합니다.
- 변환기는 데이터 구조를 유지하지만 YAML 표현은 손으로 작성한 스타일과 다를 수 있습니다.
- 반대 방향 변환이 필요하면 YAML to JSON 변환기를 사용하세요.
