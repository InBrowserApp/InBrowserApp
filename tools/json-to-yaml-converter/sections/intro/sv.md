## Vad verktyget är till för

Använd verktyget när du behöver göra om giltig JSON till YAML som är lättare att läsa, dela eller committa. Allt körs lokalt i webbläsaren.

## När det hjälper

- Konvertera API-exempel till YAML för dokumentation eller konfigurationsfiler.
- Skriva om kopierade JSON-payloads till ett format som är lättare att jämföra med ögat.
- Förbereda strukturerat innehåll för verktyg och arbetsflöden som förväntar sig YAML.

## Vad du ska tänka på

- Inmatningen måste vara giltig JSON i det här verktyget.
- Konverteraren bevarar datastrukturen, men YAML-presentationen kan ändå skilja sig från handskriven stil.
- Om du behöver gå åt andra hållet ska du använda YAML till JSON-konverteraren i stället.
