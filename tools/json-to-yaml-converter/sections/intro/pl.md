## Do czego służy to narzędzie

Użyj tego narzędzia, gdy chcesz zamienić poprawny JSON na YAML, który łatwiej czytać, udostępniać lub commitować. Wszystko działa lokalnie w przeglądarce.

## Gdzie pomaga

- Konwertować przykłady API do YAML dla dokumentacji lub plików konfiguracyjnych.
- Przepisywać skopiowane payloady JSON do formatu łatwiejszego do porównywania wzrokiem.
- Przygotowywać uporządkowaną treść dla narzędzi i workflowów, które oczekują YAML.

## Na co uważać

- W tym narzędziu wejście musi być poprawnym JSON-em.
- Konwerter zachowuje strukturę danych, ale sposób prezentacji YAML może różnić się od stylu pisanego ręcznie.
- Jeśli potrzebujesz konwersji w drugą stronę, użyj konwertera YAML na JSON.
