## למה הכלי הזה מיועד

השתמשו בכלי הזה כשצריך להפוך JSON תקין ל-YAML שקל יותר לקרוא, לשתף או לעשות לו commit. הכל רץ מקומית בדפדפן.

## איפה הוא עוזר

- להמיר דוגמאות API ל-YAML עבור תיעוד או קובצי הגדרה.
- לכתוב מחדש payloads של JSON שהועתקו לפורמט שקל יותר להשוות בעין.
- להכין תוכן מובנה לכלים ולזרימות עבודה שמצפים ל-YAML.

## למה לשים לב

- בכלי הזה הקלט חייב להיות JSON תקין.
- הממיר שומר על מבנה הנתונים, אבל הצגת ה-YAML עדיין יכולה להיות שונה מסגנון שנכתב ידנית.
- אם צריך את הכיוון ההפוך, השתמשו בממיר YAML ל-JSON.
