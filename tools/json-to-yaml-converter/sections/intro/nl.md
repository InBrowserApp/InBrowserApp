## Waar deze tool voor is

Gebruik deze tool wanneer je geldige JSON wilt omzetten naar YAML dat makkelijker te lezen, delen of committen is. Alles draait lokaal in de browser.

## Waar het helpt

- API-voorbeelden omzetten naar YAML voor documentatie of configuratiebestanden.
- Gekopieerde JSON-payloads herschrijven naar een formaat dat visueel makkelijker te vergelijken is.
- Gestructureerde inhoud voorbereiden voor tools en workflows die YAML verwachten.

## Waar je op moet letten

- In deze tool moet de invoer geldige JSON zijn.
- De converter bewaart de datastructuur, maar de YAML-weergave kan nog steeds verschillen van handgeschreven stijl.
- Als je de andere kant op wilt, gebruik dan de YAML-naar-JSON-converter.
