## Hva dette verktøyet er for

Bruk dette verktøyet når du trenger å gjøre gyldig JSON om til YAML som er enklere å lese, dele eller committe. Alt kjører lokalt i nettleseren.

## Når det hjelper

- Konvertere API-eksempler til YAML for dokumentasjon eller konfigurasjonsfiler.
- Skrive om kopierte JSON-payloads til et format som er lettere å sammenligne visuelt.
- Forberede strukturert innhold for verktøy og arbeidsflyter som forventer YAML.

## Hva du bør passe på

- Inndataene må være gyldig JSON i dette verktøyet.
- Konvertereren bevarer datastrukturen, men YAML-presentasjonen kan fortsatt avvike fra håndskrevet stil.
- Hvis du trenger å gå motsatt vei, bruk YAML til JSON-konvertereren i stedet.
