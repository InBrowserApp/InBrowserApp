## Por qué se usa Base85

Base85 empaqueta cuatro bytes en cinco caracteres imprimibles, por lo que es más denso que el hexadecimal y sigue siendo fácil de mover por registros, documentos, flujos PDF/PostScript y herramientas de depuración.

## Para qué sirve este codificador

Esta herramienta codifica texto del navegador o archivos cargados a Base85 con las variantes ASCII85 o Z85. Puedes cambiar de variante antes de codificar, confirmar qué salida espera el sistema de destino y exportar el resultado sin enviar datos a un servidor.

## Cuándo usarlo

Úsalo cuando necesites una representación segura para texto de datos binarios, quieras comprobar si un flujo espera ASCII85 o Z85, o necesites preparar cargas para copiar y pegar, documentación y utilidades sin conexión.
