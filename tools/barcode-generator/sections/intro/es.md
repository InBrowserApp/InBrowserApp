## Qué hace esta herramienta

Este generador de códigos de barras crea formatos comunes como CODE128, EAN y
UPC directamente en el navegador. Puedes cambiar el contenido, el tamaño de
las barras, la posición del texto y los colores antes de exportar el
resultado.

## Por qué resulta útil

Úsalo cuando necesites un código de barras rápido para etiquetas, referencias
de inventario, maquetas de embalaje, entradas o documentos internos. La vista
previa se actualiza al instante, así que es fácil revisar el espaciado y la
legibilidad antes de descargar.

## Cómo funciona

Elige un formato de código de barras, introduce el contenido y ajusta anchura,
altura, margen, colores y opciones de texto. Cuando la vista previa se vea
bien, descarga el código de barras como PNG, SVG, JPEG o WebP.
