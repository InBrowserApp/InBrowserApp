## RIPEMD-128 คืออะไร?

RIPEMD-128 (RACE Integrity Primitives Evaluation Message Digest) เป็นฟังก์ชันแฮชเข้ารหัสที่สร้างค่าแฮช 128 บิต (16 ไบต์) ซึ่งมักแสดงเป็นตัวเลขฐานสิบหก 32 ตัวอักษร เป็นส่วนหนึ่งของตระกูล RIPEMD ที่พัฒนาในยุโรปเพื่อเป็นทางเลือกแทน MD4/MD5

**คุณลักษณะสำคัญ:**

- **กำหนดได้**: อินพุตเดียวกันจะสร้างแฮชเดียวกันเสมอ
- **การคำนวณที่รวดเร็ว**: คำนวณได้อย่างรวดเร็วสำหรับอินพุตใดก็ตาม
- **เอฟเฟกต์หิมะถล่ม**: การเปลี่ยนแปลงเล็กน้อยในอินพุตจะสร้างเอาต์พุตที่แตกต่างอย่างมาก
- **ขนาดเอาต์พุตคงที่**: สร้างแฮช 128 บิตเสมอไม่ว่าขนาดอินพุตจะเป็นเท่าใด
- **ทางเดียว**: การกู้คืนอินพุตต้นฉบับจากแฮชทำได้ยากมากในเชิงคำนวณ

**การใช้งานทั่วไป:**

- การตรวจสอบความสมบูรณ์ของข้อมูล
- การทำฟิงเกอร์พริ้นต์และการลบข้อมูลซ้ำ
- ความเข้ากันได้กับระบบเดิม
