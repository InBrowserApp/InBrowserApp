## Что такое RIPEMD-128?

RIPEMD-128 (RACE Integrity Primitives Evaluation Message Digest) — это криптографическая хеш‑функция, которая производит 128-битное (16-байтовое) хеш‑значение, обычно представляемое как 32-символьное шестнадцатеричное число. Она входит в семейство RIPEMD, разработанное в Европе как альтернатива MD4/MD5.

**Ключевые характеристики:**

- **Детерминированность**: Одинаковый ввод всегда производит одинаковый хеш
- **Быстрое вычисление**: Быстро вычисляется для любого заданного ввода
- **Эффект лавины**: Небольшие изменения во вводе производят кардинально разные выводы
- **Фиксированный размер выхода**: Всегда производит 128-битный хеш независимо от размера ввода
- **Односторонность**: Вычислительно нецелесообразно восстановить исходный ввод из хеша

**Обычные применения:**

- Проверка целостности данных
- Фингерпринтинг и дедупликация
- Совместимость с устаревшими системами
