## 이 도구는 언제 쓰나요

이 도구는 압축되었거나 지저분한 JSON을 문서, 테스트, 티켓, PR에 붙여 넣기
전에 깔끔하고 읽기 쉬운 형태로 정리할 때 사용합니다. 모든 처리는 브라우저
안에서 로컬로 실행됩니다.

## 잘 맞는 상황

- API 요청과 응답 payload를 확인할 때.
- 내보낸 JSON 파일을 공유 전에 다시 포맷할 때.
- 데이터를 바꾸지 않고 깊게 중첩된 객체와 배열을 읽을 때.

## 주의할 점

- 이 포매터는 유효한 JSON만 처리합니다.
- 주석, 후행 쉼표, 그 밖의 JavaScript 전용 문법은 실패합니다.
- 포맷팅은 공백만 바꾸며, 키 순서나 값은 수정하지 않습니다.
