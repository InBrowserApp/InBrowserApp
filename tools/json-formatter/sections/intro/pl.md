## Do czego służy to narzędzie

Użyj tego narzędzia, aby zamienić zbity lub nieuporządkowany JSON w czysty,
czytelny wynik przed wklejeniem go do dokumentacji, testów, zgłoszeń lub pull
requestów. Wszystko działa lokalnie w przeglądarce.

## Gdzie pomaga

- Sprawdzanie payloadów żądań i odpowiedzi API.
- Ponowne formatowanie wyeksportowanych plików JSON przed udostępnieniem.
- Czytanie głęboko zagnieżdżonych obiektów i tablic bez zmieniania danych.

## Na co uważać

- Formatter akceptuje tylko poprawny JSON.
- Komentarze, końcowe przecinki i inna składnia właściwa tylko dla JavaScript
  zakończą się błędem.
- Formatowanie zmienia tylko białe znaki; nie zmienia kolejności kluczy ani
  wartości.
