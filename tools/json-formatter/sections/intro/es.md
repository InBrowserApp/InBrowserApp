## Para qué sirve esta herramienta

Usa esta herramienta para convertir JSON compacto o desordenado en una salida
limpia y legible antes de pegarla en documentación, pruebas, tickets o pull
requests. Todo se ejecuta localmente en el navegador.

## Dónde ayuda

- Inspeccionar payloads de solicitudes y respuestas de API.
- Reformatear archivos JSON exportados antes de compartirlos.
- Leer objetos y arrays muy anidados sin cambiar los datos.

## Qué tener en cuenta

- El formateador solo acepta JSON válido.
- Los comentarios, las comas finales y otras sintaxis propias de JavaScript
  fallarán.
- El formateo solo cambia los espacios en blanco; no reordena claves ni
  modifica valores.
