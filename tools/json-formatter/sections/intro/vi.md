## Công cụ này dùng để làm gì

Dùng công cụ này để biến JSON nén hoặc lộn xộn thành đầu ra sạch và dễ đọc
trước khi dán vào tài liệu, bài test, ticket hoặc pull request. Mọi thứ đều
chạy cục bộ trong trình duyệt.

## Khi nào hữu ích

- Kiểm tra payload của request và response API.
- Định dạng lại các tệp JSON đã xuất trước khi chia sẻ.
- Đọc object và array lồng sâu mà không thay đổi dữ liệu.

## Điều cần lưu ý

- Trình định dạng chỉ chấp nhận JSON hợp lệ.
- Comment, dấu phẩy cuối và các cú pháp chỉ có trong JavaScript sẽ thất bại.
- Việc định dạng chỉ thay đổi khoảng trắng; không sắp xếp lại key hoặc sửa giá
  trị.
