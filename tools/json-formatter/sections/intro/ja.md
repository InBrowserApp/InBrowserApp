## このツールの用途

このツールは、圧縮された JSON や整っていない JSON を、ドキュメント、
テスト、チケット、PR に貼り付ける前に、読みやすい整形済みの出力へ変換
するためのものです。すべての処理はブラウザ内でローカルに実行されます。

## 向いている場面

- API のリクエストやレスポンスのペイロードを確認する。
- 共有前に書き出した JSON ファイルを整形し直す。
- データを変えずに深くネストしたオブジェクトや配列を読みやすくする。

## 注意したいこと

- このフォーマッタが受け付けるのは有効な JSON のみです。
- コメント、末尾カンマ、その他 JavaScript 固有の構文は失敗します。
- 整形で変わるのは空白だけで、キー順や値は変更されません。
