## เครื่องมือนี้ใช้ทำอะไร

ใช้เครื่องมือนี้เพื่อแปลง JSON ที่ถูกบีบอัดหรือจัดรูปแบบไม่ดีให้เป็นผลลัพธ์ที่
อ่านง่าย ก่อนนำไปวางในเอกสาร การทดสอบ ticket หรือ pull request ทุกอย่างทำงาน
ภายในเบราว์เซอร์แบบ local

## เหมาะกับสถานการณ์ไหน

- ตรวจสอบ payload ของ request และ response จาก API
- จัดรูปแบบไฟล์ JSON ที่ export มาใหม่ก่อนแชร์
- อ่าน object และ array ที่ซ้อนลึกได้โดยไม่เปลี่ยนข้อมูล

## สิ่งที่ควรระวัง

- ตัว formatter รับเฉพาะ JSON ที่ถูกต้องเท่านั้น
- comment, trailing comma และ syntax อื่นที่เป็นแบบ JavaScript เท่านั้นจะไม่ผ่าน
- การจัดรูปแบบจะเปลี่ยนเฉพาะช่องว่าง ไม่ได้เรียง key ใหม่หรือแก้ค่า
