## Alat ini untuk apa

Gunakan alat ini untuk mengubah JSON yang padat atau berantakan menjadi
keluaran yang bersih dan mudah dibaca sebelum ditempel ke dokumentasi,
pengujian, tiket, atau pull request. Semuanya berjalan secara lokal di browser.

## Kapan alat ini membantu

- Memeriksa payload request dan response API.
- Memformat ulang file JSON hasil ekspor sebelum dibagikan.
- Membaca objek dan array yang sangat bertingkat tanpa mengubah data.

## Hal yang perlu diperhatikan

- Formatter ini hanya menerima JSON yang valid.
- Komentar, trailing comma, dan sintaks khusus JavaScript lainnya akan gagal.
- Pemformatan hanya mengubah whitespace; urutan key dan nilai tetap sama.
