## Vad verktyget är till för

Använd verktyget för att förvandla kompakt eller stökig JSON till ren och
lättläst output innan du klistrar in den i dokumentation, tester, ärenden
eller pull requests. Allt körs lokalt i webbläsaren.

## När det hjälper

- Granska API-anropens request- och responsepayloads.
- Formatera om exporterade JSON-filer innan du delar dem.
- Läsa djupt nästlade objekt och arrayer utan att ändra datan.

## Vad du ska tänka på

- Formatteraren accepterar bara giltig JSON.
- Kommentarer, efterföljande kommatecken och annan JavaScript-specifik syntax
  misslyckas.
- Formateringen ändrar bara whitespace; den ändrar inte nyckelordning eller
  värden.
