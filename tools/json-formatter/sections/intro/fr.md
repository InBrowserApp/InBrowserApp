## À quoi sert cet outil

Utilisez cet outil pour transformer un JSON compact ou désordonné en une sortie
propre et lisible avant de le coller dans une documentation, des tests, un
ticket ou une pull request. Tout s'exécute localement dans le navigateur.

## Quand il est utile

- Inspecter les charges utiles des requêtes et réponses API.
- Reformater des fichiers JSON exportés avant de les partager.
- Lire des objets et tableaux profondément imbriqués sans modifier les données.

## Ce qu'il faut garder en tête

- Le formateur n'accepte que du JSON valide.
- Les commentaires, les virgules finales et les autres syntaxes propres à
  JavaScript échoueront.
- Le formatage ne change que les espaces ; il ne réordonne pas les clés et ne
  modifie pas les valeurs.
