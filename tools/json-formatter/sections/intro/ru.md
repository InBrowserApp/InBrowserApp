## Для чего нужен этот инструмент

Используйте этот инструмент, чтобы превратить сжатый или неаккуратный JSON в
чистый и читаемый вид перед вставкой в документацию, тесты, тикеты или pull
request. Все выполняется локально в браузере.

## Где он полезен

- Проверять payload запросов и ответов API.
- Переформатировать экспортированные JSON-файлы перед тем, как делиться ими.
- Читать глубоко вложенные объекты и массивы, не меняя данные.

## Что важно учитывать

- Форматтер принимает только корректный JSON.
- Комментарии, висячие запятые и другой синтаксис только для JavaScript не
  пройдут.
- Форматирование меняет только пробельные символы; порядок ключей и значения
  не меняются.
