## ما فائدة هذه الأداة

استخدم هذه الأداة لتحويل JSON المضغوط أو غير المرتب إلى مخرجات نظيفة وسهلة
القراءة قبل لصقه في التوثيق أو الاختبارات أو التذاكر أو طلبات السحب. كل شيء
يعمل محليًا داخل المتصفح.

## أين تكون مفيدة

- فحص حمولات طلبات واستجابات API.
- إعادة تنسيق ملفات JSON المصدرة قبل مشاركتها.
- قراءة الكائنات والمصفوفات المتداخلة بعمق بدون تغيير البيانات.

## ما الذي يجب الانتباه له

- لا يقبل المنسق إلا JSON صالحًا.
- ستفشل التعليقات والفواصل اللاحقة وأي صياغات خاصة بـ JavaScript.
- التنسيق يغيّر المسافات البيضاء فقط، ولا يعيد ترتيب المفاتيح ولا يعدّل
  القيم.
