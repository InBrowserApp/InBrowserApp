## Hva dette verktøyet er for

Bruk dette verktøyet til å gjøre kompakt eller rotete JSON om til ren og
lesbar output før du limer det inn i dokumentasjon, tester, saker eller pull
requests. Alt kjører lokalt i nettleseren.

## Når det hjelper

- Inspisere payloads i API-forespørsler og -svar.
- Omformatere eksporterte JSON-filer før du deler dem.
- Lese dypt nestede objekter og arrayer uten å endre dataene.

## Hva du bør passe på

- Formattereren godtar bare gyldig JSON.
- Kommentarer, hengende komma og annen JavaScript-spesifikk syntaks vil feile.
- Formateringen endrer bare mellomrom og linjeskift; den endrer ikke
  nøkkelrekkefølge eller verdier.
