## Para que serve esta ferramenta

Use esta ferramenta para transformar JSON compacto ou bagunçado em uma saída
limpa e legível antes de colar em documentação, testes, tickets ou pull
requests. Tudo roda localmente no navegador.

## Onde ela ajuda

- Inspecionar payloads de requisições e respostas de API.
- Reformatar arquivos JSON exportados antes de compartilhá-los.
- Ler objetos e arrays profundamente aninhados sem alterar os dados.

## O que observar

- O formatador aceita apenas JSON válido.
- Comentários, vírgulas finais e outras sintaxes exclusivas de JavaScript
  falham.
- A formatação altera apenas os espaços em branco; não reordena chaves nem
  modifica valores.
