## A cosa serve questo strumento

Usa questo strumento per trasformare JSON compatto o disordinato in un output
pulito e leggibile prima di incollarlo in documentazione, test, ticket o pull
request. Tutto viene eseguito localmente nel browser.

## Dove aiuta

- Ispezionare i payload di richieste e risposte API.
- Riformattare file JSON esportati prima di condividerli.
- Leggere oggetti e array molto annidati senza modificare i dati.

## Cosa tenere presente

- Il formatter accetta solo JSON valido.
- Commenti, virgole finali e altra sintassi valida solo in JavaScript non
  funzionano.
- La formattazione cambia solo gli spazi; non riordina le chiavi né modifica i
  valori.
