## למה הכלי הזה מיועד

השתמשו בכלי הזה כדי להפוך JSON דחוס או מבולגן לפלט נקי וקריא לפני שמדביקים אותו
לתיעוד, בדיקות, כרטיסים או pull requests. הכל רץ מקומית בדפדפן.

## איפה הוא עוזר

- לבדוק payloads של בקשות ותגובות API.
- לפרמט מחדש קובצי JSON מיוצאים לפני שיתוף.
- לקרוא אובייקטים ומערכים מקוננים מאוד בלי לשנות את הנתונים.

## למה לשים לב

- הפורמטור מקבל רק JSON תקין.
- הערות, פסיקים נגררים ותחביר אחר ששייך רק ל-JavaScript ייכשלו.
- הפורמט משנה רק רווחים ותווי לבן; הוא לא מסדר מחדש מפתחות ולא משנה ערכים.
