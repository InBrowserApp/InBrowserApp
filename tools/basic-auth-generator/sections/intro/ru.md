## Что такое Basic Auth?

Basic Auth помещает `username:password` в заголовок `Authorization` после кодирования в Base64. Это простой и широко поддерживаемый механизм, но Base64 — это только кодирование, а не шифрование.

## Что создаёт этот инструмент

- Заголовок `Authorization: Basic ...`, который можно вставить в API-клиент.
- Готовый пример `curl` для быстрого теста.
- Всё выполняется локально в браузере.

## Что важно помнить

- Всегда используйте HTTPS при передаче данных Basic Auth.
- Любой, кто увидит заголовок, сможет декодировать исходные имя пользователя и пароль.
- Basic Auth удобен для внутренних инструментов, staging-окружений и быстрых проверок API.
