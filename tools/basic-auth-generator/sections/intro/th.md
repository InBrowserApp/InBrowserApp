## Basic Auth คืออะไร

Basic Auth จะนำ `username:password` ไปเข้ารหัสแบบ Base64 แล้วใส่ไว้ในส่วนหัว `Authorization` วิธีนี้เรียบง่ายและรองรับอย่างแพร่หลาย แต่ Base64 เป็นเพียงการเข้ารหัสเชิงรูปแบบ ไม่ใช่การเข้ารหัสเพื่อความปลอดภัย

## เครื่องมือนี้สร้างอะไรให้

- ส่วนหัว `Authorization: Basic ...` ที่นำไปวางใน API client ได้ทันที
- ตัวอย่าง `curl` ที่พร้อมรันสำหรับการทดสอบอย่างรวดเร็ว
- ทุกอย่างทำงานในเบราว์เซอร์แบบ local

## สิ่งที่ควรระวัง

- ควรใช้ HTTPS ทุกครั้งเมื่อส่งข้อมูล Basic Auth
- ใครก็ตามที่เห็นส่วนหัวนี้สามารถถอดกลับเป็นชื่อผู้ใช้และรหัสผ่านเดิมได้
- Basic Auth เหมาะกับเครื่องมือภายใน สภาพแวดล้อม staging และการตรวจสอบ API แบบรวดเร็ว
