## מהו Basic Auth?

Basic Auth מכניס את `username:password` לכותרת `Authorization` לאחר קידוד Base64. זו שיטה פשוטה ונתמכת מאוד, אבל Base64 הוא רק קידוד ולא הצפנה.

## מה הכלי הזה מייצר

- כותרת `Authorization: Basic ...` שאפשר להדביק בלקוחות API.
- דוגמת `curl` מוכנה להרצה לבדיקות מהירות.
- הכול רץ מקומית בתוך הדפדפן.

## מה חשוב לזכור

- השתמשו תמיד ב-HTTPS כששולחים פרטי Basic Auth.
- כל מי שרואה את הכותרת יכול לפענח את שם המשתמש והסיסמה המקוריים.
- Basic Auth נוח לכלים פנימיים, לסביבות staging ולבדיקות API מהירות.
