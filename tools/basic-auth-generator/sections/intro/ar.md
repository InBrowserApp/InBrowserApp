## ما هو Basic Auth؟

يقوم Basic Auth بوضع `username:password` داخل ترويسة `Authorization` بعد ترميزه بصيغة Base64. هذه الطريقة بسيطة ومدعومة على نطاق واسع، لكن Base64 ترميز فقط وليس تشفيرًا.

## ماذا تولّد هذه الأداة؟

- ترويسة `Authorization: Basic ...` يمكنك لصقها في عملاء API.
- مثال `curl` جاهز للتشغيل من أجل الاختبار السريع.
- كل شيء يعمل محليًا داخل المتصفح.

## ما الذي يجب الانتباه إليه؟

- استخدم HTTPS دائمًا عند إرسال بيانات Basic Auth.
- أي شخص يرى هذه الترويسة يستطيع فك اسم المستخدم وكلمة المرور الأصليين.
- Basic Auth مناسب للأدوات الداخلية وبيئات staging وفحوصات API السريعة.
