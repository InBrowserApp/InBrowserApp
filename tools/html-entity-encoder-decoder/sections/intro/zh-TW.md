## 這個工具能做什麼

- 將一般文字編碼為命名、十進位或十六進位形式的 HTML 實體。
- 把已經實體化的片段解碼回可讀文字。
- 全程都在瀏覽器本機執行，輸入內容不會離開裝置。

## 什麼時候使用它

- 在把 HTML 貼到文件、模板或示範前，先跳脫特殊字元。
- 檢查包含 `&amp;`、`&#60;` 或 `&#x3C;` 的複製標記片段。
- 依照相容性需求比較命名、十進位與十六進位實體格式。

## 關於實體格式的說明

- 命名實體最容易閱讀，但不是每個字元都有命名形式。
- 十進位與十六進位實體可以表示任何 Unicode 字元，包括 emoji。
- 解碼時，未知或無效的實體會保留原樣，不會猜測替換。
