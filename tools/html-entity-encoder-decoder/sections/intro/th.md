## เครื่องมือนี้ทำอะไรได้บ้าง

- เข้ารหัสข้อความธรรมดาให้เป็น HTML entity แบบชื่อ แบบฐานสิบ หรือแบบ
  ฐานสิบหก
- ถอดรหัสข้อความที่เป็น entity แล้วกลับมาเป็นข้อความที่อ่านได้
- ทุกอย่างทำงานในเบราว์เซอร์ของคุณแบบโลคอล ข้อมูลจึงไม่ออกจากอุปกรณ์

## ใช้เมื่อไร

- escape อักขระพิเศษก่อนนำ HTML ไปวางในเอกสาร เทมเพลต หรือเดโม
- ตรวจสอบมาร์กอัปที่คัดลอกมาและมี `&amp;`, `&#60;` หรือ `&#x3C;`
- เปรียบเทียบ entity แบบชื่อ แบบฐานสิบ และแบบฐานสิบหกเพื่อรองรับความ
  เข้ากันได้

## หมายเหตุเกี่ยวกับรูปแบบ entity

- entity แบบชื่ออ่านง่ายที่สุด แต่ไม่ใช่ทุกอักขระจะมีชื่อกำกับ
- entity แบบฐานสิบและฐานสิบหกแทน Unicode ได้ทุกตัวอักษร รวมถึง emoji
- entity ที่ไม่รู้จักหรือไม่ถูกต้องจะคงเดิมเมื่อถอดรหัส
