## Cosa fa questo strumento

- Codifica testo normale in entità HTML con formato nominato, decimale o
  esadecimale.
- Decodifica frammenti già codificati di nuovo in testo leggibile.
- Tutto viene eseguito localmente nel browser, quindi i tuoi dati restano sul
  dispositivo.

## Quando usarlo

- Esegui l'escape dei caratteri speciali prima di incollare HTML in
  documentazione, template o demo.
- Controlla markup copiato che contiene `&amp;`, `&#60;` o `&#x3C;`.
- Confronta entità nominate, decimali ed esadecimali per esigenze di
  compatibilità.

## Note sui formati di entità

- Le entità nominate sono le più leggibili, ma non tutti i caratteri ne hanno
  una.
- Le entità decimali ed esadecimali possono rappresentare qualsiasi carattere
  Unicode, emoji inclusi.
- Le entità sconosciute o non valide restano invariate durante la decodifica.
