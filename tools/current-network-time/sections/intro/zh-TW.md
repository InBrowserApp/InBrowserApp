## 這個工具適合做什麼

用這個工具可以把你的裝置時鐘與網路時間進行比對。它會從
Cloudflare 的 trace 端點讀取時間戳記，依請求往返延遲的中點估算網路時鐘，並在瀏覽器中顯示結果。

## 適合哪些場景

- 檢查本機系統時鐘是快了還是慢了。
- 在排查 TLS、權杖、排程或日誌問題前，先確認是否存在時間漂移。
- 不安裝 NTP 工具時，快速取得一個以網路為基準的參考時間。

## 需要注意什麼

- 顯示的偏移量是估算值，會受到網路延遲影響。
- 如果 trace 請求失敗，工具會先退回顯示本機時鐘，直到下次同步成功。
- 如果要做系統層級的精確修正，請調整裝置的時間同步設定或 NTP 組態。
