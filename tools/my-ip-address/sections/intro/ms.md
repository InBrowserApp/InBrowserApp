## Bagaimana Kami Mendapatkan Alamat IP Anda

Alat ini mengambil alamat IP anda dengan membuat permintaan kepada perkhidmatan pihak ketiga termasuk Cloudflare, geojs.io, ip.sb, dan ipify.org. Sila ambil perhatian bahawa ini bukan perkhidmatan luar talian dan memerlukan sambungan rangkaian kepada pelayan luaran untuk mendapatkan maklumat IP awam anda.
