## Bagaimana Kami Mendapatkan Alamat IP Anda

Alat ini mengambil alamat IP Anda dengan membuat permintaan ke layanan pihak ketiga termasuk Cloudflare, geojs.io, ip.sb, dan ipify.org. Harap dicatat bahwa ini bukan layanan offline dan memerlukan konektivitas jaringan ke server eksternal untuk mengambil informasi IP publik Anda.
