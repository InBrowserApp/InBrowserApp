Erstelle QR-Codes für alltägliche Aufgaben zum Teilen, ohne den Inhalt an einen Server zu senden. Der Generator unterstützt einfachen Text und URLs, WLAN-Zugangsdaten, vCard-Kontaktkarten, SMS- und Telefonaktionen, E-Mail-Links, Kartenkoordinaten und Kalenderereignisse.

## Wann du ihn verwendest

Nutze dieses Tool, wenn du einen scanbaren Code für einen Flyer, einen Support-Desk, ein Unterrichtsblatt, ein Veranstaltungsschild, ein Besuchernetzwerk oder eine schnelle Übergabe von Gerät zu Gerät brauchst. Die Nutzdatenvorschau zeigt den exakten Text, der kodiert wird, sodass du vor dem Herunterladen bestätigen kannst, ob der Code eine URL, eine `WIFI:`-Zeichenfolge, einen `mailto:`-Link, eine vCard, einen `geo:`-URI oder einen Kalendereintrag enthält.

## Optionen, die das Scannen beeinflussen

Die Fehlerkorrektur legt fest, wie viel vom QR-Code beschädigt oder verdeckt sein kann und trotzdem noch gescannt wird. Verwende Mittel für die meisten Codes, Hoch, wenn der Code klein gedruckt oder in der Nähe eines Logos platziert wird, und Niedrig nur, wenn es wichtiger ist, dichte Daten kompakt zu halten, als robust zu bleiben. Lass genügend Ruhezone um den Code und stelle sicher, dass Vorder- und Hintergrundfarben einen starken Kontrast haben.

## Datenschutz und Ausgabe

Die Generierung erfolgt in deinem Browser. WLAN-Passwörter, Kontaktdaten und Ereignistexte bleiben auf deinem Gerät. Lade PNG, JPG oder WebP für Dokumente, Präsentationen und Chat-Apps herunter; verwende SVG, wenn der QR-Code in Design-Tools skaliert oder in mehreren Größen gedruckt wird.
