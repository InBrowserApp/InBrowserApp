Créez des codes QR pour les tâches de partage du quotidien sans envoyer le contenu à un serveur. Le générateur prend en charge le texte brut et les URL, les informations de connexion Wi-Fi, les fiches contact vCard, les actions SMS et d’appel téléphonique, les liens e-mail, les coordonnées cartographiques et les événements de calendrier.

## Quand l’utiliser

Utilisez cet outil lorsque vous avez besoin d’un code scannable pour un prospectus, un service d’assistance, un support de cours, une signalétique d’événement, un réseau visiteurs ou un transfert rapide d’un appareil à un autre. L’aperçu du contenu affiche le texte exact qui sera encodé, ce qui vous permet de vérifier si le code contient une URL, une chaîne `WIFI:`, un lien `mailto:`, une vCard, un URI `geo:` ou une entrée de calendrier avant le téléchargement.

## Options qui influencent la lecture

Le paramètre de correction d’erreur détermine la part du code QR qui peut être endommagée ou masquée tout en restant lisible. Utilisez le niveau moyen pour la plupart des codes, le niveau élevé lorsque le code risque d’être imprimé en petit ou placé près d’un logo, et le niveau faible uniquement lorsque garder des données denses compactes compte plus que la robustesse. Conservez une zone de silence suffisante autour du code et assurez-vous que les couleurs de premier plan et d’arrière-plan offrent un fort contraste.

## Confidentialité et sortie

La génération s’effectue dans votre navigateur. Les mots de passe Wi-Fi, les coordonnées de contact et le texte des événements restent sur votre appareil. Téléchargez en PNG, JPG ou WebP pour les documents, les présentations et les applications de chat ; utilisez SVG lorsque le code QR sera redimensionné dans des outils de conception ou imprimé à plusieurs tailles.
