สร้าง QR code สำหรับงานแชร์ในชีวิตประจำวันโดยไม่ส่งเนื้อหาไปยังเซิร์ฟเวอร์ ตัวสร้างนี้รองรับข้อความธรรมดาและ URL, รายละเอียดสำหรับเข้าร่วม Wi-Fi, บัตรรายชื่อติดต่อ vCard, การทำงานผ่าน SMS และโทรศัพท์, ลิงก์อีเมล, พิกัดแผนที่ และกิจกรรมปฏิทิน

## ควรใช้เมื่อใด

ใช้เครื่องมือนี้เมื่อคุณต้องการโค้ดที่สแกนได้สำหรับใบปลิว จุดบริการช่วยเหลือ เอกสารแจกในชั้นเรียน ป้ายงานอีเวนต์ เครือข่ายสำหรับผู้เยี่ยมชม หรือการส่งต่อข้อมูลอย่างรวดเร็วระหว่างอุปกรณ์ ตัวอย่างข้อมูลจะแสดงข้อความจริงที่จะถูกเข้ารหัส คุณจึงตรวจสอบได้ก่อนดาวน์โหลดว่าโค้ดมี URL, สตริง `WIFI:`, ลิงก์ `mailto:`, vCard, URI `geo:`, หรือรายการปฏิทินหรือไม่

## ตัวเลือกที่มีผลต่อการสแกน

การตั้งค่าการแก้ไขข้อผิดพลาดควบคุมว่า QR code จะเสียหายหรือถูกบดบังได้มากแค่ไหนแล้วยังสแกนได้ ใช้ระดับปานกลางสำหรับโค้ดส่วนใหญ่ ใช้ระดับสูงเมื่อโค้ดอาจถูกพิมพ์ขนาดเล็กหรือวางใกล้โลโก้ และใช้ระดับต่ำเฉพาะเมื่อการทำให้ข้อมูลหนาแน่นมีขนาดกะทัดรัดสำคัญกว่าความทนทาน เว้นพื้นที่ว่างรอบโค้ดให้เพียงพอ และตรวจสอบให้แน่ใจว่าสีด้านหน้าและสีพื้นหลังมีคอนทราสต์ชัดเจน

## ความเป็นส่วนตัวและเอาต์พุต

การสร้างเกิดขึ้นในเบราว์เซอร์ของคุณ รหัสผ่าน Wi-Fi รายละเอียดติดต่อ และข้อความกิจกรรมจะอยู่บนอุปกรณ์ของคุณ ดาวน์โหลด PNG, JPG หรือ WebP สำหรับเอกสาร สไลด์ และแอปแชต ใช้ SVG เมื่อ QR code จะถูกปรับขนาดในเครื่องมือออกแบบหรือใช้กับงานพิมพ์หลายขนาด
