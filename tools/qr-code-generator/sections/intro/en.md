Create QR codes for everyday sharing tasks without sending the content to a server. The generator supports plain text and URLs, Wi-Fi join details, vCard contact cards, SMS and phone actions, email links, map coordinates, and calendar events.

## When to use it

Use this tool when you need a scannable code for a flyer, support desk, classroom handout, event sign, visitor network, or quick device-to-device handoff. The payload preview shows the exact text that will be encoded, so you can confirm whether the code contains a URL, `WIFI:` string, `mailto:` link, vCard, `geo:` URI, or calendar entry before downloading.

## Options that affect scanning

The error correction setting controls how much of the QR code can be damaged or obscured and still scan. Use medium for most codes, high when the code may be printed small or placed near a logo, and low only when keeping dense data compact matters more than resilience. Keep enough quiet zone around the code, and make sure foreground and background colors have strong contrast.

## Privacy and output

Generation happens in your browser. Wi-Fi passwords, contact details, and event text stay on your device. Download PNG, JPG, or WebP for documents, slides, and chat apps; use SVG when the QR code will be scaled in design tools or printed at multiple sizes.
