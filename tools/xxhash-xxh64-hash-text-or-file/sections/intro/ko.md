## xxHash (XXH64)란?

xxHash는 좋은 분포 특성을 유지하면서 속도와 성능에 중점을 둔 매우 빠른 비암호화 해시 알고리즘입니다. XXH64는 64비트(8바이트) 해시 값을 생성하는 64비트 변형으로, 일반적으로 16자리 16진수로 표시됩니다.

**주요 특징:**

- **매우 빠름**: 속도에 최적화되어 암호화 해시 함수보다 훨씬 빠름
- **결정론적**: 동일한 입력은 항상 동일한 해시를 생성
- **좋은 분포**: 해시 테이블을 위한 우수한 해시 분포 제공
- **비암호화**: 보안 목적에는 적합하지 않으며 성능을 위해 설계됨
- **더 큰 출력**: 64비트 해시는 32비트 변형보다 더 나은 충돌 저항성 제공
- **플랫폼 최적화**: 최대 속도를 위해 사용 가능할 때 SIMD 명령어 사용

**일반적인 용도:**

- 해시 테이블 및 데이터 구조
- 파일 무결성 검사(비보안)
- 데이터 중복 제거
- 데이터 전송용 체크섬
- 성능 중심 애플리케이션
- 데이터베이스 인덱싱
- 캐시 키 생성
