## 什麼是 xxHash (XXH64)？

xxHash 是一種極快的非加密雜湊演算法，專注於速度和效能，同時保持良好的分佈特性。XXH64 是產生 64 位元（8 位元組）雜湊值的 64 位元變體，通常顯示為 16 個字元的十六進位數。

**主要特徵：**

- **極快速度**：針對速度最佳化，比加密雜湊函數快得多
- **確定性**：相同的輸入總是產生相同的雜湊
- **良好分佈**：為雜湊表提供優秀的雜湊分佈
- **非加密**：不適用於安全目的，專為效能設計
- **更大輸出**：64 位元雜湊比 32 位元變體提供更好的碰撞抗性
- **平台最佳化**：在可用時使用 SIMD 指令以獲得最大速度

**常見用途：**

- 雜湊表和資料結構
- 檔案完整性檢查（非安全）
- 資料去重
- 資料傳輸校驗和
- 效能關鍵應用
- 資料庫索引
- 快取鍵生成
