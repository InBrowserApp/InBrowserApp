## URL 인코딩이란?

URL 인코딩(퍼센트 인코딩이라고도 함)은 특수 문자를 인터넷을 통해 안전하게 전송할 수 있는 형식으로 변환하는 방법입니다. URL은 특정 문자만 포함할 수 있으므로 허용되지 않는 문자는 인코딩해야 합니다.

**작동 방식:**

- 특수 문자는 `%`와 그 뒤에 16진수 ASCII 코드로 변환됩니다
- 예: 공백은 `%20`이 되고, `{'@'}`는 `%40`이 됩니다
- 문자(A-Z, a-z), 숫자(0-9), 일부 기호(- \_ . ~)만 인코딩이 필요하지 않습니다

**일반적인 예:**

- 공백 → `%20`
- `{'@'}` → `%40`
- `#` → `%23`
- `&` → `%26`
- `?` → `%3F`

**왜 필요한가:**

- URL에는 특별한 의미를 가진 예약 문자가 있습니다
- 데이터가 올바르게 전송되도록 보장합니다
- URL 구조와의 충돌을 방지합니다
- 웹 폼과 API 호출에 필요합니다
