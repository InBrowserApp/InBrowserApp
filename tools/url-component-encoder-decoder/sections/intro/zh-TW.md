## 什麼是 URL 編碼？

URL 編碼（也稱為百分號編碼）是將特殊字元轉換為可以安全在網際網路上傳輸的格式的方法。URL 只能包含某些字元，所以任何不被允許的字元都必須進行編碼。

**工作原理：**

- 特殊字元被轉換為 `%` 後跟其十六進位 ASCII 碼
- 例如：空格變成 `%20`，`@` 變成 `%40`
- 只有字母（A-Z, a-z）、數字（0-9）和一些符號（- \_ . ~）不需要編碼

**常見範例：**

- 空格 → `%20`
- `@` → `%40`
- `#` → `%23`
- `&` → `%26`
- `?` → `%3F`

**為什麼需要：**

- URL 有具有特殊含義的保留字元
- 確保資料正確傳輸
- 防止與 URL 結構發生衝突
- Web 表單和 API 呼叫所必需
