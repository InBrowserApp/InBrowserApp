## การเข้ารหัส URL คืออะไร?

การเข้ารหัส URL (เรียกอีกอย่างว่าการเข้ารหัสเปอร์เซ็นต์) เป็นวิธีการแปลงอักขระพิเศษให้อยู่ในรูปแบบที่สามารถส่งผ่านอินเทอร์เน็ตได้อย่างปลอดภัย URL สามารถมีได้เฉพาะอักขระบางตัวเท่านั้น ดังนั้นอักขระใดก็ตามที่ไม่ได้รับอนุญาตจะต้องถูกเข้ารหัส

**วิธีการทำงาน:**

- อักขระพิเศษจะถูกแปลงเป็น `%` ตามด้วยรหัส ASCII ฐานสิบหกของมัน
- ตัวอย่าง: ช่องว่างกลายเป็น `%20`, `{'@'}` กลายเป็น `%40`
- มีเพียงตัวอักษร (A-Z, a-z), ตัวเลข (0-9) และสัญลักษณ์บางตัว (- \_ . ~) เท่านั้นที่ไม่ต้องเข้ารหัส

**ตัวอย่างทั่วไป:**

- ช่องว่าง → `%20`
- `{'@'}` → `%40`
- `#` → `%23`
- `&` → `%26`
- `?` → `%3F`

**เหตุผลที่จำเป็น:**

- URL มีอักขระสงวนที่มีความหมายพิเศษ
- รับประกันว่าข้อมูลถูกส่งอย่างถูกต้อง
- ป้องกันความขัดแย้งกับโครงสร้าง URL
- จำเป็นสำหรับฟอร์มเว็บและการเรียก API
