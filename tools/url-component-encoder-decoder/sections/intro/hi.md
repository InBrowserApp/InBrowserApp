## URL एन्कोडिंग क्या है?

URL एन्कोडिंग (प्रतिशत एन्कोडिंग भी कहलाता है) विशेष अक्षरों को एक ऐसे प्रारूप में परिवर्तित करने की विधि है जो इंटरनेट पर सुरक्षित रूप से प्रसारित की जा सकती है। URL में केवल कुछ निश्चित अक्षर हो सकते हैं, इसलिए कोई भी अनुमति न दिया गया अक्षर एन्कोड होना चाहिए।

**यह कैसे काम करता है:**

- विशेष अक्षरों को `%` के साथ उनके हेक्साडेसिमल ASCII कोड में परिवर्तित किया जाता है
- उदाहरण: स्पेस `%20` बन जाता है, `{'@'}` `%40` बन जाता है
- केवल अक्षर (A-Z, a-z), संख्याएं (0-9), और कुछ प्रतीक (- \_ . ~) को एन्कोडिंग की आवश्यकता नहीं

**सामान्य उदाहरण:**

- स्पेस → `%20`
- `{'@'}` → `%40`
- `#` → `%23`
- `&` → `%26`
- `?` → `%3F`

**यह क्यों आवश्यक है:**

- URL में विशेष अर्थों वाले आरक्षित अक्षर होते हैं
- सुनिश्चित करता है कि डेटा सही तरीके से प्रसारित हो
- URL संरचना के साथ संघर्ष को रोकता है
- वेब फॉर्म और API कॉल के लिए आवश्यक
