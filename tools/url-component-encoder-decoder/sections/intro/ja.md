## URL エンコーディングとは？

URL エンコーディング（パーセントエンコーディングとも呼ばれる）は、特殊文字をインターネット上で安全に送信できる形式に変換する方法です。URL は特定の文字のみを含むことができるため、許可されていない文字はエンコードする必要があります。

**仕組み：**

- 特殊文字は `%` とその後に続く 16 進数の ASCII コードに変換されます
- 例：スペースは `%20` になり、`{'@'}` は `%40` になります
- 文字（A-Z, a-z）、数字（0-9）、一部の記号（- \_ . ~）のみエンコードは不要です

**一般的な例：**

- スペース → `%20`
- `{'@'}` → `%40`
- `#` → `%23`
- `&` → `%26`
- `?` → `%3F`

**なぜ必要か：**

- URL には特別な意味を持つ予約文字があります
- データが正しく送信されることを保証します
- URL 構造との競合を防ぎます
- Web フォームと API 呼び出しに必要です
