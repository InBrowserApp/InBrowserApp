## ما هو ترميز URL؟

ترميز URL (يُسمى أيضاً ترميز النسبة المئوية) هو طريقة لتحويل الأحرف الخاصة إلى تنسيق يمكن نقله بأمان عبر الإنترنت. يمكن أن تحتوي عناوين URL فقط على أحرف معينة، لذلك يجب ترميز أي حرف غير مسموح.

**كيف يعمل:**

- يتم تحويل الأحرف الخاصة إلى `%` متبوعاً برمز ASCII السادس عشر
- مثال: المسافة تصبح `%20`، `{'@'}` تصبح `%40`
- فقط الأحرف (A-Z, a-z) والأرقام (0-9) وبعض الرموز (- \_ . ~) لا تحتاج ترميز

**أمثلة شائعة:**

- مسافة → `%20`
- `{'@'}` → `%40`
- `#` → `%23`
- `&` → `%26`
- `?` → `%3F`

**لماذا مطلوب:**

- عناوين URL لها أحرف محجوزة بمعاني خاصة
- يضمن نقل البيانات بشكل صحيح
- يمنع التعارض مع بنية URL
- مطلوب لنماذج الويب واستدعاءات API
