## מה זה קידוד URL?

קידוד URL (נקרא גם קידוד אחוזים) הוא שיטה להמרת תווים מיוחדים לפורמט שניתן להעביר בבטחה דרך האינטרנט. URL יכולים להכיל רק תווים מסוימים, לכן כל תו שאינו מותר חייב להיות מקודד.

**איך זה עובד:**

- תווים מיוחדים מומרים ל-`%` ואחריו קוד ASCII הקסדצימלי שלהם
- דוגמה: רווח הופך ל-`%20`, `@` הופך ל-`%40`
- רק אותיות (A-Z, a-z), מספרים (0-9) וכמה סמלים (- \_ . ~) לא צריכים קידוד

**דוגמאות נפוצות:**

- רווח → `%20`
- `@` → `%40`
- `#` → `%23`
- `&` → `%26`
- `?` → `%3F`

**למה זה נדרש:**

- ל-URL יש תווים שמורים עם משמעויות מיוחדות
- מבטיח שנתונים מועברים נכון
- מונע קונפליקטים עם מבנה ה-URL
- נדרש לטפסי אינטרנט וקריאות API
