## Wat is Whirlpool?

Whirlpool is een cryptografische hashfunctie. Ze produceert een 512-bit (64-byte) hashwaarde, meestal weergegeven als een hexadecimale string van 128 tekens.

**Belangrijkste kenmerken:**

- **Deterministisch**: Dezelfde invoer levert altijd dezelfde hash op
- **Lawine-effect**: Kleine wijzigingen in de invoer geven zeer verschillende uitvoer
- **Botsingsbestendig**: Ontworpen om het vinden van botsingen moeilijk te maken

**Veelvoorkomende toepassingen:**

- Integriteitscontroles van gegevens
- Digitale handtekeningen
- Veilige identifiers
