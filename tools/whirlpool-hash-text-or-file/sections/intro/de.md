## Was ist Whirlpool?

Whirlpool ist eine kryptografische Hash-Funktion. Sie erzeugt einen 512-Bit-(64-Byte-)Hashwert, der typischerweise als 128-stellige Hexadezimalzeichenkette dargestellt wird.

**Hauptmerkmale:**

- **Deterministisch**: Dieselbe Eingabe erzeugt immer denselben Hash
- **Lawineneffekt**: Kleine Änderungen der Eingabe erzeugen sehr unterschiedliche Ausgaben
- **Kollisionsresistent**: Entwickelt, um Kollisionen schwer auffindbar zu machen

**Häufige Verwendungen:**

- Datenintegritätsprüfungen
- Digitale Signaturen
- Sichere Kennungen
