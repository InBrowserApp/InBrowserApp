## O que é Whirlpool?

Whirlpool é uma função hash criptográfica. Ela produz um valor hash de 512 bits (64 bytes), normalmente mostrado como uma string hexadecimal de 128 caracteres.

**Características principais:**

- **Determinístico**: A mesma entrada sempre produz o mesmo hash
- **Efeito avalanche**: Pequenas mudanças na entrada produzem saídas muito diferentes
- **Resistente a colisões**: Projetada para tornar colisões difíceis de encontrar

**Usos comuns:**

- Verificação de integridade de dados
- Assinaturas digitais
- Identificadores seguros
