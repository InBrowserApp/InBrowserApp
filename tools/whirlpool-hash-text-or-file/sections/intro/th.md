## Whirlpool คืออะไร?

Whirlpool เป็นฟังก์ชันแฮชเชิงเข้ารหัส ให้ค่าแฮช 512 บิต (64 ไบต์) โดยมักแสดงเป็นสตริงเลขฐานสิบหก 128 ตัวอักษร

**คุณลักษณะสำคัญ:**

- **กำหนดได้**: อินพุตเดียวกันให้แฮชเหมือนเดิมเสมอ
- **เอฟเฟกต์หิมะถล่ม**: การเปลี่ยนแปลงเล็กน้อยในอินพุตทำให้ได้ผลลัพธ์ที่ต่างมาก
- **ต้านทานการชน**: ออกแบบมาให้หาการชนได้ยาก

**การใช้งานทั่วไป:**

- การตรวจสอบความสมบูรณ์ของข้อมูล
- ลายเซ็นดิจิทัล
- ตัวระบุที่ปลอดภัย
