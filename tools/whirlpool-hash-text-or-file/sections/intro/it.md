## Cos'è Whirlpool?

Whirlpool è una funzione hash crittografica. Produce un valore hash di 512 bit (64 byte), di solito mostrato come una stringa esadecimale di 128 caratteri.

**Caratteristiche chiave:**

- **Deterministica**: Lo stesso input produce sempre lo stesso hash
- **Effetto valanga**: Piccoli cambiamenti nell'input producono output molto diversi
- **Resistente alle collisioni**: Progettata per rendere difficile trovare collisioni

**Usi comuni:**

- Verifiche di integrità dei dati
- Firme digitali
- Identificatori sicuri
