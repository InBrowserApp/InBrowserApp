## Vad är Whirlpool?

Whirlpool är en kryptografisk hashfunktion. Den producerar ett 512-bitars (64-byte) hashvärde, vanligtvis visat som en 128-teckens hexsträng.

**Nyckelegenskaper:**

- **Deterministisk**: Samma inmatning ger alltid samma hash
- **Lavineffekt**: Små förändringar i inmatningen ger mycket olika utdata
- **Kollisionsresistent**: Utformad för att göra kollisioner svåra att hitta

**Vanliga användningar:**

- Dataintegritetskontroller
- Digitala signaturer
- Säkra identifierare
