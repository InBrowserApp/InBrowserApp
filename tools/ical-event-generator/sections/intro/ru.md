## Создавайте календарные файлы прямо в браузере

Этот инструмент генерирует стандартные файлы событий `.ics` прямо в браузере. Вы можете задавать события по времени или на весь день, выбирать стратегию работы с часовыми поясами, добавлять напоминания и экспортировать итоговую запись календаря без синхронизации данных с сервером.

### Зачем использовать

- Подходит, когда вам нужен только календарный файл, а не полный сценарий с аккаунтом календаря.
- Позволяет держать чувствительные расписания локально и всё равно получать стандартное вложение события.
- Даёт возможность настроить правила повторения и напоминания до загрузки итогового файла `.ics`.

### Рекомендуемый процесс

1. Заполните название события, место, заметки и при необходимости ссылку.
2. Выберите диапазон события и решите, экспортировать ли метки времени в `UTC` или сохранить исходный часовой пояс через `TZID`.
3. Добавляйте повторения и напоминания только при необходимости, затем скачайте файл и прикрепите его там, где делитесь событием.

### Примечания

- Вывод в `UTC` обычно является самым безопасным вариантом для широкой совместимости с календарями.
- Вывод с `TZID` сохраняет исходный контекст расписания для клиентов, которые понимают именованные часовые пояса.
- Для событий на весь день форма оставляет конечную дату включительно, хотя в ICS-файле она сохраняется как исключающая дата окончания.
