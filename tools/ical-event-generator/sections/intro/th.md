# สร้างไฟล์ปฏิทินได้โดยไม่ต้องออกจากเบราว์เซอร์

เครื่องมือนี้สร้างไฟล์เหตุการณ์ `.ics` มาตรฐานได้โดยตรงในเบราว์เซอร์ของคุณ คุณสามารถกำหนดอีเวนต์แบบมีเวลา หรืออีเวนต์ทั้งวัน เลือกวิธีจัดการเขตเวลา เพิ่มการเตือน และส่งออกข้อมูลปฏิทินสุดท้ายได้โดยไม่ต้องซิงก์ข้อมูลไปยังเซิร์ฟเวอร์

## ทำไมจึงควรใช้

- เหมาะเมื่อคุณต้องการเพียงไฟล์ปฏิทิน โดยไม่ต้องเข้าสู่ขั้นตอนเต็มรูปแบบของบัญชีปฏิทิน
- ช่วยเก็บตารางเวลาที่มีความละเอียดอ่อนไว้ในเครื่อง และยังคงสร้างไฟล์แนบอีเวนต์ตามมาตรฐานได้
- คุณสามารถปรับกฎการทำซ้ำและเวลาการเตือนได้ก่อนดาวน์โหลดไฟล์ `.ics` ฉบับสุดท้าย

## ขั้นตอนที่แนะนำ

1. กรอกสรุปอีเวนต์ สถานที่ บันทึกย่อ และ URL อ้างอิงถ้ามี
2. เลือกช่วงเวลาของอีเวนต์ จากนั้นตัดสินใจว่าจะส่งออกเป็นเวลา `UTC` หรือเก็บเขตเวลาเดิมด้วย `TZID`
3. เพิ่มกฎการทำซ้ำและการเตือนเฉพาะเมื่อจำเป็น แล้วดาวน์โหลดไฟล์ไปแนบในที่ที่คุณแชร์อีเวนต์

## หมายเหตุ

- เอาต์พุตแบบ `UTC` มักเป็นตัวเลือกที่ปลอดภัยที่สุดหากคุณต้องการความเข้ากันได้กับปฏิทินอย่างกว้างขวาง
- เอาต์พุตแบบ `TZID` จะเก็บบริบทการตั้งเวลาเดิมไว้สำหรับไคลเอนต์ที่เข้าใจเขตเวลาที่มีชื่อ
- สำหรับอีเวนต์ทั้งวัน แบบฟอร์มจะเก็บวันที่สิ้นสุดแบบรวมวันสุดท้ายไว้ แต่ไฟล์ ICS จะบันทึกเป็นวันที่สิ้นสุดแบบไม่รวมวันสุดท้าย
