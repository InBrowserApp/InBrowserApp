## 什么是 xxHash (XXH3 64)？

XXH3 是现代的 xxHash 算法，专为极高速度和优秀分布性而设计。XXH3 64 输出 64 位（8 字节）哈希值，通常显示为 16 个字符的十六进制字符串。它是非加密哈希，并支持可选种子以实现可复现的哈希结果。

**主要特征：**

- **极快速度**：针对大数据输入优化
- **确定性**：相同输入与种子产生相同哈希
- **非加密**：不适用于安全用途
- **良好分布**：适合哈希表与索引
- **可加种子**：可选种子用于区分哈希输出

**常见用途：**

- 哈希表和数据结构
- 文件完整性校验（非安全）
- 数据去重与分块
- 缓存键与数据库索引
- 高吞吐数据流水线
