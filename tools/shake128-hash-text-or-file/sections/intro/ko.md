## SHAKE128 (FIPS 202)이란?

SHAKE128 (FIPS 202)은 SHA-3 계열에 속하는 XOF(가변 길이 출력 함수)입니다. 고정 길이 해시와 달리 128비트 보안 강도를 유지하면서 원하는 비트 길이의 출력을 만들 수 있습니다. NIST가 FIPS 202로 표준화했으며 Keccak 스폰지 구조를 기반으로 합니다.

프로토콜, 파일 형식, 내부 체크섬 규칙이 특정 다이제스트 길이를 요구할 때 이런 유연성이 중요합니다. 이 도구에서는 일반 텍스트나 업로드한 파일을 해시하고, 8의 배수인 한 비트 단위로 출력 길이를 지정할 수 있습니다.

대표적인 용도는 프로토콜 해싱, 키 파생, 가변 길이 암호 다이제스트, 그리고 같은 입력과 출력 길이에서 항상 같은 결과가 필요한 데이터 무결성 워크플로입니다.
