## 什麼是 SHAKE128 (FIPS 202)？

SHAKE128 (FIPS 202) 是 SHA-3 家族中的可擴展輸出函數（XOF）。與固定長度雜湊不同，它可以回傳任意位數的輸出，同時提供 128 位元的安全強度。它由 NIST 在 FIPS 202 中標準化，並建立於 Keccak 海綿結構之上。

當協定、檔案格式或內部校驗規則需要特定摘要長度時，這種彈性就很重要。在這個工具中，你可以對純文字或上傳檔案進行雜湊，並以位元為單位選擇輸出長度，只要它是 8 的倍數即可。

常見用途包括協定雜湊、金鑰派生、可變長度的加密摘要，以及要求相同輸入與輸出長度必須始終產生相同結果的資料完整性流程。
