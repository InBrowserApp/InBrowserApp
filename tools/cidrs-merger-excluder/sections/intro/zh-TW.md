## What This Tool Does

這個工具會將 CIDR 區塊合併成等效的最小集合，然後扣除你放在排除清單中的任何 CIDR 區塊。它支援在同一次執行中處理 IPv4 和 IPv6，而且所有處理都會在你的瀏覽器本機完成。

## How Merge and Exclude Works

合併清單會先正規化：清除主機位元、合併重疊的網路，並在相鄰網路可用較短的 CIDR 區塊表示時將其收合。之後，會從合併後的範圍扣除排除清單。最終輸出會展開回能精確涵蓋剩餘範圍的最精簡 CIDR 清單。

## When This Is Useful

清理防火牆規則、準備雲端安全群組項目、檢查 VPN 允許清單、彙整路由表，或從較大的位址配置中移除保留範圍時都可以使用。當複製來的設定包含重疊區塊，或需要從較大的網路移除幾個較小範圍時，特別有幫助。

## Input Notes

每行輸入一個 CIDR，或用逗號分隔多個 CIDR。IPv4 和 IPv6 區塊可以一起貼上，但排除項只會套用到相同位址家族的區塊。無效項目會連同其清單與行號回報，讓你在修正大量貼上的輸入時不用猜測。
