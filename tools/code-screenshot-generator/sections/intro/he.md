# מחולל צילומי מסך לקוד

## מה הוא עושה

מחולל צילומי המסך לקוד הופך מקטע שהודבק לכרטיס קוד מלוטש שאפשר לייצא כ-SVG, PNG, JPEG, WebP או HTML עצמאי. הוא פועל בדפדפן שלכם, מחיל הדגשת תחביר עם ערכות נושא לבחירה, ומאפשר לכוונן את המסגרת לפני ההורדה.

## מתי להשתמש בו

השתמשו בו לתמונות בתיעוד, הערות גרסה, פוסטים בבלוג, מצגות, תצוגות מקדימות לרשתות חברתיות ודיווחי באגים שבהם קל יותר לסרוק מקטע מעוצב מאשר טקסט רגיל. ייצואי SVG ו-HTML נשארים חדים בתיעוד, ואילו PNG, JPEG ו-WebP שימושיים לכלים שמצפים לתמונות רסטר.

## איך לקבל ייצוא נקי

בחרו את השפה הקרובה ביותר או השאירו את זיהוי השפה במצב אוטומטי, ואז בחרו ערכת נושא עם ניגודיות מספקת ליעד שלכם. השאירו מספרי שורות פעילים כשהמקטע ישמש להפניה בדיון, והשביתו את הרקע החיצוני כשצריך נכס שקוף לעיצוב אחר.

## פרטיות ומגבלות

המקטע מעובד מקומית בדף ואינו מועלה על ידי הכלי הזה. קבצים גדולים מאוד עלולים להאט את ההדגשה ואת ייצוא הרסטר, לכן קצרו את המקטע לשורות שתרצו להציג לפני הייצוא.
