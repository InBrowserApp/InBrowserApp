# Code Screenshot Generator

## What it does

Code Screenshot Generator menukar cebisan kod yang ditampal menjadi kad kod yang kemas dan boleh dieksport sebagai SVG, PNG, JPEG, WebP, atau HTML kendiri. Ia berjalan dalam pelayar anda, menggunakan penyerlahan sintaks dengan tema yang boleh dipilih, dan membolehkan anda melaraskan bingkai sebelum memuat turun.

## When to use it

Gunakannya untuk imej dokumentasi, nota keluaran, catatan blog, pembentangan, pratonton sosial, dan laporan pepijat apabila cebisan kod bergaya lebih mudah diimbas daripada teks biasa. Eksport SVG dan HTML kekal tajam untuk dokumen, manakala PNG, JPEG, dan WebP berguna untuk alat yang memerlukan imej raster.

## How to get a clean export

Pilih bahasa yang paling hampir atau biarkan pengesanan bahasa secara automatik, kemudian pilih tema dengan kontras yang mencukupi untuk destinasi anda. Kekalkan nombor baris apabila cebisan kod akan dirujuk dalam perbincangan, dan nyahdayakan latar belakang luar apabila anda memerlukan aset lutsinar untuk reka bentuk lain.

## Privacy and limitations

Cebisan kod diproses secara setempat dalam halaman dan tidak dimuat naik oleh alat ini. Fail yang sangat besar boleh membuatkan penyerlahan dan eksport raster menjadi perlahan, jadi pangkas cebisan kod kepada baris yang ingin anda tunjukkan sebelum mengeksport.
