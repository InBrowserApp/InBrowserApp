# Code Screenshot Generator

## What it does

Code Screenshot Generator zet een geplakte snippet om in een verzorgde codekaart die kan worden geëxporteerd als SVG, PNG, JPEG, WebP of zelfstandige HTML. Het draait in je browser, past syntaxmarkering toe met selecteerbare thema's en laat je het frame afstellen voordat je downloadt.

## When to use it

Gebruik het voor documentatieafbeeldingen, release-opmerkingen, blogposts, presentaties, sociale previews en bugrapporten waarin een vormgegeven snippet gemakkelijker te scannen is dan platte tekst. SVG- en HTML-exports blijven scherp voor documentatie, terwijl PNG, JPEG en WebP handig zijn voor tools die rasterafbeeldingen verwachten.

## How to get a clean export

Kies de meest passende taal of laat taaldetectie op automatisch staan, en kies daarna een thema met genoeg contrast voor je bestemming. Laat regelnummers aan wanneer er in een discussie naar de snippet wordt verwezen, en schakel de buitenste achtergrond uit wanneer je een transparant asset voor een ander ontwerp nodig hebt.

## Privacy and limitations

De snippet wordt lokaal op de pagina verwerkt en wordt niet door deze tool geüpload. Zeer grote bestanden kunnen markering en rasterexport traag maken, dus beperk de snippet vóór het exporteren tot de regels die je wilt tonen.
