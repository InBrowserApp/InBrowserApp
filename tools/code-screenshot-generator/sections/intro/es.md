# Generador de capturas de código

## Qué hace

Generador de capturas de código convierte un fragmento pegado en una tarjeta de código pulida que se puede exportar como SVG, PNG, JPEG, WebP o HTML independiente. Se ejecuta en tu navegador, aplica resaltado de sintaxis con temas seleccionables y te permite ajustar el marco antes de descargar.

## Cuándo usarlo

Úsalo para imágenes de documentación, notas de la versión, publicaciones de blog, presentaciones, vistas previas sociales e informes de errores donde un fragmento con estilo sea más fácil de revisar que texto sin formato. Las exportaciones SVG y HTML se mantienen nítidas para documentación, mientras que PNG, JPEG y WebP son útiles para herramientas que esperan imágenes ráster.

## Cómo obtener una exportación limpia

Elige el lenguaje más cercano o deja la detección de lenguaje en automático, luego selecciona un tema con suficiente contraste para tu destino. Mantén los números de línea activados cuando el fragmento se vaya a citar en una conversación, y desactiva el fondo exterior cuando necesites un recurso transparente para otro diseño.

## Privacidad y limitaciones

Esta herramienta procesa el fragmento localmente en la página y no lo sube. Los archivos muy grandes pueden ralentizar el resaltado y la exportación ráster, así que recorta el fragmento a las líneas que quieres mostrar antes de exportar.
