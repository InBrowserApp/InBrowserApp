# Code Screenshot Generator

## What it does

Code Screenshot Generator पेस्ट किए गए स्निपेट को एक पॉलिश कोड कार्ड में बदलता है, जिसे SVG, PNG, JPEG, WebP या स्टैंडअलोन HTML के रूप में एक्सपोर्ट किया जा सकता है। यह आपके ब्राउज़र में चलता है, चुनी जा सकने वाली थीम के साथ सिंटैक्स हाइलाइटिंग लागू करता है, और डाउनलोड से पहले फ्रेम को समायोजित करने देता है।

## When to use it

इसे डॉक्यूमेंटेशन इमेज, रिलीज नोट्स, ब्लॉग पोस्ट, प्रेजेंटेशन, सोशल प्रीव्यू और बग रिपोर्ट के लिए उपयोग करें, जहां स्टाइल किया गया स्निपेट सादे टेक्स्ट की तुलना में स्कैन करना आसान होता है। SVG और HTML एक्सपोर्ट दस्तावेजों के लिए स्पष्ट बने रहते हैं, जबकि PNG, JPEG और WebP उन टूल के लिए उपयोगी हैं जिन्हें रास्टर इमेज चाहिए।

## How to get a clean export

सबसे नजदीकी भाषा चुनें या भाषा पहचान को स्वचालित रहने दें, फिर अपने गंतव्य के लिए पर्याप्त कंट्रास्ट वाली थीम चुनें। जब चर्चा में स्निपेट का संदर्भ दिया जाएगा तब लाइन नंबर चालू रखें, और किसी दूसरे डिजाइन के लिए पारदर्शी एसेट चाहिए तो बाहरी बैकग्राउंड बंद कर दें।

## Privacy and limitations

स्निपेट पेज में स्थानीय रूप से प्रोसेस होता है और यह टूल उसे अपलोड नहीं करता। बहुत बड़ी फाइलें हाइलाइटिंग और रास्टर एक्सपोर्ट को धीमा कर सकती हैं, इसलिए एक्सपोर्ट करने से पहले स्निपेट को उन्हीं लाइनों तक सीमित करें जिन्हें आप दिखाना चाहते हैं।
