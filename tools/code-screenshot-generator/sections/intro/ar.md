# Code Screenshot Generator

## ما الذي يفعله

يحوّل Code Screenshot Generator المقتطف الملصق إلى بطاقة كود مصقولة يمكن تصديرها بصيغ SVG أو PNG أو JPEG أو WebP أو كملف HTML مستقل. يعمل داخل متصفحك، ويطبّق تمييز الصياغة باستخدام سمات قابلة للاختيار، ويتيح لك ضبط الإطار قبل التنزيل.

## متى تستخدمه

استخدمه لصور التوثيق، وملاحظات الإصدار، ومنشورات المدونات، والعروض التقديمية، ومعاينات الشبكات الاجتماعية، وتقارير الأخطاء عندما يكون المقتطف المنسّق أسهل قراءة من النص العادي. تبقى صادرات SVG وHTML حادة في التوثيق، بينما تكون PNG وJPEG وWebP مفيدة للأدوات التي تتوقع صورًا نقطية.

## كيفية الحصول على تصدير نظيف

اختر أقرب لغة أو اترك اكتشاف اللغة على الوضع التلقائي، ثم اختر سمة بتباين كافٍ للوجهة التي ستستخدمها فيها. أبقِ أرقام الأسطر مفعّلة عندما سيُشار إلى المقتطف في نقاش، وعطّل الخلفية الخارجية عندما تحتاج إلى أصل شفاف لتصميم آخر.

## الخصوصية والقيود

تتم معالجة المقتطف محليًا داخل الصفحة ولا ترفعه هذه الأداة. يمكن للملفات الكبيرة جدًا أن تجعل التمييز والتصدير النقطي بطيئين، لذلك قلّص المقتطف إلى الأسطر التي تريد عرضها قبل التصدير.
