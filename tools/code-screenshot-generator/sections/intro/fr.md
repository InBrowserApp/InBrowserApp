# Code Screenshot Generator

## What it does

Code Screenshot Generator transforme un extrait collé en carte de code soignée, exportable en SVG, PNG, JPEG, WebP ou HTML autonome. Il s'exécute dans votre navigateur, applique une coloration syntaxique avec des thèmes sélectionnables et vous permet d'ajuster le cadre avant le téléchargement.

## When to use it

Utilisez-le pour les images de documentation, les notes de version, les articles de blog, les présentations, les aperçus sociaux et les rapports de bogue lorsqu'un extrait stylisé est plus facile à parcourir qu'un texte brut. Les exports SVG et HTML restent nets pour la documentation, tandis que les formats PNG, JPEG et WebP sont utiles pour les outils qui attendent des images matricielles.

## How to get a clean export

Choisissez le langage le plus proche ou laissez la détection du langage en automatique, puis sélectionnez un thème avec suffisamment de contraste pour votre destination. Gardez les numéros de ligne activés lorsque l'extrait sera cité dans une discussion, et désactivez l'arrière-plan externe si vous avez besoin d'une ressource transparente pour un autre design.

## Privacy and limitations

L'extrait est traité localement dans la page et n'est pas téléversé par cet outil. Les très grands fichiers peuvent ralentir la coloration et l'exportation matricielle ; réduisez donc l'extrait aux lignes que vous voulez montrer avant de l'exporter.
