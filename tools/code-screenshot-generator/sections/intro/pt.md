# Code Screenshot Generator

## O que ele faz

Code Screenshot Generator transforma um trecho colado em um cartão de código refinado que pode ser exportado como SVG, PNG, JPEG, WebP ou HTML independente. Ele roda no seu navegador, aplica realce de sintaxe com temas selecionáveis e permite ajustar a moldura antes de baixar.

## Quando usar

Use-o para imagens de documentação, notas de lançamento, posts de blog, apresentações, prévias sociais e relatórios de bugs em que um trecho estilizado é mais fácil de examinar do que texto simples. As exportações em SVG e HTML permanecem nítidas para documentação, enquanto PNG, JPEG e WebP são úteis para ferramentas que esperam imagens raster.

## Como obter uma exportação limpa

Escolha a linguagem mais próxima ou deixe a detecção de linguagem no automático, depois escolha um tema com contraste suficiente para o destino. Mantenha os números de linha ativados quando o trecho for referenciado em uma discussão e desative o plano de fundo externo quando precisar de um recurso transparente para outro design.

## Privacidade e limitações

O trecho é processado localmente na página e não é enviado por esta ferramenta. Arquivos muito grandes podem tornar o realce e a exportação raster lentos, então reduza o trecho às linhas que deseja mostrar antes de exportar.
