# Code Screenshot Generator

## Что делает

Code Screenshot Generator превращает вставленный фрагмент в аккуратную карточку кода, которую можно экспортировать как SVG, PNG, JPEG, WebP или отдельный HTML. Он работает в браузере, применяет подсветку синтаксиса с выбираемыми темами и позволяет настроить рамку перед скачиванием.

## Когда использовать

Используйте его для изображений в документации, заметок к релизам, постов в блогах, презентаций, превью для соцсетей и отчетов об ошибках, где стилизованный фрагмент легче просмотреть, чем обычный текст. Экспорт в SVG и HTML остается четким для документации, а PNG, JPEG и WebP полезны для инструментов, которым нужны растровые изображения.

## Как получить чистый экспорт

Выберите ближайший язык или оставьте автоматическое определение языка, затем выберите тему с достаточным контрастом для места назначения. Оставляйте номера строк включенными, когда на фрагмент будут ссылаться в обсуждении, и отключайте внешний фон, когда нужен прозрачный ресурс для другого дизайна.

## Конфиденциальность и ограничения

Фрагмент обрабатывается локально на странице и не загружается этим инструментом. Очень большие файлы могут замедлить подсветку и растровый экспорт, поэтому перед экспортом обрежьте фрагмент до строк, которые хотите показать.
