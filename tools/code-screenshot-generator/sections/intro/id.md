# Code Screenshot Generator

## Fungsinya

Code Screenshot Generator mengubah cuplikan yang ditempel menjadi kartu kode yang rapi dan dapat diekspor sebagai SVG, PNG, JPEG, WebP, atau HTML mandiri. Alat ini berjalan di browser Anda, menerapkan penyorotan sintaks dengan tema yang dapat dipilih, dan memungkinkan Anda menyesuaikan bingkai sebelum mengunduh.

## Kapan menggunakannya

Gunakan untuk gambar dokumentasi, catatan rilis, posting blog, presentasi, pratinjau sosial, dan laporan bug saat cuplikan bergaya lebih mudah dipindai daripada teks polos. Ekspor SVG dan HTML tetap tajam untuk dokumentasi, sementara PNG, JPEG, dan WebP berguna untuk alat yang membutuhkan gambar raster.

## Cara mendapatkan ekspor yang bersih

Pilih bahasa yang paling mendekati atau biarkan deteksi bahasa tetap otomatis, lalu pilih tema dengan kontras yang cukup untuk tujuan Anda. Pertahankan nomor baris saat cuplikan akan dirujuk dalam diskusi, dan nonaktifkan latar belakang luar saat Anda membutuhkan aset transparan untuk desain lain.

## Privasi dan batasan

Cuplikan diproses secara lokal di halaman dan tidak diunggah oleh alat ini. File yang sangat besar dapat membuat penyorotan dan ekspor raster lambat, jadi pangkas cuplikan ke baris yang ingin Anda tampilkan sebelum mengekspor.
