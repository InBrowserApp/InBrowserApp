# Code Screenshot Generator

## 기능

Code Screenshot Generator는 붙여넣은 스니펫을 세련된 코드 카드로 바꾸고 SVG, PNG, JPEG, WebP 또는 독립 실행형 HTML로 내보낼 수 있게 해줍니다. 브라우저에서 실행되며, 선택 가능한 테마로 구문 강조를 적용하고 다운로드 전에 프레임을 조정할 수 있습니다.

## 사용 시점

스타일이 적용된 스니펫이 일반 텍스트보다 훑어보기 쉬운 문서 이미지, 릴리스 노트, 블로그 글, 프레젠테이션, 소셜 미리보기, 버그 보고서에 사용하세요. SVG와 HTML 내보내기는 문서에서 선명하게 유지되고, PNG, JPEG, WebP는 래스터 이미지를 요구하는 도구에 유용합니다.

## 깔끔하게 내보내는 방법

가장 가까운 언어를 선택하거나 언어 감지를 자동으로 둔 다음, 목적지에 충분한 대비를 제공하는 테마를 고르세요. 토론에서 스니펫을 참조할 때는 줄 번호를 켜 두고, 다른 디자인에 사용할 투명 에셋이 필요하면 바깥 배경을 비활성화하세요.

## 개인정보 보호 및 제한 사항

스니펫은 페이지 안에서 로컬로 처리되며 이 도구로 업로드되지 않습니다. 매우 큰 파일은 강조 표시와 래스터 내보내기를 느리게 만들 수 있으므로, 내보내기 전에 보여주고 싶은 줄만 남기도록 스니펫을 다듬으세요.
