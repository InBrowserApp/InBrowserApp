# Code Screenshot Generator

## What it does

Code Screenshot Generator trasforma uno snippet incollato in una scheda di codice rifinita che può essere esportata come SVG, PNG, JPEG, WebP o HTML autonomo. Funziona nel tuo browser, applica l'evidenziazione della sintassi con temi selezionabili e ti permette di regolare la cornice prima del download.

## When to use it

Usalo per immagini di documentazione, note di rilascio, post di blog, presentazioni, anteprime social e segnalazioni di bug in cui uno snippet stilizzato è più facile da leggere del testo semplice. Le esportazioni SVG e HTML restano nitide per la documentazione, mentre PNG, JPEG e WebP sono utili per strumenti che richiedono immagini raster.

## How to get a clean export

Scegli il linguaggio più vicino o lascia attivo il rilevamento automatico, poi seleziona un tema con contrasto sufficiente per la tua destinazione. Mantieni attivi i numeri di riga quando lo snippet sarà citato in una discussione e disattiva lo sfondo esterno quando ti serve una risorsa trasparente per un altro design.

## Privacy and limitations

Lo snippet viene elaborato localmente nella pagina e non viene caricato da questo strumento. File molto grandi possono rallentare l'evidenziazione e l'esportazione raster, quindi riduci lo snippet alle righe che vuoi mostrare prima di esportare.
