# Code Screenshot Generator

## ทำอะไรได้บ้าง

Code Screenshot Generator แปลงสนิปเป็ตที่วางไว้ให้เป็นการ์ดโค้ดที่ดูเรียบร้อย และส่งออกได้เป็น SVG, PNG, JPEG, WebP หรือ HTML แบบสแตนด์อโลน เครื่องมือนี้ทำงานในเบราว์เซอร์ของคุณ ใช้การไฮไลต์ไวยากรณ์พร้อมธีมที่เลือกได้ และให้คุณปรับกรอบก่อนดาวน์โหลด

## ควรใช้เมื่อใด

ใช้สำหรับรูปภาพประกอบเอกสาร บันทึกประจำรุ่น บทความบล็อก งานนำเสนอ ภาพตัวอย่างบนโซเชียล และรายงานบั๊ก เมื่อสนิปเป็ตที่จัดรูปแบบแล้วอ่านสแกนได้ง่ายกว่าข้อความล้วน การส่งออกเป็น SVG และ HTML ยังคมชัดสำหรับเอกสาร ส่วน PNG, JPEG และ WebP เหมาะกับเครื่องมือที่ต้องการรูปภาพแรสเตอร์

## วิธีส่งออกให้ดูสะอาด

เลือกภาษาที่ใกล้เคียงที่สุดหรือปล่อยให้การตรวจจับภาษาเป็นอัตโนมัติ จากนั้นเลือกธีมที่มีคอนทราสต์พอสำหรับปลายทางของคุณ เปิดเลขบรรทัดไว้เมื่อจะอ้างถึงสนิปเป็ตในการสนทนา และปิดพื้นหลังด้านนอกเมื่อคุณต้องการแอสเซ็ตโปร่งใสสำหรับงานออกแบบอื่น

## ความเป็นส่วนตัวและข้อจำกัด

สนิปเป็ตจะถูกประมวลผลภายในหน้าเว็บบนเครื่องของคุณ และเครื่องมือนี้จะไม่อัปโหลดข้อมูล ไฟล์ที่ใหญ่มากอาจทำให้การไฮไลต์และการส่งออกเป็นแรสเตอร์ช้าลง ดังนั้นควรตัดสนิปเป็ตให้เหลือเฉพาะบรรทัดที่ต้องการแสดงก่อนส่งออก
