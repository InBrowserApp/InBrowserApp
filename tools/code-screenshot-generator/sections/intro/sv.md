# Code Screenshot Generator

## What it does

Code Screenshot Generator gör ett inklistrat kodavsnitt till ett polerat kodkort som kan exporteras som SVG, PNG, JPEG, WebP eller fristående HTML. Det körs i din webbläsare, använder syntaxmarkering med valbara teman och låter dig justera ramen innan du laddar ner.

## When to use it

Använd det för dokumentationsbilder, versionsanteckningar, blogginlägg, presentationer, sociala förhandsvisningar och felrapporter där ett formgivet kodavsnitt är lättare att överblicka än oformaterad text. SVG- och HTML-exporter förblir skarpa i dokumentation, medan PNG, JPEG och WebP är användbara för verktyg som förväntar sig rasterbilder.

## How to get a clean export

Välj det närmaste språket eller låt språkidentifiering vara automatisk, och välj sedan ett tema med tillräcklig kontrast för din destination. Behåll radnummer när kodavsnittet ska refereras i en diskussion, och inaktivera den yttre bakgrunden när du behöver en transparent resurs för en annan design.

## Privacy and limitations

Kodavsnittet bearbetas lokalt på sidan och laddas inte upp av det här verktyget. Mycket stora filer kan göra syntaxmarkering och rasterexport långsamma, så korta ner kodavsnittet till de rader du vill visa innan du exporterar.
