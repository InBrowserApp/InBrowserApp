أدوات الشبكة هي نقطة انطلاق لمهام الشبكات التي تعمل داخل المتصفح. استخدمها عندما تعرف نوع المشكلة التي تحلها لكنك تريد طريقًا أسرع إلى الأداة المناسبة: فحص نطاق IP، أو الاستعلام عن سجلات DNS، أو فك ترميز نطاق دولي، أو البحث عن منفذ، أو التحقق من شهادة أو بصمة مفتاح SSH.

## ما يمكنك فعله

- العمل مع IPv4 وIPv6 وكتل CIDR ونطاقات العناوين وعناوين link-local المشتقة من MAC.
- الاستعلام عن سجلات DNS وDNS العكسي عبر أدوات بحث مناسبة للمتصفح.
- البحث عن رموز حالة HTTP وأنواع MIME وأرقام المنافذ والوقت الشبكي الحالي.
- فحص تفاصيل الشهادات والمفاتيح العامة دون إرسال المادة المصدرية إلى خادم.

## اختيار الأداة المناسبة

ابدأ بـ **IP وCIDR** عندما يكون الإدخال عنوانًا أو نطاقًا أو شبكة فرعية أو كتلة توجيه. استخدم **DNS والنطاقات** للسجلات واستعلامات PTR وتحويل IDN/Punycode. استخدم **مراجع البروتوكولات** عندما تحتاج إلى جدول بحث سريع. استخدم **المفاتيح والشهادات** عندما تكون المادة المصدرية شهادة TLS أو مفتاحًا عامًا أو إدخال مفتاح SSH مصرّح به.

## ملاحظات الخصوصية

تعمل معظم الأدوات في هذه المجموعة بالكامل داخل متصفحك. الأدوات التي تحتاج إلى بيانات شبكة عامة، مثل استعلامات DNS أو معلومات IP، قد تتصل بالمحلل أو خدمة البحث اللازمة للإجابة عن ذلك الاستعلام. تجنّب لصق الأسرار في أدوات البحث العامة، وفضّل الأدوات المحلية فقط لفحص الشهادات والمفاتيح عندما تكون المادة حساسة.
