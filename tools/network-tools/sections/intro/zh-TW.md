網路工具是處理瀏覽器型網路任務的起點。當你知道要解決的問題類型，但想更快找到合適工具時，可以從這裡開始：檢查 IP 範圍、查詢 DNS 記錄、解碼國際化網域、查詢連接埠，或檢查憑證與 SSH 金鑰指紋。

## 你可以做什麼

- 處理 IPv4、IPv6、CIDR 區塊、位址範圍，以及由 MAC 衍生的鏈路本地位址。
- 透過適合瀏覽器使用的查詢工具查詢 DNS 與反向 DNS 記錄。
- 查詢 HTTP 狀態碼、MIME 類型、連接埠號碼與目前網路時間。
- 不將來源材料傳送到伺服器，也能檢查憑證與公開金鑰詳細資料。

## 選擇合適工具

當輸入是位址、範圍、子網路或路由區塊時，請從 **IP 與 CIDR** 開始。記錄、PTR 查詢與 IDN/Punycode 轉換請使用 **DNS 與網域**。需要快速查表時，請使用 **通訊協定參考**。來源材料是 TLS 憑證、公開金鑰或 SSH authorized key 項目時，請使用 **金鑰與憑證**。

## 隱私注意事項

此集合中的多數工具都完全在你的瀏覽器中執行。需要公用網路資料的工具，例如 DNS 查詢或 IP 資訊，可能會連線到回答該查詢所需的解析器或查詢服務。避免將機密貼到公開查詢工具；當材料較敏感時，檢查憑證與金鑰請優先使用僅限本機的工具。
