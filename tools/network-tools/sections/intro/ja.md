ネットワークツールは、ブラウザー上でネットワーク関連の作業を始めるための入口です。解決したい問題の種類は分かっていて、適切なユーティリティにすばやくたどり着きたいときに使えます。IP 範囲の確認、DNS レコードの照会、国際化ドメインのデコード、ポートの検索、証明書や SSH 鍵フィンガープリントの確認に利用できます。

## できること

- IPv4、IPv6、CIDR ブロック、アドレス範囲、MAC 由来のリンクローカルアドレスを扱えます。
- ブラウザーで使いやすいルックアップツールを通じて、DNS と逆引き DNS レコードを照会できます。
- HTTP ステータスコード、MIME タイプ、ポート番号、現在のネットワーク時刻を調べられます。
- 元の素材をサーバーに送信せずに、証明書と公開鍵の詳細を確認できます。

## 適切なツールの選び方

入力がアドレス、範囲、サブネット、またはルーティングブロックの場合は、**IP と CIDR** から始めます。レコード、PTR ルックアップ、IDN/Punycode 変換には **DNS とドメイン** を使います。すばやく参照できる表が必要なときは **プロトコルリファレンス** を使います。元の素材が TLS 証明書、公開鍵、または SSH authorized key エントリの場合は **鍵と証明書** を使います。

## プライバシーに関する注意

このコレクションのほとんどのツールは、完全にブラウザー内で実行されます。DNS ルックアップや IP 情報など、公開ネットワークデータが必要なツールは、その問い合わせに回答するために必要なリゾルバーやルックアップサービスに接続する場合があります。公開ルックアップツールに機密情報を貼り付けることは避け、素材が機密の場合は証明書と鍵の確認にはローカルのみで動作するツールを優先してください。
