네트워크 도구는 브라우저 기반 네트워킹 작업의 시작점입니다. 해결하려는 문제의 종류는 알고 있지만 적합한 유틸리티로 더 빠르게 이동하고 싶을 때 사용하세요. IP 범위를 검사하거나, DNS 레코드를 조회하거나, 국제화 도메인을 디코딩하거나, 포트를 찾아보거나, 인증서 또는 SSH 키 지문을 확인할 수 있습니다.

## 할 수 있는 작업

- IPv4, IPv6, CIDR 블록, 주소 범위, MAC 파생 링크 로컬 주소로 작업합니다.
- 브라우저 친화적인 조회 도구를 통해 DNS 및 역방향 DNS 레코드를 조회합니다.
- HTTP 상태 코드, MIME 타입, 포트 번호, 현재 네트워크 시간을 조회합니다.
- 원본 자료를 서버로 보내지 않고 인증서와 공개 키 세부 정보를 검사합니다.

## 적합한 도구 선택

입력이 주소, 범위, 서브넷 또는 라우팅 블록이면 **IP 및 CIDR**부터 시작하세요. 레코드, PTR 조회, IDN/Punycode 변환에는 **DNS 및 도메인**을 사용하세요. 빠른 조회 표가 필요할 때는 **프로토콜 참조**를 사용하세요. 원본 자료가 TLS 인증서, 공개 키 또는 SSH authorized key 항목이면 **키 및 인증서**를 사용하세요.

## 개인정보 보호 참고 사항

이 모음의 대부분 도구는 브라우저에서 완전히 실행됩니다. DNS 조회 또는 IP 정보처럼 공용 네트워크 데이터가 필요한 도구는 해당 질의에 답하는 데 필요한 리졸버 또는 조회 서비스에 접속할 수 있습니다. 공개 조회 도구에 비밀 정보를 붙여넣지 말고, 자료가 민감한 경우 인증서 및 키 검사에는 로컬 전용 도구를 선호하세요.
