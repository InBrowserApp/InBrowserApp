नेटवर्क टूल्स ब्राउज़र-आधारित नेटवर्किंग कार्यों के लिए शुरुआती जगह है। इसका उपयोग तब करें जब आपको पता हो कि आप किस तरह की समस्या हल कर रहे हैं, लेकिन सही यूटिलिटी तक तेज़ रास्ता चाहिए: IP रेंज की जाँच करना, DNS रिकॉर्ड क्वेरी करना, अंतरराष्ट्रीयकृत डोमेन डिकोड करना, पोर्ट देखना, या प्रमाणपत्र अथवा SSH key fingerprint जाँचना।

## आप क्या कर सकते हैं

- IPv4, IPv6, CIDR ब्लॉक, address ranges, और MAC-व्युत्पन्न link-local addresses के साथ काम करें।
- ब्राउज़र-अनुकूल lookup tools के ज़रिए DNS और reverse DNS रिकॉर्ड क्वेरी करें।
- HTTP status codes, MIME types, पोर्ट नंबर, और वर्तमान नेटवर्क समय देखें।
- स्रोत सामग्री को सर्वर पर भेजे बिना प्रमाणपत्र और public-key विवरणों की जाँच करें।

## सही टूल चुनना

जब इनपुट कोई पता, रेंज, subnet, या routing block हो, तो **IP और CIDR** से शुरू करें। records, PTR lookups, और IDN/Punycode conversion के लिए **DNS और डोमेन** इस्तेमाल करें। जब आपको तुरंत lookup table चाहिए, तो **प्रोटोकॉल संदर्भ** इस्तेमाल करें। जब स्रोत सामग्री TLS प्रमाणपत्र, सार्वजनिक कुंजी, या SSH authorized key entry हो, तो **कुंजियाँ और प्रमाणपत्र** इस्तेमाल करें।

## गोपनीयता नोट्स

इस संग्रह के अधिकांश टूल पूरी तरह आपके ब्राउज़र में चलते हैं। जिन टूल्स को सार्वजनिक नेटवर्क डेटा चाहिए, जैसे DNS lookups या IP जानकारी, वे उस क्वेरी का उत्तर देने के लिए आवश्यक resolver या lookup service से संपर्क कर सकते हैं। सार्वजनिक lookup tools में secrets पेस्ट करने से बचें, और सामग्री संवेदनशील होने पर प्रमाणपत्र और कुंजी जाँच के लिए केवल-स्थानीय टूल्स को प्राथमिकता दें।
