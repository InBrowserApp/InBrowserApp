เครื่องมือเครือข่ายเป็นจุดเริ่มต้นสำหรับงานเครือข่ายที่ทำในเบราว์เซอร์ ใช้เมื่อคุณรู้ชนิดของปัญหาที่กำลังแก้ แต่ต้องการไปยังยูทิลิตี้ที่เหมาะสมได้เร็วขึ้น เช่น ตรวจสอบช่วง IP, สอบถาม DNS record, ถอดรหัสโดเมนแบบ IDN, ค้นหาพอร์ต หรือตรวจสอบใบรับรองหรือลายนิ้วมือคีย์ SSH

## สิ่งที่คุณทำได้

- ทำงานกับ IPv4, IPv6, บล็อก CIDR, ช่วงที่อยู่ และที่อยู่ link-local ที่ได้จาก MAC
- สอบถาม DNS และ reverse DNS records ผ่านเครื่องมือค้นหาที่เหมาะกับเบราว์เซอร์
- ค้นหารหัสสถานะ HTTP, ประเภท MIME, หมายเลขพอร์ต และเวลาเครือข่ายปัจจุบัน
- ตรวจสอบรายละเอียดใบรับรองและกุญแจสาธารณะโดยไม่ส่งแหล่งข้อมูลไปยังเซิร์ฟเวอร์

## การเลือกเครื่องมือที่เหมาะสม

เริ่มด้วย **IP และ CIDR** เมื่ออินพุตเป็นที่อยู่ ช่วง subnet หรือบล็อก routing ใช้ **DNS และโดเมน** สำหรับ records, การค้นหา PTR และการแปลง IDN/Punycode ใช้ **ข้อมูลอ้างอิงโปรโตคอล** เมื่อคุณต้องการตารางค้นหาแบบรวดเร็ว ใช้ **กุญแจและใบรับรอง** เมื่อแหล่งข้อมูลเป็นใบรับรอง TLS, กุญแจสาธารณะ หรือรายการ authorized key ของ SSH

## หมายเหตุความเป็นส่วนตัว

เครื่องมือส่วนใหญ่ในคอลเล็กชันนี้ทำงานทั้งหมดในเบราว์เซอร์ของคุณ เครื่องมือที่ต้องใช้ข้อมูลเครือข่ายสาธารณะ เช่น การค้นหา DNS หรือข้อมูล IP อาจติดต่อ resolver หรือบริการค้นหาที่จำเป็นต่อการตอบ query นั้น หลีกเลี่ยงการวางข้อมูลลับลงในเครื่องมือค้นหาสาธารณะ และเลือกใช้เครื่องมือที่ทำงานเฉพาะในเครื่องสำหรับการตรวจสอบใบรับรองและกุญแจเมื่อข้อมูลมีความอ่อนไหว
