网络工具是浏览器网络任务的起点。当您知道要解决的问题类型，但希望更快找到合适的实用工具时，可以使用它：检查 IP 范围、查询 DNS 记录、解码国际化域名、查找端口，或检查证书或 SSH 密钥指纹。

## 您可以做什么

- 处理 IPv4、IPv6、CIDR 块、地址范围，以及由 MAC 派生的链路本地地址。
- 通过适合浏览器使用的查询工具查询 DNS 和反向 DNS 记录。
- 查找 HTTP 状态码、MIME 类型、端口号和当前网络时间。
- 检查证书和公钥详情，而无需将源内容发送到服务器。

## 选择合适的工具

当输入是地址、范围、子网或路由块时，从 **IP 和 CIDR** 开始。对于记录、PTR 查询和 IDN/Punycode 转换，请使用 **DNS 和域名**。需要快速查询表时，请使用 **协议参考**。当源内容是 TLS 证书、公钥或 SSH authorized key 条目时，请使用 **密钥和证书**。

## 隐私说明

该集合中的大多数工具完全在您的浏览器中运行。需要公共网络数据的工具，例如 DNS 查询或 IP 信息，可能会联系回答该查询所需的解析器或查询服务。避免将机密粘贴到公共查询工具中；当内容敏感时，请优先使用仅在本地运行的工具检查证书和密钥。
