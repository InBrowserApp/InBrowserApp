Herramientas de red es un punto de partida para tareas de red basadas en navegador. Úsalo cuando sepas qué tipo de problema estás resolviendo pero quieras llegar más rápido a la utilidad adecuada: inspeccionar un rango IP, consultar registros DNS, decodificar un dominio internacionalizado, buscar un puerto o comprobar la huella de un certificado o una clave SSH.

## Qué puedes hacer

- Trabajar con IPv4, IPv6, bloques CIDR, rangos de direcciones y direcciones de enlace local derivadas de MAC.
- Consultar registros DNS y DNS inverso mediante herramientas de búsqueda compatibles con el navegador.
- Buscar códigos de estado HTTP, tipos MIME, números de puerto y la hora de red actual.
- Inspeccionar detalles de certificados y claves públicas sin enviar el material de origen a un servidor.

## Elegir la herramienta adecuada

Empieza con **IP y CIDR** cuando la entrada sea una dirección, rango, subred o bloque de enrutamiento. Usa **DNS y dominios** para registros, búsquedas PTR y conversión IDN/Punycode. Usa **Referencias de protocolos** cuando necesites una tabla de consulta rápida. Usa **Claves y certificados** cuando el material de origen sea un certificado TLS, una clave pública o una entrada de clave autorizada SSH.

## Notas de privacidad

La mayoría de las herramientas de esta colección se ejecutan por completo en tu navegador. Las herramientas que necesitan datos de red públicos, como búsquedas DNS o información de IP, pueden contactar con el resolvedor o el servicio de búsqueda requerido para responder a esa consulta. Evita pegar secretos en herramientas de búsqueda públicas y prefiere las herramientas solo locales para inspeccionar certificados y claves cuando el material sea sensible.
