## Base64 とは？

Base64 は、メール本文、JSON ブロブ、小さな data URL など、テキストベースのチャネルでバイナリ互換のペイロードを運ぶ必要があるときに役立ちます。これはエンコードの層であり、セキュリティの層ではありません。

## 使用する場面

- API が Base64 文字列を返す、または期待する場合の手軽なデバッグ。
- ログやペイロード用に、ブラウザのテキストを安全な転送形式へ変換する。
- 貼り付けた Base64 ブロブが期待どおりの内容にデコードされるかを確認する。

## 注意すべき点

- Base64 はデータサイズを約 3 分の 1 増加させます。
- 元の値を暗号化したり隠したりするものではありません。
- パディング不正やコピー＆ペーストの不備は、通常デコードエラーとして現れます。
