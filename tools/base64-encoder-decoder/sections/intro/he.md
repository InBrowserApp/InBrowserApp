## מהו Base64?

Base64 שימושי כאשר ערוץ מבוסס טקסט צריך לשאת מטענים ידידותיים לבינארי, כמו גופי דוא"ל, בלובים של JSON או data URL קטנים. זוהי שכבת קידוד, לא שכבת אבטחה.

## מתי להשתמש בו

- ניפוי באגים מהיר כאשר API מחזיר או מצפה למחרוזות Base64.
- המרת טקסט מהדפדפן לפורמט הובלה בטוח עבור יומנים או מטענים.
- בדיקה האם בלוב Base64 שהודבק מתפענח לתוכן שאתם מצפים לו.

## מה כדאי לזכור

- Base64 מגדיל את הגודל בכשליש בערך.
- הוא אינו מצפין ואינו מסתיר את הערך המקורי.
- ריפוד לא חוקי או העתקה והדבקה שבורה מופיעים בדרך כלל כשגיאת פענוח.
