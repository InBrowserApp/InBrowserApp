## Base64 คืออะไร?

Base64 มีประโยชน์เมื่อช่องทางที่ใช้ข้อความจำเป็นต้องส่งข้อมูลที่เป็นมิตรกับไบนารี เช่น เนื้อหาอีเมล บล็อบ JSON หรือ data URL ขนาดเล็ก มันเป็นชั้นการเข้ารหัส ไม่ใช่ชั้นความปลอดภัย

## เมื่อใดควรใช้

- ดีบักอย่างรวดเร็วเมื่อ API ส่งคืนหรือคาดหวังสตริง Base64
- แปลงข้อความจากเบราว์เซอร์เป็นรูปแบบการขนส่งที่ปลอดภัยสำหรับบันทึกหรือเพย์โหลด
- ตรวจสอบว่าบล็อบ Base64 ที่วางจะถอดรหัสเป็นเนื้อหาที่คุณคาดหวังหรือไม่

## สิ่งที่ควรคำนึงถึง

- Base64 เพิ่มขนาดข้อมูลประมาณหนึ่งในสาม
- มันไม่ได้เข้ารหัสลับหรือซ่อนค่าต้นฉบับ
- การเติมเต็มที่ไม่ถูกต้องหรือการคัดลอก-วางที่ขาดหาย มักจะปรากฏเป็นข้อผิดพลาดในการถอดรหัส
