## Apa itu Tipe MIME?

Tipe MIME (Multipurpose Internet Mail Extensions) adalah cara standar untuk menunjukkan sifat dan format dokumen atau file. Mereka digunakan oleh browser web dan server untuk menentukan cara menangani file. Tipe MIME terdiri dari tipe dan subtipe, dipisahkan oleh garis miring (misalnya, text/html). Kategori utama adalah:

- **Aplikasi:** File aplikasi mencakup JSON, PDF, arsip ZIP, dan format data biner atau terstruktur lainnya.
- **Audio:** Format audio mencakup MP3, WAV, OGG, dan format file suara lainnya.
- **Font:** File font mencakup TTF, OTF, WOFF, dan format font lainnya yang digunakan untuk tipografi web.
- **Gambar:** Format gambar mencakup JPEG, PNG, GIF, SVG, dan format file gambar lainnya.
- **Pesan:** Format pesan mencakup pesan email dan protokol berbasis pesan lainnya.
- **Model:** Format model 3D mencakup GLTF, OBJ, STL, dan format file model 3D lainnya.
- **Teks:** Format teks mencakup HTML, CSS, JavaScript, teks biasa, CSV, dan format berbasis teks lainnya.
- **Video:** Format video mencakup MP4, WebM, AVI, dan format file video lainnya.
