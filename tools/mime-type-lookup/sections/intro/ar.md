## ما هو نوع MIME؟

أنواع MIME (ملحقات البريد الإلكتروني متعددة الأغراض) هي طريقة قياسية للإشارة إلى طبيعة وتنسيق المستند أو الملف. يتم استخدامها بواسطة متصفحات الويب والخوادم لتحديد كيفية التعامل مع الملفات. تتكون أنواع MIME من نوع ونوع فرعي، مفصولة بشرطة مائلة (على سبيل المثال، text/html). الفئات الرئيسية هي:

- **التطبيق:** تتضمن ملفات التطبيق JSON و PDF وأرشيفات ZIP وتنسيقات البيانات الثنائية أو المهيكلة الأخرى.
- **الصوت:** تتضمن تنسيقات الصوت MP3 و WAV و OGG وتنسيقات ملفات الصوت الأخرى.
- **الخط:** تتضمن ملفات الخط TTF و OTF و WOFF وتنسيقات الخطوط الأخرى المستخدمة في طباعة الويب.
- **الصورة:** تتضمن تنسيقات الصور JPEG و PNG و GIF و SVG وتنسيقات ملفات الصور الأخرى.
- **الرسالة:** تتضمن تنسيقات الرسائل رسائل البريد الإلكتروني والبروتوكولات الأخرى القائمة على الرسائل.
- **النموذج:** تتضمن تنسيقات النماذج ثلاثية الأبعاد GLTF و OBJ و STL وتنسيقات ملفات النماذج ثلاثية الأبعاد الأخرى.
- **النص:** تتضمن تنسيقات النص HTML و CSS و JavaScript والنص العادي و CSV والتنسيقات الأخرى القائمة على النص.
- **الفيديو:** تتضمن تنسيقات الفيديو MP4 و WebM و AVI وتنسيقات ملفات الفيديو الأخرى.
