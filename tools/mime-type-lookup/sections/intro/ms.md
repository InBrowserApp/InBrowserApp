## Apakah Jenis MIME?

Jenis MIME (Sambungan Mel Internet Pelbagai Guna) ialah cara piawai untuk menunjukkan sifat dan format dokumen atau fail. Ia digunakan oleh pelayar web dan pelayan untuk menentukan cara mengendalikan fail. Jenis MIME terdiri daripada jenis dan subjenis, dipisahkan oleh garis miring (contohnya, text/html). Kategori utama ialah:

- **Aplikasi:** Fail aplikasi termasuk JSON, PDF, arkib ZIP dan format data binari atau berstruktur lain.
- **Audio:** Format audio termasuk MP3, WAV, OGG dan format fail bunyi lain.
- **Fon:** Fail fon termasuk TTF, OTF, WOFF dan format fon lain yang digunakan untuk tipografi web.
- **Imej:** Format imej termasuk JPEG, PNG, GIF, SVG dan format fail imej lain.
- **Mesej:** Format mesej termasuk mesej e-mel dan protokol berasaskan mesej lain.
- **Model:** Format model 3D termasuk GLTF, OBJ, STL dan format fail model 3D lain.
- **Teks:** Format teks termasuk HTML, CSS, JavaScript, teks biasa, CSV dan format berasaskan teks lain.
- **Video:** Format video termasuk MP4, WebM, AVI dan format fail video lain.
