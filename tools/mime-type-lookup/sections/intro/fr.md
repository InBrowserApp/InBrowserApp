## Qu'est-ce qu'un Type MIME ?

Les types MIME (Extensions Multi-Usage du Courrier Internet) sont un moyen standard d'indiquer la nature et le format d'un document ou d'un fichier. Ils sont utilisés par les navigateurs web et les serveurs pour déterminer comment gérer les fichiers. Les types MIME consistent en un type et un sous-type, séparés par une barre oblique (par exemple, text/html). Les catégories principales sont :

- **Application :** Les fichiers d'application incluent JSON, PDF, archives ZIP et d'autres formats de données binaires ou structurés.
- **Audio :** Les format audio incluent MP3, WAV, OGG et d'autres formats de fichiers audio.
- **Police :** Les fichiers de police incluent TTF, OTF, WOFF et d'autres formats de police utilisés pour la typographie web.
- **Image :** Les formats d'image incluent JPEG, PNG, GIF, SVG et d'autres formats de fichiers image.
- **Message :** Les formats de message incluent les messages électroniques et d'autres protocoles basés sur les messages.
- **Modèle :** Les formats de modèle 3D incluent GLTF, OBJ, STL et d'autres formats de fichiers de modèle 3D.
- **Texte :** Les formats de texte incluent HTML, CSS, JavaScript, texte brut, CSV et d'autres formats basés sur le texte.
- **Vidéo :** Les formats vidéo incluent MP4, WebM, AVI et d'autres formats de fichiers vidéo.
