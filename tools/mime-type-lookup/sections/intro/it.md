## Cos'è un Tipo MIME?

I tipi MIME (Multipurpose Internet Mail Extensions) sono un modo standard per indicare la natura e il formato di un documento o file. Sono utilizzati dai browser web e dai server per determinare come gestire i file. I tipi MIME consistono in un tipo e un sottotipo, separati da una barra (ad esempio, text/html). Le categorie principali sono:

- **Applicazione:** I file applicazione includono JSON, PDF, archivi ZIP e altri formati di dati binari o strutturati.
- **Audio:** I formati audio includono MP3, WAV, OGG e altri formati di file audio.
- **Carattere:** I file carattere includono TTF, OTF, WOFF e altri formati di carattere utilizzati per la tipografia web.
- **Immagine:** I formati immagine includono JPEG, PNG, GIF, SVG e altri formati di file immagine.
- **Messaggio:** I formati messaggio includono messaggi di posta elettronica e altri protocolli basati su messaggi.
- **Modello:** I formati modello 3D includono GLTF, OBJ, STL e altri formati di file modello 3D.
- **Testo:** I formati testo includono HTML, CSS, JavaScript, testo semplice, CSV e altri formati basati su testo.
- **Video:** I formati video includono MP4, WebM, AVI e altri formati di file video.
