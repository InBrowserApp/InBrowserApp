## 什么是 Unicode 转义？

Unicode 转义将字符转换为表示其 Unicode 码位的编码序列。当源代码、配置文件或数据格式无法直接包含某些字符时，这种转换至关重要。

**常见转义格式：**

- `\uXXXX` — JavaScript / JSON，大多数编程语言中使用
- `\u{XXXXX}` — ES6+ JavaScript，支持增补字符，无需代理对
- `&#xXXXX;` / `&#DDDD;` — 十六进制或十进制形式的 HTML 实体
- `U+XXXX` — 文档中使用的标准 Unicode 表示法
- `\xXX` / `%XX` — UTF-8 字节级编码，常用于 URL 和类 C 语言
- `\UXXXXXXXX` — Python 8 位格式，可表示任意码位
- `0xXXXX` — 十六进制字面量表示法

## 何时使用此工具

- 在要求 ASCII 安全编码的源代码或配置文件中嵌入非 ASCII 字符
- 通过检查底层 Unicode 码位来调试乱码文本
- 在不同语言或格式之间迁移时转换转义表示法
- 为需要实体编码字符的 JSON、HTML 或 URL 上下文准备文本

## 工作原理

在左侧输入或粘贴纯文本，工具会使用所选格式对非 ASCII 字符进行转义。在右侧粘贴转义文本，工具会自动检测并同时解码所有支持的格式。所有操作均在浏览器本地完成。
