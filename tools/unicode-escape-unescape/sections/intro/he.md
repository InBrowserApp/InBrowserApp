## מהו קידוד Unicode?

קידוד Unicode ממיר תווים לרצפים מקודדים המייצגים את נקודות הקוד שלהם ב־Unicode. הדבר חיוני כאשר קוד מקור, קובצי תצורה או פורמטים של נתונים אינם יכולים להכיל תווים מסוימים ישירות.

**פורמטים נפוצים של קידוד:**

- `\uXXXX` — JavaScript / JSON, נפוץ ברוב שפות התכנות
- `\u{XXXXX}` — JavaScript מגרסת ES6 ומעלה, תומך בתווים משלימים ללא זוגות תחליפיים
- `&#xXXXX;` / `&#DDDD;` — ישויות HTML בפורמט הקסדצימלי או עשרוני
- `U+XXXX` — סימון Unicode תקני המשמש בתיעוד
- `\xXX` / `%XX` — קידוד ברמת בתים של UTF-8, נפוץ בכתובות URL ובשפות דמויות C
- `\UXXXXXXXX` — פורמט Python בן 8 ספרות לכל נקודת קוד
- `0xXXXX` — סימון ליטרל הקסדצימלי

## מתי להשתמש בכלי זה

- הטמעת תווים שאינם ASCII בקוד מקור או בקובצי תצורה הדורשים קידוד תואם ASCII
- ניפוי באגים בטקסט משובש על ידי בדיקת נקודות הקוד הבסיסיות של Unicode
- המרה בין סימוני קידוד שונים בעת מעבר בין שפות או פורמטים
- הכנת טקסט להקשרי JSON, HTML או URL הדורשים תווים מקודדים כישויות
