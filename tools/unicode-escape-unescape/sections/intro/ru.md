## Что такое экранирование Unicode?

Экранирование Unicode преобразует символы в закодированные последовательности, представляющие их кодовые точки Unicode. Это необходимо, когда исходный код, файлы конфигурации или форматы данных не могут содержать определённые символы напрямую.

**Распространённые форматы экранирования:**

- `\uXXXX` — JavaScript / JSON, используется в большинстве языков программирования
- `\u{XXXXX}` — ES6+ JavaScript, поддерживает дополнительные символы без суррогатных пар
- `&#xXXXX;` / `&#DDDD;` — HTML-сущности в шестнадцатеричной или десятичной форме
- `U+XXXX` — стандартная нотация Unicode, используемая в документации
- `\xXX` / `%XX` — побайтовое кодирование UTF-8, распространено в URL и C-подобных языках
- `\UXXXXXXXX` — 8-значный формат Python для любой кодовой точки
- `0xXXXX` — шестнадцатеричная литеральная нотация

## Когда использовать этот инструмент

- Встраивание не-ASCII символов в исходный код или файлы конфигурации, требующие ASCII-безопасного кодирования
- Отладка искажённого текста путём анализа базовых кодовых точек Unicode
- Конвертация между различными форматами экранирования при переносе между языками или форматами
- Подготовка текста для JSON, HTML или URL-контекстов, требующих кодирования символов в виде сущностей

## Как это работает

Введите или вставьте обычный текст слева, и инструмент экранирует не-ASCII символы в выбранном формате. Вставьте экранированный текст справа — он автоматически определит и декодирует все поддерживаемые форматы одновременно. Всё выполняется локально в браузере.
