## 什麼是 Unicode 轉義？

Unicode 轉義是將字元轉換為代表其 Unicode 碼位的編碼序列。當原始碼、設定檔或資料格式無法直接包含某些字元時，這項功能不可或缺。

**常見的轉義格式：**

- `\uXXXX` — JavaScript / JSON，大多數程式語言皆採用
- `\u{XXXXX}` — ES6+ JavaScript，無需代理對即可支援增補字元
- `&#xXXXX;` / `&#DDDD;` — HTML 實體，十六進位或十進位形式
- `U+XXXX` — 文件中使用的標準 Unicode 表示法
- `\xXX` / `%XX` — UTF-8 位元組級編碼，常見於 URL 和 C 系語言
- `\UXXXXXXXX` — Python 8 位格式，可表示任意碼位
- `0xXXXX` — 十六進位字面值表示法

## 何時使用此工具

- 在要求 ASCII 安全編碼的原始碼或設定檔中嵌入非 ASCII 字元
- 透過檢查底層 Unicode 碼位來偵錯亂碼文字
- 在不同語言或格式之間移植時，轉換不同的轉義表示法
- 為需要實體編碼字元的 JSON、HTML 或 URL 情境準備文字

## 運作方式

在左側輸入或貼上純文字，工具會使用所選格式將非 ASCII 字元轉義。在右側貼上轉義文字，工具會自動偵測並同時解碼所有支援的格式。所有處理皆在瀏覽器中本機完成。
