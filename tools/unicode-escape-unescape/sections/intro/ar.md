## ما هو تهريب يونيكود؟

تهريب يونيكود يحوّل الأحرف إلى تسلسلات مُرمَّزة تمثّل نقاط الشفرة الخاصة بها في يونيكود. هذا ضروري عندما لا يمكن للشفرة المصدرية أو ملفات الإعدادات أو صيغ البيانات أن تحتوي على أحرف معيّنة مباشرة.

**صيغ التهريب الشائعة:**

- `\uXXXX` — JavaScript / JSON، تُستخدم في معظم لغات البرمجة
- `\u{XXXXX}` — ES6+‎ JavaScript، تدعم الأحرف التكميلية دون أزواج بديلة
- `&#xXXXX;` / `&#DDDD;` — كيانات HTML بالصيغة الست عشرية أو العشرية
- `U+XXXX` — ترميز يونيكود القياسي المُستخدم في التوثيق
- `\xXX` / `%XX` — ترميز على مستوى بايتات UTF-8، شائع في عناوين URL واللغات الشبيهة بـ C
- `\UXXXXXXXX` — صيغة Python ذات 8 أرقام لأي نقطة شفرة
- `0xXXXX` — ترميز حرفي ست عشري

## متى تستخدم هذه الأداة

- تضمين أحرف غير ASCII في الشفرة المصدرية أو ملفات الإعدادات التي تتطلب ترميزًا آمنًا بنظام ASCII
- تصحيح النصوص المشوّهة عبر فحص نقاط شفرة يونيكود الأساسية
- التحويل بين صيغ التهريب المختلفة عند النقل بين اللغات أو الصيغ
- تحضير النصوص لسياقات JSON أو HTML أو URL التي تحتاج إلى أحرف مُرمَّزة بكيانات

## كيف تعمل

اكتب أو الصق نصًّا عاديًّا على اليسار وستقوم الأداة بتهريب الأحرف غير ASCII باستخدام الصيغة المحددة. الصق نصًّا مُهرَّبًا على اليمين وستكتشف الأداة تلقائيًّا جميع الصيغ المدعومة وتفكّ ترميزها في آنٍ واحد. كل شيء يعمل محليًّا في المتصفح.
