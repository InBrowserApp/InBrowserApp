## 유니코드 이스케이프란?

유니코드 이스케이프는 문자를 유니코드 코드 포인트를 나타내는 인코딩된 시퀀스로 변환합니다. 소스 코드, 설정 파일 또는 데이터 형식에 특정 문자를 직접 포함할 수 없을 때 필수적입니다.

**주요 이스케이프 형식:**

- `\uXXXX` — JavaScript / JSON, 대부분의 프로그래밍 언어에서 사용
- `\u{XXXXX}` — ES6+ JavaScript, 서로게이트 쌍 없이 보충 문자 지원
- `&#xXXXX;` / `&#DDDD;` — 16진수 또는 10진수 형태의 HTML 엔티티
- `U+XXXX` — 문서에서 사용되는 표준 유니코드 표기법
- `\xXX` / `%XX` — UTF-8 바이트 수준 인코딩, URL 및 C 계열 언어에서 일반적
- `\UXXXXXXXX` — 모든 코드 포인트를 위한 Python 8자리 형식
- `0xXXXX` — 16진수 리터럴 표기법

## 이 도구를 사용하는 경우

- ASCII 안전 인코딩이 필요한 소스 코드나 설정 파일에 비ASCII 문자를 삽입할 때
- 깨진 텍스트의 유니코드 코드 포인트를 검사하여 디버깅할 때
- 언어나 형식 간 이식 시 서로 다른 이스케이프 표기법을 변환할 때
- JSON, HTML 또는 URL 컨텍스트에 엔티티 인코딩된 문자가 필요한 텍스트를 준비할 때

## 작동 방식

왼쪽에 일반 텍스트를 입력하거나 붙여넣으면 선택한 형식으로 비ASCII 문자가 이스케이프됩니다. 오른쪽에 이스케이프된 텍스트를 붙여넣으면 지원되는 모든 형식을 자동으로 감지하여 동시에 디코딩합니다. 모든 처리는 브라우저에서 로컬로 실행됩니다.
