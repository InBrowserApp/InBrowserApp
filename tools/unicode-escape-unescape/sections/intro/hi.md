## यूनिकोड एस्केपिंग क्या है?

यूनिकोड एस्केपिंग वर्णों को एन्कोडेड अनुक्रमों में बदलती है जो उनके यूनिकोड कोड पॉइंट्स का प्रतिनिधित्व करते हैं। यह तब आवश्यक होता है जब सोर्स कोड, कॉन्फ़िगरेशन फ़ाइलें या डेटा प्रारूप कुछ वर्णों को सीधे शामिल नहीं कर सकते।

**सामान्य एस्केप प्रारूप:**

- `\uXXXX` — JavaScript / JSON, अधिकांश प्रोग्रामिंग भाषाओं में उपयोग किया जाता है
- `\u{XXXXX}` — ES6+ JavaScript, सरोगेट जोड़ों के बिना पूरक वर्णों का समर्थन करता है
- `&#xXXXX;` / `&#DDDD;` — हेक्साडेसिमल या दशमलव रूप में HTML एंटिटी
- `U+XXXX` — दस्तावेज़ीकरण में उपयोग किया जाने वाला मानक यूनिकोड नोटेशन
- `\xXX` / `%XX` — UTF-8 बाइट-स्तरीय एन्कोडिंग, URL और C-जैसी भाषाओं में सामान्य
- `\UXXXXXXXX` — किसी भी कोड पॉइंट के लिए Python 8-अंकीय प्रारूप
- `0xXXXX` — हेक्स लिटरल नोटेशन

## इस टूल का उपयोग कब करें

- सोर्स कोड या कॉन्फ़िग फ़ाइलों में गैर-ASCII वर्ण एम्बेड करना जिन्हें ASCII-सुरक्षित एन्कोडिंग की आवश्यकता होती है
- अंतर्निहित यूनिकोड कोड पॉइंट्स की जांच करके विकृत पाठ को डीबग करना
- भाषाओं या प्रारूपों के बीच पोर्ट करते समय विभिन्न एस्केप नोटेशन के बीच रूपांतरण करना
- JSON, HTML या URL संदर्भों के लिए पाठ तैयार करना जिन्हें एंटिटी-एन्कोडेड वर्णों की आवश्यकता होती है

## यह कैसे काम करता है

बाईं ओर सादा पाठ टाइप करें या पेस्ट करें और टूल चयनित प्रारूप का उपयोग करके गैर-ASCII वर्णों को एस्केप करेगा। दाईं ओर एस्केप किया गया पाठ पेस्ट करें और यह सभी समर्थित प्रारूपों को स्वतः पहचान कर एक साथ डिकोड करेगा। सब कुछ ब्राउज़र में स्थानीय रूप से चलता है।
