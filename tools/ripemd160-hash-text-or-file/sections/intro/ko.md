## RIPEMD-160이란?

RIPEMD-160(RACE Integrity Primitives Evaluation Message Digest)은 160비트(20바이트) 해시 값을 생성하는 암호화 해시 함수로, 일반적으로 40자리 16진수로 표현됩니다. 1996년 Hans Dobbertin, Antoon Bosselaers, Bart Preneel이 유럽 RACE 프로젝트의 일환으로 개발했습니다.

**주요 특징:**

- **결정론적**: 동일한 입력은 항상 동일한 해시를 생성
- **빠른 계산**: 주어진 입력에 대해 합리적으로 빠르게 계산
- **눈사태 효과**: 입력의 작은 변화가 완전히 다른 출력을 생성
- **고정 출력 크기**: 입력 크기에 관계없이 항상 160비트 해시를 생성
- **2라인 병렬 구조**: 보안 강화를 위해 두 개의 병렬 계산 라인 사용

**보안 상태:**
✅ **RIPEMD-160은 암호학적으로 안전한 것으로 간주됩니다**. 알려진 실용적인 공격이 없습니다. 좋은 보안 마진을 제공하며 160비트 해시가 충분한 암호화 애플리케이션에 여전히 권장됩니다.

**일반적인 용도:**

- 비트코인 주소 생성(Base58Check 인코딩)
- 디지털 서명 및 인증서
- 데이터 무결성 검증
- 160비트 해시가 필요한 암호화 프로토콜
- 필요할 때 SHA-1의 대안

**다른 알고리즘과의 비교:**

- MD5와 SHA-1보다 안전
- SHA-256보다 작은 출력(160비트 vs 256비트)
- 좋은 성능 특성
- 암호화 커뮤니티에서 잘 연구되고 신뢰받음

**권장 사용처:**

- 160비트 해시 보안이 필요한 애플리케이션
- 비트코인 관련 암호화 작업
- RIPEMD-160이 지정된 레거시 시스템 호환성
