## RIPEMD-160とは？

RIPEMD-160（RACE Integrity Primitives Evaluation Message Digest）は、160ビット（20バイト）のハッシュ値を生成する暗号学的ハッシュ関数で、通常40文字の16進数として表示されます。1996年にHans Dobbertin、Antoon Bosselaers、Bart Preneelによって欧州RACEプロジェクトの一部として開発されました。

**主な特徴：**

- **決定論的**：同じ入力は常に同じハッシュを生成
- **高速計算**：任意の入力に対して合理的に高速に計算
- **雪崩効果**：入力の小さな変化が劇的に異なる出力を生成
- **固定出力サイズ**：入力サイズに関係なく常に160ビットハッシュを生成
- **2ライン並列構造**：セキュリティ強化のために2つの並列計算ラインを使用

**セキュリティ状況：**
✅ **RIPEMD-160は暗号学的に安全とされています**。既知の実用的な攻撃はありません。良好なセキュリティマージンを提供し、160ビットハッシュで十分な暗号学的アプリケーションに引き続き推奨されています。

**一般的な用途：**

- ビットコインアドレス生成（Base58Checkエンコーディング）
- デジタル署名と証明書
- データ整合性検証
- 160ビットハッシュを必要とする暗号プロトコル
- 必要に応じたSHA-1の代替

**他のアルゴリズムとの比較：**

- MD5やSHA-1より安全
- SHA-256より小さい出力（160ビット vs 256ビット）
- 良好な性能特性
- 暗号学コミュニティで十分に研究され信頼されている

**推奨される用途：**

- 160ビットハッシュセキュリティを必要とするアプリケーション
- ビットコイン関連の暗号操作
- RIPEMD-160が指定されているレガシーシステムとの互換性
