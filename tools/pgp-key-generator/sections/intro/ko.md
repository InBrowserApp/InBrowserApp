# PGP 키 생성기

이 도구를 사용하면 브라우저에서 직접 OpenPGP 키 쌍을 만들 수 있습니다. ASCII 아머 형식의 공개 키, 개인 키, 해지 인증서, 키 ID, 지문을 생성하므로 키 자료를 서버로 보내지 않고도 암호화 메일, 파일 암호화, 릴리스 서명 또는 계정 복구 워크플로를 설정할 수 있습니다.

## 사용 시점

PGP 키는 비대칭 암호화가 필요할 때 유용합니다. 다른 사람은 공개 키를 사용해 내게 보낼 데이터를 암호화하거나 서명을 검증하고, 개인 키는 데이터를 복호화하고 서명을 만드는 데 사용됩니다. 브라우저 기반 생성기는 짧은 설정 세션, 데모 또는 결과가 즉시 필요한 로컬 워크플로에 편리합니다.

## 키 쌍 생성 방법

이름, 이메일 또는 둘 다 입력하여 키에 알아보기 쉬운 사용자 ID를 부여하세요. 업무, 프로젝트 또는 릴리스 서명용 키를 구분하고 싶다면 선택적 댓글을 추가하세요. 최신 OpenPGP 소프트웨어에는 ECC를 선택하고, 오래된 도구와의 호환성이 필요할 때는 RSA를 선택하세요. 암호구는 선택 사항이지만 보관하려는 개인 키에는 강력히 권장됩니다.

## 키 유형과 만료

ECC는 Curve25519를 사용하며 compact하고 빠르기 때문에 기본값입니다. RSA는 호환성을 위해 2048, 3072, 4096비트로 제공됩니다. 만료는 일 단위로 설정됩니다. 0은 적극적으로 관리하고 해지할 수 있는 키에만 사용하세요. 만료 기간이 짧을수록 장기적인 위험을 줄이고 키 교체 습관을 만들기 쉽습니다.

## 개인 키를 안전하게 다루기

공개 키, 개인 키, 해지 인증서를 각각 별도의 파일로 다운로드하세요. 개인 키는 암호화된 비밀번호 관리자 또는 안전한 오프라인 저장소에 백업하고, 개인 키를 분실하거나 노출했을 때 키를 폐기할 수 있도록 해지 인증서는 별도 위치에 보관하세요. 공개 키를 게시하기 전에 신뢰할 수 있는 채널을 통해 지문을 비교하세요.
