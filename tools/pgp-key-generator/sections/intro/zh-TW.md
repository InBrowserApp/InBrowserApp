# PGP 金鑰產生器

使用此工具可直接在瀏覽器中建立 OpenPGP 金鑰對。它會產生 ASCII-armored 公開金鑰、私密金鑰、撤銷憑證、金鑰 ID 和指紋，讓你可以設定加密郵件、檔案加密、版本簽署或帳號復原流程，而不必把金鑰資料傳送到伺服器。

## 何時使用

當你需要非對稱密碼學時，PGP 金鑰很有用：其他人使用你的公開金鑰加密資料給你或驗證簽章，而你的私密金鑰則用來解密資料和建立簽章。瀏覽器型產生器適合短時間的設定工作、示範，或想要立即取得結果的本機流程。

## 如何產生金鑰對

輸入名稱、電子郵件或兩者，讓金鑰具備可辨識的使用者 ID。如果你想區分工作、專案或版本簽署金鑰，也可以加入選用註解。現代 OpenPGP 軟體請選擇 ECC；需要相容舊工具時則選擇 RSA。密碼片語是選用項目，但如果你打算保留任何私鑰，強烈建議設定。

## 金鑰類型與到期時間

ECC 使用 Curve25519，因為它精簡且快速，所以是預設選項。RSA 提供 2048、3072 和 4096 位元以支援相容性。到期時間以天為單位；只有在你會主動管理且能撤銷金鑰時，才使用 0。較短的到期期間可降低長期風險，也更容易養成輪換習慣。

## 安全處理私鑰

請將公開金鑰、私密金鑰和撤銷憑證分別下載成獨立檔案。將私鑰備份在加密的密碼管理器或安全的離線儲存空間，並把撤銷憑證存放在不同位置，以便在私鑰遺失或外洩時停用該金鑰。發布公開金鑰前，請透過可信任的管道比對指紋。
