# מחולל מפתחות PGP

השתמשו בכלי הזה כדי ליצור זוג מפתחות OpenPGP ישירות בדפדפן שלכם. הוא מפיק מפתח ציבורי משוריין, מפתח פרטי, תעודת ביטול, מזהה מפתח וטביעת אצבע, כדי שתוכלו להגדיר תהליכי עבודה לדואר מוצפן, הצפנת קבצים, חתימת גרסאות או שחזור חשבון בלי לשלוח את חומר המפתח לשרת.

## מתי להשתמש בו

מפתחות PGP שימושיים כשצריך הצפנה אסימטרית: אנשים אחרים משתמשים במפתח הציבורי שלכם כדי להצפין עבורכם נתונים או לאמת חתימות, בעוד שהמפתח הפרטי שלכם מפענח נתונים ויוצר חתימות. מחולל מבוסס דפדפן נוח להפעלות הגדרה קצרות, הדגמות או תהליכי עבודה מקומיים שבהם רוצים את התוצאה מיד.

## איך ליצור זוג מפתחות

הזינו שם, אימייל או את שניהם כדי שלמפתח יהיה מזהה משתמש ברור. הוסיפו הערה אופציונלית אם אתם רוצים להפריד בין מפתחות לעבודה, לפרויקט או לחתימת גרסאות. בחרו ECC לתוכנות OpenPGP מודרניות, או RSA כשנדרשת תאימות לכלים ישנים יותר. ביטוי סיסמה הוא אופציונלי, אבל מומלץ מאוד לכל מפתח פרטי שאתם מתכוונים לשמור.

## סוגי מפתחות ותפוגה

ECC משתמש ב-Curve25519 והוא ברירת המחדל כי הוא קומפקטי ומהיר. RSA זמין ב-2048, 3072 ו-4096 ביטים לצורך תאימות. התפוגה מוגדרת בימים; השתמשו ב-0 רק למפתחות שאתם מנהלים באופן פעיל ויכולים לבטל. תקופות תפוגה קצרות יותר מפחיתות סיכון לטווח ארוך ומקלות על הרגלי רוטציה.

## טיפול בטוח במפתחות פרטיים

הורידו את המפתח הציבורי, המפתח הפרטי ותעודת הביטול כקבצים נפרדים. גבו את המפתח הפרטי במנהל סיסמאות מוצפן או באחסון לא מקוון מאובטח, ושמרו את תעודת הביטול במקום נפרד כדי שתוכלו להוציא את המפתח משימוש אם המפתח הפרטי יאבד או ייחשף. לפני פרסום מפתח ציבורי, השוו את טביעת האצבע דרך ערוץ מהימן.
