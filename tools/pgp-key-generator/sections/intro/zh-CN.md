# PGP 密钥生成器

使用此工具可直接在浏览器中创建 OpenPGP 密钥对。它会生成 ASCII 装甲格式的公钥、私钥、吊销证书、密钥 ID 和指纹，帮助你设置加密邮件、文件加密、发布签名或账号恢复流程，而无需将密钥材料发送到服务器。

## 何时使用

当你需要非对称加密时，PGP 密钥很有用：其他人使用你的公钥为你加密数据或验证签名，而你的私钥用于解密数据和创建签名。基于浏览器的生成器适合短时间设置、演示，或需要立即得到结果的本地工作流。

## 如何生成密钥对

输入姓名、电子邮件或两者都填，让密钥拥有可识别的用户 ID。如果你想区分工作、项目或发布签名密钥，可以添加可选注释。现代 OpenPGP 软件建议选择 ECC；如果需要兼容旧工具，则选择 RSA。密码短语是可选的，但如果你打算保留私钥，强烈建议设置。

## 密钥类型和过期时间

ECC 使用 Curve25519，是默认选择，因为它紧凑且速度快。RSA 提供 2048、3072 和 4096 位选项以满足兼容性需求。过期时间以天为单位；仅当你会主动管理并能够吊销密钥时，才使用 0。较短的过期周期可以降低长期风险，也更容易养成轮换习惯。

## 安全处理私钥

将公钥、私钥和吊销证书分别下载为独立文件。把私钥备份到加密密码管理器或安全的离线存储中，并将吊销证书保存在单独位置，以便在私钥丢失或暴露时停用该密钥。发布公钥前，请通过可信渠道核对指纹。
