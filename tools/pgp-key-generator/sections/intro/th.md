# ตัวสร้างคีย์ PGP

ใช้เครื่องมือนี้เพื่อสร้างคู่คีย์ OpenPGP โดยตรงในเบราว์เซอร์ของคุณ เครื่องมือนี้สร้างคีย์สาธารณะแบบ armor, คีย์ส่วนตัว, ใบรับรองการเพิกถอน, ID คีย์ และ fingerprint เพื่อให้คุณตั้งค่าเวิร์กโฟลว์สำหรับอีเมลเข้ารหัส การเข้ารหัสไฟล์ การลงนามรีลีส หรือการกู้คืนบัญชีได้โดยไม่ต้องส่งข้อมูลคีย์ไปยังเซิร์ฟเวอร์

## ควรใช้เมื่อใด

คีย์ PGP มีประโยชน์เมื่อคุณต้องใช้การเข้ารหัสแบบไม่สมมาตร: ผู้อื่นใช้คีย์สาธารณะของคุณเพื่อเข้ารหัสข้อมูลให้คุณหรือตรวจสอบลายเซ็น ส่วนคีย์ส่วนตัวของคุณใช้ถอดรหัสข้อมูลและสร้างลายเซ็น ตัวสร้างบนเบราว์เซอร์สะดวกสำหรับการตั้งค่าระยะสั้น การสาธิต หรือเวิร์กโฟลว์ในเครื่องที่ต้องการผลลัพธ์ทันที

## วิธีสร้างคู่คีย์

ป้อนชื่อ อีเมล หรือทั้งสองอย่าง เพื่อให้คีย์มี ID ผู้ใช้ที่จดจำได้ เพิ่มความคิดเห็นแบบเลือกได้หากต้องการแยกคีย์สำหรับงาน โปรเจกต์ หรือการลงนามรีลีส เลือก ECC สำหรับซอฟต์แวร์ OpenPGP สมัยใหม่ หรือเลือก RSA เมื่อต้องการความเข้ากันได้กับเครื่องมือรุ่นเก่า Passphrase เป็นตัวเลือก แต่ขอแนะนำอย่างยิ่งสำหรับคีย์ส่วนตัวใดก็ตามที่คุณตั้งใจจะเก็บไว้

## ประเภทคีย์และการหมดอายุ

ECC ใช้ Curve25519 และเป็นค่าเริ่มต้นเพราะมีขนาดกะทัดรัดและรวดเร็ว RSA มีให้เลือกที่ 2048, 3072 และ 4096 บิตเพื่อความเข้ากันได้ การหมดอายุกำหนดเป็นจำนวนวัน ใช้ 0 เฉพาะกับคีย์ที่คุณดูแลอย่างจริงจังและสามารถเพิกถอนได้ ระยะเวลาหมดอายุที่สั้นกว่าจะลดความเสี่ยงระยะยาวและช่วยให้การหมุนเวียนคีย์ทำได้ง่ายขึ้น

## การจัดการคีย์ส่วนตัวอย่างปลอดภัย

ดาวน์โหลดคีย์สาธารณะ คีย์ส่วนตัว และใบรับรองการเพิกถอนเป็นไฟล์แยกกัน สำรองคีย์ส่วนตัวไว้ในตัวจัดการรหัสผ่านที่เข้ารหัสหรือพื้นที่จัดเก็บออฟไลน์ที่ปลอดภัย และเก็บใบรับรองการเพิกถอนไว้แยกต่างหาก เพื่อให้คุณเลิกใช้งานคีย์ได้หากคีย์ส่วนตัวสูญหายหรือถูกเปิดเผย ก่อนเผยแพร่คีย์สาธารณะ ให้เปรียบเทียบ fingerprint ผ่านช่องทางที่เชื่อถือได้
