## Apa yang dilakukan alat ini

Image Metadata Cleaner menghapus blok metadata umum dari file JPEG, PNG,
dan WebP langsung di browser. Alat ini membersihkan data EXIF, XMP,
IPTC, komentar, dan profil ICC yang bisa berisi detail kamera,
informasi lokasi, riwayat edit, atau metadata penulis, sambil tetap
menjaga format file aslinya.

## Kasus penggunaan yang cocok

- Membagikan foto atau tangkapan layar tanpa data lokasi atau perangkat.
- Membersihkan aset hasil ekspor sebelum dikirim ke klien, formulir,
  atau issue tracker.
- Tetap memakai alur kerja lokal di browser saat Anda hanya perlu
  menghapus metadata, bukan memakai editor penuh.

## Hal yang tidak berubah

- Alat ini tidak mengubah ukuran, memotong, memutar, atau mengedit isi
  gambar secara langsung.
- Alat ini tidak mengubah format; JPEG tetap JPEG, PNG tetap PNG, dan
  WebP tetap WebP.
- Alat ini hanya menghapus blok metadata yang didukung, jadi gunakan
  editor khusus jika Anda memerlukan pembersihan yang lebih luas atau
  pemrosesan batch.
