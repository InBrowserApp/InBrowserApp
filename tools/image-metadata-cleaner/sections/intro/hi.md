## यह टूल क्या करता है

Image Metadata Cleaner ब्राउज़र में सीधे JPEG, PNG और WebP फ़ाइलों से
सामान्य मेटाडेटा ब्लॉक हटाता है। यह EXIF, XMP, IPTC, टिप्पणियाँ और
ICC प्रोफ़ाइल डेटा साफ करता है, जिनमें कैमरा विवरण, स्थान जानकारी,
एडिट इतिहास या लेखक संबंधी मेटाडेटा हो सकता है, और साथ ही मूल फ़ाइल
फ़ॉर्मेट को बनाए रखता है।

## किन स्थितियों में उपयोगी है

- फ़ोटो या स्क्रीनशॉट साझा करने से पहले स्थान और डिवाइस जैसी जानकारी
  हटानी हो।
- क्लाइंट, फ़ॉर्म या issue tracker को भेजने से पहले exported images को
  साफ करना हो।
- जब केवल metadata removal चाहिए और पूरा image editor नहीं चाहिए।

## यह क्या नहीं बदलता

- यह छवि का आकार नहीं बदलता, crop नहीं करता, rotate नहीं करता और
  चित्र की सामग्री को सीधे edit नहीं करता।
- यह फ़ॉर्मेट नहीं बदलता; JPEG, JPEG ही रहता है, PNG, PNG ही रहता है,
  और WebP, WebP ही रहता है।
- यह सिर्फ supported metadata blocks हटाता है, इसलिए broader cleanup
  या batch processing के लिए dedicated editor बेहतर रहेगा।
