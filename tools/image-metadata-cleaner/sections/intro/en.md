## What this tool does

Image Metadata Cleaner removes common metadata blocks from JPEG, PNG,
and WebP files directly in your browser. It strips EXIF, XMP, IPTC,
comment, and ICC profile data that can contain camera details, location
info, editing history, or authoring metadata, while keeping the
original file format.

## Good use cases

- Sharing photos or screenshots without attached location or device
  details.
- Cleaning exported assets before sending them to clients, forms, or
  issue trackers.
- Keeping a browser-local workflow when you only need metadata removal,
  not a full editor.

## What it does not change

- It does not resize, crop, rotate, or otherwise edit the picture
  itself.
- It does not convert between formats; JPEG stays JPEG, PNG stays PNG,
  and WebP stays WebP.
- It only removes supported metadata blocks, so use a dedicated editor
  if you need broader cleanup or batch processing.
