## เครื่องมือนี้ทำอะไร

Image Metadata Cleaner จะลบบล็อกเมตาดาตาทั่วไปออกจากไฟล์ JPEG, PNG
และ WebP ได้โดยตรงในเบราว์เซอร์ เครื่องมือนี้จะลบข้อมูล EXIF,
XMP, IPTC, ความเห็น และโปรไฟล์ ICC ที่อาจมีรายละเอียดกล้อง ข้อมูล
ตำแหน่ง ประวัติการแก้ไข หรือข้อมูลผู้สร้าง โดยยังคงรูปแบบไฟล์เดิมไว้

## เหมาะกับการใช้งานแบบไหน

- แชร์รูปภาพหรือภาพหน้าจอโดยไม่แนบข้อมูลตำแหน่งหรือข้อมูลอุปกรณ์
- ล้างไฟล์ที่ export แล้วก่อนส่งให้ลูกค้า แบบฟอร์ม หรือ issue
  tracker
- ต้องการ workflow แบบ local ในเบราว์เซอร์ เมื่อคุณต้องการแค่ลบ
  เมตาดาตา ไม่ใช่ใช้ตัวแก้ไขภาพแบบเต็ม

## สิ่งที่เครื่องมือนี้ไม่เปลี่ยน

- เครื่องมือนี้ไม่ปรับขนาด ครอบ หมุน หรือแก้ไขตัวภาพโดยตรง
- เครื่องมือนี้ไม่แปลงฟอร์แมต; JPEG ยังเป็น JPEG, PNG ยังเป็น PNG
  และ WebP ยังเป็น WebP
- เครื่องมือนี้ลบเฉพาะบล็อกเมตาดาตาที่รองรับเท่านั้น ถ้าต้องการล้าง
  ข้อมูลให้กว้างกว่านี้หรือทำงานแบบเป็นชุด ควรใช้โปรแกรมเฉพาะทาง
