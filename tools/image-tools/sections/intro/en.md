## What this collection covers

Image Tools gathers the image utilities that are already available in the
Astro rewrite. Use it as a focused directory when you know you need to work
with an image file but are not sure whether the next step is conversion,
compression, inspection, generation, or code reading.

## When to use it

Start here when a workflow involves pictures, screenshots, icons, SVG artwork,
barcodes, QR codes, color palettes, or image metadata. The page helps you move
from a broad intent, such as making an image smaller or changing its format, to
the specific tool that performs that task.

## How the directory works

The cards are populated from the current tool registry for the image category.
Search looks across tool names, descriptions, slugs, and tags. Workflow filters
group tools by the job they support: converting formats, optimizing files,
inspecting details, creating new image assets, or scanning codes from images.

## Privacy and limitations

The linked tools are designed for local browser workflows where possible, so
files stay in your current tab instead of being uploaded to a remote service.
This page is a directory rather than an editor: choose a tool card to open the
actual converter, optimizer, reader, or generator. New image tools will appear
after they are migrated and registered in the Astro app.
