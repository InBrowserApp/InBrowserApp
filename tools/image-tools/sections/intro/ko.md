## 이 컬렉션에서 다루는 내용

이미지 도구는 Astro 재작성 버전에서 이미 사용할 수 있는 이미지 유틸리티를
모아 둡니다. 이미지 파일을 작업해야 한다는 것은 알지만 다음 단계가
변환, 압축, 검사, 생성 또는 코드 판독 중 무엇인지 확실하지 않을 때
집중된 디렉터리로 사용하세요.

## 언제 사용하나요

사진, 스크린샷, 아이콘, SVG 아트워크, 바코드, QR 코드, 색상 팔레트
또는 이미지 메타데이터가 포함된 워크플로라면 여기에서 시작하세요.
이 페이지는 이미지를 더 작게 만들거나 형식을 바꾸는 것처럼 넓은
의도를 해당 작업을 수행하는 구체적인 도구로 이어 줍니다.

## 디렉터리 작동 방식

카드는 이미지 카테고리의 현재 도구 레지스트리에서 채워집니다. 검색은
도구 이름, 설명, 슬러그, 태그 전반을 살펴봅니다. 워크플로 필터는
지원하는 작업별로 도구를 묶습니다. 형식 변환, 파일 최적화, 세부 정보
검사, 새 이미지 자산 생성 또는 이미지에서 코드 스캔입니다.

## 개인정보 보호 및 제한 사항

연결된 도구는 가능한 경우 로컬 브라우저 워크플로용으로 설계되어
파일이 원격 서비스에 업로드되지 않고 현재 탭 안에 머뭅니다. 이
페이지는 편집기가 아니라 디렉터리입니다. 실제 변환기, 최적화 도구,
리더 또는 생성기를 열려면 도구 카드를 선택하세요. 새 이미지 도구는
Astro 앱으로 마이그레이션되고 등록된 후 표시됩니다.
