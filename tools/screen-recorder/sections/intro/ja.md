# 画面録画

ブラウザーで選択した画面、ウィンドウ、またはタブを、動画をサーバーへアップロードせずに録画できます。このツールはブラウザーの Screen Capture API と MediaRecorder API を使用するため、ダウンロードするまで録画データはローカルに保持されます。

## 使いどころ

短いデモ、バグ報告、操作手順の共有、QA メモ、または軽量なブラウザーワークフローで十分な社内向けの簡単な動画に使えます。録画を開始する前に、タブ音声またはシステム音声を含めるようブラウザーに要求し、必要に応じてマイク音声もミックスできます。

## プライバシーとブラウザー対応

利用できるキャプチャ元と音声オプションはブラウザーによって決まります。ブラウザーによっては現在のタブの音声しか共有できなかったり、HTTPS が必要だったり、一時停止や録画そのものに対応していなかったりします。権限が拒否された場合、ストリームは保持されず、別の設定で再試行できます。

## 安定した録画のためのヒント

開始する前に関係のないキャプチャセッションを閉じ、必要最小限のキャプチャ元を選び、音声が重要な場合は短いテストを行ってください。録画は現在のページセッション内にのみ保持されるため、クリアする前に結果をダウンロードしてください。
