# Запись экрана

Записывайте выбранный в браузере экран, окно или вкладку без загрузки видео на
сервер. Инструмент использует API Screen Capture и MediaRecorder вашего
браузера, поэтому запись остается локальной, пока вы ее не скачаете.

## Когда это использовать

Используйте рекордер для коротких демонстраций, отчетов об ошибках, пошаговых
обзоров, заметок QA или быстрых внутренних видео, когда достаточно легкого
браузерного процесса. Перед началом записи можно попросить браузер включить
звук вкладки или системы, а также при необходимости добавить микрофон.

## Конфиденциальность и поддержка браузеров

Браузер сам определяет, какие источники захвата и параметры звука доступны.
Некоторые браузеры передают звук только из текущей вкладки, некоторым требуется
HTTPS, а некоторые вообще не поддерживают паузу или запись. Если доступ
отклонен, поток не сохраняется, и вы можете повторить попытку с другими
настройками.

## Советы для надежной записи

Перед началом закройте посторонние сеансы захвата, выберите минимальный
подходящий источник и сделайте короткую пробу, если звук важен. Скачайте
результат перед очисткой, потому что записи хранятся только в текущем сеансе
страницы.
