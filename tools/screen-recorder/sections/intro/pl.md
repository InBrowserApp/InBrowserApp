# Rejestrator ekranu

Nagrywaj wybrany w przeglądarce ekran, okno lub kartę bez przesyłania wideo na
serwer. Narzędzie korzysta z interfejsów API Screen Capture i MediaRecorder w
Twojej przeglądarce, więc nagranie pozostaje lokalne, dopóki go nie pobierzesz.

## Kiedy go używać

Użyj rejestratora do krótkich demonstracji, zgłoszeń błędów, instruktaży, notatek
QA lub szybkich filmów wewnętrznych, gdy wystarczy lekki przepływ pracy w
przeglądarce. Przed rozpoczęciem nagrywania możesz poprosić przeglądarkę o
dołączenie dźwięku z karty lub systemu oraz opcjonalnie domiksować mikrofon.

## Prywatność i obsługa w przeglądarkach

To przeglądarka decyduje, które źródła przechwytywania i opcje dźwięku są
dostępne. Niektóre przeglądarki udostępniają dźwięk tylko z bieżącej karty,
niektóre wymagają HTTPS, a inne w ogóle nie obsługują wstrzymywania ani
nagrywania. Jeśli uprawnienie zostanie odrzucone, żaden strumień nie zostanie
zachowany i możesz spróbować ponownie z innymi ustawieniami.

## Wskazówki dotyczące niezawodnych nagrań

Zamknij niepowiązane sesje przechwytywania przed rozpoczęciem, wybierz najmniejsze
przydatne źródło i wykonaj krótki test, gdy dźwięk ma znaczenie. Pobierz wynik
przed jego wyczyszczeniem, ponieważ nagrania są przechowywane tylko w bieżącej
sesji strony.
