# เครื่องบันทึกหน้าจอ

บันทึกหน้าจอ หน้าต่าง หรือแท็บที่เลือกผ่านเบราว์เซอร์ โดยไม่ต้องอัปโหลดวิดีโอ
ไปยังเซิร์ฟเวอร์ เครื่องมือนี้ใช้ Screen Capture และ MediaRecorder APIs ของ
เบราว์เซอร์ ดังนั้นการบันทึกจะอยู่ในเครื่องจนกว่าคุณจะดาวน์โหลด

## ควรใช้เมื่อใด

ใช้เครื่องบันทึกนี้สำหรับเดโมสั้น ๆ รายงานบั๊ก คำแนะนำแบบทีละขั้นตอน
บันทึก QA หรือวิดีโอภายในแบบรวดเร็ว เมื่อเวิร์กโฟลว์เบราว์เซอร์ที่เบาและง่าย
เพียงพอต่อการใช้งาน คุณสามารถขอให้เบราว์เซอร์รวมเสียงจากแท็บหรือเสียงระบบ
และเลือกผสมเสียงไมโครโฟนก่อนเริ่มบันทึกได้

## ความเป็นส่วนตัวและการรองรับของเบราว์เซอร์

เบราว์เซอร์เป็นผู้กำหนดว่าแหล่งจับภาพและตัวเลือกเสียงใดพร้อมใช้งาน
บางเบราว์เซอร์แชร์เสียงได้เฉพาะแท็บปัจจุบัน บางเบราว์เซอร์ต้องใช้ HTTPS
และบางเบราว์เซอร์อาจไม่รองรับการหยุดชั่วคราวหรือการบันทึกเลย
หากไม่ได้รับอนุญาต ระบบจะไม่เก็บสตรีมไว้ และคุณสามารถลองใหม่ด้วยการตั้งค่าอื่น

## เคล็ดลับสำหรับการบันทึกที่เชื่อถือได้

ปิดเซสชันการจับภาพอื่นที่ไม่เกี่ยวข้องก่อนเริ่ม เลือกแหล่งที่เล็กที่สุด
แต่ยังเพียงพอต่อการใช้งาน และทดสอบสั้น ๆ เมื่อเสียงเป็นสิ่งสำคัญ
ดาวน์โหลดผลลัพธ์ก่อนล้างข้อมูล เพราะการบันทึกจะถูกเก็บไว้เฉพาะในเซสชัน
ของหน้าปัจจุบันเท่านั้น
