## Обзор

JSON Diff Path сравнивает два JSON-документа и превращает каждое структурное изменение в читаемую запись пути с выводом JSONPath и JSON Pointer.

## Когда использовать

Используйте его, когда нужно проверить изменения в API-данных, просмотреть миграции конфигурации или сгенерировать операции RFC 6902 JSON Patch для автоматизации.

## Как это работает

Инструмент разбирает оба JSON-ввода, вычисляет изменения `add`, `remove` и `replace`, а затем позволяет фильтровать эти операции и переключаться между списком путей и выводом JSON Patch в одной и той же панели результатов.
