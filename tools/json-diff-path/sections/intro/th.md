## ภาพรวม

JSON Diff Path เปรียบเทียบเอกสาร JSON สองชุดและแปลงการเปลี่ยนแปลงเชิงโครงสร้างทุกจุด
ให้เป็นรายการ path ที่อ่านง่าย พร้อมเอาต์พุตทั้ง JSONPath และ JSON Pointer

## ควรใช้เมื่อใด

ใช้เมื่อคุณต้องการตรวจสอบการเปลี่ยนแปลงของ API payload, ตรวจดูการย้ายคอนฟิก, หรือสร้าง
การดำเนินการ RFC 6902 JSON Patch สำหรับการทำงานอัตโนมัติ

## วิธีการทำงาน

เครื่องมือนี้จะแยกวิเคราะห์ JSON ทั้งสองฝั่ง คำนวณการเปลี่ยนแปลงแบบ `add`, `remove`,
และ `replace` จากนั้นให้คุณกรองการดำเนินการเหล่านั้นและสลับระหว่างรายการ path กับเอาต์พุต
JSON Patch ได้ในแผงผลลัพธ์เดียวกัน
