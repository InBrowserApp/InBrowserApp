## סקירה כללית

JSON Diff Path משווה בין שני מסמכי JSON והופך כל שינוי מבני לרשומת נתיב ברורה, עם פלט JSONPath ו-JSON Pointer.

## מתי להשתמש בזה

השתמשו בו כשצריך לסקור שינויים ב-payloads של API, לבדוק מעברי תצורה, או ליצור פעולות RFC 6902 JSON Patch לאוטומציה.

## איך זה עובד

הכלי מנתח את שני קלטי ה-JSON, מחשב שינויים מסוג `add`, `remove` ו-`replace`, ואז מאפשר לסנן את הפעולות האלה ולעבור בין רשימת נתיבים לבין פלט JSON Patch באותו אזור תוצאות.
