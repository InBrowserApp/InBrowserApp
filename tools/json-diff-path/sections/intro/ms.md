## Overview

JSON Diff Path membandingkan dua dokumen JSON dan menukar setiap perubahan struktur menjadi rekod laluan yang mudah dibaca, dengan output JSONPath dan JSON Pointer.

## When To Use It

Gunakannya apabila anda perlu menyemak perubahan payload API, meneliti migrasi konfigurasi, atau menjana operasi RFC 6902 JSON Patch untuk automasi.

## How It Works

Alat ini menghurai kedua-dua input JSON, mengira perubahan `add`, `remove`, dan `replace`, kemudian membolehkan anda menapis operasi itu dan bertukar antara senarai laluan serta output JSON Patch dalam panel hasil yang sama.
