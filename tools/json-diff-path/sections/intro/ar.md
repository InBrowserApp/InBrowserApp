## نظرة عامة

JSON Diff Path يقارن مستندَي JSON ويحوّل كل تغيير بنيوي إلى سجل مسار مقروء مع مخرجات JSONPath وJSON Pointer.

## متى يُستخدم

استخدمه عندما تحتاج إلى مراجعة تغييرات حمولة واجهة برمجة التطبيقات، أو فحص عمليات ترحيل الإعدادات، أو إنشاء عمليات RFC 6902 JSON Patch للأتمتة.

## كيف يعمل

تحلل الأداة كلا مدخلي JSON، ثم تحسب تغييرات `add` و`remove` و`replace`، وبعد ذلك تتيح لك تصفية تلك العمليات والتبديل بين قائمة المسارات ومخرجات JSON Patch في لوحة النتائج نفسها.
