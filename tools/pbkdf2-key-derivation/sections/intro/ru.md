## Что такое PBKDF2?

PBKDF2 (Password-Based Key Derivation Function 2) выводит криптографический ключ из пароля с использованием соли и множества итераций. Это замедляет перебор и даёт разные ключи при изменении соли.

**Ключевые моменты:**

- Использует HMAC с выбранным хешем (SHA-1/SHA-256 и т. д.)
- Больше итераций — выше вычислительные затраты
- Длину вывода можно настроить

**Рекомендации:**

- Используйте уникальную случайную соль
- Повышайте число итераций в пределах допустимой производительности
- Для новых систем рассмотрите Argon2 или scrypt
