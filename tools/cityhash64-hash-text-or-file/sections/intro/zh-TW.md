## 什麼是 CityHash64？

CityHash64 是 Google 推出的快速非加密雜湊演算法，會產生 64 位元（8 位元組）的值。當你需要為文字或檔案建立精簡且可重現的指紋，而且速度比加密安全性更重要時，它很實用。

**主要特徵：**

- **快速且確定**：相同的輸入與種子永遠會產生相同的 64 位元雜湊值
- **非加密**：請勿將 CityHash64 用於密碼、簽章、權杖或防篡改完整性檢查
- **支援種子**：留空即可使用標準 CityHash64；需要獨立的帶種子雜湊空間時，可輸入十進位或 `0x` 十六進位種子
- **本機處理**：文字與檔案會在瀏覽器中雜湊；上傳的檔案不會送到伺服器
- **多種編碼**：結果會以十六進位、Base64、十進位與二進位值顯示

**常見用途：**

- 雜湊表與資料結構
- 非安全用途的檔案指紋
- 資料去重與分桶
- 快取鍵與分片鍵
- 已使用 CityHash64 的系統回歸測試資料
- 資料庫索引
