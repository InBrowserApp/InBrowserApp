## CityHash64란 무엇인가요?

CityHash64는 Google에서 만든 빠른 비암호화 해시 알고리즘으로, 64비트(8바이트) 값을 생성합니다. 텍스트나 파일에 대해 작고 결정적인 지문이 필요하고 암호화 보안보다 속도가 더 중요할 때 유용합니다.

**주요 특징:**

- **빠르고 결정적**: 같은 입력과 시드는 항상 같은 64비트 해시를 생성합니다
- **비암호화**: 비밀번호, 서명, 토큰 또는 변조 방지 무결성 검사에는 CityHash64를 사용하지 마세요
- **시드 지원**: 표준 CityHash64를 사용하려면 시드를 비워 두거나, 별도의 시드 해시 공간이 필요할 때 10진수 또는 `0x` 16진수 시드를 입력하세요
- **로컬 처리**: 텍스트와 파일은 브라우저에서 해시 처리되며, 업로드한 파일은 서버로 전송되지 않습니다
- **여러 인코딩**: 결과는 16진수, Base64, 10진수, 2진수 값으로 표시됩니다

**일반적인 용도:**

- 해시 테이블과 데이터 구조
- 보안 목적이 아닌 파일 지문
- 데이터 중복 제거 및 버킷 분류
- 캐시 키와 샤딩 키
- 이미 CityHash64를 사용하는 시스템의 회귀 테스트 픽스처
- 데이터베이스 인덱싱
