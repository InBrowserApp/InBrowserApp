## 什麼是 SHA-512/256（FIPS 180-4）？

SHA-512/256（FIPS 180-4）是 SHA-2 家族的一種加密雜湊函數，輸出 256 位元（32 位元組）的摘要，通常表示為 64 個字元的十六進位字串。它基於 SHA-512，使用不同的初始值並截斷輸出，在保留 SHA-512 內部結構的同時提供更短的摘要。

**主要特徵：**

- **確定性**：相同的輸入總是產生相同的雜湊
- **快速計算**：對於任何給定輸入都能快速計算
- **雪崩效應**：輸入的微小變化會產生截然不同的輸出
- **不可逆**：從雜湊反推原始輸入在計算上是不可行的
- **抗碰撞**：很難找到產生相同雜湊的兩個不同輸入

**常見用途：**

- 數位簽章和憑證
- 完整性校驗與檢查碼
- HMAC 建構
- 檔案驗證與資料流程
- 需要 256 位元雜湊的協定
