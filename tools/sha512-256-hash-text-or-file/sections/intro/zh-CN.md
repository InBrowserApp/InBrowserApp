## 什么是 SHA-512/256（FIPS 180-4）？

SHA-512/256（FIPS 180-4）是 SHA-2 家族的一种加密散列函数，输出 256 位（32 字节）的摘要，通常表示为 64 个字符的十六进制字符串。它基于 SHA-512，使用不同的初始值并截断输出，在保留 SHA-512 内部结构的同时提供更短的摘要。

**主要特征：**

- **确定性**：相同的输入总是产生相同的散列
- **快速计算**：对于任何给定输入都能快速计算
- **雪崩效应**：输入的微小变化会产生截然不同的输出
- **不可逆**：从散列反推原始输入在计算上是不可行的
- **抗碰撞**：很难找到产生相同散列的两个不同输入

**常见用途：**

- 数字签名和证书
- 完整性校验与校验和
- HMAC 构造
- 文件验证与数据流水线
- 需要 256 位散列的协议
