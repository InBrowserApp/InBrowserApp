## Grabar audio del navegador sin conexión

Usa la grabadora de audio para capturar una nota de voz rápida, una prueba de micrófono, un borrador de narración o una muestra de sonido sin salir del navegador. La herramienta solicita acceso al micrófono solo cuando inicias una grabación; después puedes pausar, reanudar, detener, previsualizar y descargar el resultado.

## Usos prácticos

Es útil para comprobar si un micrófono funciona, guardar un recordatorio hablado, grabar una muestra temporal de pronunciación o crear un clip corto para adjuntarlo a otro flujo de trabajo. Como la grabadora se ejecuta del lado del cliente, también es una opción cómoda cuando no quieres instalar una app de audio de escritorio para una toma simple.

## Privacidad y formatos del navegador

La grabación se realiza mediante la API MediaRecorder del navegador. El audio permanece local en la página mientras lo grabas y lo previsualizas; InBrowser.App no sube el flujo del micrófono. El tipo de archivo final depende de la compatibilidad del navegador, por lo que un navegador puede descargar WebM u OGG mientras otro genera M4A.

## Consejos para grabaciones limpias

Usa un entorno silencioso, mantén un nivel de entrada de micrófono razonable y haz una toma breve de prueba antes de grabar algo importante. Si la página no puede iniciar la grabación, comprueba que el sitio esté abierto mediante HTTPS o localhost y que el permiso del micrófono esté habilitado para la pestaña actual del navegador.
