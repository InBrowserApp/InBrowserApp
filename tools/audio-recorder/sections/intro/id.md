## Rekam Audio Browser Secara Offline

Gunakan Perekam Audio untuk menangkap catatan suara cepat, tes mikrofon, draf narasi, atau sampel suara tanpa meninggalkan browser. Alat ini meminta akses mikrofon hanya saat Anda mulai merekam, lalu memungkinkan Anda menjeda, melanjutkan, menghentikan, mempratinjau dan mengunduh hasilnya.

## Penggunaan Praktis

Alat ini berguna untuk memeriksa apakah mikrofon berfungsi, mengumpulkan pengingat lisan, merekam sampel pelafalan sementara, atau membuat klip pendek untuk dilampirkan ke alur kerja lain. Karena perekam berjalan di sisi klien, alat ini juga menjadi opsi praktis saat Anda tidak ingin memasang aplikasi audio desktop untuk rekaman sederhana.

## Privasi Dan Format Browser

Perekaman terjadi melalui API MediaRecorder milik browser. Audio tetap lokal di halaman saat Anda merekam dan mempratinjaunya; InBrowser.App tidak mengunggah stream mikrofon. Jenis file akhir bergantung pada dukungan browser, jadi satu browser dapat mengunduh WebM atau OGG sementara browser lain menghasilkan M4A.

## Tips Untuk Rekaman Jernih

Gunakan lingkungan yang tenang, jaga level input mikrofon tetap wajar, dan buat rekaman uji singkat sebelum merekam sesuatu yang penting. Jika halaman tidak dapat memulai perekaman, pastikan situs dibuka melalui HTTPS atau localhost dan izin mikrofon diizinkan untuk tab browser saat ini.
