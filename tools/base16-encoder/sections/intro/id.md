## Apa itu Base16 (Hex)?

Base16, juga dikenal sebagai heksadesimal (hex), adalah encoding biner-ke-teks yang merepresentasikan setiap byte dengan dua karakter 0-9 dan A-F. Umum digunakan untuk inspeksi byte, checksum, dan identifikasi.

**Penggunaan umum:**

- Debugging dan logging byte mentah
- Mewakili hash, kunci, dan ID
- Menyalin/menempel data biner di kanal teks
