## Fungsi alat ini

Penampil EXIF membaca metadata gambar langsung di browser Anda dan
mengelompokkan hasilnya ke dalam detail gambar dasar, pengaturan kamera,
bidang GPS, dan data tertanam lanjutan. Alat ini mendukung format foto umum
seperti JPEG, PNG, HEIC, TIFF, WebP, dan GIF ketika browser dan parser metadata
dapat membacanya.

## Contoh penggunaan yang tepat

- Memeriksa kamera, lensa, eksposur, atau perangkat lunak penyuntingan yang
  tercatat dalam foto.
- Memeriksa koordinat GPS sebelum menerbitkan atau meneruskan gambar.
- Menyalin cuplikan metadata JSON yang sudah disanitasi untuk debugging, audit
  aset, atau dokumentasi.

## Catatan privasi

**Pengingat privasi:** Metadata gambar dapat menyertakan lokasi presisi, perangkat, dan detail waktu. Tinjau sebelum membagikan foto secara publik.

Metadata dapat mengungkap lokasi persis, waktu pengambilan, model perangkat,
perangkat lunak pembuat, atau bidang hak cipta. Penampil ini tidak mengunggah
gambar, tetapi juga tidak menghapus metadata; gunakan pembersih metadata gambar
sebelum membagikan foto jika bidang tersebut mengungkap lebih banyak informasi
daripada yang Anda inginkan.
