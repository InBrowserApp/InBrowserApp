## Vad är JSON Schema-validering?

Använd det här verktyget när du vill ha snabb feedback medan du designar nyttolaster, felsöker API-exempel eller kontrollerar om en schemaändring bryter exempeldata. Allt körs lokalt i webbläsaren, så den råa JSON-en lämnar aldrig sidan.

## Var det passar bra

- Granska exempelnyttolaster i API-dokumentation.
- Validera mockdata under frontend-arbete.
- Kontrollera format-känsliga fält som `uuid`, `email` eller `date-time`.

## Vad det inte ersätter

- Auktorisering på serversidan och affärsregler.
- Kontraktskontroller som beror på fjärrreferenser eller applikationstillstånd.
- Fullständig CI-validering mot hela ditt schemaset.
