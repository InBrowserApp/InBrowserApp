## Apa itu validasi JSON Schema?

Gunakan alat ini ketika Anda menginginkan umpan balik cepat saat merancang payload, men-debug contoh API, atau memeriksa apakah perubahan skema merusak data sampel. Semua berjalan secara lokal di browser, sehingga JSON mentah tidak pernah meninggalkan halaman.

## Di mana ia cocok digunakan

- Meninjau contoh payload dalam dokumentasi API.
- Memvalidasi data tiruan selama pengerjaan frontend.
- Memeriksa bidang sensitif format seperti `uuid`, `email`, atau `date-time`.

## Apa yang tidak digantikannya

- Otorisasi sisi server dan aturan bisnis.
- Pemeriksaan kontrak yang bergantung pada referensi jarak jauh atau status aplikasi.
- Validasi CI penuh terhadap seluruh kumpulan skema Anda.
