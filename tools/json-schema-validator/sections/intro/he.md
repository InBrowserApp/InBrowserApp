## מהו אימות JSON Schema?

השתמש בכלי זה כאשר אתה רוצה משוב מהיר בזמן עיצוב מטענים, ניפוי באגים של דוגמאות API או בדיקה האם שינוי בסכמה שובר נתוני דוגמה. הכל פועל מקומית בדפדפן, כך שה-JSON הגולמי לעולם אינו עוזב את הדף.

## היכן זה מתאים היטב

- סקירת מטעני דוגמה בתיעוד API.
- אימות נתוני דמה במהלך עבודת frontend.
- בדיקת שדות הרגישים לפורמט כגון `uuid`, `email` או `date-time`.

## מה זה לא מחליף

- הרשאה בצד השרת וכללי עסקים.
- בדיקות חוזה התלויות בהפניות מרוחקות או במצב היישום.
- אימות CI מלא מול כל מערך הסכמות שלך.
