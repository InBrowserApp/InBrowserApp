## Czym jest walidacja JSON Schema?

Użyj tego narzędzia, gdy chcesz szybko otrzymać informację zwrotną podczas projektowania payloadów, debugowania przykładów API lub sprawdzania, czy zmiana schematu nie psuje przykładowych danych. Wszystko działa lokalnie w przeglądarce, więc surowy JSON nigdy nie opuszcza strony.

## Gdzie dobrze pasuje

- Przeglądanie przykładowych payloadów w dokumentacji API.
- Walidacja danych testowych podczas pracy nad frontendem.
- Sprawdzanie pól wrażliwych na format, takich jak `uuid`, `email` czy `date-time`.

## Czego nie zastępuje

- Autoryzacji po stronie serwera i reguł biznesowych.
- Kontroli kontraktów zależnych od zdalnych odwołań lub stanu aplikacji.
- Pełnej walidacji CI dla całego zestawu schematów.
