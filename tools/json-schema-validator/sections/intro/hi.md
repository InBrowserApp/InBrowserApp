## JSON Schema सत्यापन क्या है?

जब आप पेलोड डिज़ाइन करते समय, API उदाहरणों को डीबग करते समय, या यह जाँचते समय कि क्या स्कीमा परिवर्तन नमूना डेटा को तोड़ता है, त्वरित फ़ीडबैक चाहते हैं तो इस टूल का उपयोग करें। सब कुछ ब्राउज़र में स्थानीय रूप से चलता है, इसलिए कच्चा JSON कभी पेज नहीं छोड़ता।

## यह कहाँ अच्छी तरह फिट होता है

- API दस्तावेज़ों में उदाहरण पेलोड की समीक्षा।
- फ्रंटएंड कार्य के दौरान मॉक डेटा का सत्यापन।
- `uuid`, `email`, या `date-time` जैसे फ़ॉर्मेट-संवेदनशील फ़ील्ड की जाँच।

## यह क्या प्रतिस्थापित नहीं करता

- सर्वर-साइड प्राधिकरण और व्यावसायिक नियम।
- अनुबंध जाँचें जो दूरस्थ संदर्भों या एप्लिकेशन स्थिति पर निर्भर करती हैं।
- आपके पूरे स्कीमा सेट के विरुद्ध पूर्ण CI सत्यापन।
