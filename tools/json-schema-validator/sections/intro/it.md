## Cos'è la convalida JSON Schema?

Usa questo strumento quando vuoi un feedback rapido durante la progettazione di payload, il debug di esempi API o la verifica se una modifica allo schema rompe i dati di esempio. Tutto viene eseguito localmente nel browser, quindi il JSON grezzo non lascia mai la pagina.

## Dove si adatta bene

- Revisione di payload di esempio nella documentazione API.
- Convalida di dati mock durante il lavoro frontend.
- Verifica di campi sensibili al formato come `uuid`, `email` o `date-time`.

## Cosa non sostituisce

- L'autorizzazione lato server e le regole di business.
- I controlli di contratto che dipendono da riferimenti remoti o stato dell'applicazione.
- La convalida CI completa rispetto all'intero set di schemi.
