## Что такое проверка JSON Schema?

Используйте этот инструмент, когда вам нужна быстрая обратная связь при проектировании полезных нагрузок, отладке примеров API или проверке того, не ломает ли изменение схемы образцы данных. Всё работает локально в браузере, поэтому исходный JSON никогда не покидает страницу.

## Где он хорошо подходит

- Просмотр примеров полезных нагрузок в документации API.
- Проверка имитационных данных при работе над фронтендом.
- Проверка чувствительных к формату полей, таких как `uuid`, `email` или `date-time`.

## Что он не заменяет

- Серверную авторизацию и бизнес-правила.
- Контрактные проверки, зависящие от удалённых ссылок или состояния приложения.
- Полную CI-валидацию по всему набору ваших схем.
