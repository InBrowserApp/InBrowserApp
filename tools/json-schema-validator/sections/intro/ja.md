## JSON Schema の検証とは？

ペイロードの設計、API の例のデバッグ、またはスキーマの変更がサンプルデータを壊さないかをチェックする際に、迅速なフィードバックが欲しいときにこのツールを使ってください。すべてブラウザ内でローカルに実行されるため、生の JSON はページを離れることがありません。

## 適している場面

- API ドキュメントの例ペイロードのレビュー。
- フロントエンド作業中のモックデータの検証。
- `uuid`、`email`、`date-time` などのフォーマットに敏感なフィールドのチェック。

## 代替できないもの

- サーバーサイドの認可とビジネスルール。
- リモート参照やアプリケーション状態に依存する契約チェック。
- スキーマセット全体に対する完全な CI 検証。
