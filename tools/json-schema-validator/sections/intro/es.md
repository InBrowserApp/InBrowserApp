## ¿Qué es la validación con JSON Schema?

Usa esta herramienta cuando necesites comentarios rápidos al diseñar cargas útiles, depurar ejemplos de API o comprobar si un cambio de esquema rompe los datos de muestra. Todo se ejecuta localmente en el navegador, así que el JSON sin procesar nunca sale de la página.

## Dónde encaja bien

- Revisar ejemplos de cargas útiles en la documentación de API.
- Validar datos simulados durante el trabajo de frontend.
- Comprobar campos sensibles al formato como `uuid`, `email` o `date-time`.

## Lo que no reemplaza

- Autorización del lado del servidor y reglas de negocio.
- Comprobaciones de contrato que dependen de referencias remotas o estado de la aplicación.
- Validación completa en CI contra todo tu conjunto de esquemas.
