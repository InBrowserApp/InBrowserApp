## Apakah pengesahan JSON Schema?

Gunakan alat ini apabila anda mahukan maklum balas pantas semasa mereka bentuk muatan, menyahpepijat contoh API atau memeriksa sama ada perubahan skema merosakkan data sampel. Segala-galanya berjalan secara setempat dalam pelayar, jadi JSON mentah tidak pernah meninggalkan halaman.

## Di mana ia sesuai

- Menyemak contoh muatan dalam dokumentasi API.
- Mengesahkan data tiruan semasa kerja frontend.
- Memeriksa medan sensitif format seperti `uuid`, `email` atau `date-time`.

## Apa yang ia tidak gantikan

- Kebenaran sebelah pelayan dan peraturan perniagaan.
- Pemeriksaan kontrak yang bergantung pada rujukan jauh atau keadaan aplikasi.
- Pengesahan CI penuh terhadap keseluruhan set skema anda.
