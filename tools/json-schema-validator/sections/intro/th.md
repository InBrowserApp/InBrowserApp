## การตรวจสอบ JSON Schema คืออะไร?

ใช้เครื่องมือนี้เมื่อคุณต้องการข้อเสนอแนะที่รวดเร็วในขณะออกแบบเพย์โหลด ดีบักตัวอย่าง API หรือตรวจสอบว่าการเปลี่ยนแปลงสคีมาทำให้ข้อมูลตัวอย่างเสียหายหรือไม่ ทุกอย่างทำงานในเบราว์เซอร์ในเครื่อง ดังนั้น JSON ดิบจึงไม่เคยออกจากหน้านี้

## เหมาะกับการใช้งานที่ไหน

- ตรวจสอบเพย์โหลดตัวอย่างในเอกสาร API
- ตรวจสอบข้อมูลจำลองระหว่างการทำงานฝั่ง frontend
- ตรวจสอบฟิลด์ที่ไวต่อรูปแบบ เช่น `uuid`, `email` หรือ `date-time`

## สิ่งที่มันไม่ได้แทนที่

- การอนุญาตฝั่งเซิร์ฟเวอร์และกฎทางธุรกิจ
- การตรวจสอบสัญญาที่ขึ้นอยู่กับการอ้างอิงระยะไกลหรือสถานะแอปพลิเคชัน
- การตรวจสอบ CI แบบเต็มกับชุดสคีมาทั้งหมดของคุณ
