## ما هو التحقق من JSON Schema؟

استخدم هذه الأداة عندما تريد ملاحظات سريعة أثناء تصميم الحمولات أو تصحيح أمثلة API أو التحقق مما إذا كان تغيير المخطط يكسر بيانات العينة. يتم تشغيل كل شيء محليًا في المتصفح، لذلك لا يغادر JSON الخام الصفحة أبدًا.

## أين يناسب جيدًا

- مراجعة أمثلة الحمولات في وثائق API.
- التحقق من البيانات الوهمية أثناء العمل على الواجهة الأمامية.
- فحص الحقول الحساسة للصيغة مثل `uuid` أو `email` أو `date-time`.

## ما لا يحل محله

- التفويض من جانب الخادم وقواعد العمل.
- فحوصات العقد التي تعتمد على المراجع البعيدة أو حالة التطبيق.
- التحقق الكامل في CI مقابل مجموعة المخططات بالكامل.
