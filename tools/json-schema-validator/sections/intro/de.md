## Was ist JSON-Schema-Validierung?

Verwenden Sie dieses Tool, wenn Sie schnelles Feedback beim Entwerfen von Payloads, beim Debuggen von API-Beispielen oder beim Prüfen wünschen, ob eine Schemaänderung Beispieldaten beschädigt. Alles läuft lokal im Browser, sodass das rohe JSON die Seite nie verlässt.

## Wo es gut passt

- Überprüfen von Beispiel-Payloads in API-Dokumentation.
- Validieren von Mock-Daten während der Frontend-Arbeit.
- Prüfen formatabhängiger Felder wie `uuid`, `email` oder `date-time`.

## Was es nicht ersetzt

- Serverseitige Autorisierung und Geschäftsregeln.
- Vertragsprüfungen, die von Remote-Referenzen oder Anwendungszustand abhängen.
- Vollständige CI-Validierung gegen Ihren gesamten Schemasatz.
