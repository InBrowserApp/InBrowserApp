## Что такое Data URI?

Data URI (или data URL) встраивает небольшие файлы прямо в текст. Формат: `data:[mime][;charset][;base64],data`.

**Обычно используется для:**

- Встраивания изображений или шрифтов в HTML/CSS
- Хранения небольших ресурсов в JSON/настройках

**Примечания:**

- Подходит для небольших файлов; большие строки могут замедлять страницу
- Base64 часто используется для бинарных данных

### Пример

```text
data:image/svg+xml;base64,PHN2ZyB4bWxucz0i...
```

Часть до запятой описывает файл: его MIME-тип и то, используется ли Base64. Часть после запятой содержит закодированные данные.

### Когда этот конвертер особенно полезен

- Нужно превратить локальный файл в строку, которую можно вставить в HTML, CSS, JSON или шаблон письма
- Нужно быстро собрать самодостаточный пример без отдельного хостинга ресурса
- Нужно проверить определённый MIME-тип перед вставкой результата в другой инструмент

### Практические ограничения

- Data URI лучше всего подходят для небольших файлов: иконок, маленьких изображений и коротких фрагментов
- Base64 добавляет примерно 33% накладных расходов, поэтому итоговая строка длиннее исходного файла
- Очень длинные строки неудобно вставлять в формы, конфиги или редакторы с ограничениями по размеру
