## Data URI とは？

Data URI（data URL）は小さなファイルをテキスト内に直接埋め込みます。形式：`data:[mime][;charset][;base64],data`。

**主な用途：**

- HTML/CSS に画像やフォントをインライン埋め込み
- JSON/設定に小さなリソースを保存

**注意点：**

- 小さなファイル向き。長い文字列はページを重くする場合があります
- バイナリは Base64 が一般的です

### 例

```text
data:image/svg+xml;base64,PHN2ZyB4bWxucz0i...
```

カンマより前の部分には、MIME タイプや Base64 を使っているかどうかなど、ファイルの情報が入ります。カンマより後ろの部分がエンコードされた本体です。

### この変換ツールが向いている場面

- ローカルファイルを HTML、CSS、JSON、メール用マークアップに埋め込める文字列へ変換したいとき
- アセットを別途ホストせずに、自己完結したデモやサンプルをすばやく作りたいとき
- 他のツールへ貼り付ける前に、検出された MIME タイプを確認したいとき

### 実用上の制限

- Data URI はアイコン、小さな画像、短い断片などの小さなファイルに向いています
- Base64 ではおよそ 33% サイズが増えるため、最終的な文字列は元ファイルより大きくなります
- 文字列が長すぎると、長さ制限のあるフォーム、設定ファイル、エディタには貼り付けにくくなります
