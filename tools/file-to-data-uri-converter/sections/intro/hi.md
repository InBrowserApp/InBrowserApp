## Data URI क्या है?

Data URI (या data URL) छोटे फ़ाइलों को सीधे टेक्स्ट में एम्बेड करता है। फ़ॉर्मेट: `data:[mime][;charset][;base64],data`.

**सामान्य उपयोग:**

- HTML/CSS में इमेज या फ़ॉन्ट इनलाइन
- JSON/कॉन्फ़िग में छोटे एसेट्स स्टोर करना

**नोट्स:**

- छोटे फ़ाइलों के लिए बेहतर; बड़ी स्ट्रिंग्स पेज को धीमा कर सकती हैं
- बाइनरी डेटा के लिए Base64 आम है

### उदाहरण

```text
data:image/svg+xml;base64,PHN2ZyB4bWxucz0i...
```

कॉमा से पहले वाला हिस्सा फ़ाइल का विवरण देता है, जैसे उसका MIME प्रकार और क्या वह Base64 का उपयोग करती है। कॉमा के बाद वाला हिस्सा एन्कोड किया गया डेटा होता है।

### यह कन्वर्टर कब उपयोगी है

- किसी लोकल फ़ाइल को HTML, CSS, JSON या ईमेल मार्कअप में एम्बेड करने लायक स्ट्रिंग में बदलना हो
- एसेट को कहीं और होस्ट किए बिना जल्दी से self-contained डेमो बनाना हो
- परिणाम को किसी दूसरे टूल में पेस्ट करने से पहले पहचाने गए MIME प्रकार की जांच करनी हो

### व्यावहारिक सीमाएँ

- Data URI छोटे फ़ाइलों, जैसे आइकन, छोटी इमेज या छोटे स्निपेट के लिए सबसे अच्छा काम करता है
- Base64 लगभग 33% अतिरिक्त आकार जोड़ता है, इसलिए अंतिम स्ट्रिंग मूल फ़ाइल से बड़ी होती है
- बहुत लंबी स्ट्रिंग्स को आकार-सीमित फ़ॉर्म, कॉन्फ़िग या एडिटर में पेस्ट करना असुविधाजनक हो सकता है
