## 什麼是 Data URI？

Data URI（也叫 data URL）可以把小檔案直接嵌入到文字中。格式：`data:[mime][;charset][;base64],data`。

**常見用途：**

- 在 HTML/CSS 中內嵌圖片或字型
- 在 JSON/設定中保存小資源

**注意：**

- 適合小檔案，過大會讓頁面變慢
- 二進位通常使用 Base64

### 範例

```text
data:image/svg+xml;base64,PHN2ZyB4bWxucz0i...
```

逗號前面的部分描述檔案資訊，例如 MIME 類型以及是否使用 Base64。逗號後面的部分是編碼後的檔案內容。

### 何時適合使用這個轉換器

- 將本機檔案快速轉成可嵌入 HTML、CSS、JSON 或郵件範本的字串
- 製作不需要額外託管資源的自包含示範或範例
- 在把結果貼到其他工具之前，先檢查偵測到的 MIME 類型

### 實際限制

- Data URI 更適合圖示、小圖片或短片段這類小型檔案
- Base64 大約會增加 33% 體積，因此最終字串會比原始檔案更大
- 字串過長時，在有長度限制的表單、設定或編輯器中貼上會比較麻煩
