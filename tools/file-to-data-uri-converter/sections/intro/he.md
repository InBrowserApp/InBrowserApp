## מה זה Data URI?

Data URI (או data URL) מטמיע קבצים קטנים ישירות בטקסט. פורמט: `data:[mime][;charset][;base64],data`.

**שימושים נפוצים:**

- הטמעת תמונות או גופנים בתוך HTML/CSS
- שמירת נכסים קטנים ב-JSON/קונפיגים

**הערות:**

- מתאים לקבצים קטנים; מחרוזות ארוכות עלולות להאט את הדף
- Base64 נפוץ לנתונים בינריים

### דוגמה

```text
data:image/svg+xml;base64,PHN2ZyB4bWxucz0i...
```

החלק שלפני הפסיק מתאר את הקובץ, למשל סוג ה-MIME והאם נעשה שימוש ב-Base64. החלק שאחרי הפסיק הוא המטען המקודד.

### מתי הממיר הזה מתאים במיוחד

- כאשר רוצים להפוך קובץ מקומי למחרוזת שאפשר להטמיע בתוך HTML, CSS, JSON או תבנית אימייל
- כאשר רוצים ליצור במהירות דמו עצמאי בלי לאחסן את הנכס במקום אחר
- כאשר רוצים לבדוק את סוג ה-MIME שזוהה לפני שמדביקים את התוצאה לכלי אחר

### מגבלות מעשיות

- Data URI מתאים בעיקר לקבצים קטנים כמו אייקונים, תמונות קטנות או קטעים קצרים
- Base64 מוסיף בערך 33% תקורה, ולכן המחרוזת הסופית גדולה יותר מהקובץ המקורי
- מחרוזות ארוכות מאוד עלולות להיות לא נוחות להדבקה בטפסים, בקבצי תצורה או בעורכים עם מגבלות גודל
