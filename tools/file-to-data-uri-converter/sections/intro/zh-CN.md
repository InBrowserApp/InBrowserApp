## 什么是 Data URI？

Data URI（也叫 data URL）可以把小文件直接嵌入到文本中。格式：`data:[mime][;charset][;base64],data`。

**常见用途：**

- 在 HTML/CSS 中内联图片或字体
- 在 JSON/配置中保存小资源

**注意：**

- 适合小文件，过大会让页面变慢
- 二进制通常使用 Base64

### 示例

```text
data:image/svg+xml;base64,PHN2ZyB4bWxucz0i...
```

逗号前面的部分描述文件信息，例如 MIME 类型以及是否使用 Base64。逗号后面的部分是编码后的文件内容。

### 何时适合使用这个转换器

- 将本地文件快速转换成可嵌入 HTML、CSS、JSON 或邮件模板的字符串
- 制作无需额外托管资源的自包含演示或示例
- 在把结果粘贴到其他工具之前，先检查识别出的 MIME 类型

### 实际限制

- Data URI 更适合图标、小图片或短片段这类小文件
- Base64 大约会增加 33% 体积，因此最终字符串会比原文件更大
- 字符串过长时，在带有长度限制的表单、配置或编辑器中粘贴会比较麻烦
