## Data URI คืออะไร?

Data URI (หรือ data URL) ฝังไฟล์ขนาดเล็กลงในข้อความโดยตรง รูปแบบ: `data:[mime][;charset][;base64],data`.

**การใช้งานทั่วไป:**

- ฝังรูปภาพหรือฟอนต์ใน HTML/CSS
- เก็บทรัพยากรขนาดเล็กใน JSON/คอนฟิก

**หมายเหตุ:**

- เหมาะกับไฟล์เล็ก; สตริงยาวอาจทำให้หน้าเว็บช้าลง
- Base64 ใช้บ่อยกับข้อมูลไบนารี

### ตัวอย่าง

```text
data:image/svg+xml;base64,PHN2ZyB4bWxucz0i...
```

ส่วนก่อนเครื่องหมายจุลภาคใช้บอกข้อมูลของไฟล์ เช่น ประเภท MIME และมีการใช้ Base64 หรือไม่ ส่วนหลังเครื่องหมายจุลภาคคือข้อมูลไฟล์ที่ถูกเข้ารหัสแล้ว

### ตัวแปลงนี้เหมาะใช้เมื่อใด

- ต้องการแปลงไฟล์ในเครื่องให้เป็นสตริงที่ฝังใน HTML, CSS, JSON หรือเทมเพลตอีเมลได้
- ต้องการสร้างเดโมแบบ self-contained อย่างรวดเร็วโดยไม่ต้องโฮสต์ไฟล์ไว้ที่อื่น
- ต้องการตรวจสอบประเภท MIME ที่ตรวจพบก่อนนำผลลัพธ์ไปวางในเครื่องมืออื่น

### ข้อจำกัดในการใช้งานจริง

- Data URI เหมาะกับไฟล์ขนาดเล็ก เช่น ไอคอน รูปภาพเล็ก หรือสตริงสั้น ๆ
- Base64 เพิ่มขนาดประมาณ 33% ทำให้สตริงสุดท้ายใหญ่กว่าไฟล์ต้นฉบับ
- หากสตริงยาวมาก การนำไปวางในฟอร์ม คอนฟิก หรือเอดิเตอร์ที่มีข้อจำกัดด้านขนาดอาจไม่สะดวก
