## ما هو Data URI؟

Data URI (أو data URL) يدمج الملفات الصغيرة مباشرة داخل النص. الصيغة: `data:[mime][;charset][;base64],data`.

**الاستخدامات الشائعة:**

- تضمين الصور أو الخطوط داخل HTML/CSS
- حفظ الموارد الصغيرة في JSON/الإعدادات

**ملاحظات:**

- مناسب للملفات الصغيرة؛ السلاسل الطويلة قد تبطئ الصفحة
- Base64 شائع للبيانات الثنائية

### مثال

```text
data:image/svg+xml;base64,PHN2ZyB4bWxucz0i...
```

الجزء الذي يسبق الفاصلة يصف الملف، مثل نوع MIME وما إذا كان يستخدم Base64. أما الجزء الذي يلي الفاصلة فهو المحتوى المُرمَّز.

### متى يكون هذا المحول مناسبًا

- تحويل ملف محلي إلى سلسلة يمكن تضمينها داخل HTML أو CSS أو JSON أو قوالب البريد الإلكتروني
- إنشاء عرض توضيحي مكتفٍ ذاتيًا بسرعة من دون استضافة المورد في مكان آخر
- التحقق من نوع MIME المكتشف قبل لصق النتيجة في أداة أخرى

### حدود عملية

- يكون Data URI أنسب للملفات الصغيرة مثل الأيقونات والصور الصغيرة والمقتطفات القصيرة
- يضيف Base64 زيادة تقارب 33%، لذلك تكون السلسلة النهائية أكبر من الملف الأصلي
- قد يكون من المزعج لصق السلاسل الطويلة جدًا في النماذج أو الإعدادات أو المحررات ذات حدود الحجم
