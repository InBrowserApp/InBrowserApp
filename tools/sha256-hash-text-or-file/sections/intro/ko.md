## SHA-256이란?

SHA-256(Secure Hash Algorithm 256비트)은 256비트(32바이트) 해시 값을 생성하는 암호화 해시 함수로, 일반적으로 64자리 16진수로 표현됩니다. NSA에서 설계하고 NIST에서 발표한 SHA-2 해시 함수 계열의 일부입니다.

**주요 특징:**

- **결정론적**: 동일한 입력은 항상 동일한 해시를 생성
- **빠른 계산**: 주어진 입력에 대해 빠르게 계산
- **눈사태 효과**: 입력의 작은 변화가 완전히 다른 출력을 생성
- **비가역적**: 해시를 역산하여 원본 입력을 찾는 것은 계산적으로 불가능
- **충돌 저항성**: 동일한 해시를 생성하는 두 개의 다른 입력을 찾기 매우 어려움

**일반적인 용도:**

- 디지털 서명 및 인증서
- 블록체인 및 암호화폐(비트코인은 SHA-256 사용)
- 비밀번호 저장(적절한 솔팅과 함께)
- 파일 무결성 검증
- 작업 증명 알고리즘
