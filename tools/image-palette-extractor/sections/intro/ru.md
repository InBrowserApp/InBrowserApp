## Что делает этот инструмент

Инструмент извлечения палитры находит доминирующие цвета в изображении прямо
в вашем браузере. Он анализирует изображение, группирует визуально похожие
пиксели и возвращает практичную палитру со значениями HEX, RGB, HSL и
процентной долей для каждого цвета.

## Подходящие задачи

- Получить цвета бренда или продукта из скриншота, логотипа, фотографии или
  макета.
- Быстро собрать CSS-палитру для посадочной страницы, миниатюры или передачи
  дизайна.
- Сравнить, какую долю изображения занимает доминирующий цвет и какую -
  вспомогательные акценты.
- Работать с приватными изображениями, не отправляя файл на сервер.

## Варианты экспорта

Результат можно скопировать как простой список HEX, пользовательские свойства
CSS или JSON. Формат CSS полезен, когда нужны переменные вроде `--palette-1`,
а JSON сохраняет форматы цвета и долю доминирования вместе для скриптов или
автоматизации дизайна.

## На что обратить внимание

- Извлечение палитры приблизительное. Оно предназначено для получения полезных
  визуальных групп, а не полного списка цветов каждого пикселя.
- Прозрачные пиксели по умолчанию игнорируются, чтобы значки и вырезанные
  изображения не искажали палитру; отключите это, если сама прозрачность
  является частью иллюстрации.
- Точный режим качества анализирует больше пикселей и может работать
  медленнее с очень большими изображениями.
