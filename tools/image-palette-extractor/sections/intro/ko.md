## 이 도구의 기능

이미지 팔레트 추출기는 브라우저에서 직접 이미지의 주요 색상을 찾습니다.
사진을 샘플링하고 시각적으로 비슷한 픽셀을 그룹화한 뒤, 각 색상에
대한 HEX, RGB, HSL, 백분율 값을 포함한 실용적인 팔레트를 반환합니다.

## 좋은 활용 사례

- 스크린샷, 로고, 사진 또는 목업에서 브랜드나 제품 색상을 추출합니다.
- 랜딩 페이지, 썸네일 또는 디자인 인계용 CSS 팔레트를 빠르게 만듭니다.
- 이미지가 하나의 주요 색상과 보조 포인트 색상에 얼마나 좌우되는지
  비교합니다.
- 파일을 서버로 보내지 않고 비공개 이미지로 작업합니다.

## 내보내기 옵션

결과는 일반 HEX 목록, CSS 사용자 지정 속성 또는 JSON으로 복사할 수
있습니다. CSS 형식은 `--palette-1` 같은 변수를 원할 때 유용하고, JSON은
스크립트나 디자인 자동화를 위해 색상 형식과 점유 비율을 함께 보존합니다.

## 주의할 점

- 팔레트 추출은 근사값입니다. 모든 픽셀 색상의 완전한 목록이 아니라
  유용한 시각적 그룹을 만드는 데 목적이 있습니다.
- 아이콘과 컷아웃이 팔레트를 왜곡하지 않도록 투명 픽셀은 기본적으로
  무시됩니다. 투명도 자체가 아트워크의 일부라면 이 설정을 끄세요.
- 정밀 품질 설정은 더 많은 픽셀을 샘플링하므로 매우 큰 이미지에서는
  느릴 수 있습니다.
