## ما الذي تفعله هذه الأداة

يعثر مستخرج لوحات ألوان الصور على الألوان السائدة في الصورة مباشرة داخل
متصفحك. يأخذ عينات من الصورة، ويجمع البكسلات المتشابهة بصريًا، ويعيد
لوحة عملية تتضمن قيم HEX وRGB وHSL والنسب المئوية لكل لون.

## حالات استخدام مناسبة

- استخراج ألوان العلامة التجارية أو المنتج من لقطة شاشة أو شعار أو صورة فوتوغرافية أو نموذج تصميم.
- إنشاء لوحة CSS سريعة لصفحة هبوط أو صورة مصغرة أو تسليم تصميم.
- مقارنة مدى اعتماد الصورة على لون سائد واحد مقابل الألوان الداعمة.
- العمل مع الصور الخاصة دون إرسال الملف إلى خادم.

## خيارات التصدير

يمكن نسخ النتيجة كقائمة HEX عادية أو خصائص CSS مخصصة أو JSON. يكون تنسيق
CSS مفيدًا عندما تريد متغيرات مثل `--palette-1`، بينما يحتفظ JSON بتنسيقات
الألوان ونسبة الهيمنة معًا للنصوص البرمجية أو أتمتة التصميم.

## أشياء يجب الانتباه إليها

- استخراج اللوحة تقريبي. الغرض منه إنتاج مجموعات بصرية مفيدة، وليس جردًا كاملًا لكل لون بكسل.
- يتم تجاهل البكسلات الشفافة افتراضيًا حتى لا تشوّه الأيقونات والعناصر المقصوصة اللوحة؛ أوقف ذلك عندما تكون الشفافية نفسها جزءًا من العمل الفني.
- تأخذ إعدادات الجودة الدقيقة عينات من بكسلات أكثر، وقد تكون أبطأ مع الصور الكبيرة جدًا.
