## यह टूल क्या करता है

Image Palette Extractor आपके ब्राउज़र में सीधे किसी इमेज के प्रमुख रंग
ढूंढता है। यह इमेज को सैंपल करता है, दिखने में मिलते-जुलते पिक्सेल को
समूहित करता है, और हर रंग के लिए HEX, RGB, HSL, और प्रतिशत मानों वाला
व्यावहारिक पैलेट लौटाता है।

## अच्छे उपयोग के मामले

- स्क्रीनशॉट, लोगो, फोटो, या मॉकअप से ब्रांड या उत्पाद रंग निकालें।
- लैंडिंग पेज, थंबनेल, या डिजाइन हैंडऑफ के लिए जल्दी CSS पैलेट बनाएं।
- तुलना करें कि किसी इमेज का कितना हिस्सा एक प्रमुख रंग से बनता है और
  कितना सहायक एक्सेंट रंगों से।
- निजी इमेज के साथ फ़ाइल को सर्वर पर भेजे बिना काम करें।

## निर्यात विकल्प

परिणाम को साधारण HEX सूची, CSS custom properties, या JSON के रूप में कॉपी किया
जा सकता है। CSS प्रारूप तब उपयोगी होता है जब आपको `--palette-1` जैसे variables
चाहिए हों, जबकि JSON स्क्रिप्ट या डिजाइन ऑटोमेशन के लिए रंग प्रारूपों और
प्रमुखता अनुपात को साथ रखता है।

## ध्यान रखने योग्य बातें

- पैलेट निकालना अनुमानित होता है। इसका उद्देश्य उपयोगी दृश्य समूह बनाना है,
  हर पिक्सेल रंग की पूरी सूची नहीं।
- पारदर्शी पिक्सेल डिफ़ॉल्ट रूप से अनदेखे किए जाते हैं ताकि आइकन और कटआउट
  पैलेट को तिरछा न करें; जब पारदर्शिता स्वयं कलाकृति का हिस्सा हो तो इसे बंद
  करें।
- सटीक गुणवत्ता सेटिंग अधिक पिक्सेल सैंपल करती है और बहुत बड़े चित्रों पर
  धीमी हो सकती है।
