## Для чего нужен этот инструмент

Используйте этот инструмент, когда нужно превратить корректный JSON в CSV, который легче читать, делиться им или коммитить. Все выполняется локально в браузере.

## Где он полезен

- Преобразовывать примеры API в CSV для документации или конфигурационных файлов.
- Переписывать скопированные JSON-payload'ы в формат, который проще сравнивать глазами.
- Подготавливать структурированный контент для инструментов и процессов, которые ожидают CSV.

## Что важно учитывать

- В этом инструменте входные данные должны быть корректным JSON.
- Конвертер сохраняет структуру данных, но представление CSV может отличаться от написанного вручную стиля.
- Если нужно преобразование в обратную сторону, используйте конвертер CSV в JSON.
