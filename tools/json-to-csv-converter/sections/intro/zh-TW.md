## 這個工具適合做什麼

當你需要把有效 JSON 轉成適合表格、匯出或匯入流程的 CSV 時，可以使用這個工具。所有處理都在瀏覽器本地完成。

## 適合哪些場景

- 把 JSON 物件陣列轉換成分析、營運或客服團隊更容易開啟的表格列。
- 在分享 CSV 匯出之前，調整分隔符號、引號、表頭和公式跳脫。
- 為只接受 CSV、不接受原始 JSON 的系統準備輕量的表格資料。

## 需要注意什麼

- 這個工具裡的輸入必須是合法 JSON。
- 當你需要帶欄位名稱的輸出時，最適合傳入物件陣列；如果只是原始列資料，也可以使用陣列的陣列。
- 如果你要反向轉換，請使用 CSV 轉 JSON 工具。
