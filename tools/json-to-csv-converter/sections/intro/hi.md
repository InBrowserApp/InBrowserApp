## यह टूल किस काम आता है

जब आपको वैध JSON को ऐसे CSV में बदलना हो जिसे पढ़ना, साझा करना या commit करना आसान हो, तब इस टूल का उपयोग करें। सब कुछ ब्राउज़र में लोकली चलता है।

## कहाँ उपयोगी है

- API examples को documentation या config files के लिए CSV में बदलने के लिए।
- कॉपी किए गए JSON payloads को ऐसे format में लिखने के लिए जिसे आँख से compare करना आसान हो।
- उन tools और workflows के लिए structured content तैयार करने के लिए जो CSV चाहते हैं।

## किन बातों का ध्यान रखें

- इस टूल में input वैध JSON होना चाहिए।
- converter data structure को सुरक्षित रखता है, लेकिन CSV presentation हाथ से लिखे गए style से अलग हो सकती है।
- अगर आपको उल्टी दिशा में जाना है, तो CSV to JSON converter का उपयोग करें।
