## Công cụ này dùng để làm gì

Dùng công cụ này khi bạn cần biến JSON hợp lệ thành CSV dễ đọc, dễ chia sẻ hoặc dễ commit hơn. Mọi thứ đều chạy cục bộ trong trình duyệt.

## Khi nào hữu ích

- Chuyển ví dụ API sang CSV cho tài liệu hoặc tệp cấu hình.
- Viết lại các payload JSON đã sao chép sang định dạng dễ so sánh bằng mắt hơn.
- Chuẩn bị nội dung có cấu trúc cho các công cụ và quy trình yêu cầu CSV.

## Điều cần lưu ý

- Trong công cụ này, đầu vào phải là JSON hợp lệ.
- Bộ chuyển đổi giữ nguyên cấu trúc dữ liệu, nhưng cách trình bày CSV vẫn có thể khác với kiểu viết tay.
- Nếu cần chuyển ngược lại, hãy dùng công cụ chuyển CSV sang JSON.
