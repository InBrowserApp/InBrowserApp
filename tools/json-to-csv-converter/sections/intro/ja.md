## このツールの用途

有効な JSON を、読みやすく共有しやすくコミットにも向いた CSV に変換したいときに使うツールです。すべての処理はブラウザ内でローカルに実行されます。

## 向いている場面

- API のサンプルをドキュメントや設定ファイル向けの CSV に変換する。
- コピーした JSON ペイロードを目視で比較しやすい形式に整える。
- CSV を前提とするツールやワークフロー向けに構造化データを準備する。

## 注意したいこと

- このツールでは入力は有効な JSON である必要があります。
- コンバーターはデータ構造を保ちますが、CSV の見た目は手書きのスタイルと異なることがあります。
- 逆方向に変換したい場合は、CSV から JSON へのコンバーターを使ってください。
