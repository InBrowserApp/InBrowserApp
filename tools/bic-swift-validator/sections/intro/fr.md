## Qu'est-ce que le BIC/SWIFT ?

Le BIC (Bank Identifier Code), aussi appelé code SWIFT, identifie les institutions financières dans les paiements internationaux.

### Structure du BIC

Un BIC comporte 8 ou 11 caractères : code banque (4 lettres), code pays (2 lettres), code de localisation (2 alphanumériques) et code d'agence optionnel (3 alphanumériques).

### Règles de validation

La validation vérifie la longueur, le jeu de caractères et les codes pays ISO 3166.

1. Supprimez les espaces et les tirets
2. Vérifiez que le format a 8 ou 11 caractères
3. Analysez les codes banque, pays, localisation et agence

Le code d'agence "XXX" ou un BIC à 8 caractères indique le siège.

Le second caractère du code de localisation 0 indique un BIC de test ; 1 indique un participant passif.
