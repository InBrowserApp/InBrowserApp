## 什么是文本或文件哈希？

哈希函数会将文本或文件字节转换为固定长度的摘要。相同输入和算法始终会生成相同摘要，因此当你需要无需上传私有数据的可重复指纹时，哈希很有用。

## 何时使用此工具

使用此工具可以验证下载校验和、比较两个文件是否完全相同、为文本片段快速记录指纹，或调试发布 SHA 摘要的系统。导入文件会直接对文件字节进行哈希，而文本模式会对编辑器中显示的 UTF-8 文本进行哈希。

## 如何选择算法

SHA-256 是新的完整性校验的可靠默认选择。当其他系统要求这些格式时，SHA-384 和 SHA-512 可以提供更长的 SHA-2 摘要。SHA-1 仅用于旧系统比较，不应在新的安全敏感设计中使用。

## 隐私与限制

哈希通过 Web Crypto 在你的浏览器本地运行，文件不会上传。哈希不是加密：它本身无法保护秘密，密码存储需要使用带盐值和工作因子的专用密码哈希函数。
