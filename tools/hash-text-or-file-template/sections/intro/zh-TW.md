## 什麼是文字或檔案雜湊？

雜湊函式會將文字或檔案位元組轉換為固定長度的摘要。相同的輸入與演算法一律會產生相同的摘要，因此當你需要可重複的指紋，又不想上傳私人資料時，雜湊很有用。

## 何時使用這個工具

你可以用這個工具驗證下載檔案的檢查碼、比較兩個檔案是否相同、為一段文字快速記錄指紋，或除錯會發布 SHA 摘要的系統。匯入檔案時會直接雜湊檔案位元組；文字模式則會雜湊編輯器中顯示的 UTF-8 文字。

## 選擇演算法

SHA-256 是新的完整性檢查的穩妥預設值。當其他系統需要較長的 SHA-2 摘要格式時，可使用 SHA-384 和 SHA-512。SHA-1 是為了舊系統比對而提供，但不應用於新的安全敏感設計。

## 隱私與限制

雜湊會透過 Web Crypto 在你的瀏覽器本機執行，檔案不會上傳。雜湊不是加密：它本身無法保護秘密，密碼儲存需要專用的密碼雜湊函式，並搭配鹽值與工作因子。
