## Apa itu gradien CSS?

Buat gradien linear, radial, dan konik dengan pemberhentian warna yang bisa diseret, mode blend, dan CSS siap pakai.

Gradien CSS adalah gambar yang dihasilkan browser dari transisi warna yang halus. Karena dihitung saat render, ia dapat diskalakan ke ukuran apa pun tanpa pecah dan tidak memerlukan file gambar.

Ada tiga jenis: linear (arah atau sudut), radial (lingkaran atau elips dari pusat), dan konik (sapuan mengitari pusat). Setiap gradien ditentukan oleh titik henti warna; setiap titik dapat memiliki posisi dan alfa untuk mengatur perpaduan.

Beberapa gradien bisa ditumpuk dengan memisahkannya menggunakan koma di background-image dan digabung dengan background-blend-mode untuk tekstur yang lebih kaya. CSS juga mendukung interpolasi pada ruang warna seperti oklch untuk transisi yang lebih mulus.

- Gradien linear cocok untuk pencahayaan dan kedalaman terarah.
- Gradien radial dan konik berguna untuk spotlight, cincin, dan roda warna.
- Atur posisi titik henti untuk transisi yang lebih tajam atau lebih lembut.
