## 什么是 CSS 渐变？

创建可拖拽色标的线性、径向与圆锥渐变，支持混合模式并输出可直接使用的 CSS。

CSS 渐变是由浏览器根据颜色之间的平滑过渡生成的图像。由于在渲染时计算，它可以在任意尺寸下保持清晰，并且不需要图片文件。

渐变有三种类型：线性（方向或角度）、径向（从中心发散的圆或椭圆）以及圆锥（围绕中心旋转的扇形）。每个渐变由多个色标构成；每个色标可以包含位置和透明度来控制颜色混合。

多个渐变可以在 background-image 中用逗号分隔叠加，并通过 background-blend-mode 进行混合，形成更丰富的纹理。CSS 还支持不同的插值色彩空间，例如 oklch，以获得更平滑、符合感知的过渡。

- 线性渐变适合表现光照和方向感。
- 径向与圆锥渐变常用于聚光、光环和色轮效果。
- 调整色标位置可以让过渡更清晰或更柔和。
