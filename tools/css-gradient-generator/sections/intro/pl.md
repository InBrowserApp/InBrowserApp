## Czym jest gradient CSS?

Twórz gradienty liniowe, radialne i stożkowe z przeciąganymi punktami kolorów, trybami mieszania i gotowym CSS.

Gradient CSS to obraz generowany przez przeglądarkę z płynnego przejścia między kolorami. Ponieważ jest obliczany podczas renderowania, skaluje się do dowolnego rozmiaru bez pikselizacji i nie wymaga plików graficznych.

Są trzy typy: liniowy (kierunek lub kąt), radialny (okrąg lub elipsa z centrum) i stożkowy (obrót wokół centrum). Każdy gradient jest zdefiniowany przez przystanki kolorów; każdy przystanek może zawierać pozycję i alfa, aby kontrolować mieszanie.

Wiele gradientów można nakładać, oddzielając je przecinkami w background-image, i łączyć za pomocą background-blend-mode, aby uzyskać bogatsze tekstury. CSS obsługuje też interpolację w przestrzeniach barw, takich jak oklch, dla gładszych przejść.

- Gradienty liniowe świetnie nadają się do światła i kierunkowej głębi.
- Gradienty radialne i stożkowe są przydatne do reflektorów, pierścieni i kół barw.
- Przesuwaj pozycje przystanków, aby wyostrzyć lub zmiękczyć przejścia.
