## 什麼是 CSS 漸層？

建立可拖曳色標的線性、放射與圓錐漸層，支援混合模式並輸出可直接使用的 CSS。

CSS 漸層是由瀏覽器根據顏色之間的平滑過渡生成的影像。由於在渲染時計算，它可在任意尺寸下保持清晰，且不需要圖片檔案。

漸層有三種型態：線性（方向或角度）、放射（由中心發散的圓或橢圓）以及圓錐（繞中心旋轉的扇形）。每個漸層由多個色標構成；每個色標可包含位置與透明度來控制顏色混合。

多個漸層可在 background-image 中以逗號分隔堆疊，並透過 background-blend-mode 混合以形成更豐富的質感。CSS 也支援不同的插值色彩空間，例如 oklch，以獲得更平滑、符合感知的過渡。

- 線性漸層適合表現光線與方向感。
- 放射與圓錐漸層常用於聚光、光環與色輪效果。
- 調整色標位置可讓過渡更銳利或更柔和。
