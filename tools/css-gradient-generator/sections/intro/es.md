## ¿Qué es un gradiente CSS?

Crea gradientes lineales, radiales y cónicos con paradas de color arrastrables, modos de mezcla y CSS listo para usar.

Un gradiente CSS es una imagen generada por el navegador a partir de una transición suave entre colores. Como se calcula en tiempo de renderizado, escala a cualquier tamaño sin pixelarse y no requiere archivos de imagen.

Hay tres tipos: lineal (dirección o ángulo), radial (círculo o elipse desde un centro) y cónico (barrido alrededor de un centro). Cada gradiente se define con paradas de color; cada parada puede incluir posición y alfa para controlar la mezcla.

Se pueden apilar varios gradientes separándolos con comas en background-image y combinarlos con background-blend-mode para crear texturas más ricas. CSS también permite interpolación en espacios de color como oklch para transiciones perceptualmente más suaves.

- Los gradientes lineales son ideales para iluminación y profundidad direccional.
- Los gradientes radiales y cónicos sirven para focos, anillos y ruedas de color.
- Ajusta las posiciones de las paradas para transiciones más nítidas o más suaves.
