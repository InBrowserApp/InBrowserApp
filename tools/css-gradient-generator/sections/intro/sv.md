## Vad är en CSS-gradient?

Skapa linjära, radiella och koniska gradienter med dragbara färgstopp, blandningslägen och färdig CSS.

En CSS-gradient är en bild som webbläsaren skapar från en mjuk övergång mellan färger. Eftersom den beräknas vid rendering skalar den till alla storlekar utan att bli pixlig och kräver inga bildfiler.

Det finns tre typer: linjär (riktning eller vinkel), radiell (cirkel eller ellips från ett centrum) och konisk (svep runt ett centrum). Varje gradient definieras av färgstopp; varje stopp kan ange position och alfa för att styra blandningen.

Flera gradienter kan staplas genom att separera dem med kommatecken i background-image och kombineras med background-blend-mode för rikare texturer. CSS stödjer också interpolation i färgrymder som oklch för mjukare övergångar.

- Linjär gradient är bra för ljus och riktad djupkänsla.
- Radiella och koniska gradienter passar för spotlight, ringar och färghjul.
- Justera stoppens positioner för skarpare eller mjukare övergångar.
