## Qué hace

El Convertidor de PDF a imagen renderiza cada página de un PDF local como imágenes PNG, JPG o WebP directamente en tu navegador. Puedes previsualizar una página, ajustar el DPI para obtener una salida más nítida o más pequeña, y exportar la página actual o todas las páginas como un ZIP.

## Cuándo usarlo

Úsalo cuando un sitio web, una herramienta de diseño, una aplicación de chat o un flujo de trabajo documental necesite archivos de imagen en lugar de un PDF. Resulta útil para convertir presentaciones en imágenes de vista previa, extraer capturas de páginas para revisión o preparar páginas individuales para sistemas de carga que no aceptan archivos PDF.

## Consejos para exportaciones limpias

Empieza con 144 o 200 DPI para compartir en general, y usa 300 DPI cuando el texto deba mantenerse nítido en impresión o al inspeccionarlo de cerca. PNG conserva bordes y texto sin pérdida, mientras que JPG o WebP pueden producir archivos más pequeños en documentos con muchas imágenes. Un DPI más alto y los PDF largos pueden usar más memoria, así que descarga la página actual cuando solo necesites una página.

## Privacidad y límites

Los archivos permanecen en la sesión actual del navegador y no se suben a un servidor. Los PDF protegidos con contraseña o dañados pueden no cargarse, y las páginas muy grandes pueden verse limitadas por los límites de canvas y memoria de tu navegador.
