# Random Password Generator

Generate passwords, passphrases, grouped codes, and PINs locally in your browser.

## Choose the Right Format

Use the mode that matches how the secret will be typed, read, or shared.

## Tune the Output

Adjust the character mix, separators, grouping, and similar-character filtering.

## Export and Share

Copy the current result, download it, or open it full screen before handing it off.
