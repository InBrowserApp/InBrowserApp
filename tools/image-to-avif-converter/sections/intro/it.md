## Cos'è AVIF

AVIF è un formato immagine moderno che può ridurre la dimensione dei file
mantenendo una buona qualità visiva. Supporta la trasparenza e offre un'elevata
efficienza di compressione, perciò spesso è una scelta solida per foto,
immagini di prodotto e risorse web che devono restare nitide senza sprecare
banda.

## Perché usare questo convertitore

Usa questo strumento quando ti servono immagini più leggere e pronte per il web
senza inviare file a un server. Converte le immagini localmente nel tuo
browser e ti permette di regolare qualità, velocità dell'encoder, modalità
lossless e scala dell'output prima del download. Quando converti più immagini,
lo strumento prepara anche uno ZIP per salvare facilmente il batch.

## Come funziona

1. Carica uno o più file immagine.
2. Scegli qualità AVIF, velocità, modalità lossless e scala dell'output.
3. Converti i file in un worker del browser.
4. Visualizza l'anteprima di ogni risultato, scarica singoli file AVIF oppure
   scarica lo ZIP.

## Consigli per risultati migliori

- Usa un valore di qualità intorno a 70-80 per foto e screenshot come punto di
  partenza pratico.
- Abbassa la qualità quando la dimensione del file conta più dei dettagli fini.
- Valori di velocità più bassi possono produrre file più piccoli, ma richiedono
  più tempo con immagini grandi o batch.
- Usa la modalità lossless per diagrammi, risorse UI o immagini in cui i colori
  esatti sono importanti.
- Mantieni la scala al 100% per una conversione solo di formato, oppure riducila
  quando l'immagine verrà visualizzata più piccola nella pagina.
- Confronta le dimensioni del file originale e dell'output, perché alcune
  immagini molto piccole o già compresse potrebbero non diventare più leggere
  dopo la codifica AVIF.
