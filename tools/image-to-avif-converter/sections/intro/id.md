## Apa Itu AVIF

AVIF adalah format gambar modern yang dapat mengurangi ukuran file sambil
tetap menjaga kualitas visual yang baik. Format ini mendukung transparansi dan
efisiensi kompresi yang tinggi, sehingga sering menjadi pilihan kuat untuk
foto, gambar produk, dan aset web yang harus tetap tajam tanpa membuang
bandwidth.

## Mengapa Menggunakan Konverter Ini

Gunakan alat ini saat Anda membutuhkan gambar siap web yang lebih kecil tanpa
mengirim file ke server. Alat ini mengonversi gambar secara lokal di browser
Anda, memungkinkan Anda menyesuaikan kualitas, kecepatan encoder, mode
lossless, dan skala output sebelum diunduh. Saat Anda mengonversi beberapa
gambar, alat ini juga menyiapkan ZIP agar batch mudah disimpan.

## Cara Kerjanya

1. Unggah satu atau beberapa file gambar.
2. Pilih kualitas AVIF, kecepatan, mode lossless, dan skala output.
3. Konversi file dengan worker di browser.
4. Pratinjau setiap hasil, unduh file AVIF satu per satu, atau unduh ZIP.

## Tips untuk Hasil yang Lebih Baik

- Gunakan nilai kualitas sekitar 70-80 untuk foto dan tangkapan layar sebagai
  titik awal praktis.
- Turunkan kualitas ketika ukuran file lebih penting daripada detail halus.
- Nilai kecepatan yang lebih rendah dapat menghasilkan file yang lebih kecil,
  tetapi membutuhkan waktu lebih lama pada gambar besar atau batch.
- Gunakan mode lossless untuk diagram, aset UI, atau gambar yang ketepatan
  warnanya penting.
- Pertahankan skala pada 100% untuk konversi format saja, atau kurangi ketika
  gambar akan ditampilkan lebih kecil di halaman.
- Bandingkan ukuran file asli dan output karena beberapa gambar yang sangat
  kecil atau sudah dikompresi mungkin tidak menjadi lebih kecil setelah
  pengenkodean AVIF.
