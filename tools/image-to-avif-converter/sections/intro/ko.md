## AVIF란 무엇인가요

AVIF는 좋은 시각 품질을 유지하면서 파일 크기를 줄일 수 있는 최신
이미지 형식입니다. 투명도와 높은 압축 효율을 지원하므로, 대역폭을
낭비하지 않으면서 선명도를 유지해야 하는 사진, 제품 이미지, 웹
자산에 적합한 선택이 되는 경우가 많습니다.

## 이 변환기를 사용하는 이유

파일을 서버로 보내지 않고 더 작은 웹용 이미지가 필요할 때 이 도구를
사용하세요. browser에서 로컬로 이미지를 변환하고, 다운로드 전에
품질, 인코더 속도, 무손실 모드, 출력 크기 비율을 조정할 수 있습니다.
여러 이미지를 변환하면 일괄 저장이 쉽도록 도구가 ZIP도 준비합니다.

## 작동 방식

1. 이미지 파일을 하나 이상 업로드합니다.
2. AVIF 품질, 속도, 무손실 모드, 출력 크기 비율을 선택합니다.
3. browser worker에서 파일을 변환합니다.
4. 각 결과를 미리 보고, 개별 AVIF 파일을 다운로드하거나 ZIP을 다운로드합니다.

## 더 나은 결과를 위한 팁

- 사진과 스크린샷에는 실용적인 시작점으로 품질 값을 70-80 정도로
  사용하세요.
- 세밀한 디테일보다 파일 크기가 더 중요할 때는 품질을 낮추세요.
- 속도 값이 낮을수록 더 작은 파일을 만들 수 있지만, 큰 이미지나
  일괄 작업에서는 시간이 더 오래 걸립니다.
- 정확한 색상이 중요한 다이어그램, UI 자산, 이미지에는 무손실 모드를
  사용하세요.
- 형식만 변환하려면 크기 비율을 100%로 유지하고, 이미지가 페이지에서
  더 작게 표시될 경우에는 줄이세요.
- 일부 아주 작은 이미지나 이미 압축된 이미지는 AVIF 인코딩 후에도
  더 작아지지 않을 수 있으므로 원본과 출력 파일 크기를 비교하세요.
