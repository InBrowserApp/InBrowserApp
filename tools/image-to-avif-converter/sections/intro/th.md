## AVIF คืออะไร

AVIF เป็นรูปแบบรูปภาพสมัยใหม่ที่ช่วยลดขนาดไฟล์ได้โดยยังคงคุณภาพการมองเห็นที่ดี
รองรับความโปร่งใสและมีประสิทธิภาพการบีบอัดสูง จึงมักเป็นตัวเลือกที่ดีสำหรับ
รูปถ่าย รูปภาพสินค้า และแอสเซ็ตเว็บที่ต้องคมชัดโดยไม่สิ้นเปลืองแบนด์วิดท์

## ทำไมจึงใช้ตัวแปลงนี้

ใช้เครื่องมือนี้เมื่อคุณต้องการรูปภาพสำหรับเว็บที่มีขนาดเล็กลง
โดยไม่ต้องส่งไฟล์ไปยังเซิร์ฟเวอร์ เครื่องมือนี้แปลงรูปภาพในเครื่องภายใน
browser ของคุณ ให้คุณปรับคุณภาพ ความเร็วของตัวเข้ารหัส โหมดไม่สูญเสียข้อมูล
และสเกลผลลัพธ์ก่อนดาวน์โหลด เมื่อคุณแปลงหลายรูปภาพ เครื่องมือจะเตรียม ZIP
ให้ด้วยเพื่อให้บันทึกทั้งชุดได้ง่าย

## วิธีทำงาน

1. อัปโหลดไฟล์รูปภาพหนึ่งไฟล์หรือหลายไฟล์
2. เลือกคุณภาพ AVIF ความเร็ว โหมดไม่สูญเสียข้อมูล และสเกลผลลัพธ์
3. แปลงไฟล์ใน worker ของ browser
4. ดูตัวอย่างผลลัพธ์แต่ละรายการ ดาวน์โหลดไฟล์ AVIF แยกกัน หรือดาวน์โหลด ZIP

## เคล็ดลับเพื่อผลลัพธ์ที่ดีขึ้น

- ใช้ค่าคุณภาพประมาณ 70-80 สำหรับรูปถ่ายและภาพหน้าจอเป็นจุดเริ่มต้นที่ใช้งานได้จริง
- ลดคุณภาพลงเมื่อขนาดไฟล์สำคัญกว่ารายละเอียดเล็กๆ น้อยๆ
- ค่าความเร็วที่ต่ำลงอาจสร้างไฟล์ที่เล็กกว่า แต่จะใช้เวลานานขึ้นกับรูปภาพขนาดใหญ่หรือการแปลงเป็นชุด
- ใช้โหมดไม่สูญเสียข้อมูลสำหรับไดอะแกรม แอสเซ็ต UI หรือรูปภาพที่ต้องรักษาสีให้ตรงเดิม
- คงสเกลไว้ที่ 100% สำหรับการแปลงเฉพาะรูปแบบ หรือลดลงเมื่อรูปภาพจะแสดงในขนาดเล็กลงบนหน้าเว็บ
- เปรียบเทียบขนาดไฟล์ต้นฉบับกับผลลัพธ์ เพราะรูปภาพที่เล็กมากหรือถูกบีบอัดอยู่แล้วบางไฟล์อาจไม่เล็กลงหลังเข้ารหัส AVIF
