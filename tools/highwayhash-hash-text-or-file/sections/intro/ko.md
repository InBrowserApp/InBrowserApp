## HighwayHash란 무엇인가요?

HighwayHash는 Google이 높은 처리량의 핑거프린팅과 무결성 검사를 위해 설계한 빠른 키 기반 해시 함수입니다. 256비트 키를 사용하며, 같은 텍스트 또는 파일 입력에서 64비트, 128비트 또는 256비트 출력을 생성할 수 있습니다.

## 사용 시점

- 캐시 키, 객체 ID, 샤딩 또는 내부 조회 테이블을 위한 결정적 키 기반 체크섬을 만듭니다.
- 폭넓은 암호학적 호환성보다 속도가 더 중요할 때 같은 키로 파일이나 텍스트 페이로드를 비교합니다.
- 무결성 워크플로에 더 큰 비밀번호용이 아닌 해시가 유용할 때 128비트 또는 256비트 핑거프린트를 생성합니다.

## 키 및 출력 옵션

키는 `0x000102030405060708090a0b0c0d0e0f101112131415161718191a1b1c1d1e1f`처럼 정확히 32바이트의 16진수 데이터로 입력하세요. `0x` 접두사는 선택 사항이며, 이 도구는 긴 키를 더 쉽게 읽을 수 있도록 공백, 콜론, 하이픈, 밑줄을 허용합니다. 키를 비워 두면 라이브러리 기본 키를 사용합니다. 빠른 확인에는 편리하지만 비밀로 취급해서는 안 됩니다.

## 보안 참고 사항

HighwayHash는 HMAC, 디지털 서명 또는 비밀번호 해싱을 대체하지 않습니다. 표준 암호학적 검증이 필요한 시스템 간 진위 증명이 아니라, 빠른 키 기반 핑거프린트와 체크섬 워크플로에 사용하세요.
