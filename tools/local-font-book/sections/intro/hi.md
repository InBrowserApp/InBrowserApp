## Local Font Access क्या है?

Local Font Access एक ब्राउज़र API है जो डिवाइस पर इंस्टॉल फ़ॉन्ट्स सूचीबद्ध करता है।

यह टूल आपको परिणाम खोजने, संबंधित फ़ॉन्ट रूपों की तुलना करने और चुने गए फ़ॉन्ट के लिए CSS स्निपेट कॉपी करने देता है।

यह केवल सुरक्षित संदर्भ और समर्थित ब्राउज़रों में काम करता है और उपयोगकर्ता अनुमति (local-fonts) मांगता है।

API FontData लौटाता है जिसमें family, fullName, postscriptName और style होते हैं।

### मुख्य बिंदु

- इसका उपयोग वर्तमान डिवाइस पर CSS `font-family` स्टैक के लिए जरूरी सटीक नामों की पुष्टि करने में करें।
- कॉल यूज़र जेस्चर से शुरू होने चाहिए।
- Permissions Policy कुछ साइटों पर एक्सेस रोक सकती है।
- यह टूल लोकल पर चलता है और फ़ॉन्ट अपलोड नहीं करता।
