## Local Font Access คืออะไร?

Local Font Access เป็น API ของเบราว์เซอร์ที่แสดงฟอนต์ที่ติดตั้งในอุปกรณ์

เครื่องมือนี้ช่วยให้คุณค้นหาในผลลัพธ์ เปรียบเทียบรูปแบบที่เกี่ยวข้อง และคัดลอกโค้ด CSS ของฟอนต์ที่เลือกได้

ทำงานได้เฉพาะในบริบทที่ปลอดภัยและเบราว์เซอร์ที่รองรับ และต้องมีสิทธิ์ผู้ใช้ (local-fonts)

API จะคืนค่า FontData ที่มี family, fullName, postscriptName และ style

### ประเด็นสำคัญ

- ใช้เครื่องมือนี้เพื่อตรวจสอบชื่อที่ถูกต้องสำหรับสแตก CSS `font-family` บนอุปกรณ์ปัจจุบัน
- ต้องเรียกผ่านการโต้ตอบของผู้ใช้เท่านั้น
- Permissions Policy อาจบล็อกการเข้าถึงในบางเว็บไซต์
- เครื่องมือนี้ทำงานในเครื่องและไม่อัปโหลดฟอนต์
