## Что такое Local Font Access?

Local Font Access — это API браузера для перечисления установленных на устройстве шрифтов.

Этот инструмент позволяет искать по результатам, сравнивать связанные начертания и копировать CSS-фрагмент для выбранного шрифта.

Он работает только в безопасном контексте и поддерживаемых браузерах и требует разрешения (local-fonts).

API возвращает FontData с полями family, fullName, postscriptName и style.

### Главное

- Используйте его, чтобы проверить точные имена, которые нужны для стека CSS `font-family` на текущем устройстве.
- Вызовы должны инициироваться пользовательским действием.
- Permissions Policy может блокировать доступ на некоторых сайтах.
- Этот инструмент работает локально и не загружает шрифты.
