## ما هو Local Font Access؟

Local Font Access هي واجهة API للمتصفح تعرض الخطوط المثبتة على جهازك.

تتيح لك هذه الأداة البحث في النتائج ومقارنة الأوجه المرتبطة ونسخ مقتطف CSS للخط الذي تختاره.

تعمل فقط في سياق آمن وعلى المتصفحات المدعومة وتتطلب إذن المستخدم (local-fonts).

تعيد API كائنات FontData تتضمن family وfullName وpostscriptName وstyle.

### نقاط مهمة

- استخدمها لتأكيد الأسماء الدقيقة التي تحتاجها في مكدس CSS `font-family` على الجهاز الحالي.
- يجب أن تُستدعى عبر تفاعل المستخدم.
- قد تمنعها Permissions Policy في بعض المواقع.
- هذه الأداة تعمل محليًا ولا ترفع الخطوط.
