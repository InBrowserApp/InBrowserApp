## מה זה Local Font Access?

Local Font Access הוא API של הדפדפן שמציג גופנים מותקנים במכשיר.

הכלי הזה מאפשר לך לחפש בתוצאות, להשוות בין וריאציות קשורות ולהעתיק קטע CSS עבור הגופן שבחרת.

פועל רק בהקשר מאובטח ובדפדפנים נתמכים ודורש הרשאת משתמש (local-fonts).

ה-API מחזיר FontData עם family, fullName, postscriptName ו-style.

### נקודות חשובות

- השתמשו בו כדי לאשר את השמות המדויקים הדרושים למחסנית CSS `font-family` במכשיר הנוכחי.
- יש להפעיל קריאות באמצעות מחווה של המשתמש.
- Permissions Policy יכולה לחסום גישה בחלק מהאתרים.
- הכלי פועל מקומית ולא מעלה גופנים.
