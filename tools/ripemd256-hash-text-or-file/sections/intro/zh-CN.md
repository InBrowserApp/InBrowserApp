## 什么是 RIPEMD-256？

RIPEMD-256（RACE 完整性原语评估消息摘要）是一种加密散列函数，产生 256 位（32 字节）的散列值，通常呈现为 64 个字符的十六进制数。它是由欧洲开发的 RIPEMD 散列函数家族的一部分，作为 MD4/MD5 的替代方案。

**主要特征：**

- **确定性**：相同的输入总是产生相同的散列
- **快速计算**：对于任何给定输入都能快速计算
- **雪崩效应**：输入的微小变化会产生截然不同的输出
- **固定输出大小**：无论输入大小如何，始终产生 256 位散列
- **单向性**：从散列反推原始输入在计算上是不可行的

**常见用途：**

- 数据完整性检查
- 指纹识别与去重
- 遗留系统兼容性
