## 什麼是 RIPEMD-256？

RIPEMD-256（RACE 完整性原語評估訊息摘要）是一種加密雜湊函數，產生 256 位元（32 位元組）的雜湊值，通常呈現為 64 個字元的十六進位數。它是歐洲開發的 RIPEMD 雜湊函數家族的一部分，作為 MD4/MD5 的替代方案。

**主要特徵：**

- **確定性**：相同的輸入總是產生相同的雜湊
- **快速計算**：對於任何給定輸入都能快速計算
- **雪崩效應**：輸入的微小變化會產生截然不同的輸出
- **固定輸出大小**：無論輸入大小如何，始終產生 256 位元雜湊
- **單向性**：從雜湊反推原始輸入在計算上是不可行的

**常見用途：**

- 資料完整性檢查
- 指紋識別與去重
- 遺留系統相容性
