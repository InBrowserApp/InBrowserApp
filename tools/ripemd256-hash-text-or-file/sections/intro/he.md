## מה זה RIPEMD-256?

RIPEMD-256‏ (RACE Integrity Primitives Evaluation Message Digest) הוא פונקציית האש קריפטוגרפית שמייצרת ערך האש באורך 256 ביט (32 בייט), בדרך כלל מוצג כמספר הקסדצימלי בן 64 תווים. הוא חלק ממשפחת RIPEMD שפותחה באירופה כחלופה ל‑MD4/MD5.

**מאפיינים עיקריים:**

- **דטרמיניסטי**: אותו קלט תמיד מייצר אותו האש
- **חישוב מהיר**: מהיר לחישוב עבור כל קלט נתון
- **אפקט מפולת**: שינויים קטנים בקלט מייצרים פלטים שונים בתכלית
- **גודל פלט קבוע**: תמיד מייצר האש באורך 256 ביט ללא קשר לגודל הקלט
- **חד‑כיווניות**: שחזור הקלט המקורי מתוך ההאש אינו מעשי חישובית

**שימושים נפוצים:**

- בדיקות שלמות נתונים
- טביעת אצבע והסרת כפילויות
- תאימות למערכות ישנות
