## RIPEMD-256とは？

RIPEMD-256（RACE Integrity Primitives Evaluation Message Digest）は、256ビット（32バイト）のハッシュ値を生成する暗号学的ハッシュ関数で、通常64文字の16進数として表示されます。MD4/MD5の代替として欧州で開発されたRIPEMDファミリーの一部です。

**主な特徴：**

- **決定論的**：同じ入力は常に同じハッシュを生成
- **高速計算**：任意の入力に対して高速に計算
- **雪崩効果**：入力の小さな変化が大きく異なる出力を生成
- **固定出力サイズ**：入力サイズに関係なく常に256ビット
- **一方向性**：ハッシュから元の入力を復元することは計算上困難

**一般的な用途：**

- データ整合性チェック
- フィンガープリンティングと重複排除
- レガシーシステム互換
