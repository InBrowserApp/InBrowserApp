## RIPEMD-256란?

RIPEMD-256(RACE Integrity Primitives Evaluation Message Digest)는 256비트(32바이트) 해시 값을 생성하는 암호화 해시 함수로, 일반적으로 64자리 16진수로 표현됩니다. MD4/MD5의 대안으로 유럽에서 개발된 RIPEMD 계열의 일부입니다.

**주요 특징:**

- **결정론적**: 동일한 입력은 항상 동일한 해시를 생성
- **빠른 계산**: 주어진 입력에 대해 빠르게 계산
- **눈사태 효과**: 입력의 작은 변화가 완전히 다른 출력을 생성
- **고정된 출력 크기**: 입력 크기와 무관하게 항상 256비트 해시
- **단방향성**: 해시로부터 원본 입력을 복구하는 것은 계산적으로 어렵습니다

**일반적인 용도:**

- 데이터 무결성 검사
- 핑거프린팅 및 중복 제거
- 레거시 시스템 호환성
