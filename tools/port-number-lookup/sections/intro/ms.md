## Apakah Nombor Port?

Nombor port ialah integer tidak bertanda 16-bit (0-65535) yang digunakan
untuk mengenal pasti proses atau perkhidmatan tertentu pada peranti
rangkaian. Port membenarkan pelbagai perkhidmatan rangkaian berjalan pada
alamat IP yang sama.

### Port Sistem (0-1023)

Juga dipanggil "port terkenal", dikhaskan untuk perkhidmatan biasa seperti
HTTP (80), HTTPS (443), SSH (22).

### Port Berdaftar (1024-49151)

Digunakan oleh aplikasi dan perkhidmatan yang berdaftar dengan IANA, seperti
MySQL (3306), PostgreSQL (5432).

### Port Dinamik (49152-65535)

Juga dipanggil "port sementara", digunakan untuk sambungan sementara oleh
aplikasi klien.
