## Was ist eine Portnummer?

Eine Portnummer ist eine 16-Bit-Ganzzahl ohne Vorzeichen (0-65535), die
verwendet wird, um bestimmte Prozesse oder Dienste auf einem Netzwerkgerat
zu identifizieren. Ports ermoglichen es, mehrere Netzwerkdienste auf
derselben IP-Adresse auszufuhren.

### Systemports (0-1023)

Auch "bekannte Ports" genannt, reserviert fur gangige Dienste wie HTTP (80),
HTTPS (443), SSH (22).

### Registrierte Ports (1024-49151)

Verwendet von Anwendungen und Diensten, die bei der IANA registriert sind,
wie MySQL (3306), PostgreSQL (5432).

### Dynamische Ports (49152-65535)

Auch "ephemere Ports" genannt, verwendet fur temporare Verbindungen durch
Client-Anwendungen.
