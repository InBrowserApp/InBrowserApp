## Vad ar ett Portnummer?

Ett portnummer ar ett 16-bitars heltal utan tecken (0-65535) som anvands for
att identifiera specifika processer eller tjanster pa en natverksenhet.
Portar tillater flera natverkstjanster att koras pa samma IP-adress.

### Systemportar (0-1023)

Kallas ocksa "valkanda portar", reserverade for vanliga tjanster som HTTP
(80), HTTPS (443), SSH (22).

### Registrerade Portar (1024-49151)

Anvands av applikationer och tjanster registrerade hos IANA, sasom MySQL
(3306), PostgreSQL (5432).

### Dynamiska Portar (49152-65535)

Kallas ocksa "efemara portar", anvands for tillfalliga anslutningar av
klientapplikationer.
