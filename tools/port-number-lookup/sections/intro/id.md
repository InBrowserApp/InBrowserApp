## Apa itu Nomor Port?

Nomor port adalah bilangan bulat tak bertanda 16-bit (0-65535) yang
digunakan untuk mengidentifikasi proses atau layanan tertentu pada perangkat
jaringan. Port memungkinkan beberapa layanan jaringan berjalan pada alamat
IP yang sama.

### Port Sistem (0-1023)

Juga disebut "port terkenal", dicadangkan untuk layanan umum seperti HTTP
(80), HTTPS (443), SSH (22).

### Port Terdaftar (1024-49151)

Digunakan oleh aplikasi dan layanan yang terdaftar di IANA, seperti MySQL
(3306), PostgreSQL (5432).

### Port Dinamis (49152-65535)

Juga disebut "port efemeral", digunakan untuk koneksi sementara oleh
aplikasi klien.
