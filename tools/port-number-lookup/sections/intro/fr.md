## Qu'est-ce qu'un Numero de Port ?

Un numero de port est un entier non signe de 16 bits (0-65535) utilise pour
identifier des processus ou services specifiques sur un appareil en reseau.
Les ports permettent a plusieurs services reseau de fonctionner sur la meme
adresse IP.

### Ports Systeme (0-1023)

Egalement appeles "ports bien connus", reserves aux services courants comme
HTTP (80), HTTPS (443), SSH (22).

### Ports Enregistres (1024-49151)

Utilises par les applications et services enregistres aupres de l'IANA, tels
que MySQL (3306), PostgreSQL (5432).

### Ports Dynamiques (49152-65535)

Egalement appeles "ports ephemeres", utilises pour les connexions
temporaires par les applications clientes.
