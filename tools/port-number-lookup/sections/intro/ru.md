## Что такое номер порта?

Номер порта - это 16-битное беззнаковое целое число (0-65535), используемое
для идентификации конкретных процессов или служб на сетевом устройстве.
Порты позволяют нескольким сетевым службам работать на одном IP-адресе.

### Системные порты (0-1023)

Также называются "общеизвестными портами", зарезервированы для
распространенных служб, таких как HTTP (80), HTTPS (443), SSH (22).

### Зарегистрированные порты (1024-49151)

Используются приложениями и службами, зарегистрированными в IANA, такими как
MySQL (3306), PostgreSQL (5432).

### Динамические порты (49152-65535)

Также называются "эфемерными портами", используются для временных соединений
клиентскими приложениями.
