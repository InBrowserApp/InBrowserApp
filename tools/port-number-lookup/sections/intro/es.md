## Que es un Numero de Puerto?

Un numero de puerto es un entero sin signo de 16 bits (0-65535) utilizado
para identificar procesos o servicios especificos en un dispositivo en red.
Los puertos permiten que multiples servicios de red se ejecuten en la misma
direccion IP.

### Puertos del Sistema (0-1023)

Tambien llamados "puertos conocidos", reservados para servicios comunes como
HTTP (80), HTTPS (443), SSH (22).

### Puertos Registrados (1024-49151)

Utilizados por aplicaciones y servicios registrados en IANA, como MySQL
(3306), PostgreSQL (5432).

### Puertos Dinamicos (49152-65535)

Tambien llamados "puertos efimeros", utilizados para conexiones temporales
por aplicaciones cliente.
