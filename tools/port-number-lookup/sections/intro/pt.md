## O que e um Numero de Porta?

Um numero de porta e um inteiro sem sinal de 16 bits (0-65535) usado para
identificar processos ou servicos especificos em um dispositivo de rede. As
portas permitem que varios servicos de rede sejam executados no mesmo
endereco IP.

### Portas do Sistema (0-1023)

Tambem chamadas de "portas bem conhecidas", reservadas para servicos comuns
como HTTP (80), HTTPS (443), SSH (22).

### Portas Registradas (1024-49151)

Usadas por aplicativos e servicos registrados na IANA, como MySQL (3306),
PostgreSQL (5432).

### Portas Dinamicas (49152-65535)

Tambem chamadas de "portas efemeras", usadas para conexoes temporarias por
aplicativos cliente.
