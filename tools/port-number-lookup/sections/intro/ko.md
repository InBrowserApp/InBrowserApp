## 포트 번호란?

포트 번호는 네트워크 장치에서 특정 프로세스나 서비스를 식별하는 데 사용되는 16비트 부호 없는 정수 (0-65535)입니다. 포트를
통해 여러 네트워크 서비스가 동일한 IP 주소에서 실행될 수 있습니다.

### 시스템 포트 (0-1023)

"잘 알려진 포트"라고도 하며, HTTP (80), HTTPS (443), SSH (22)와 같은 일반 서비스용으로 예약되어 있습니다.

### 등록된 포트 (1024-49151)

IANA에 등록된 애플리케이션 및 서비스에서 사용합니다. 예: MySQL (3306), PostgreSQL (5432).

### 동적 포트 (49152-65535)

"임시 포트"라고도 하며, 클라이언트 애플리케이션의 임시 연결에 사용됩니다.
