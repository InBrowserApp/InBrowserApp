## หมายเลขพอร์ตคืออะไร?

หมายเลขพอร์ตคือจำนวนเต็มไม่มีเครื่องหมาย 16 บิต (0-65535)
ที่ใช้ระบุกระบวนการหรือบริการเฉพาะบนอุปกรณ์เครือข่าย
พอร์ตช่วยให้บริการเครือข่ายหลายรายการทำงานบนที่อยู่ IP เดียวกันได้

### พอร์ตระบบ (0-1023)

เรียกอีกอย่างว่า "พอร์ตที่รู้จักกันดี" สงวนไว้สำหรับบริการทั่วไป เช่น HTTP
(80), HTTPS (443), SSH (22)

### พอร์ตที่ลงทะเบียน (1024-49151)

ใช้โดยแอปพลิเคชันและบริการที่ลงทะเบียนกับ IANA เช่น MySQL (3306), PostgreSQL
(5432)

### พอร์ตไดนามิก (49152-65535)

เรียกอีกอย่างว่า "พอร์ตชั่วคราว"
ใช้สำหรับการเชื่อมต่อชั่วคราวโดยแอปพลิเคชันไคลเอนต์
