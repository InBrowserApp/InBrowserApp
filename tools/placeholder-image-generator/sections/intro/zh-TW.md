## 這個工具能做什麼

這個佔位圖產生器可以直接在瀏覽器中建立自訂尺寸、純色或漸層背景、帶有置中文字的佔位圖。你可以將結果匯出為 PNG、SVG、JPEG 或 WebP，並支援 2x、3x Retina 高解析版本。

## 它有什麼用

它適合用在設計稿佔位、CMS 初始內容、空狀態頁面、社群媒體版型規劃，或 API 回傳前的佔位素材。當你需要一張尺寸與風格都可控的快速圖片時，不必再打開設計軟體。

## 使用方式

先選擇預設尺寸，或手動輸入寬高。接著設定純色或漸層背景，視需要加入自訂文字，再調整字體大小、匯出倍率與品質。預覽會即時更新，你可以直接下載適合目前工作的格式。

## 使用建議

- 如果希望自動顯示尺寸資訊，可以將文字輸入框留空。
- 橫幅或封面類佔位圖更適合使用漸層，通用型佔位圖更適合純色背景。
- 需要可無限縮放時優先匯出 SVG；需要點陣圖時再選擇 PNG、JPEG 或 WebP。
