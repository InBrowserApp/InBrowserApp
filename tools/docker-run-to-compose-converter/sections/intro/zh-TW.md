## 工具作用

Docker Run 轉 Compose 轉換器 會在瀏覽器中把一個或多個 `docker run` 命令轉換成
`docker-compose.yml` 檔案。你可以先檢查產生的 YAML，再複製或下載到自己的專案裡。

## 適用場景

當你想把一次性的容器啟動命令整理成可重複使用的 Compose 服務定義，或者想確認連接埠、
掛載、環境變數和資源參數會如何映射到 Compose 時，這個工具最合適。

## 使用預期

常見選項如連接埠、環境變數、重新啟動策略、健康檢查、掛載和資源限制都會盡量保留。
不支援或含義不明確的參數會以警告列出，方便你在使用前再檢查一次輸出。
