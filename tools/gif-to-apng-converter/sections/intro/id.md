## Apa yang Dilakukan Konversi GIF ke APNG?

Alat ini mengubah file GIF animasi menjadi file APNG, yaitu file PNG yang dapat berisi frame animasi. Pekerjaan tetap berlangsung secara lokal di browser, sehingga GIF asli tidak perlu diunggah ke server.

## Kasus Penggunaan yang Baik

- Memindahkan animasi GIF pendek ke alur kerja berbasis PNG.
- Mengekspor batch aset GIF sebagai file APNG untuk pengujian.
- Menyesuaikan kecepatan animasi, skala, atau perilaku pengulangan sebelum membagikan output.

## Hal yang Perlu Diperhatikan

- Output APNG menggunakan ekstensi `.png` meskipun tetap berupa animasi.
- GIF berukuran besar atau dengan jumlah frame tinggi dapat memerlukan waktu dan mungkin menghasilkan file yang lebih besar daripada sumbernya.
- Beberapa aplikasi dan platform lama masih hanya menampilkan frame pertama dari APNG.
