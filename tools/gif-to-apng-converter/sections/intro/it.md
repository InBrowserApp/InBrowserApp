## Che cosa fa la conversione da GIF ad APNG?

Questo strumento trasforma file GIF animati in file APNG, cioe file PNG che possono contenere fotogrammi di animazione. Il lavoro resta locale nel browser, quindi i GIF originali non devono essere caricati su un server.

## Buoni casi d'uso

- Spostare brevi animazioni GIF in un flusso di lavoro basato su PNG.
- Esportare un batch di risorse GIF come file APNG per i test.
- Regolare velocita dell'animazione, scala o comportamento di ripetizione prima di condividere un output.

## A cosa prestare attenzione

- Gli output APNG usano l'estensione `.png` anche se restano animati.
- I GIF grandi o con molti fotogrammi possono richiedere tempo e produrre file piu grandi del sorgente.
- Alcune app e piattaforme meno recenti mostrano ancora solo il primo fotogramma di un APNG.
