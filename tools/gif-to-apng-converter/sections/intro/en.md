## What does GIF to APNG conversion do?

This tool turns animated GIF files into APNG files, which are PNG files that can contain animation frames. It keeps the work local in the browser, so the original GIFs do not need to be uploaded to a server.

## Good use cases

- Moving short GIF animations into a PNG-based workflow.
- Exporting a batch of GIF assets as APNG files for testing.
- Adjusting animation speed, scale, or loop behavior before sharing an output.

## What to watch for

- APNG outputs use the `.png` extension even though they remain animated.
- Large or high-frame-count GIFs can take time and may produce larger files than the source.
- Some older apps and platforms still show only the first frame of an APNG.
