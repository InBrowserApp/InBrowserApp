## SHA-384とは？

SHA-384（Secure Hash Algorithm 384ビット）は、384ビット（48バイト）のハッシュ値を生成する暗号学的ハッシュ関数で、通常96文字の16進数として表示されます。NSAによって設計され、NISTによって公開されたSHA-2ハッシュ関数ファミリーの一部です。

**主な特徴：**
- **決定論的**：同じ入力は常に同じハッシュを生成
- **高速計算**：任意の入力に対して高速に計算
- **雪崩効果**：入力の小さな変化が劇的に異なる出力を生成
- **不可逆**：ハッシュを逆算して元の入力を見つけることは計算上不可能
- **衝突耐性**：同じハッシュを生成する2つの異なる入力を見つけることは非常に困難

**一般的な用途：**
- デジタル署名と証明書
- ブロックチェーンと暗号通貨（一部のシステムでSHA-384を使用）
- パスワード保存（適切なソルト付き）
- ファイル整合性検証
- プルーフオブワークアルゴリズム
