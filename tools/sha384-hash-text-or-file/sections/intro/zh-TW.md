## 什麼是 SHA-384？

SHA-384（安全雜湊演算法 384 位元）是一種加密雜湊函數，產生 384 位元（48 位元組）的雜湊值，通常呈現為 96 個字元的十六進位數。它是由 NSA 設計並由 NIST 發布的 SHA-2 雜湊函數系列的一部分。

**主要特徵：**

- **確定性**：相同的輸入總是產生相同的雜湊
- **快速計算**：對於任何給定輸入都能快速計算
- **雪崩效應**：輸入的微小變化會產生截然不同的輸出
- **不可逆**：從雜湊反推原始輸入在計算上是不可行的
- **抗碰撞**：很難找到產生相同雜湊的兩個不同輸入

**常見用途：**

- 數位簽章和憑證
- 區塊鏈和加密貨幣（一些系統使用 SHA-384）
- 密碼儲存（配合適當的加鹽）
- 檔案完整性驗證
- 工作量證明演算法
