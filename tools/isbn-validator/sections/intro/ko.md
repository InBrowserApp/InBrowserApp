## ISBN이란?

ISBN(International Standard Book Number)은 책과 판을 식별하는 번호입니다.

- `ISBN-10`: `0-306-40615-2`
- `ISBN-13`: `978-0-306-40615-7`
- `X = 10`

### ISBN-10 검사

ISBN-10은 9자리 데이터와 1자리 체크 디지트(X는 10)로 구성됩니다.

1. 하이픈과 공백 제거
2. 가중치 10~2로 곱해 합산
3. 체크 디지트로 합계가 11의 배수가 되게 함

`10×d1 + 9×d2 + ... + 2×d9 + check ≡ 0 (mod 11)`

`0-8044-2957-X`

### ISBN-13 검사

ISBN-13은 12자리 데이터와 1자리 체크 디지트로, 보통 978 또는 979로 시작합니다.

1. 1과 3을 번갈아 곱함
2. 앞 12자리를 합산
3. 체크 디지트로 합계가 10의 배수가 되게 함

`1×d1 + 3×d2 + 1×d3 + ... + 3×d12 + check ≡ 0 (mod 10)`

`978-0-306-40615-7`

유효한 ISBN-10은 978 접두사로 ISBN-13으로 변환됩니다. 978로 시작하는 ISBN-13만 역변환됩니다.

`9780306406157` -> `0306406152`
