## VINとは

VIN（Vehicle Identification Number、車両識別番号）は、車両を一意に識別するための17文字のコードです。

- `1M8GDM9AXKP042788`
- I、O、Qの文字は使用されません
- 9番目の文字はチェックディジットです

### VINの構造

1. **WMI**（1-3桁目）：世界製造者識別コード
2. **VDS**（4-8桁目）：車両記述部
3. **チェックディジット**（9桁目）：他のすべての文字から算出
4. **VIS**（10-17桁目）：車両識別部

### チェックディジット

各文字は数値に変換されます（A=1、B=2、...I、O、Qは除く）。各桁には重みが割り当てられています。重み付き合計を11で割った余りがチェックディジットとなり、10はXで表されます。

`(w1×v1 + w2×v2 + ... + w17×v17) mod 11 = チェックディジット`

このツールはフォーマットとチェックディジットのルールのみを検証します。実際の車両登録情報の確認は行いません。
