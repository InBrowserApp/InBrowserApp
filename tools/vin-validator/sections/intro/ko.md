## VIN이란?

차량 식별 번호(VIN)는 차량을 고유하게 식별하는 17자리 코드입니다.

- `1M8GDM9AXKP042788`
- 문자 I, O, Q는 사용되지 않습니다
- 9번째 문자는 체크 디짓입니다

### VIN 구조

1. **WMI** (1-3번째 자리): 세계 제조업체 식별자
2. **VDS** (4-8번째 자리): 차량 설명 구간
3. **체크 디짓** (9번째 자리): 다른 모든 문자로부터 계산됩니다
4. **VIS** (10-17번째 자리): 차량 식별자 구간

### 체크 디짓

각 문자는 숫자로 변환됩니다 (A=1, B=2, ... I, O, Q 제외). 각 자리에는 가중치가 있습니다. 가중 합계를 11로 나눈 나머지가 체크 디짓이며, 10은 X로 표시됩니다.

`(w1×v1 + w2×v2 + ... + w17×v17) mod 11 = check digit`

이 도구는 형식과 체크 디짓 규칙만 검증합니다. 실제 차량 등록 여부는 확인하지 않습니다.
