## Para que serve esta ferramenta

Use esta ferramenta quando precisar transformar YAML válido em TOML mais fácil de ler, compartilhar ou versionar. Tudo roda localmente no navegador.

## Onde ela ajuda

- Converter exemplos de API em TOML para documentação ou arquivos de configuração.
- Reescrever payloads YAML copiados em um formato mais fácil de comparar visualmente.
- Preparar conteúdo estruturado para ferramentas e fluxos que esperam TOML.

## O que observar

- Nesta ferramenta a entrada precisa ser YAML válido.
- O conversor preserva a estrutura dos dados, mas a apresentação em TOML ainda pode diferir de um estilo escrito à mão.
- Se você precisar do caminho inverso, use o conversor de TOML para YAML.
