## 此工具顯示的內容

裝置資訊會收集你目前使用裝置中瀏覽器可看見的詳細資料。它會將結果分成瀏覽器、顯示、硬體、網路、儲存空間與功能支援區段，讓你不需要安裝診斷軟體，也能快速了解網站可以偵測到哪些資訊。

## 適用情境

當你需要偵錯回應式版面、重現支援票單、比較瀏覽器、確認 Cookie 或本機儲存空間是否可用、檢查顯示尺寸，或為錯誤回報擷取精簡 JSON 快照時，可以使用此工具。在測試 canvas、WebGL、剪貼簿、Service Worker 或依賴儲存空間的功能之前，它也很有用。

## 隱私與準確性注意事項

此工具完全在你的瀏覽器中執行，不會上傳快照。瀏覽器會刻意隱藏或四捨五入部分數值，尤其是記憶體、CPU、GPU、網路與 User-Agent 詳細資料。缺少數值通常表示瀏覽器未公開該 API、頁面不在安全內容中，或隱私設定阻擋了存取。

## 如何閱讀結果

請將資料視為瀏覽器目前對你環境的觀察結果，而不是保證完整的硬體清單。調整視窗大小或旋轉裝置後，重新整理快照即可更新視窗區域、方向與顯示數值。需要與開發者或支援團隊分享實際觀察到的值時，請使用 JSON 複製操作。
