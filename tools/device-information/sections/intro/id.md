## Apa yang Ditampilkan Alat Ini

Informasi Perangkat mengumpulkan detail yang dapat dilihat browser untuk perangkat yang Anda gunakan saat ini. Alat ini mengelompokkan hasilnya ke bagian browser, tampilan, perangkat keras, jaringan, penyimpanan, dan kapabilitas agar Anda dapat cepat melihat apa yang dapat dideteksi situs web tanpa memasang perangkat lunak diagnostik.

## Kapan Ini Membantu

Gunakan saat Anda perlu men-debug layout responsif, mereproduksi tiket dukungan, membandingkan browser, memastikan apakah cookie atau local storage tersedia, memeriksa dimensi layar, atau mengambil cuplikan JSON ringkas untuk laporan bug. Alat ini juga berguna sebelum menguji fitur canvas, WebGL, clipboard, service worker, atau fitur yang bergantung pada penyimpanan.

## Catatan Privasi dan Akurasi

Alat ini berjalan sepenuhnya di browser Anda dan tidak mengunggah cuplikan. Browser sengaja menyembunyikan atau membulatkan beberapa nilai, terutama detail memori, CPU, GPU, jaringan, dan user-agent. Nilai yang hilang biasanya berarti browser tidak mengekspos API tersebut, halaman tidak berada dalam konteks aman, atau pengaturan privasi memblokir akses.

## Cara Membaca Hasilnya

Perlakukan data sebagai tampilan browser saat ini terhadap lingkungan Anda, bukan inventaris perangkat keras yang dijamin. Ubah ukuran jendela atau putar perangkat, lalu segarkan cuplikan untuk memperbarui nilai viewport, orientasi, dan tampilan. Gunakan aksi salin JSON saat Anda perlu membagikan nilai persis yang teramati kepada pengembang atau tim dukungan.
