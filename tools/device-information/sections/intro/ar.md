## ما تعرضه هذه الأداة

تجمع معلومات الجهاز التفاصيل التي يستطيع المتصفح رؤيتها عن الجهاز الذي تستخدمه الآن. تنظّم النتائج في أقسام المتصفح والشاشة والعتاد والشبكة والتخزين والإمكانات، حتى تتمكن بسرعة من معرفة ما يستطيع موقع ويب اكتشافه دون تثبيت برامج تشخيص.

## متى تفيد

استخدمها عندما تحتاج إلى تصحيح تخطيطات متجاوبة، أو إعادة إنتاج تذاكر دعم، أو مقارنة المتصفحات، أو تأكيد ما إذا كانت ملفات تعريف الارتباط أو التخزين المحلي متاحة، أو التحقق من أبعاد الشاشة، أو التقاط لقطة JSON صغيرة لتقرير خطأ. وهي مفيدة أيضًا قبل اختبار ميزات تعتمد على canvas أو WebGL أو الحافظة أو Service Worker أو التخزين.

## ملاحظات حول الخصوصية والدقة

تعمل الأداة بالكامل داخل متصفحك ولا ترفع اللقطة. تخفي المتصفحات عمدًا بعض القيم أو تقرّبها، وخصوصًا تفاصيل الذاكرة وCPU وGPU والشبكة ووكيل المستخدم. تعني القيم المفقودة عادةً أن المتصفح لا يتيح واجهة API تلك، أو أن الصفحة ليست في سياق آمن، أو أن أحد إعدادات الخصوصية حظر الوصول.

## كيفية قراءة النتائج

تعامل مع البيانات على أنها رؤية المتصفح الحالية لبيئتك، وليست قائمة مضمونة بمكونات العتاد. غيّر حجم النافذة أو أدر الجهاز، ثم حدّث اللقطة لتحديث قيم منفذ العرض والاتجاه والشاشة. استخدم إجراء نسخ JSON عندما تحتاج إلى مشاركة القيم المرصودة بدقة مع مطوّر أو فريق دعم.
