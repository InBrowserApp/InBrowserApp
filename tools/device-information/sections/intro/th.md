## เครื่องมือนี้แสดงอะไร

เครื่องมือข้อมูลอุปกรณ์จะรวบรวมรายละเอียดที่เบราว์เซอร์มองเห็นของอุปกรณ์ที่คุณกำลังใช้อยู่ตอนนี้ โดยจัดผลลัพธ์เป็นส่วนของเบราว์เซอร์ จอแสดงผล ฮาร์ดแวร์ เครือข่าย พื้นที่เก็บข้อมูล และความสามารถ เพื่อให้คุณเห็นอย่างรวดเร็วว่าเว็บไซต์ตรวจจับอะไรได้บ้างโดยไม่ต้องติดตั้งซอฟต์แวร์วินิจฉัย

## เมื่อไรที่มีประโยชน์

ใช้เมื่อคุณต้องการดีบักเลย์เอาต์แบบตอบสนอง ทำซ้ำปัญหาจาก ticket ฝ่ายสนับสนุน เปรียบเทียบเบราว์เซอร์ ยืนยันว่าคุกกี้หรือ local storage พร้อมใช้งาน ตรวจสอบขนาดจอแสดงผล หรือบันทึกสแนปชอต JSON ขนาดกะทัดรัดสำหรับรายงานบั๊ก นอกจากนี้ยังมีประโยชน์ก่อนทดสอบฟีเจอร์ที่พึ่งพา canvas, WebGL, clipboard, service worker หรือ storage

## หมายเหตุเรื่องความเป็นส่วนตัวและความแม่นยำ

เครื่องมือนี้ทำงานทั้งหมดในเบราว์เซอร์ของคุณและไม่อัปโหลดสแนปชอต เบราว์เซอร์ตั้งใจซ่อนหรือปัดค่าบางค่า โดยเฉพาะรายละเอียดหน่วยความจำ CPU, GPU, เครือข่าย และ user-agent ค่าที่หายไปมักหมายความว่าเบราว์เซอร์ไม่ได้เปิดเผย API นั้น หน้าเว็บไม่ได้อยู่ใน secure context หรือการตั้งค่าความเป็นส่วนตัวบล็อกการเข้าถึง

## วิธีอ่านผลลัพธ์

ให้มองข้อมูลเป็นมุมมองปัจจุบันของเบราว์เซอร์ต่อสภาพแวดล้อมของคุณ ไม่ใช่รายการฮาร์ดแวร์ที่รับประกันถูกต้อง ปรับขนาดหน้าต่างหรือหมุนอุปกรณ์ แล้วรีเฟรชสแนปชอตเพื่ออัปเดตค่าวิวพอร์ต การวางแนว และจอแสดงผล ใช้คำสั่งคัดลอก JSON เมื่อคุณต้องการแชร์ค่าที่สังเกตได้จริงกับนักพัฒนาหรือทีมสนับสนุน
