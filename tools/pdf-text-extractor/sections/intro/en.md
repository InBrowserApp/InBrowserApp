## What this tool does

PDF Text Extractor reads a PDF in your browser and pulls out text that is
already selectable inside the document. The output is plain text that you can
copy or download as a TXT file for notes, search indexing, data review, or
downstream processing.

## Good use cases

- Copying text from reports, contracts, statements, manuals, or research PDFs
  without opening a desktop PDF editor.
- Checking whether a document has real embedded text before sending it to a
  search index, translation workflow, accessibility review, or automation job.
- Extracting a lightweight TXT copy for support tickets, document audits,
  content cleanup, or quick comparison with another version.
- Spotting pages that may be scans because they contain image content but no
  selectable text.

## Privacy notes

The PDF is processed locally in your browser and is not uploaded by this tool.
The extracted text can still contain personal data, contract terms, account
numbers, hidden headers, or other sensitive content. Review the result before
sharing it or pasting it into another service.

## Limitations

This tool is not an OCR engine. Scanned pages, photographed pages, flattened
image-only PDFs, and some encrypted or damaged files may produce little or no
text. Layout is normalized into plain text, so columns, tables, footnotes, and
reading order can differ from the visual page. Use a dedicated OCR or PDF
repair tool when the source document does not contain selectable text.
