## ที่อยู่ IPv6 เฉพาะภายในคืออะไร?

ที่อยู่ IPv6 เฉพาะภายใน (ULA) มีไว้สำหรับการสื่อสารภายในไซต์ เครือข่ายส่วนตัว และองค์กรที่เชื่อมต่อกัน พื้นที่แอดเดรส ULA ทั้งหมดคือ `fc00::/7` บิตที่แปดคือ **บิต L**: ค่า `1` จะเลือกช่วง `fd00::/8` แบบจัดสรรภายในที่ตัวสร้างนี้ใช้ ส่วนอีกครึ่งหนึ่งคือ `fc00::/8` ยังคงสงวนไว้สำหรับวิธีจัดสรรแบบอื่น

โดยปกติ ULA จะไม่สามารถเข้าถึงได้จากทั่วโลก แต่คำว่า “ภายใน” ไม่ได้หมายความว่าเป็นความลับหรือปลอดภัยโดยอัตโนมัติ เมื่อผู้ดูแลกำหนดค่าเส้นทางเหล่านั้น ที่อยู่เหล่านี้สามารถข้ามขอบเขตไซต์ที่เชื่อมต่อด้วยการกำหนดเส้นทาง ผ่าน VPN และผ่านการเชื่อมต่อส่วนตัวได้

## ตัวสร้าง RFC 4193 นี้สร้าง /48 อย่างไร

ตัวสร้าง RFC 4193 นี้ขอบิตสุ่ม 40 บิตพอดีจาก Web Crypto API แล้วนำมารวมกับ `fd` ผลลัพธ์คือพรีฟิกซ์ของไซต์ขนาด 48 บิตที่ไม่ซ้ำกันในทางสถิติ เช่น `fd12:3456:789a::/48` การสร้างเกิดขึ้นในเบราว์เซอร์ โดยไม่มีการเก็บที่อยู่ MAC การประทับเวลา ตัวระบุอุปกรณ์ หรือการตอบกลับจากเซิร์ฟเวอร์

Global ID มีค่าที่เป็นไปได้ `2^40` ค่า หรือประมาณ 1.1 ล้านล้านค่า การสุ่มที่ปลอดภัยทำให้โอกาสนำค่าเดิมมาใช้โดยไม่ตั้งใจต่ำ แต่ไม่สามารถรับประกันได้ว่าพรีฟิกซ์สองค่าที่สร้างแยกกันจะไม่ชนกัน บันทึก `/48` ที่เลือกไว้ในเอกสารเครือข่ายและนำกลับมาใช้ซ้ำอย่างสม่ำเสมอ

## วางแผนซับเน็ต /64 ที่ใช้ได้ทั้ง 65,536 เครือข่าย

หลังพรีฟิกซ์ของไซต์ `/48` จะเป็น Subnet ID ขนาด 16 บิต ค่าตั้งแต่ `0000` ถึง `ffff` ทำให้มีเครือข่าย `/64` ได้ 65,536 เครือข่าย ตัวอย่างเช่น Subnet ID `00a0` จะเปลี่ยน `fd12:3456:789a::/48` ให้เป็นเครือข่ายมาตรฐาน `fd12:3456:789a:a0::/64`

64 บิตที่เหลือคือ Interface ID เครื่องมือนี้วางแผนเฉพาะพรีฟิกซ์เครือข่าย ไม่ได้สร้างที่อยู่โฮสต์ `/128` หรือคำนวณตัวระบุอินเทอร์เฟซจากที่อยู่ MAC

## ULA เหมาะกับที่ใด—และไม่เหมาะกับที่ใด

ULA เหมาะกับการกำหนดที่อยู่ภายในที่คงที่ ไซต์ที่เชื่อมต่อผ่าน VPN เครือข่ายแล็บ และบริการที่ควรรักษาพรีฟิกซ์ภายในไว้ขณะใช้ IPv6 แบบโกลบอลยูนิคาสต์ด้วย ULA ไม่ใช่ไฟร์วอลล์หรือขอบเขตความปลอดภัยโดยตัวมันเอง ใช้การควบคุมการเข้าถึงตามปกติ กรองทราฟฟิก ULA ที่ไม่เหมาะสมที่ขอบเขตของไซต์ และอย่าเผยแพร่ระเบียน ULA ภายในไปยัง DNS สาธารณะ

โฮสต์หนึ่งเครื่องสามารถใช้ ULA และที่อยู่โกลบอลยูนิคาสต์พร้อมกันได้ ใช้ที่อยู่โกลบอลเพื่อให้เข้าถึงผ่านอินเทอร์เน็ตได้ และใช้พรีฟิกซ์ ULA ที่คงที่สำหรับเส้นทางส่วนตัวที่ต้องการพรีฟิกซ์ดังกล่าว
