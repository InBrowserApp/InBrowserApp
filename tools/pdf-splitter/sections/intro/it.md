## Cosa fa questo strumento

PDF Splitter ti consente di aprire un PDF nel browser, scegliere le pagine per
intervallo o per numero di pagina e generare un documento più piccolo. Puoi
estrarre le pagine selezionate in un unico PDF, dividere ogni intervallo
inserito in un PDF separato oppure dividere ogni pagina selezionata nel proprio
file e scaricare i risultati come archivio ZIP.

## Casi d'uso adatti

- Estrarre alcune pagine da un contratto, report, manuale o scansione lungo
  prima di condividerle con qualcun altro.
- Separare capitoli, fatture, moduli o sezioni di allegati in singoli file PDF.
- Rimuovere le pagine che non ti servono prima di inviare un documento a una
  copisteria, a un help desk o a un flusso di approvazione.
- Creare divisioni ripetibili con una sintassi di intervalli come
  `1-3,5,8-10` invece di fare clic manualmente su ogni pagina.

## Come funzionano gli intervalli di pagine

Usa numeri di pagina e intervalli inclusivi separati da virgole. `1-3,5,8-10`
seleziona le pagine 1, 2, 3, 5, 8, 9 e 10. Una pagina può comparire una sola
volta nell'espressione, e gli intervalli decrescenti come `7-4` vengono
rifiutati così l'ordine di output resta chiaro e prevedibile.

Per un singolo PDF di output, le pagine selezionate vengono copiate in un nuovo
documento nell'ordine indicato dall'espressione dell'intervallo. Per più PDF di
output, "un file per intervallo" mantiene insieme ogni segmento inserito, mentre
"un file per pagina" crea un PDF separato per ogni pagina selezionata.

## Note sulla privacy

Il PDF viene elaborato localmente nel tuo browser e non viene caricato da questo
strumento. I link di download generati sono URL oggetto temporanei che esistono
solo nella scheda corrente. Controlla i file risultanti prima di condividerli,
perché le pagine copiate possono comunque contenere metadati incorporati,
annotazioni, valori di moduli o contenuti nascosti dal documento originale.

## Limitazioni

I PDF crittografati, protetti da password o danneggiati potrebbero non aprirsi
nella libreria PDF lato browser. Questo strumento copia le pagine in nuovi PDF,
ma non è uno strumento di oscuramento visivo e non garantisce la rimozione di
tutti i metadati del documento. Per oscuramento legale, correzioni di
accessibilità o ottimizzazione avanzata, usa un editor PDF dedicato dopo la
divisione.
