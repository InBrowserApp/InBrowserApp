## 이 도구가 하는 일

PDF 분할기는 브라우저에서 PDF를 열고, 범위 또는 페이지 번호로 페이지를
선택한 뒤 더 작은 문서를 생성할 수 있게 해줍니다. 선택한 페이지를 하나의
PDF로 추출하거나, 입력한 각 범위를 별도 PDF로 나누거나, 선택한 모든
페이지를 각각의 파일로 분할해 결과를 ZIP 아카이브로 다운로드할 수 있습니다.

## 적합한 사용 사례

- 긴 계약서, 보고서, 매뉴얼 또는 스캔본에서 몇 페이지만 뽑아 다른 사람에게
  공유하기.
- 장, 송장, 양식 또는 첨부 섹션을 개별 PDF 파일로 분리하기.
- 문서를 인쇄소, 지원 데스크 또는 승인 워크플로에 보내기 전에 필요 없는
  페이지 제거하기.
- 모든 페이지를 수동으로 클릭하는 대신 `1-3,5,8-10` 같은 범위 구문으로
  반복 가능한 분할 만들기.

## 페이지 범위 작동 방식

쉼표로 구분한 페이지 번호와 포함 범위를 사용하세요. `1-3,5,8-10`은 1, 2,
3, 5, 8, 9, 10페이지를 선택합니다. 페이지는 표현식에 한 번만 나타날 수
있으며, `7-4`처럼 내림차순 범위는 출력 순서가 명확하고 예측 가능하도록
거부됩니다.

하나의 출력 PDF에서는 선택한 페이지가 범위 표현식에 표시된 순서대로 새 문서
하나에 복사됩니다. 여러 출력 PDF에서는 "범위마다 파일 하나"가 입력한 각
구간을 함께 유지하고, "페이지마다 파일 하나"가 선택한 각 페이지마다 별도의
PDF를 만듭니다.

## 개인정보 보호 참고 사항

PDF는 브라우저에서 로컬로 처리되며 이 도구를 통해 업로드되지 않습니다.
생성된 다운로드 링크는 현재 탭에만 존재하는 임시 객체 URL입니다. 복사된
페이지에는 원본 문서의 내장 메타데이터, 주석, 양식 값 또는 숨겨진 콘텐츠가
여전히 포함될 수 있으므로 공유 전에 결과 파일을 검토하세요.

## 제한 사항

암호화되었거나 비밀번호로 보호되었거나 손상된 PDF는 브라우저 측 PDF
라이브러리에서 열리지 않을 수 있습니다. 이 분할기는 페이지를 새 PDF로
복사하지만, 시각적 가림 처리 도구가 아니며 모든 문서 메타데이터 제거를
보장하지 않습니다. 법적 가림 처리, 접근성 보정 또는 고급 최적화가 필요하면
분할 후 전용 PDF 편집기를 사용하세요.
