## ¿Qué es un Slug?

Un slug es una versión de una cadena de texto adaptada para URLs, que se utiliza normalmente para crear URLs legibles y optimizadas para SEO. Convierte el texto eliminando caracteres especiales, reemplazando los espacios con guiones y pasándolo a minúsculas.

**Ejemplos:**

- "Hello World!" → `hello-world`
- "My Blog Post Title" → `my-blog-post-title`
- "Product: iPhone 15 Pro" → `product-iphone-15-pro`

## ¿Por qué usar Slugs?

Los slugs mejoran la legibilidad de las URLs, ayudan con el SEO y hacen que los enlaces sean más fáciles de compartir y recordar. La mayoría de los sistemas de gestión de contenido y frameworks web utilizan slugs para generar URLs limpias y descriptivas a partir de títulos de páginas o nombres de artículos.

## Transliteración Unicode

Esta herramienta admite la transliteración Unicode, lo que significa que puede manejar escrituras no latinas como chino, japonés, coreano, cirílico, árabe y más. Los caracteres no ASCII se convierten a sus equivalentes latinos más cercanos antes de generar el slug.
