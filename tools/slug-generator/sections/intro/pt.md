## O que é um Slug?

Um slug é uma versão amigável para URLs de uma string, normalmente utilizado para criar URLs legíveis e otimizados para SEO. Ele converte texto removendo caracteres especiais, substituindo espaços por hifens e convertendo para minúsculas.

**Exemplos:**

- "Hello World!" → `hello-world`
- "My Blog Post Title" → `my-blog-post-title`
- "Product: iPhone 15 Pro" → `product-iphone-15-pro`

## Por que usar Slugs?

Os slugs melhoram a legibilidade das URLs, ajudam no SEO e tornam os links mais fáceis de compartilhar e memorizar. A maioria dos sistemas de gerenciamento de conteúdo e frameworks web utiliza slugs para gerar URLs limpas e descritivas a partir de títulos de páginas ou nomes de artigos.

## Transliteração Unicode

Esta ferramenta suporta transliteração Unicode, o que significa que pode lidar com scripts não latinos como chinês, japonês, coreano, cirílico, árabe e outros. Caracteres não ASCII são convertidos para os equivalentes latinos mais próximos antes de gerar o slug.
