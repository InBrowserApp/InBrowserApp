## Что такое chmod?

`chmod` ("change mode") — это команда Unix/Linux для изменения прав доступа к файлам и каталогам. Этот калькулятор позволяет переключаться между числовыми правами вроде `755`, символьными правами вроде `rwxr-xr-x` и матрицей флажков без ручных вычислений.

## Как работают числовые права

Каждая цифра отвечает за одну роль: владелец, группа и остальные. Внутри каждой цифры `4` означает чтение, `2` запись, а `1` выполнение. Складывайте значения, чтобы получить нужное право: `7 = rwx`, `6 = rw-`, `5 = r-x` и `4 = r--`. Для каталогов бит выполнения также означает, что в каталог можно войти.

## Распространённые примеры chmod

- `chmod 755 script.sh` даёт владельцу полный доступ и позволяет всем остальным читать и выполнять файл.
- `chmod 644 notes.txt` оставляет файл доступным для записи владельцу, а всем остальным — только для чтения.
- `chmod 600 .env` часто используют для приватных секретов, потому что только владелец может читать или записывать файл.
- `chmod 775 shared-folder` удобен для командных каталогов, когда группа тоже должна иметь возможность создавать и изменять файлы.
