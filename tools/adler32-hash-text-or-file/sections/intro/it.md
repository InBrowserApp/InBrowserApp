## Cos’è Adler-32?

Adler-32 è un algoritmo di checksum veloce che produce un valore a 32 bit (di solito 8 caratteri esadecimali). È pensato per rilevare errori accidentali, non per sicurezza crittografica.

**Punti chiave:**

- **Veloce e deterministico**: Lo stesso input produce sempre lo stesso output
- **Controllo integrità**: Utile per rilevare corruzioni in trasferimento o archiviazione
- **Non crittografico**: Non usare per password, firme o anti-manomissione

**Usi comuni:**

- Verifica trasferimento file
- Verifica archivi/pacchetti
- Controlli leggeri di integrità
