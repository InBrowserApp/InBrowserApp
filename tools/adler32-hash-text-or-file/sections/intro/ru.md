## Что такое Adler-32?

Adler-32 — это быстрый алгоритм контрольной суммы, выдающий 32-битное значение (обычно 8 шестнадцатеричных символов). Он предназначен для обнаружения случайных ошибок, а не для криптографической безопасности.

**Ключевые моменты:**

- **Быстрый и детерминированный**: одинаковый ввод всегда дает одинаковый результат
- **Проверка целостности**: полезен для выявления повреждений при передаче и хранении
- **Не криптографический**: не использовать для паролей, подписей и защиты от подмены

**Типичные применения:**

- Проверка передачи файлов
- Проверка архивов/пакетов
- Легкие проверки целостности
