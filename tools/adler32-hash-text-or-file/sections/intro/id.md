## Apa itu Adler-32?

Adler-32 adalah algoritma checksum cepat yang menghasilkan nilai 32-bit (biasanya 8 karakter heksadesimal). Algoritma ini dirancang untuk mendeteksi kesalahan tidak sengaja, bukan untuk keamanan kriptografis.

**Poin penting:**

- **Cepat dan deterministik**: Input yang sama selalu menghasilkan output yang sama
- **Pemeriksaan integritas**: Berguna untuk mendeteksi korupsi saat transfer atau penyimpanan
- **Bukan kriptografis**: Jangan gunakan untuk kata sandi, tanda tangan, atau anti-manipulasi

**Penggunaan umum:**

- Pemeriksaan transfer file
- Verifikasi arsip/paket
- Pemeriksaan integritas ringan
