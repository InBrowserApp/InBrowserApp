## Adler-32 คืออะไร?

Adler-32 คืออัลกอริทึมเช็กซัมที่ทำงานได้รวดเร็วและให้ผลลัพธ์ 32 บิต (ปกติแสดงเป็นเลขฐานสิบหก 8 ตัวอักษร) ใช้สำหรับตรวจจับข้อผิดพลาดที่เกิดขึ้นโดยไม่ตั้งใจ ไม่ได้ออกแบบมาเพื่อความปลอดภัยเชิงเข้ารหัส

**ประเด็นสำคัญ:**

- **เร็วและให้ผลคงที่**: อินพุตเดิมได้เอาต์พุตเดิมเสมอ
- **ตรวจสอบความสมบูรณ์ของข้อมูล**: เหมาะกับการตรวจจับข้อมูลเสียหายระหว่างส่งหรือจัดเก็บ
- **ไม่ใช่อัลกอริทึมเข้ารหัส**: ไม่ควรใช้กับรหัสผ่าน ลายเซ็น หรือการป้องกันการแก้ไข

**การใช้งานทั่วไป:**

- ตรวจสอบการโอนไฟล์
- ตรวจสอบไฟล์บีบอัด/แพ็กเกจ
- ตรวจสอบความสมบูรณ์แบบเบา ๆ
