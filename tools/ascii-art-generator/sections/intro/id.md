## Apa yang dilakukan alat ini

Alat ini mengubah teks biasa menjadi seni ASCII menggunakan font FIGlet. Pilih
dari ratusan font bawaan untuk membuat banner teks bergaya yang seluruhnya
terdiri dari karakter keyboard.

## Kapan ini berguna

- Menambahkan header menarik ke file README dan komentar kode sumber.
- Membuat banner teks untuk layar pembuka terminal dan alat CLI.
- Menghasilkan judul dekoratif untuk email teks biasa dan pesan chat.

## Cara kerjanya

Font dimuat sesuai kebutuhan agar halaman tetap cepat. Pilih font, ketik teks
Anda, dan seni ASCII akan diperbarui secara instan. Salin hasilnya dengan satu
klik.
