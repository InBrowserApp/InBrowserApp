## Was dieses Tool macht

Dieses Tool wandelt einfachen Text mithilfe von FIGlet-Schriftarten in ASCII-Art um. Wählen Sie aus Hunderten integrierter Schriftarten, um stilisierte Textbanner zu erstellen, die ausschließlich aus Tastaturzeichen bestehen.

## Wofür es nützlich ist

- Auffällige Überschriften für README-Dateien und Quellcode-Kommentare erstellen.
- Textbanner für Terminal-Begrüßungsbildschirme und CLI-Tools erzeugen.
- Dekorative Überschriften für Nur-Text-E-Mails und Chat-Nachrichten generieren.

## So funktioniert es

Schriftarten werden bei Bedarf geladen, damit die Seite schnell bleibt. Wählen Sie eine Schriftart, geben Sie Ihren Text ein, und die ASCII-Art wird sofort aktualisiert. Kopieren Sie das Ergebnis mit einem Klick.
