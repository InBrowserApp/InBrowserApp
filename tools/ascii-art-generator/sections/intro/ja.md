## このツールについて

プレーンテキストを FIGlet フォントを使ってアスキーアートに変換するツールです。数百種類の組み込みフォントから選んで、キーボード文字だけで構成されたスタイリッシュなテキストバナーを作成できます。

## 活用シーン

- README ファイルやソースコードのコメントに目を引くヘッダーを追加する。
- ターミナルのスプラッシュスクリーンや CLI ツール用のテキストバナーを作成する。
- プレーンテキストのメールやチャットメッセージに装飾的な見出しを生成する。

## 仕組み

フォントはオンデマンドで読み込まれるため、ページの表示速度は快適です。フォントを選択してテキストを入力すると、アスキーアートが即座に更新されます。ワンクリックで結果をコピーできます。
