## O que esta ferramenta faz

Esta ferramenta converte texto simples em arte ASCII usando fontes FIGlet. Escolha
entre centenas de fontes integradas para criar banners de texto estilizados
compostos inteiramente por caracteres do teclado.

## Onde ela ajuda

- Adicionar cabeçalhos chamativos a arquivos README e comentários em código-fonte.
- Criar banners de texto para telas de abertura de terminal e ferramentas CLI.
- Gerar títulos decorativos para e-mails e mensagens de chat em texto simples.

## Como funciona

As fontes são carregadas sob demanda para que a página permaneça rápida. Selecione
uma fonte, digite seu texto e a arte ASCII é atualizada instantaneamente. Copie o
resultado com um clique.
