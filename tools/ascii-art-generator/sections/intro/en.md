## What this tool does

This tool converts plain text into ASCII art using FIGlet fonts. Choose from
hundreds of built-in fonts to create stylized text banners made entirely of
keyboard characters.

## Where it helps

- Adding eye-catching headers to README files and source-code comments.
- Creating text banners for terminal splash screens and CLI tools.
- Generating decorative headings for plain-text emails and chat messages.

## How it works

Fonts are loaded on demand so the page stays fast. Select a font, type your
text, and the ASCII art updates instantly. Copy the result with one click.
