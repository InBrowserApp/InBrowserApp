## Qué hace esta herramienta

Esta herramienta convierte texto sin formato en arte ASCII utilizando fuentes
FIGlet. Elige entre cientos de fuentes integradas para crear banners de texto
estilizados compuestos enteramente por caracteres del teclado.

## Para qué es útil

- Agregar encabezados llamativos a archivos README y comentarios en el código fuente.
- Crear banners de texto para pantallas de inicio de terminal y herramientas CLI.
- Generar encabezados decorativos para correos electrónicos de texto plano y mensajes de chat.

## Cómo funciona

Las fuentes se cargan bajo demanda para que la página se mantenga rápida.
Selecciona una fuente, escribe tu texto y el arte ASCII se actualiza al
instante. Copia el resultado con un solo clic.
