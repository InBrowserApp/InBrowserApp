## When to use it

Use this tool when a plain heading feels too flat and you need a retro text banner for a README, changelog, terminal demo, or release note.

## What this version focuses on

It keeps the workflow tight: type a short line, switch between a curated set of FIGlet fonts, adjust alignment and width, then copy or download the final block.

## Good fits

It works well for developer docs, CLI screenshots, launch posts, event slates, and any text-first layout that benefits from a lightweight display treatment.
