## 이 도구의 기능

이 도구는 FIGlet 폰트를 사용하여 일반 텍스트를 ASCII 아트로 변환합니다. 수백
가지 내장 폰트 중에서 선택하여 키보드 문자만으로 구성된 스타일리시한 텍스트
배너를 만들 수 있습니다.

## 활용 사례

- README 파일이나 소스 코드 주석에 눈에 띄는 헤더 추가.
- 터미널 스플래시 화면 및 CLI 도구용 텍스트 배너 제작.
- 일반 텍스트 이메일이나 채팅 메시지에 장식적인 제목 생성.

## 작동 방식

폰트는 필요할 때 로드되므로 페이지가 빠르게 유지됩니다. 폰트를 선택하고
텍스트를 입력하면 ASCII 아트가 즉시 업데이트됩니다. 클릭 한 번으로 결과를
복사할 수 있습니다.
