## Qué hace el Lector de códigos de barras

El Lector de códigos de barras extrae el valor almacenado en códigos comunes
unidimensionales y bidimensionales desde una imagen local o desde tu cámara.
Admite QR Code, Data Matrix, PDF417, Aztec, Code 128, Code 39, EAN, UPC, ITF,
Codabar y otros formatos gestionados por el escáner en el navegador.

## Cuándo usarlo

Úsalo cuando tengas una etiqueta de producto, una etiqueta de envío, un boleto,
una etiqueta de inventario, una foto de un envase, una captura de pantalla o una
imagen de documento, y necesites inspeccionar el valor codificado exacto antes
de actuar sobre él. El contenido decodificado permanece visible para que puedas
copiarlo, abrir enlaces compatibles o compararlo con la imagen de origen.

## Privacidad y comportamiento de la cámara

Las imágenes se decodifican localmente en el navegador y no se suben. El
escaneo con cámara usa el aviso de permiso de cámara del navegador y luego se
detiene automáticamente después de decodificar un código. También puedes detener
la cámara manualmente en cualquier momento.

## Consejos para mejores escaneos

- Usa una imagen clara donde se vea el código completo y no esté estirado.
- Mantén las etiquetas planas, nítidas y bien iluminadas al escanear con la
  cámara.
- En códigos de barras unidimensionales largos, deja espacio libre en ambos
  extremos de las barras.
- Si una foto no se escanea, recorta alrededor del código o prueba con una
  imagen de mayor resolución.
