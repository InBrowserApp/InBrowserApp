## מה עושה קורא הברקודים

קורא הברקודים מחלץ את הערך המאוחסן בקודים חד-ממדיים ודו-ממדיים
נפוצים מתוך תמונה מקומית או מהמצלמה שלכם. הוא תומך ב-QR Code,
Data Matrix, PDF417, Aztec, Code 128, Code 39, EAN, UPC, ITF,
Codabar ובפורמטים נוספים שהסורק בדפדפן יודע לטפל בהם.

## מתי להשתמש בו

השתמשו בו כשיש לכם תווית מוצר, תווית משלוח, כרטיס, תג מלאי,
צילום אריזה, צילום מסך או תמונת מסמך, ואתם צריכים לבדוק את הערך
המקודד המדויק לפני פעולה. התוכן שפוענח נשאר גלוי כדי שתוכלו להעתיק
אותו, לפתוח קישורים נתמכים או להשוות אותו לתמונת המקור.

## פרטיות והתנהגות המצלמה

תמונות מפוענחות מקומית בדפדפן ואינן מועלות. סריקה במצלמה משתמשת
בבקשת הרשאת המצלמה של הדפדפן, ואז נעצרת אוטומטית אחרי שקוד
מפוענח. אפשר גם לעצור את המצלמה ידנית בכל זמן.

## טיפים לסריקות טובות יותר

- השתמשו בתמונה ברורה שבה כל הקוד גלוי ואינו מעוות.
- בעת סריקה במצלמה, שמרו על תוויות שטוחות, חדות ומוארות היטב.
- בברקודים חד-ממדיים ארוכים, השאירו שטח ריק משני קצות הפסים.
- אם צילום אינו נסרק, בצעו חיתוך סביב הקוד או נסו תמונה ברזולוציה
  גבוהה יותר.
