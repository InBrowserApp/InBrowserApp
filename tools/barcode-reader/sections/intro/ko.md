## 바코드 리더의 기능

바코드 리더는 로컬 이미지나 카메라에서 일반적인 1차원 및 2차원 코드에
저장된 값을 추출합니다. QR Code, Data Matrix, PDF417, Aztec, Code 128,
Code 39, EAN, UPC, ITF, Codabar 및 브라우저 내 스캐너가 처리하는 기타
형식을 지원합니다.

## 사용하면 좋은 경우

제품 라벨, 배송 라벨, 티켓, 재고 태그, 포장 사진, 스크린샷 또는 문서
이미지가 있고, 작업하기 전에 정확히 인코딩된 값을 확인해야 할 때
사용하세요. 해독된 콘텐츠는 계속 표시되므로 복사하거나, 지원되는 링크를
열거나, 원본 이미지와 비교할 수 있습니다.

## 개인정보 보호 및 카메라 동작

이미지는 브라우저에서 로컬로 해독되며 업로드되지 않습니다. 카메라 스캔은
브라우저의 카메라 권한 요청을 사용하며, 코드가 해독되면 자동으로
중지됩니다. 언제든지 카메라를 수동으로 중지할 수도 있습니다.

## 더 나은 스캔을 위한 팁

- 전체 코드가 보이고 늘어나지 않은 선명한 이미지를 사용하세요.
- 카메라로 스캔할 때 라벨을 평평하고 선명하며 밝게 유지하세요.
- 긴 1차원 바코드의 경우 막대 양쪽 끝에 여백을 남기세요.
- 사진이 스캔되지 않으면 코드 주변을 자르거나 더 높은 해상도의 이미지를
  사용해 보세요.
