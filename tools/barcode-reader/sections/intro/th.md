## เครื่องอ่านบาร์โค้ดทำอะไร

เครื่องอ่านบาร์โค้ดดึงค่าที่เก็บอยู่ในโค้ดหนึ่งมิติและสองมิติทั่วไปจากรูปภาพในเครื่อง
หรือจากกล้องของคุณ รองรับ QR Code, Data Matrix, PDF417, Aztec, Code 128,
Code 39, EAN, UPC, ITF, Codabar และรูปแบบอื่นๆ ที่เครื่องสแกนในเบราว์เซอร์จัดการได้

## ควรใช้เมื่อใด

ใช้เมื่อคุณมีป้ายสินค้า ป้ายขนส่ง ตั๋ว แท็กสินค้าคงคลัง รูปถ่ายบรรจุภัณฑ์
ภาพหน้าจอ หรือรูปภาพเอกสาร และต้องตรวจสอบค่าที่เข้ารหัสไว้อย่างแม่นยำก่อนดำเนินการกับค่านั้น
เนื้อหาที่ถอดรหัสแล้วจะยังแสดงอยู่เพื่อให้คุณคัดลอก เปิดลิงก์ที่รองรับ
หรือเปรียบเทียบกับรูปภาพต้นฉบับได้

## ความเป็นส่วนตัวและพฤติกรรมของกล้อง

รูปภาพจะถูกถอดรหัสในเครื่องภายในเบราว์เซอร์และไม่ถูกอัปโหลด
การสแกนด้วยกล้องใช้หน้าต่างขอสิทธิ์กล้องของเบราว์เซอร์
จากนั้นจะหยุดโดยอัตโนมัติหลังจากถอดรหัสโค้ดได้
คุณยังหยุดกล้องเองได้ทุกเมื่อ

## เคล็ดลับเพื่อการสแกนที่ดีขึ้น

- ใช้รูปภาพที่ชัดเจน ซึ่งมองเห็นโค้ดทั้งหมดและไม่ถูกยืด
- เมื่อสแกนด้วยกล้อง ให้ป้ายอยู่ราบ คมชัด และมีแสงเพียงพอ
- สำหรับบาร์โค้ดหนึ่งมิติที่ยาว ให้เว้นพื้นที่ว่างที่ปลายทั้งสองด้านของแท่งบาร์
- หากรูปถ่ายสแกนไม่ได้ ให้ครอบรอบโค้ดหรือลองใช้รูปภาพความละเอียดสูงขึ้น
