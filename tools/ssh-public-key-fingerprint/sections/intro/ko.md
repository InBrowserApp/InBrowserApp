## SSH 공개 키 지문이란?

SSH 공개 키 지문은 공개 키 blob의 짧은 다이제스트입니다. `authorized_keys`, 서버 인벤토리, 배포 워크플로에서 키를 신뢰하기 전에 비교할 수 있는 간결한 값을 제공합니다.

OpenSSH는 일반적으로 `SHA256:...` 같은 SHA-256 지문을 표시합니다. 오래된 문서와 일부 감사에서는 여전히 콜론으로 구분된 MD5 지문을 사용합니다. 이 도구는 최신 SSH 출력과 레거시 기록을 모두 대조할 수 있도록 두 형식을 모두 표시하며, 키를 어디에도 전송하지 않습니다.

단일 공개 키, 여러 `authorized_keys` 줄, 또는 SSH2 공개 키 블록을 붙여넣으세요. 파서는 주석과 authorized_keys 옵션을 건너뛰고 실제 SSH 키 blob을 읽은 뒤 브라우저에서 로컬로 지문을 계산합니다.

- 복사한 공개 키가 팀원이 공유한 지문과 일치하는지 확인합니다.
- `authorized_keys` 항목을 서버 접근 목록과 비교합니다.
- 지문을 복사하기 전에 키 유형, 키 크기, 곡선, 주석을 확인합니다.
