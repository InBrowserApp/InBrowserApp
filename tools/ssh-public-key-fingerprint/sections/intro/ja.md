## SSH 公開鍵フィンガープリントとは？

SSH 公開鍵フィンガープリントは、公開鍵 blob の短いダイジェストです。`authorized_keys`、サーバー台帳、デプロイ手順で鍵を信頼する前に比較できる、コンパクトな値を提供します。

OpenSSH では一般的に `SHA256:...` のような SHA-256 フィンガープリントが表示されます。古いドキュメントや一部の監査では、今でもコロン区切りの MD5 フィンガープリントが使われます。このツールは両方を表示するため、鍵をどこにも送信せずに、最新の SSH 出力とレガシー記録のどちらにも照合できます。

単一の公開鍵、複数の `authorized_keys` 行、または SSH2 公開鍵ブロックを貼り付けてください。パーサーはコメントと authorized_keys オプションをスキップし、実際の SSH 鍵 blob を読み取り、ブラウザー内でローカルにフィンガープリントを計算します。

- コピーした公開鍵が、チームメイトから共有されたフィンガープリントと一致することを確認します。
- `authorized_keys` エントリをサーバーのアクセスリストと比較します。
- フィンガープリントをコピーする前に、鍵タイプ、鍵サイズ、曲線、コメントを確認します。
