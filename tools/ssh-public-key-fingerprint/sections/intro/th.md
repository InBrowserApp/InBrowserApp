## ลายนิ้วมือคีย์สาธารณะ SSH คืออะไร?

ลายนิ้วมือคีย์สาธารณะ SSH คือค่า digest สั้นๆ ของบล็อบคีย์สาธารณะ ช่วยให้คุณมีค่าขนาดกะทัดรัดสำหรับเปรียบเทียบก่อนเชื่อถือคีย์ใน `authorized_keys`, รายการสินทรัพย์เซิร์ฟเวอร์ หรือเวิร์กโฟลว์การปรับใช้

OpenSSH มักแสดงลายนิ้วมือ SHA-256 เช่น `SHA256:...` เอกสารเก่าและการตรวจสอบบางอย่างยังใช้ลายนิ้วมือ MD5 ที่คั่นด้วยเครื่องหมายโคลอน เครื่องมือนี้แสดงทั้งสองแบบ เพื่อให้คุณจับคู่ได้ทั้งเอาต์พุต SSH สมัยใหม่และบันทึกเดิม โดยไม่ต้องส่งคีย์ออกไปที่ใด

วางคีย์สาธารณะรายการเดียว หลายบรรทัดจาก `authorized_keys` หรือบล็อกคีย์สาธารณะ SSH2 ตัวแยกวิเคราะห์จะข้ามความคิดเห็นและตัวเลือก authorized_keys อ่านบล็อบคีย์ SSH จริง แล้วคำนวณลายนิ้วมือภายในเบราว์เซอร์ของคุณ

- ตรวจสอบว่าคีย์สาธารณะที่คัดลอกมาตรงกับลายนิ้วมือที่เพื่อนร่วมทีมแชร์ไว้
- เปรียบเทียบรายการใน `authorized_keys` กับรายการสิทธิ์เข้าถึงเซิร์ฟเวอร์
- ตรวจสอบชนิดคีย์ ขนาดคีย์ curve และความคิดเห็นก่อนคัดลอกลายนิ้วมือ
