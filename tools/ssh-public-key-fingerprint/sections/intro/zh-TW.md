## 什麼是 SSH 公開金鑰指紋？

SSH 公開金鑰指紋是公開金鑰 blob 的短摘要。它提供一個精簡值，讓你在信任 `authorized_keys`、伺服器清單或部署流程中的金鑰之前進行比對。

OpenSSH 通常會顯示像 `SHA256:...` 這樣的 SHA-256 指紋。較舊的文件和部分稽核仍使用以冒號分隔的 MD5 指紋。此工具會同時顯示兩者，讓你不必將金鑰傳送到任何地方，也能比對現代 SSH 輸出與舊版記錄。

貼上單一公開金鑰、多行 `authorized_keys`，或 SSH2 公開金鑰區塊。解析器會略過註解和 authorized_keys 選項，讀取真正的 SSH 金鑰 blob，並在你的瀏覽器本機計算指紋。

- 驗證複製的公開金鑰是否符合隊友分享的指紋。
- 將 `authorized_keys` 項目與伺服器存取清單比對。
- 在複製指紋前檢查金鑰類型、金鑰大小、曲線和註解。
