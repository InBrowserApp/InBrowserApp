## ケースコンバーターとは？

ケースコンバーターは、プログラミングや文章で使われる命名規則間でテキストを変換するツールです。変数名を手で打ち直す代わりに、一度貼り付けるだけですべてのバリエーションを即座に取得できます。

## 対応するケーススタイル

- **camelCase** と **PascalCase** — JavaScript、TypeScript、Java、C# で一般的。
- **snake_case** と **SCREAMING_SNAKE_CASE** — Python、Ruby、C の定数で使用。
- **kebab-case** と **SCREAMING-KEBAB-CASE** — CSS クラス名、URL、CLI フラグで使用。
- **dot.case** と **path/case** — 設定キーやファイルパスで使用。
- **Title Case**、**Sentence case**、**UPPERCASE**、**lowercase** — 文章や表示テキスト向け。

## 活用シーン

- 異なる命名規則を持つ言語間で変数名や定数名をリネームする場合。
- プレーンテキストから URL スラッグ、CSS クラス名、環境変数キーを生成する場合。
- 検索・置換リファクタリングの前に不統一な命名を正規化する場合。
