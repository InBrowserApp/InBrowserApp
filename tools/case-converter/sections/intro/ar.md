## ما هو محوّل حالة الأحرف؟

محوّل حالة الأحرف يحوّل النص بين اصطلاحات التسمية المستخدمة في البرمجة والكتابة. بدلًا من إعادة كتابة اسم متغيّر يدويًّا، الصقه مرة واحدة واحصل على جميع الأنماط فورًا.

## أنماط الحالة المدعومة

- **camelCase** و**PascalCase** — شائعان في JavaScript وTypeScript وJava وC#.
- **snake_case** و**SCREAMING_SNAKE_CASE** — مفضّلان في Python وRuby وثوابت C.
- **kebab-case** و**SCREAMING-KEBAB-CASE** — يُستخدمان في أسماء أصناف CSS وعناوين URL وخيارات سطر الأوامر.
- **dot.case** و**path/case** — يظهران في مفاتيح الإعدادات ومسارات الملفات.
- **Title Case** و**Sentence case** و**UPPERCASE** و**lowercase** — للنصوص النثرية والعرض.

## متى تستخدمه

- إعادة تسمية المتغيّرات أو الثوابت عبر لغات ذات اصطلاحات مختلفة.
- توليد عناوين URL مختصرة أو أسماء أصناف CSS أو مفاتيح متغيّرات البيئة من نص عادي.
- توحيد التسمية غير المتّسقة قبل إجراء بحث واستبدال لإعادة هيكلة الشيفرة.
