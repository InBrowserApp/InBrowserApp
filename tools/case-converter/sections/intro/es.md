## ¿Qué es un conversor de mayúsculas y minúsculas?

Un conversor de mayúsculas y minúsculas transforma texto entre las convenciones de nomenclatura utilizadas en programación y redacción. En lugar de volver a escribir el nombre de una variable a mano, pégalo una vez y obtén todas las variantes al instante.

## Estilos de capitalización compatibles

- **camelCase** y **PascalCase** — comunes en JavaScript, TypeScript, Java y C#.
- **snake_case** y **SCREAMING_SNAKE_CASE** — preferidos en Python, Ruby y constantes de C.
- **kebab-case** y **SCREAMING-KEBAB-CASE** — usados en nombres de clases CSS, URLs y opciones de CLI.
- **dot.case** y **path/case** — presentes en claves de configuración y rutas de archivos.
- **Title Case**, **Sentence case**, **UPPERCASE** y **lowercase** — para prosa y texto de visualización.

## Cuándo usarlo

- Renombrar variables o constantes entre lenguajes con diferentes convenciones.
- Generar slugs de URL, nombres de clases CSS o claves de variables de entorno a partir de texto simple.
- Normalizar nomenclatura inconsistente antes de una refactorización con buscar y reemplazar.
