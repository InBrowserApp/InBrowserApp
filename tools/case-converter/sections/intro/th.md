## ตัวแปลงรูปแบบตัวอักษรคืออะไร?

ตัวแปลงรูปแบบตัวอักษรจะแปลงข้อความระหว่างรูปแบบการตั้งชื่อที่ใช้ในการเขียนโปรแกรมและงานเขียนทั่วไป แทนที่จะพิมพ์ชื่อตัวแปรใหม่ด้วยมือ เพียงวางข้อความครั้งเดียวก็จะได้ทุกรูปแบบทันที

## รูปแบบตัวอักษรที่รองรับ

- **camelCase** และ **PascalCase** — ใช้กันทั่วไปใน JavaScript, TypeScript, Java และ C#
- **snake_case** และ **SCREAMING_SNAKE_CASE** — นิยมใช้ใน Python, Ruby และค่าคงที่ในภาษา C
- **kebab-case** และ **SCREAMING-KEBAB-CASE** — ใช้ในชื่อคลาส CSS, URL และแฟล็ก CLI
- **dot.case** และ **path/case** — พบในคีย์การตั้งค่าและเส้นทางไฟล์
- **Title Case**, **Sentence case**, **UPPERCASE** และ **lowercase** — สำหรับข้อความร้อยแก้วและข้อความแสดงผล

## เมื่อใดที่ควรใช้

- เปลี่ยนชื่อตัวแปรหรือค่าคงที่ข้ามภาษาโปรแกรมที่มีรูปแบบการตั้งชื่อต่างกัน
- สร้าง URL slug, ชื่อคลาส CSS หรือคีย์ตัวแปรสภาพแวดล้อมจากข้อความธรรมดา
- ทำให้การตั้งชื่อที่ไม่สม่ำเสมอเป็นมาตรฐานก่อนการค้นหาและแทนที่ในการรีแฟกเตอร์
