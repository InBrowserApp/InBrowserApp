## 대소문자 변환기란?

대소문자 변환기는 프로그래밍과 글쓰기에서 사용되는 명명 규칙 간에 텍스트를 변환합니다. 변수명을 직접 다시 입력하는 대신, 한 번 붙여넣기만 하면 모든 변형을 즉시 얻을 수 있습니다.

## 지원하는 케이스 스타일

- **camelCase** 및 **PascalCase** — JavaScript, TypeScript, Java, C#에서 주로 사용됩니다.
- **snake_case** 및 **SCREAMING_SNAKE_CASE** — Python, Ruby, C 상수에서 주로 사용됩니다.
- **kebab-case** 및 **SCREAMING-KEBAB-CASE** — CSS 클래스명, URL, CLI 플래그에서 사용됩니다.
- **dot.case** 및 **path/case** — 설정 키와 파일 경로에서 사용됩니다.
- **Title Case**, **Sentence case**, **UPPERCASE**, **lowercase** — 산문과 표시 텍스트에 사용됩니다.

## 사용 시기

- 서로 다른 규칙을 사용하는 언어 간에 변수나 상수의 이름을 변경할 때.
- 일반 텍스트에서 URL 슬러그, CSS 클래스명 또는 환경 변수 키를 생성할 때.
- 찾기 및 바꾸기 리팩터링 전에 일관되지 않은 이름을 정규화할 때.
