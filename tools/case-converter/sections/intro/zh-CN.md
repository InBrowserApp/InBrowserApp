## 什么是大小写转换器？

大小写转换器可以在编程和写作中使用的各种命名约定之间转换文本。无需手动重新输入变量名，只需粘贴一次即可立即获得所有变体。

## 支持的大小写风格

- **camelCase** 和 **PascalCase** —— 常用于 JavaScript、TypeScript、Java 和 C#。
- **snake_case** 和 **SCREAMING_SNAKE_CASE** —— Python、Ruby 和 C 常量的首选。
- **kebab-case** 和 **SCREAMING-KEBAB-CASE** —— 用于 CSS 类名、URL 和 CLI 参数。
- **dot.case** 和 **path/case** —— 常见于配置键和文件路径。
- **Title Case**、**Sentence case**、**UPPERCASE** 和 **lowercase** —— 用于散文和展示文本。

## 使用场景

- 在使用不同命名约定的编程语言之间重命名变量或常量。
- 从纯文本生成 URL slug、CSS 类名或环境变量键。
- 在搜索替换重构之前统一不一致的命名。
