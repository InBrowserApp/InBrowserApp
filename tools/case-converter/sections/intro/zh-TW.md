## 什麼是大小寫轉換器？

大小寫轉換器可在程式設計和寫作中使用的各種命名慣例之間轉換文字。不需要手動重新輸入變數名稱，只要貼上一次即可立即取得所有變體。

## 支援的大小寫樣式

- **camelCase** 和 **PascalCase** — 常用於 JavaScript、TypeScript、Java 和 C#。
- **snake_case** 和 **SCREAMING_SNAKE_CASE** — Python、Ruby 和 C 常數中偏好使用。
- **kebab-case** 和 **SCREAMING-KEBAB-CASE** — 用於 CSS 類別名稱、URL 和 CLI 旗標。
- **dot.case** 和 **path/case** — 常見於設定檔鍵值和檔案路徑。
- **Title Case**、**Sentence case**、**UPPERCASE** 和 **lowercase** — 用於散文和顯示文字。

## 使用時機

- 在不同命名慣例的程式語言之間重新命名變數或常數。
- 從純文字產生 URL slug、CSS 類別名稱或環境變數鍵。
- 在搜尋與取代重構前，統一不一致的命名。
