## מהו ממיר רישיות?

ממיר רישיות מתרגם טקסט בין מוסכמות השמות המקובלות בתכנות ובכתיבה. במקום להקליד מחדש שם משתנה ידנית, הדביקו אותו פעם אחת וקבלו את כל הווריאציות מיד.

## סגנונות רישיות נתמכים

- **camelCase** ו-**PascalCase** — נפוצים ב-JavaScript,‏ TypeScript,‏ Java ו-C#.
- **snake_case** ו-**SCREAMING_SNAKE_CASE** — מועדפים ב-Python,‏ Ruby וקבועי C.
- **kebab-case** ו-**SCREAMING-KEBAB-CASE** — בשימוש בשמות מחלקות CSS, כתובות URL ודגלי CLI.
- **dot.case** ו-**path/case** — מופיעים במפתחות תצורה ונתיבי קבצים.
- **Title Case**,‏ **Sentence case**,‏ **UPPERCASE** ו-**lowercase** — לטקסט רציף ותצוגה.

## מתי להשתמש בכלי

- שינוי שמות משתנים או קבועים בין שפות עם מוסכמות שונות.
- יצירת slug לכתובות URL, שמות מחלקות CSS או מפתחות משתני סביבה מטקסט רגיל.
- תקנון שמות לא עקביים לפני ביצוע חיפוש-והחלפה בעת שיפוץ קוד.
