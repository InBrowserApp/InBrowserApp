Gere identificadores CUID2 localmente no navegador sem enviar o lote atual para outro serviço. Esta ferramenta é útil quando você precisa de IDs públicos compactos para registros, URLs, links de convite, dados de teste ou placeholders no cliente, e quer controlar diretamente a quantidade e o comprimento.

## O Que Diferencia O CUID2

O CUID2 foi projetado para reduzir colisões em sistemas distribuídos e ainda assim permanecer amigável para URLs. Cada valor começa com uma letra minúscula, usa apenas caracteres base36 em minúsculas e combina contadores, impressão do host e entropia aleatória antes do hash final.

## Escolha Quantidade E Comprimento

Use saídas mais curtas quando quiser slugs compactos para demos, dados de teste ou links temporários. Aumente o comprimento quando precisar de mais margem para registros duradouros ou cargas distribuídas maiores, e aumente a quantidade quando quiser gerar um lote inteiro de uma só vez.

## Copie Ou Exporte O Lote Final

Revise a lista gerada e depois copie-a ou baixe-a como arquivo de texto quando o formato estiver correto. Como tudo roda localmente, os identificadores permanecem no navegador até que você decida usá-los ou compartilhá-los.
