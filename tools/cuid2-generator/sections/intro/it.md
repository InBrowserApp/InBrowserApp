Genera identificatori CUID2 localmente nel browser senza inviare il lotto corrente a un altro servizio. Questo strumento è utile quando ti servono ID pubblici compatti per record, URL, link di invito, dati di test o segnaposto lato client e vuoi controllare direttamente quantità e lunghezza.

## Che Cosa Rende Diverso CUID2

CUID2 è progettato per ridurre le collisioni nei sistemi distribuiti restando allo stesso tempo adatto agli URL. Ogni valore inizia con una lettera minuscola, usa solo caratteri base36 minuscoli e combina contatori, impronta dell’host ed entropia casuale prima dell’hash finale.

## Scegli Quantità E Lunghezza

Usa output più corti quando vuoi slug compatti per demo, dati di test o link temporanei. Aumenta la lunghezza quando ti serve più margine per record di lunga durata o carichi distribuiti più ampi, e aumenta la quantità quando vuoi generare un intero lotto in un solo passaggio.

## Copia O Esporta Il Lotto Finale

Controlla l’elenco generato, poi copialo o scaricalo come file di testo quando il formato ti sembra corretto. Poiché tutto viene eseguito in locale, gli identificatori restano nel browser finché non decidi di usarli o condividerli.
