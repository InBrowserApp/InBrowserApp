Genera identificadores CUID2 localmente en tu navegador sin enviar el lote actual a otro servicio. Esta herramienta es útil cuando necesitas IDs públicos compactos para registros, URLs, enlaces de invitación, datos de prueba o marcadores del lado del cliente y quieres controlar directamente la cantidad y la longitud.

## Qué Hace Distinto A CUID2

CUID2 está diseñado para reducir colisiones en sistemas distribuidos y seguir siendo apto para URLs. Cada valor empieza con una letra minúscula, usa solo caracteres base36 en minúsculas y combina contadores, huellas del host y entropía aleatoria antes del hash final.

## Elige Cantidad Y Longitud

Usa salidas más cortas cuando necesites slugs compactos para demos, datos de prueba o enlaces temporales. Aumenta la longitud cuando quieras más margen para registros duraderos o cargas distribuidas mayores, y sube la cantidad cuando quieras crear un lote completo de una sola vez.

## Copia O Exporta El Lote Final

Revisa la lista generada y luego cópiala o descárgala como archivo de texto cuando el formato te parezca correcto. Como todo se ejecuta localmente, los identificadores permanecen en tu navegador hasta que decidas usarlos o compartirlos.
