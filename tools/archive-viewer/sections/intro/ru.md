Просмотр архивов позволяет проверить сжатый файл до его извлечения. Этот инструмент открывает файлы ZIP, TAR, GZ, TGZ и TAR.GZ прямо в браузере, чтобы вы могли убедиться, что находится внутри, просмотреть папки, проверить читаемые файлы и скачать только нужный элемент.

## Когда использовать

Используйте его, когда получаете сжатый пакет и хотите быстро посмотреть содержимое без распаковки всего архива. Это удобно для проверки релизных пакетов, скачанных шаблонов, пакетов журналов, снимков исходного кода или вложения `.gz` из одного файла.

## Конфиденциальность и обработка файлов

Содержимое архива читается локально в сеансе вашего браузера. Файл не загружается в InBrowser.App. Для больших текстовых элементов предпросмотр ограничен, чтобы страница оставалась отзывчивой; скачайте элемент, если нужно проверить файл полностью.

## Поддерживаемые форматы архивов

Средство просмотра поддерживает стандартные архивы ZIP, несжатые файлы TAR, одиночные файлы, сжатые GZIP, и архивы TAR, упакованные в GZIP (`.tgz` или `.tar.gz`). Защищенные паролем или зашифрованные архивы не поддерживаются в этом первом этапе переписывания.

## Поведение предпросмотра

Файлы, похожие на текст, например JSON, Markdown, журналы, исходный код, CSV, XML, YAML и TOML, можно просматривать с подсветкой синтаксиса, если доступен подходящий язык. Распространенные файлы изображений можно просматривать визуально. Другие двоичные файлы остаются доступными для скачивания, но инструмент не будет пытаться их отобразить.
