# 先把正則跑通，再放進程式

正則最常見的問題不是完全寫不出來，而是「差一點就對了」。這個工具把測試文字、匹配結果與取代輸出放在同一頁裡，方便你在真正用到程式、清洗規則或批次流程之前先驗證一次。

## 這裡可以做什麼

- 直接貼上範例文字，立刻看到匹配高亮
- 切換常用旗標，例如全域、多行與忽略大小寫
- 檢查每次匹配拿到了哪些擷取群組與命名群組
- 預覽 `$1`、`$<name>` 這類取代引用最後會產生什麼結果

## 適合用在這些情境

- 清洗訂單編號、日誌行、匯出文字這類結構化片段
- 在真正大量取代之前，先確認規則不會誤傷內容
- 排查為什麼某個模式匹配太多、太少，或完全匹配不到
