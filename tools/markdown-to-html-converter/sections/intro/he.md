## למה להשתמש בממיר הזה

Markdown מצוין לכתיבת תיעוד, הערות שחרור ותוכן קליל, אבל יעדי פרסום בדרך כלל דורשים HTML. הכלי הזה שומר על שני הצדדים גלויים, כך שאפשר לכתוב או לייבא Markdown, לבדוק את המקטע שנוצר ולצפות בתצוגה מעוצבת לפני הייצוא.

## מה אפשר לעשות כאן

- להדביק Markdown ישירות או לייבא קובץ טקסט פשוט.
- להפעיל סינון כשצריך פלט בטוח יותר לתצוגה מקדימה ולשיתוף.
- לבדוק בו זמנית את התצוגה המרונדרת ואת מקטע ה-HTML המדויק.
- להעתיק, להוריד, להדפיס או להמשיך לערוך בלי לצאת מהדפדפן.

## HTML מסונן מול HTML גולמי

הסינון מסיר תגיות ומאפיינים לא בטוחים מהתוצאה שנוצרת. מומלץ להשאיר אותו פעיל עבור פרסום שוטף, בלוקי CMS, קטעי דוא"ל ותיעוד משותף. כבו אותו רק כשצריך ממש לבדוק את המקטע הגולמי שמחזיר מפרש ה-Markdown.

## מקרי שימוש נפוצים

- להפוך חלקים מ-README לבלוקי HTML שניתנים להטמעה.
- להכין תוכן CMS מטיוטות Markdown.
- לבדוק את ה-markup שנוצר לפני שמדביקים אותו לבונה אתרים.
- להדפיס תצוגה מעוצבת לצורך סקירה או אישור תוכן.
