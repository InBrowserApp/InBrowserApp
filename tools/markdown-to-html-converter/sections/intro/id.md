## Mengapa Menggunakan Konverter Ini

Markdown sangat cocok untuk menyusun dokumentasi, catatan rilis, dan konten ringan, tetapi tujuan penerbitan biasanya membutuhkan HTML. Alat ini menampilkan keduanya sekaligus sehingga Anda bisa menulis atau mengimpor Markdown, memeriksa fragmen yang dihasilkan, dan meninjau pratinjau bergaya sebelum mengekspor.

## Apa Yang Bisa Anda Lakukan Di Sini

- Tempel Markdown langsung atau impor file teks biasa.
- Aktifkan pembersihan saat Anda menginginkan output yang lebih aman untuk pratinjau dan berbagi.
- Tinjau pratinjau yang sudah dirender dan fragmen HTML yang tepat secara bersamaan.
- Salin, unduh, cetak, atau lanjutkan iterasi tanpa meninggalkan browser.

## HTML Yang Dibersihkan Dan HTML Mentah

Pembersihan menghapus tag dan atribut yang tidak aman dari hasil yang dihasilkan. Biarkan tetap aktif untuk penerbitan rutin, blok CMS, cuplikan email, dan dokumentasi bersama. Nonaktifkan hanya ketika Anda benar-benar perlu memeriksa fragmen mentah yang dikembalikan parser Markdown.

## Kasus Penggunaan Umum

- Mengubah bagian README menjadi blok HTML yang bisa disematkan.
- Menyiapkan konten CMS dari draf Markdown.
- Meninjau markup yang dihasilkan sebelum menempelkannya ke site builder.
- Mencetak pratinjau bergaya untuk peninjauan atau persetujuan konten.
