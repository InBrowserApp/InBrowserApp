## 为什么要使用这个转换器

Markdown 很适合撰写文档、发布说明和轻量内容，但真正发布时通常需要 HTML。这个工具把两边都放在眼前：你可以编写或导入 Markdown，检查生成的片段，再在导出前确认样式化预览。

## 你可以在这里做什么

- 直接粘贴 Markdown，或导入纯文本文件。
- 当你希望预览和分享时更安全，可以开启清理功能。
- 同时查看渲染后的预览和精确的 HTML 片段。
- 在浏览器内复制、下载、打印，或继续迭代内容。

## 清理后的 HTML 与原始 HTML

清理会从生成结果中移除不安全的标签和属性。对于日常发布、CMS 内容块、邮件片段和共享文档，建议保持开启。只有在你确实需要检查 Markdown 解析器返回的原始片段时，再关闭它。

## 典型使用场景

- 把 README 片段转换成可嵌入的 HTML 区块。
- 将 Markdown 草稿整理为 CMS 内容。
- 在粘贴到站点构建器之前先检查生成的标记。
- 打印带样式的预览用于内容审阅或确认。
