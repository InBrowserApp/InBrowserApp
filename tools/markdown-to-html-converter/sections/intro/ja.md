## このコンバーターを使う理由

Markdown はドキュメント、リリースノート、軽量なコンテンツの下書きに向いていますが、公開先では通常 HTML が必要になります。このツールは両方を同時に見せてくれるので、Markdown を書くまたは読み込み、生成された断片を確認し、書き出す前にスタイル付きプレビューを確かめられます。

## ここでできること

- Markdown を直接貼り付ける、またはプレーンテキストファイルを読み込む。
- プレビューや共有用により安全な出力がほしいときはサニタイズを有効にする。
- レンダリング済みプレビューと正確な HTML 断片を同時に確認する。
- ブラウザーを離れずにコピー、ダウンロード、印刷、再編集を続ける。

## サニタイズ済み HTML と生の HTML

サニタイズは生成結果から危険なタグや属性を取り除きます。通常の公開、CMS ブロック、メール断片、共有ドキュメントでは有効のまま使うのがおすすめです。Markdown パーサーが返した生の断片を明示的に確認したいときだけ無効にしてください。

## 代表的な利用例

- README の一部を埋め込み用 HTML ブロックに変換する。
- Markdown の下書きから CMS 用コンテンツを整える。
- サイトビルダーに貼り付ける前に生成マークアップを確認する。
- 内容確認や承認のためにスタイル付きプレビューを印刷する。
