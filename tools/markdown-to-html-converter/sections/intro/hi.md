## इस कन्वर्टर का उपयोग क्यों करें

Markdown दस्तावेज़, रिलीज़ नोट्स और हल्के कंटेंट के ड्राफ्ट के लिए बहुत अच्छा है, लेकिन प्रकाशित करने वाली जगहों को आम तौर पर HTML चाहिए होता है। यह टूल दोनों पक्ष एक साथ दिखाता है ताकि आप Markdown लिख या आयात कर सकें, बने हुए अंश को जाँच सकें और निर्यात से पहले स्टाइल वाला पूर्वावलोकन देख सकें।

## आप यहाँ क्या कर सकते हैं

- Markdown सीधे पेस्ट करें या साधारण टेक्स्ट फ़ाइल आयात करें।
- जब आपको पूर्वावलोकन और साझा करने के लिए अधिक सुरक्षित आउटपुट चाहिए हो, तब सैनिटाइज़ेशन चालू करें।
- रेंडर किया गया पूर्वावलोकन और सटीक HTML अंश एक साथ देखें।
- ब्राउज़र छोड़े बिना कॉपी, डाउनलोड, प्रिंट या आगे संपादन करें।

## सैनिटाइज़ किया हुआ और कच्चा HTML

सैनिटाइज़ेशन तैयार आउटपुट से असुरक्षित टैग और एट्रिब्यूट हटाता है। सामान्य प्रकाशन, CMS ब्लॉक, ईमेल स्निपेट और साझा दस्तावेज़ों के लिए इसे चालू रखें। इसे केवल तब बंद करें जब आपको Markdown parser से लौटे कच्चे अंश को विशेष रूप से देखना हो।

## सामान्य उपयोग के मामले

- README के हिस्सों को एम्बेड करने योग्य HTML ब्लॉक में बदलना।
- Markdown ड्राफ्ट से CMS कंटेंट तैयार करना।
- साइट बिल्डर में चिपकाने से पहले बने हुए markup की समीक्षा करना।
- सामग्री समीक्षा या स्वीकृति के लिए स्टाइल वाला पूर्वावलोकन प्रिंट करना।
