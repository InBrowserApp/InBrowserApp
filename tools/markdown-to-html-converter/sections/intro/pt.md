## Por Que Usar Este Conversor

Markdown é excelente para rascunhar documentação, notas de versão e conteúdo leve, mas os destinos de publicação normalmente exigem HTML. Esta ferramenta mantém os dois lados visíveis para que você escreva ou importe Markdown, confira o fragmento gerado e valide uma pré-visualização estilizada antes de exportar.

## O Que Você Pode Fazer Aqui

- Colar Markdown diretamente ou importar um arquivo de texto simples.
- Ativar a sanitização quando quiser uma saída mais segura para pré-visualização e compartilhamento.
- Revisar ao mesmo tempo a pré-visualização renderizada e o fragmento HTML exato.
- Copiar, baixar, imprimir ou continuar iterando sem sair do navegador.

## HTML Sanitizado x HTML Bruto

A sanitização remove tags e atributos inseguros do resultado gerado. Mantenha-a ativada para publicações rotineiras, blocos de CMS, trechos de e-mail e documentação compartilhada. Desative apenas quando você precisar inspecionar o fragmento bruto devolvido pelo parser de Markdown.

## Casos De Uso Típicos

- Transformar seções de README em blocos HTML incorporáveis.
- Preparar conteúdo de CMS a partir de rascunhos em Markdown.
- Revisar a marcação gerada antes de colá-la em um construtor de sites.
- Imprimir uma prévia estilizada para revisão ou aprovação de conteúdo.
