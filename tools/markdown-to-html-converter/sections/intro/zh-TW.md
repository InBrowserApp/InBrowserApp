## 為什麼要使用這個轉換器

Markdown 很適合撰寫文件、發佈說明與輕量內容，但真正發佈時通常需要 HTML。這個工具把兩邊都放在眼前：你可以撰寫或匯入 Markdown，檢查產生的片段，再在匯出前確認樣式化預覽。

## 你可以在這裡做什麼

- 直接貼上 Markdown，或匯入純文字檔。
- 當你希望預覽與分享時更安全，可以開啟清理功能。
- 同時查看渲染後的預覽與精確的 HTML 片段。
- 在瀏覽器內複製、下載、列印，或繼續迭代內容。

## 清理後的 HTML 與原始 HTML

清理會從產生結果中移除不安全的標籤與屬性。對於日常發佈、CMS 內容區塊、郵件片段與共享文件，建議保持開啟。只有在你確實需要檢查 Markdown 解析器回傳的原始片段時，再關閉它。

## 典型使用情境

- 將 README 片段轉成可嵌入的 HTML 區塊。
- 把 Markdown 草稿整理成 CMS 內容。
- 在貼到網站建置器之前先檢查產生的標記。
- 列印帶樣式的預覽，用於內容審閱或確認。
