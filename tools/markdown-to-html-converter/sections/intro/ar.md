## لماذا تستخدم هذا المحول

يُعد Markdown رائعًا لكتابة التوثيق وملاحظات الإصدار والمحتوى الخفيف، لكن وجهات النشر تحتاج غالبًا إلى HTML. هذه الأداة تُبقي الجانبين أمامك: تكتب أو تستورد Markdown، وتفحص الجزء الناتج، ثم تراجع معاينة منسقة قبل التصدير.

## ما الذي يمكنك فعله هنا

- لصق Markdown مباشرة أو استيراد ملف نصي عادي.
- تشغيل التنقية عندما تريد ناتجًا أكثر أمانًا للمعاينة والمشاركة.
- مراجعة المعاينة المعروضة والجزء الدقيق من HTML في الوقت نفسه.
- النسخ والتنزيل والطباعة أو متابعة التحرير من داخل المتصفح.

## HTML المنقّى مقابل HTML الخام

تزيل التنقية الوسوم والسمات غير الآمنة من النتيجة المولدة. أبقها مفعلة في سير النشر المعتاد وكتل CMS ومقتطفات البريد والمستندات المشتركة. عطّلها فقط عندما تحتاج فعلًا إلى فحص الجزء الخام الذي يعيده محلل Markdown.

## حالات استخدام شائعة

- تحويل أقسام README إلى كتل HTML قابلة للتضمين.
- تجهيز محتوى CMS انطلاقًا من مسودات Markdown.
- مراجعة الوسوم الناتجة قبل لصقها في أداة بناء المواقع.
- طباعة معاينة منسقة لمراجعة المحتوى أو اعتماده.
