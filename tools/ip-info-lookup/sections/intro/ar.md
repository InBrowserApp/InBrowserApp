## ما الذي تبحث عنه هذه الأداة

تحل أداة البحث عن معلومات IP عنوان IPv4 أو عنوان IPv6 أو نطاقًا أو URL، وتعرض البيانات الوصفية العامة التي يمكن لخدمات الإنترنت الإبلاغ عنها لكل عنوان. تكون مفيدة عندما تحتاج إلى فحص الوجهة التي يشير إليها نطاق، أو معرفة الشبكة المالكة لعنوان، أو اسم مضيف DNS العكسي الموجود، أو ما إذا كانت سجلات IPv4 وIPv6 تؤدي إلى مزودين مختلفين.

## كيف تعمل عمليات البحث عن النطاقات وعناوين URL

عند إدخال نطاق أو URL، تستخرج الأداة اسم المضيف وتستعلم من محلل DNS-over-HTTPS المحدد عن سجلات A وAAAA. ثم يتم إثراء كل عنوان مُرجع بشكل منفصل، لذلك يمكن للنطاقات ثنائية المكدس أن تعرض بلدانًا أو أرقام ASN أو مزودي ISP أو أسماء مضيفين أو مناطق زمنية مختلفة بين IPv4 وIPv6.

## معنى النتائج

تأتي حقول الموقع وISP من مزودي بيانات IP وصفية عامة مثل geojs.io وip.sb، بينما تأتي أسماء المضيفين من عمليات بحث PTR في DNS العكسي عندما تكون متاحة. تصف هذه السجلات كيف ترى قواعد البيانات العامة العنوان، لا الموقع الفعلي الدقيق لشخص أو جهاز.

## ملاحظات الخصوصية والدقة

يعمل البحث في متصفحك ويرسل طلبات DNS وبيانات IP وصفية إلى خدمات تابعة لجهات خارجية. قد تجعل شبكات VPN والوكلاء وشبكات CDN وشبكات الهاتف المحمول ومنصات السحابة الموقع أو المؤسسة المبلغ عنها مختلفة عن المستخدم النهائي أو الخادم الذي تتوقعه. الحقول الفارغة طبيعية للعناوين الخاصة أو المحجوزة أو المخصصة حديثا أو ذات التوثيق المحدود.
