## 此工具会查询哪些信息

IP Info Lookup 会解析 IPv4 地址、IPv6 地址、域名或 URL，并显示互联网服务可为每个地址报告的公共元数据。当你需要查看域名指向何处、哪个网络拥有某个地址、是否存在反向 DNS 主机名，或 IPv4 与 IPv6 记录是否指向不同提供商时，它会很有用。

## 域名和 URL 查询如何工作

输入域名或 URL 时，此工具会提取主机名，并向所选 DNS-over-HTTPS 解析器查询 A 和 AAAA 记录。每个返回的地址随后会分别补充信息，因此双栈域名可以显示 IPv4 和 IPv6 对应的不同国家/地区、ASN、ISP、主机名或时区。

## 结果含义

位置和 ISP 字段来自 geojs.io、ip.sb 等公共 IP 元数据提供商，而主机名在可用时来自反向 DNS PTR 查询。这些记录描述的是公共数据库如何看待该地址，而不是某个人或设备的精确物理位置。

## 隐私和准确性说明

查询在你的浏览器中运行，并会向第三方服务发送 DNS 和 IP 元数据请求。VPN、代理、CDN、移动网络和云平台都可能导致报告的位置或组织与你预期的最终用户或服务器不同。对于私有、保留、新分配或记录较少的地址，字段为空是正常现象。
