## 이 도구가 조회하는 항목

IP 정보 조회는 IPv4 주소, IPv6 주소, 도메인 또는 URL을 확인하고 인터넷 서비스가 각 주소에 대해 제공할 수 있는 공개 메타데이터를 보여줍니다. 도메인이 어디를 가리키는지, 어떤 네트워크가 주소를 소유하는지, 어떤 역방향 DNS 호스트 이름이 있는지, 또는 IPv4와 IPv6 레코드가 서로 다른 제공업체로 이어지는지 확인해야 할 때 유용합니다.

## 도메인 및 URL 조회 작동 방식

도메인 또는 URL을 입력하면 도구가 호스트 이름을 추출하고 선택한 DNS-over-HTTPS 리졸버에 A 및 AAAA 레코드를 모두 질의합니다. 반환된 각 주소에는 개별적으로 추가 정보가 붙으므로 듀얼 스택 도메인은 IPv4와 IPv6에서 서로 다른 국가, ASN, ISP, 호스트 이름 또는 시간대를 표시할 수 있습니다.

## 결과의 의미

위치 및 ISP 필드는 geojs.io, ip.sb 같은 공개 IP 메타데이터 제공업체에서 가져오며, 호스트 이름은 사용 가능한 경우 역방향 DNS PTR 조회에서 가져옵니다. 이러한 레코드는 공개 데이터베이스가 주소를 어떻게 보는지를 설명하며, 사람이나 기기의 정확한 물리적 위치를 의미하지는 않습니다.

## 개인정보 보호 및 정확도 참고 사항

조회는 브라우저에서 실행되며 DNS 및 IP 메타데이터 요청을 타사 서비스로 보냅니다. VPN, 프록시, CDN, 모바일 네트워크, 클라우드 플랫폼 때문에 보고된 위치나 조직이 예상한 최종 사용자 또는 서버와 다를 수 있습니다. 사설 주소, 예약된 주소, 새로 할당된 주소 또는 문서화가 부족한 주소에서는 필드가 비어 있는 것이 정상입니다.
