## Qué consulta esta herramienta

IP Info Lookup resuelve una dirección IPv4, una dirección IPv6, un dominio o una URL y muestra los metadatos públicos que los servicios de internet pueden informar para cada dirección. Es útil cuando necesitas inspeccionar a dónde apunta un dominio, qué red es propietaria de una dirección, qué nombre de host de DNS inverso existe o si los registros IPv4 e IPv6 llevan a proveedores diferentes.

## Cómo funcionan las consultas de dominios y URL

Cuando introduces un dominio o una URL, la herramienta extrae el nombre de host y consulta el resolvedor DNS-over-HTTPS seleccionado para obtener registros A y AAAA. Cada dirección devuelta se enriquece por separado, por lo que los dominios de doble pila pueden mostrar países, ASN, ISP, nombres de host o zonas horarias diferentes para IPv4 e IPv6.

## Qué significan los resultados

Los campos de ubicación e ISP proceden de proveedores públicos de metadatos de IP como geojs.io e ip.sb, mientras que los nombres de host proceden de consultas PTR de DNS inverso cuando están disponibles. Estos registros describen cómo las bases de datos públicas ven la dirección, no la ubicación física exacta de una persona o dispositivo.

## Notas sobre privacidad y precisión

La consulta se ejecuta en tu navegador y envía solicitudes de DNS y metadatos de IP a servicios de terceros. Las VPN, los proxies, las CDN, las redes móviles y las plataformas en la nube pueden hacer que la ubicación u organización indicada difiera del usuario final o del servidor que esperabas. Los campos vacíos son normales en direcciones privadas, reservadas, asignadas recientemente o con poca documentación.
