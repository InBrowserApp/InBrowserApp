## เครื่องมือนี้ค้นหาอะไร

IP Info Lookup จะแปลงที่อยู่ IPv4, ที่อยู่ IPv6, โดเมน หรือ URL และแสดงเมตาดาต้าสาธารณะที่บริการอินเทอร์เน็ตสามารถรายงานได้สำหรับแต่ละที่อยู่ เครื่องมือนี้มีประโยชน์เมื่อคุณต้องการตรวจสอบว่าโดเมนชี้ไปที่ใด เครือข่ายใดเป็นเจ้าของที่อยู่ มี hostname จาก reverse DNS ใดอยู่ หรือระเบียน IPv4 และ IPv6 นำไปยังผู้ให้บริการที่ต่างกันหรือไม่

## การค้นหาโดเมนและ URL ทำงานอย่างไร

เมื่อคุณป้อนโดเมนหรือ URL เครื่องมือจะดึง hostname ออกมาและสอบถามตัวแปลง DNS-over-HTTPS ที่เลือกสำหรับทั้งระเบียน A และ AAAA จากนั้นแต่ละที่อยู่ที่ส่งคืนมาจะถูกเสริมข้อมูลแยกกัน ดังนั้นโดเมนแบบ dual-stack จึงอาจแสดงประเทศ, ASN, ISP, hostname หรือเขตเวลาที่ต่างกันสำหรับ IPv4 และ IPv6

## ผลลัพธ์หมายถึงอะไร

ช่องตำแหน่งและ ISP มาจากผู้ให้บริการเมตาดาต้า IP สาธารณะ เช่น geojs.io และ ip.sb ส่วน hostname มาจากการค้นหา reverse DNS PTR เมื่อมีข้อมูล ระเบียนเหล่านี้อธิบายว่าฐานข้อมูลสาธารณะมองเห็นที่อยู่นั้นอย่างไร ไม่ใช่ตำแหน่งทางกายภาพที่แน่นอนของบุคคลหรืออุปกรณ์

## หมายเหตุเรื่องความเป็นส่วนตัวและความแม่นยำ

การค้นหาทำงานในเบราว์เซอร์ของคุณและส่งคำขอ DNS กับเมตาดาต้า IP ไปยังบริการของบุคคลที่สาม VPN, proxy, CDN, เครือข่ายมือถือ และแพลตฟอร์มคลาวด์อาจทำให้ตำแหน่งหรือองค์กรที่รายงานแตกต่างจากผู้ใช้ปลายทางหรือเซิร์ฟเวอร์ที่คุณคาดไว้ ช่องว่างเป็นเรื่องปกติสำหรับที่อยู่ส่วนตัว ที่อยู่สงวน ที่อยู่ที่เพิ่งจัดสรร หรือที่อยู่ที่มีเอกสารประกอบน้อย
