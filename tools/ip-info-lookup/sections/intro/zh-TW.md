## 此工具會查詢哪些資訊

IP Info Lookup 會解析 IPv4 位址、IPv6 位址、網域或 URL，並顯示網際網路服務可針對每個位址回報的公開中繼資料。當你需要檢查網域指向何處、哪個網路擁有某個位址、是否存在反向 DNS 主機名稱，或 IPv4 與 IPv6 記錄是否導向不同服務提供者時，這項工具很有用。

## 網域與 URL 查詢如何運作

輸入網域或 URL 時，工具會擷取主機名稱，並向所選的 DNS-over-HTTPS 解析器查詢 A 與 AAAA 記錄。每個回傳的位址都會分別補充資訊，因此雙協定堆疊網域可針對 IPv4 與 IPv6 顯示不同的國家/地區、ASN、ISP、主機名稱或時區。

## 如何理解結果

位置與 ISP 欄位來自 geojs.io 和 ip.sb 等公開 IP 中繼資料提供者，主機名稱則在可用時來自反向 DNS PTR 查詢。這些記錄描述的是公開資料庫如何看待該位址，而不是某個人或裝置的精確實體位置。

## 隱私與準確性注意事項

查詢會在你的瀏覽器中執行，並將 DNS 與 IP 中繼資料請求傳送給第三方服務。VPN、代理伺服器、CDN、行動網路與雲端平台，都可能讓回報的位置或組織與你預期的終端使用者或伺服器不同。私人、保留、新近分配或文件記錄較少的位址出現空白欄位是正常現象。
