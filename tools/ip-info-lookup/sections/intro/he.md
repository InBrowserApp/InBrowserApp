## מה הכלי הזה בודק

IP Info Lookup פותר כתובת IPv4, כתובת IPv6, דומיין או URL ומציג את המטא-נתונים הציבוריים ששירותי אינטרנט יכולים לדווח עבור כל כתובת. הוא שימושי כשצריך לבדוק לאן דומיין מצביע, איזו רשת מחזיקה בכתובת, איזה שם מארח קיים ב-DNS הפוך, או האם רשומות IPv4 ו-IPv6 מובילות לספקים שונים.

## איך פועלות בדיקות דומיין ו-URL

כשמזינים דומיין או URL, הכלי מחלץ את שם המארח ושולח שאילתה לפותר DNS-over-HTTPS שנבחר עבור רשומות A ו-AAAA. לאחר מכן כל כתובת שהוחזרה מועשרת בנפרד, כך שדומיינים בתצורת dual-stack יכולים להציג מדינות, ASNs, ספקי אינטרנט, שמות מארח או אזורי זמן שונים עבור IPv4 ו-IPv6.

## מה משמעות התוצאות

שדות המיקום וספק האינטרנט מגיעים מספקי מטא-נתוני IP ציבוריים כגון geojs.io ו-ip.sb, בעוד ששמות מארח מגיעים מבדיקות PTR של DNS הפוך כשהם זמינים. הרשומות האלה מתארות כיצד מסדי נתונים ציבוריים רואים את הכתובת, ולא את המיקום הפיזי המדויק של אדם או מכשיר.

## הערות פרטיות ודיוק

הבדיקה רצה בדפדפן שלכם ושולחת בקשות DNS ומטא-נתוני IP לשירותי צד שלישי. VPN, שרתי proxy, רשתות CDN, רשתות סלולריות ופלטפורמות ענן יכולים לגרום לכך שהמיקום או הארגון המדווחים יהיו שונים ממשתמש הקצה או מהשרת שציפיתם לראות. שדות ריקים הם מצב רגיל עבור כתובות פרטיות, שמורות, שהוקצו לאחרונה או מתועדות באופן חלקי.
