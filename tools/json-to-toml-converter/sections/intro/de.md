## Wofür dieses Tool gedacht ist

Verwende dieses Tool, wenn du gültiges JSON in TOML umwandeln musst, das sich leichter lesen, teilen oder committen lässt. Alles läuft lokal im Browser.

## Wo es hilft

- API-Beispiele in TOML für Doku oder Konfigurationsdateien umwandeln.
- Kopierte JSON-Payloads in ein Format überführen, das sich visuell leichter vergleichen lässt.
- Strukturierte Inhalte für Tools und Workflows vorbereiten, die TOML erwarten.

## Worauf du achten solltest

- Die Eingabe muss in diesem Tool gültiges JSON sein.
- Der Konverter erhält die Datenstruktur, aber die TOML-Darstellung kann sich von handgeschriebenem Stil unterscheiden.
- Wenn du in die andere Richtung konvertieren willst, nutze stattdessen den TOML-zu-JSON-Konverter.
