## Para qué sirve esta herramienta

Usa esta herramienta cuando necesites convertir JSON válido en TOML que sea más fácil de leer, compartir o versionar. Todo se ejecuta localmente en el navegador.

## Dónde ayuda

- Convertir ejemplos de API a TOML para documentación o archivos de configuración.
- Reescribir payloads JSON copiados a un formato más fácil de comparar visualmente.
- Preparar contenido estructurado para herramientas y flujos que esperan TOML.

## Qué tener en cuenta

- En esta herramienta la entrada debe ser JSON válido.
- El conversor conserva la estructura de datos, pero la presentación TOML puede diferir de un estilo escrito a mano.
- Si necesitas ir en la otra dirección, usa el conversor de TOML a JSON.
