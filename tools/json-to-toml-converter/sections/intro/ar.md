## ما فائدة هذه الأداة

استخدم هذه الأداة عندما تحتاج إلى تحويل JSON صالح إلى TOML أسهل في القراءة والمشاركة والإيداع في المستودع. كل شيء يعمل محليًا داخل المتصفح.

## أين تكون مفيدة

- تحويل أمثلة API إلى TOML مناسب للتوثيق أو ملفات الإعداد.
- إعادة كتابة حمولات JSON المنسوخة إلى صيغة أسهل للمقارنة بالعين.
- تجهيز محتوى منظم للأدوات وسير العمل التي تتوقع TOML.

## ما الذي يجب الانتباه له

- يجب أن يكون الإدخال في هذه الأداة JSON صالحًا.
- يحافظ المحول على بنية البيانات، لكن طريقة عرض TOML قد تختلف عن الأسلوب المكتوب يدويًا.
- إذا كنت تحتاج إلى الاتجاه العكسي، فاستخدم أداة تحويل TOML إلى JSON.
