## What ICO Is

ICO is the icon file format used by Windows and browsers for favicons and app
icons. A single ICO can bundle multiple square sizes so the platform can pick
the closest match.

- Use ICO when you need a classic browser favicon or Windows shortcut icon.
- Bundle multiple sizes in one file so you do not have to export separate image
  files.
- Start from a square, high-resolution source image for the cleanest result.
