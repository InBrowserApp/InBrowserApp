## ما هو محول OpenAPI إلى TypeScript؟

يحوّل محول OpenAPI إلى TypeScript مستند OpenAPI 3.x إلى أنواع TypeScript مولدة مباشرة في متصفحك. وهو مفيد عندما تريد معاينة سريعة للأنواع، أو ملف تعريف قابلًا للتنزيل، أو طريقة آمنة لاختبار خيارات `openapi-typescript` من دون إرسال schema إلى خادم.

## متى تستخدمه

استخدم هذه الأداة عندما يكون لديك بالفعل schema OpenAPI بصيغة JSON أو YAML وتريد نماذج typed لطلباتك واستجاباتك لتطبيقات الواجهة الأمامية أو نماذج SDK أو مراجعات API. وهي مفيدة بشكل خاص لمقارنة خيارات التوليد قبل اعتماد الناتج في مستودعك.

## قبل التوليد

تدعم هذه النسخة المعاد بناؤها للمتصفح مستندات OpenAPI 3.0 و3.1 المجمعة. إذا كان schema ما يزال يحتوي على أهداف `$ref` خارجية، فقم بضمها أو إدراجها أولًا، ثم ولّد مخرجات TypeScript النهائية هنا.
