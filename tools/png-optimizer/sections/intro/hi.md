## PNG Optimizer क्या करता है

PNG Optimizer PNG images को lossless Oxipng optimization से compress करता है। यह visible pixels बदले बिना, जहां संभव हो, PNG structure, compression settings, palette data, और alpha information को rewrite करता है।

## इसे कब उपयोग करें

Screenshots, UI assets, diagrams, icons, या transparent graphics publish करने से पहले इसका उपयोग करें, जहां PNG सही output format है। यह तब उपयोगी है जब आपको छोटी file चाहिए, लेकिन quality loss, color shifts, या किसी अलग image format में conversion स्वीकार नहीं कर सकते।

## महत्वपूर्ण Options

Compression level नियंत्रित करता है कि Oxipng कितना काम करता है। Lower levels तेजी से result देते हैं, जबकि higher levels अधिक combinations आजमाते हैं और बड़ी images पर अधिक समय ले सकते हैं। PNG interlacing पुराने viewers को loading के दौरान coarse preview दिखाने में मदद कर सकता है, लेकिन इससे size भी बढ़ सकता है। Alpha optimization आम तौर पर on रखना अच्छा है, क्योंकि यह केवल lossless transparent-pixel improvements लागू करता है।

## Privacy और File Handling

Image आपके browser में locally process होती है। Tool preview और download के लिए temporary browser object URL बनाता है, और PNG को server पर upload नहीं करता।

## Practical Notes

Lossless PNG optimization से हर file छोटी होगी, इसकी guarantee नहीं है। कुछ images पहले से optimized होती हैं, और कभी-कभी अलग PNG layout same size या थोड़ा बड़ा हो सकता है। ऐसे में original file रखें या lower compression level आजमाएं।
