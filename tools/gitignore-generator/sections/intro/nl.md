# Wat is .gitignore?

Een .gitignore-bestand specificeert opzettelijk niet-gevolgde bestanden die Git moet negeren. Bestanden die al door Git worden gevolgd, worden niet beïnvloed.

- Build-uitvoer (node_modules/, dist/, build/)
- IDE- en editorbestanden (.idea/, .vscode/)
- Besturingssysteembestanden (.DS_Store, Thumbs.db)

## Template source

Deze tool gebruikt sjablonen uit de officiële GitHub gitignore-repository, met dekking voor meer dan 150 talen, frameworks en omgevingen.
