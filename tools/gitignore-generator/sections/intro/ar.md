# ما هو .gitignore؟

يحدد ملف .gitignore الملفات غير المتعقبة عمداً التي يجب على Git تجاهلها. الملفات التي يتتبعها Git بالفعل لا تتأثر.

- مخرجات البناء (node_modules/، dist/، build/)
- ملفات IDE والمحرر (.idea/، .vscode/)
- ملفات نظام التشغيل (.DS_Store، Thumbs.db)

## Template source

تستخدم هذه الأداة قوالب من مستودع gitignore الرسمي على GitHub، وتغطي أكثر من 150 لغة وإطار عمل وبيئة.
