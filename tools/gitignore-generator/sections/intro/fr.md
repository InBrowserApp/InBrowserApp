## Qu'est-ce que .gitignore ?

Un fichier .gitignore spécifie les fichiers intentionnellement non suivis que Git doit ignorer. Les fichiers déjà suivis par Git ne sont pas affectés.

- Sorties de compilation (node_modules/, dist/, build/)
- Fichiers IDE et éditeur (.idea/, .vscode/)
- Fichiers du système d'exploitation (.DS_Store, Thumbs.db)

## Template source

Cet outil utilise des modèles du dépôt officiel gitignore de GitHub, couvrant plus de 150 langages, frameworks et environnements.
