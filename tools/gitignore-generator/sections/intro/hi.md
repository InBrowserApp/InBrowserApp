## .gitignore क्या है?

एक .gitignore फ़ाइल जानबूझकर अनट्रैक की गई फ़ाइलों को निर्दिष्ट करती है जिन्हें Git को अनदेखा करना चाहिए। Git द्वारा पहले से ट्रैक की गई फ़ाइलें प्रभावित नहीं होती हैं।

- बिल्ड आउटपुट (node_modules/, dist/, build/)
- IDE और संपादक फ़ाइलें (.idea/, .vscode/)
- ऑपरेटिंग सिस्टम फ़ाइलें (.DS_Store, Thumbs.db)
