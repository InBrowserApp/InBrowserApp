## .gitignore คืออะไร?

ไฟล์ .gitignore ระบุไฟล์ที่ไม่ได้ติดตามโดยตั้งใจซึ่ง Git ควรเพิกเฉย ไฟล์ที่ Git ติดตามอยู่แล้วจะไม่ได้รับผลกระทบ

- ผลลัพธ์การ build (node_modules/, dist/, build/)
- ไฟล์ IDE และเอดิเตอร์ (.idea/, .vscode/)
- ไฟล์ระบบปฏิบัติการ (.DS_Store, Thumbs.db)
