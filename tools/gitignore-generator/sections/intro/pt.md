# O que é .gitignore?

Um arquivo .gitignore especifica arquivos intencionalmente não rastreados que o Git deve ignorar. Arquivos já rastreados pelo Git não são afetados.

- Saídas de build (node_modules/, dist/, build/)
- Arquivos de IDE e editor (.idea/, .vscode/)
- Arquivos do sistema operacional (.DS_Store, Thumbs.db)

## Template source

Esta ferramenta usa modelos do repositório oficial gitignore do GitHub, cobrindo mais de 150 linguagens, frameworks e ambientes.
