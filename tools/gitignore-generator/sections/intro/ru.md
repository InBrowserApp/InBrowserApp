# Что такое .gitignore?

Файл .gitignore указывает намеренно неотслеживаемые файлы, которые Git должен игнорировать. Файлы, уже отслеживаемые Git, не затрагиваются.

- Результаты сборки (node_modules/, dist/, build/)
- Файлы IDE и редактора (.idea/, .vscode/)
- Файлы операционной системы (.DS_Store, Thumbs.db)

## Template source

Этот инструмент использует шаблоны из официального репозитория gitignore GitHub, охватывая более 150 языков, фреймворков и сред.
