# Vad är .gitignore?

En .gitignore-fil specificerar avsiktligt ospårade filer som Git ska ignorera. Filer som redan spåras av Git påverkas inte.

- Byggutdata (node_modules/, dist/, build/)
- IDE- och redigerarfiler (.idea/, .vscode/)
- Operativsystemfiler (.DS_Store, Thumbs.db)

## Template source

Detta verktyg använder mallar från det officiella GitHub gitignore-repositoriet, som täcker 150+ språk, ramverk och miljöer.
