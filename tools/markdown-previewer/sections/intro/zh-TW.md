## 這個工具能做什麼

Markdown 預覽器會在你的瀏覽器中本地渲染 Markdown，方便你在發布前檢查標題層級、清單、表格、程式碼區塊、連結以及內嵌 HTML。這個 rewrite 版本把編輯與審閱放得更近，補上了即時目錄，也支援匯出獨立 HTML，便於分享或列印，同時不需要把草稿送到伺服器。

## 什麼時候適合用它

它適合處理 README、版本更新說明、內部文件、操作手冊、部落格草稿，以及任何需要快速確認呈現效果的 Markdown 內容。淨化開關可以幫助你對比更安全的瀏覽器預覽與原始 HTML 輸出，而紙本與深色兩種主題則更方便分別檢查列印場景與暗色閱讀場景。

## 工作流程建議

編輯階段先使用分欄模式，同時觀察原始內容與渲染結果；檢查節奏、留白與標題結構時，再切換到僅預覽模式。你可以直接匯入現有的 `.md` 檔案，先載入範例文件測試表格與圍欄程式碼區塊，再在目錄結構與最終標題都確認無誤後匯出 HTML。
