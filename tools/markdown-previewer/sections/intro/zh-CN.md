## 这个工具能做什么

Markdown 预览器会在你的浏览器里本地渲染 Markdown，方便你在发布前检查标题层级、列表、表格、代码块、链接以及内联 HTML。这个 rewrite 版本把编辑和审阅放得更近，补上了实时目录，还支持导出独立 HTML，便于分享或打印，同时不需要把草稿发送到服务器。

## 什么时候适合用它

它适合处理 README、更新日志、内部文档、操作手册、博客草稿，以及任何需要快速确认呈现效果的 Markdown 内容。净化开关可以帮助你对比更安全的浏览器预览和原始 HTML 输出，而简洁与深色两种主题则更方便分别检查清爽预览和暗色阅读场景。

## 工作流建议

编辑阶段先用分栏模式，同时观察源码和渲染结果；检查节奏、留白和标题结构时，再切到仅预览模式。你可以直接导入现有的 `.md` 文件，先加载示例文档测试表格和围栏代码块，再在目录结构和最终标题都确认无误后导出 HTML。
