# Wat is Markdown?

Markdown is een lichtgewicht opmaaktaal voor het schrijven van opgemaakte tekst in platte tekstbestanden. Het houdt de bron leesbaar terwijl het nog steeds rijke opmaak ondersteunt wanneer het wordt weergegeven.

**Gemeenschappelijke syntaxis:**

- `#` voor koppen, `*` of `_` voor nadruk, `-` voor lijsten.
- Backticks voor inline code en afgeschermde codeblokken voor fragmenten.
- `[text](url)` voor links en `![alt](url)` voor afbeeldingen.

Markdown wordt veel gebruikt voor README-bestanden, documentatie, notities en blogposts. De meeste platforms renderen Markdown naar HTML; CommonMark en GitHub Flavored Markdown voegen tabellen, takenlijsten en andere extensies toe. Bestanden gebruiken meestal de extensie `.md` of `.markdown`, en het platte-tekstformaat werkt goed met versiebeheer.

## Praktische tips

- Houd opschoning ingeschakeld als de Markdown HTML van een onbekende bron bevat.
- Gebruik consistente kopniveaus, zodat de gegenereerde omtrek bruikbaar blijft.
- Schakel over naar de HTML-weergave wanneer u het exacte geëxporteerde document nodig heeft.
