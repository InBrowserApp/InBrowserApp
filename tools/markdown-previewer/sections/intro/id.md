# Apa itu Markdown?

Markdown adalah bahasa markup ringan untuk menulis teks berformat di file teks biasa. Format ini menjaga sumber tetap mudah dibaca sambil tetap mendukung tampilan yang kaya saat dirender.

**Sintaks umum:**

- `#` untuk judul, `*` atau `_` untuk penekanan, `-` untuk daftar.
- Backtick untuk kode sebaris dan blok kode berpagar untuk cuplikan.
- `[text](url)` untuk tautan dan `![alt](url)` untuk gambar.

Markdown banyak digunakan untuk file README, dokumentasi, catatan, dan posting blog. Sebagian besar platform merender Markdown ke HTML; CommonMark dan GitHub Flavored Markdown menambahkan tabel, daftar tugas, dan ekstensi lainnya. File biasanya menggunakan ekstensi `.md` atau `.markdown`, dan format teks biasa bekerja baik dengan kontrol versi.

## Mengapa Menggunakan Pratinjau?

- Tinjau judul, tabel, daftar tugas, pagar kode, dan tautan sebelum dipublikasikan.
- Temukan HTML yang ditempel atau kebiasaan memformat tanpa meninggalkan browser.
- Ekspor file HTML mandiri saat Anda memerlukan snapshot yang ramah cetak.

## Apa Yang Dibantu Alat Ini

- File README, dokumen, log perubahan, catatan rilis, dan konten handoff.
- Catatan, draf blog, panduan internal, dan artikel basis pengetahuan.
- Pemeriksaan terakhir sebelum mengirim Markdown ke sistem lain atau alur kerja rekan satu tim.

## Tip Praktis

- Tetap aktifkan sanitasi ketika Markdown berisi HTML dari sumber yang tidak dikenal.
- Gunakan tingkat judul yang konsisten sehingga kerangka yang dihasilkan tetap berguna.
- Beralih ke tampilan HTML saat Anda memerlukan dokumen yang diekspor secara persis.
