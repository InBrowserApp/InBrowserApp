## SHA3-224 (FIPS 202) คืออะไร?

SHA3-224 (FIPS 202) (Secure Hash Algorithm 224-bit) เป็นฟังก์ชันแฮชเข้ารหัสที่สร้างค่าแฮช 224-บิต (28-ไบต์) ซึ่งมักแสดงเป็นตัวเลขฐานสิบหก 56 ตัวอักษร เป็นส่วนหนึ่งของตระกูลฟังก์ชันแฮช SHA-3 ที่ได้รับการมาตรฐานโดย NIST (FIPS 202)

**คุณลักษณะสำคัญ:**

- **กำหนดได้**: อินพุตเดียวกันจะสร้างแฮชเดียวกันเสมอ
- **การคำนวณที่รวดเร็ว**: คำนวณได้อย่างรวดเร็วสำหรับอินพุตใดก็ตาม
- **เอฟเฟกต์หิมะถล่ม**: การเปลี่ยนแปลงเล็กน้อยในอินพุตจะสร้างเอาต์พุตที่แตกต่างอย่างมาก
- **ไม่สามารถย้อนกลับได้**: ไม่สามารถคำนวณเพื่อย้อนกลับแฮชเพื่อหาอินพุตต้นฉบับได้
- **ต้านทานการชน**: หาอินพุตสองตัวที่แตกต่างกันแต่สร้างแฮชเดียวกันได้ยากมาก

**การใช้งานทั่วไป:**

- ลายเซ็นดิจิทัลและใบรับรอง
- บล็อกเชนและสกุลเงินดิจิทัล (บางระบบใช้ SHA3-224 (FIPS 202))
- การจัดเก็บรหัสผ่าน (ด้วยการเกลือที่เหมาะสม)
- การตรวจสอบความสมบูรณ์ของไฟล์
- อัลกอริทึม proof-of-work
