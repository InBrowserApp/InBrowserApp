## Выберите Поддерживаемый Язык Контента

Переключатель языка управляет именно сгенерированным текстом-заполнителем, а не языком интерфейса вокруг. Этот инструмент показывает только те языки, для которых в текущем генераторе есть lorem-данные, поэтому список вариантов намеренно уже, чем полный языковой список сайта.
