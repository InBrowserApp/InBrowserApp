## Elegir La Forma De Salida Adecuada

Usa Palabras cuando solo necesites etiquetas cortas, chips, contadores o marcadores compactos de interfaz. Cambia a Oraciones para tarjetas, resúmenes, texto de ayuda o muestras de tablas. Elige Párrafos cuando quieras bloques más largos para artículos, landing pages, vistas previas del editor o pruebas de densidad de contenido más realistas. El campo de cantidad siempre sigue el modo seleccionado, así que representa palabras, oraciones o párrafos.

## Elegir Un Idioma De Contenido Compatible

El selector de idioma controla el texto de relleno generado, no el idioma de la interfaz que lo rodea. Esta herramienta solo muestra idiomas que actualmente incluyen conjuntos de datos lorem dentro del generador, por lo que las opciones son intencionalmente más limitadas que el menú completo de idiomas del sitio. Si estás probando layouts traducidos, usa el idioma compatible más cercano para validar la estructura y sustitúyelo por contenido real antes de publicar.
