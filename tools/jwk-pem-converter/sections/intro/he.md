## מהי המרה בין JWK ל‑PEM?

JWK ‏(JSON Web Key) הוא חומר מפתח במבנה JSON המשמש ב-JOSE/JWT, נקודות קצה של JWKS והגדרות serverless או דפדפן. הוא קל לקריאה עבור תוכנה, אך פחות מקובל בכלי CLI ובתשתיות שמצפות לקובצי מפתח.

PEM עוטף נתוני מפתח DER עם תוויות BEGIN/END, וזה הפורמט ש-OpenSSL, כלי TLS, שערי API ו-SDKs רבים מבקשים בדרך כלל.

הממיר הזה מגשר בין הפורמטים באופן מקומי בדפדפן שלך. הוא מטפל במכלי מפתחות RSA, EC ‏(P-256/384/521) ו-OKP, מאפשר לבחור PEM ציבורי מסוג SPKI או פרטי מסוג PKCS8 כשמתחילים מ-JWK, ויכול להחזיר בלוקי PEM נתמכים ל-JSON JWK מסודר או קומפקטי.

השתמש בפלט ציבורי כשצריך רק אימות או הפצה. המרות פרטיות מציגות חומר מפתח פרטי על המסך ובהורדות, לכן התייחס לתוצאה כסוד וסגור את הלשונית בסיום.

- העבר מפתחות בין JWKS/הגדרות JSON לבין קובצי PEM בסגנון OpenSSL.
- חלץ מפתח ציבורי לפני שיתוף עם מאמתי JWT, שערים או לקוחות.
- המר מקומית בלי להעלות חומר מפתח לשרת.
