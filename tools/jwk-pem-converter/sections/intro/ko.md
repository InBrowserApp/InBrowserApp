## JWK ↔ PEM 변환이란?

JWK(JSON Web Key)는 JOSE/JWT, JWKS 엔드포인트, serverless 또는 브라우저 설정에서 사용하는 JSON 형태의 키 자료입니다. 소프트웨어가 읽기 쉽지만, 키 파일을 기대하는 CLI나 인프라에서는 덜 받아들여질 수 있습니다.

PEM은 DER 키 데이터를 BEGIN/END 라벨로 감싸는 형식이며, OpenSSL, TLS 도구, API 게이트웨이, 많은 SDK가 보통 요구하는 형태입니다.

이 변환기는 브라우저 안에서 로컬로 두 형식을 연결합니다. RSA, EC(P-256/384/521), OKP 키 컨테이너를 처리하고, JWK에서 시작할 때 공개 SPKI 또는 개인 PKCS8 PEM을 선택할 수 있으며, 지원되는 PEM 블록을 보기 좋은 또는 압축된 JWK JSON으로 되돌릴 수 있습니다.

검증이나 배포만 필요하면 공개 출력을 사용하세요. 개인 키 변환은 개인 키 자료를 화면과 다운로드에 노출하므로 결과를 비밀처럼 다루고 작업이 끝나면 탭을 닫으세요.

- JWKS/JSON 설정과 OpenSSL 스타일 PEM 파일 사이에서 키를 옮깁니다.
- JWT 검증기, 게이트웨이, 클라이언트와 공유하기 전에 공개 키를 추출합니다.
- 키 자료를 서버에 업로드하지 않고 로컬에서 변환합니다.
