## การแปลง JWK ↔ PEM คืออะไร?

JWK (JSON Web Key) เป็นรูปแบบ JSON สำหรับกุญแจคริปโตที่ใช้ในระบบ JOSE/JWT สามารถแทนกุญแจ RSA, EC หรือ OKP และอาจอยู่ใน JWK Set (JWKS) ได้

PEM คือกุญแจ ASN.1/DER ที่เข้ารหัสด้วย Base64 พร้อมบรรทัดหัวเช่น BEGIN PUBLIC KEY หรือ BEGIN PRIVATE KEY และพบได้ทั่วไปใน TLS, OpenSSL และ SDK ต่าง ๆ

เครื่องมือนี้แปลงกุญแจได้ทั้งสองทิศทาง โดยคงวัสดุกุญแจไว้เมื่อเลือกเอาต์พุตสาธารณะ (SPKI) หรือส่วนตัว (PKCS8) รูปแบบที่รองรับรวมถึง RSA, EC (P-256/384/521) และคอนเทนเนอร์คีย์ OKP และทำงานทั้งหมดในเบราว์เซอร์ของคุณ

เลือก JWK → PEM เมื่อไลบรารี เกตเวย์ หรือ CLI ต้องการไฟล์กุญแจแบบ OpenSSL เลือก PEM → JWK เมื่อต้องใส่กุญแจลงใน JWKS ส่งผ่านคอนฟิกแบบ JSON หรือใช้งานในสภาพแวดล้อมเบราว์เซอร์หรือ serverless การแปลงกุญแจส่วนตัวยังคงเก็บข้อมูลกุญแจส่วนตัวไว้ ดังนั้นหากปลายทางต้องการเพียงกุญแจสาธารณะ ก็ควรแชร์เฉพาะผลลัพธ์แบบสาธารณะ

- ใช้กุญแจ JWK/JWKS กับระบบที่รับเฉพาะ PEM
- ส่งออกกุญแจ PEM สำหรับไลบรารี JWT เกตเวย์ API หรือการกระจายกุญแจ
- แชร์กุญแจสาธารณะอย่างปลอดภัยโดยไม่เปิดเผยข้อมูลกุญแจส่วนตัว
