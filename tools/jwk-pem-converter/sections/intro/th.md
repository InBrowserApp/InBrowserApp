## การแปลง JWK ↔ PEM คืออะไร?

JWK (JSON Web Key) คือข้อมูลคีย์ในรูปแบบ JSON ที่ใช้กับ JOSE/JWT, endpoint ของ JWKS และการตั้งค่า serverless หรือเบราว์เซอร์ ซอฟต์แวร์อ่านได้ง่าย แต่ CLI และระบบโครงสร้างพื้นฐานที่ต้องการไฟล์คีย์อาจไม่รองรับเสมอไป

PEM ห่อข้อมูลคีย์ DER ด้วยป้าย BEGIN/END ซึ่งเป็นรูปแบบที่ OpenSSL, เครื่องมือ TLS, API gateway และ SDK จำนวนมากมักต้องการ

ตัวแปลงนี้เชื่อมสองรูปแบบในเบราว์เซอร์ของคุณโดยตรง รองรับคอนเทนเนอร์คีย์ RSA, EC (P-256/384/521) และ OKP ให้เลือก PEM แบบ public SPKI หรือ private PKCS8 เมื่อเริ่มจาก JWK และแปลงบล็อก PEM ที่รองรับกลับเป็น JWK JSON แบบจัดรูปหรือแบบย่อได้

ใช้ผลลัพธ์แบบ public เมื่อต้องการเพียงการตรวจสอบหรือการแจกจ่าย การแปลง private จะแสดงข้อมูลคีย์ส่วนตัวบนหน้าจอและในไฟล์ดาวน์โหลด ดังนั้นให้ถือผลลัพธ์เป็นความลับและปิดแท็บเมื่อเสร็จแล้ว

- ย้ายคีย์ระหว่างการตั้งค่า JWKS/JSON และไฟล์ PEM แบบ OpenSSL
- ดึง public key ก่อนแชร์กับ JWT verifier, gateway หรือ client
- แปลงในเครื่องโดยไม่อัปโหลดข้อมูลคีย์ไปยังเซิร์ฟเวอร์
