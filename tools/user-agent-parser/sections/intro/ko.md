## User-Agent란?

User-Agent(UA) 문자열은 요청을 보내는 브라우저나 앱을 식별하며 보통 브라우저, OS, 디바이스, 엔진 정보를 포함합니다. UA는 위조될 수 있으므로 보안 신호가 아닌 참고 정보로 사용하세요.

### 이 파서가 보여 주는 정보

이 도구는 붙여 넣은 UA 문자열을 브라우저 안에서 로컬로 분석하고, 결과를 브라우저, 운영체제, 엔진, 기기, CPU, JSON 출력으로 나누어 보여 줍니다. 어떤 내용도 업로드되지 않습니다.

### 예시

다음과 같은 일반적인 Windows용 Chrome 문자열을 붙여 넣어 보세요.

```text
Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/115.0.0.0 Safari/537.36
```

결과에는 Windows 10의 Chrome 115, Blink 엔진, amd64 CPU 아키텍처가 표시되어야 합니다.

### 중요한 참고 사항

최신 브라우저는 점점 더 Client Hints에 의존하므로, 복사한 UA 문자열만으로는 실제 요청에서 사이트가 볼 수 있는 모든 정보를 알 수 없습니다.
