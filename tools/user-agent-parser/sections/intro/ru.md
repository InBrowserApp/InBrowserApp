## Что такое User-Agent?

Строка User-Agent (UA) идентифицирует браузер или приложение, выполняющее запрос, и обычно содержит сведения о браузере, ОС, устройстве и движке. Поскольку UA можно подделать, используйте её как подсказку, а не как сигнал безопасности.

### Что показывает этот парсер

Этот инструмент локально разбирает вставленную строку UA прямо в браузере и группирует результат по браузеру, операционной системе, движку, устройству, CPU и JSON-выводу. Ничего не отправляется.

### Пример

Вставьте, например, такую типичную строку Chrome в Windows:

```text
Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/115.0.0.0 Safari/537.36
```

Результат должен определить Chrome 115 в Windows 10, движок Blink и архитектуру CPU amd64.

### Важное замечание

Современные браузеры всё чаще полагаются на Client Hints, поэтому скопированная строка UA может не показывать всё, что сайт видит при реальном запросе.
