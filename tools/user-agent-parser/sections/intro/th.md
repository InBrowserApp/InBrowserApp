## User-Agent คืออะไร?

สตริง User-Agent (UA) ใช้ระบุเบราว์เซอร์หรือแอปที่ส่งคำขอ และมักมีข้อมูลเบราว์เซอร์ ระบบปฏิบัติการ อุปกรณ์ และเอนจิน เนื่องจากสามารถปลอมได้ ให้ใช้เป็นข้อมูลอ้างอิง ไม่ใช่สัญญาณความปลอดภัย

### ตัวแยกวิเคราะห์นี้แสดงอะไรบ้าง

เครื่องมือนี้จะแยกวิเคราะห์สตริง UA ที่คุณวางไว้ภายในเบราว์เซอร์แบบโลคัล และจัดผลลัพธ์เป็นหมวดเบราว์เซอร์ ระบบปฏิบัติการ เอนจิน อุปกรณ์ CPU และเอาต์พุต JSON โดยไม่มีการอัปโหลดข้อมูลใด ๆ

### ตัวอย่าง

ลองวางสตริง Chrome บน Windows ที่พบบ่อยแบบนี้:

```text
Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/115.0.0.0 Safari/537.36
```

ผลลัพธ์ควรระบุได้ว่าเป็น Chrome 115 บน Windows 10 พร้อมเอนจิน Blink และสถาปัตยกรรม CPU แบบ amd64

### ข้อควรรู้

เบราว์เซอร์สมัยใหม่พึ่งพา Client Hints มากขึ้นเรื่อย ๆ ดังนั้นสตริง UA ที่คัดลอกมาอาจไม่แสดงข้อมูลทั้งหมดที่เว็บไซต์เห็นได้ในคำขอจริง
