## User-Agent とは？

User-Agent（UA）文字列はリクエストを行うブラウザやアプリを識別し、通常はブラウザ、OS、デバイス、エンジン情報を含みます。UA は偽装できるため、セキュリティ判断ではなく参考情報として扱ってください。

### このパーサーで分かること

このツールは、貼り付けた UA 文字列をブラウザ内でローカルに解析し、ブラウザ、OS、エンジン、デバイス、CPU、JSON 出力に分けて結果を表示します。内容は送信されません。

### 例

次のような一般的な Windows 版 Chrome の文字列を貼り付けてみてください。

```text
Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/115.0.0.0 Safari/537.36
```

解析結果では、Windows 10 上の Chrome 115、Blink エンジン、amd64 CPU アーキテクチャが判別されるはずです。

### 重要な注意点

最近のブラウザは Client Hints への依存が強くなっているため、コピーした UA 文字列だけでは実際のリクエスト時にサイトが把握できる情報をすべて表せないことがあります。
