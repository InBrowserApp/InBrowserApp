## מהו User-Agent?

מחרוזת User-Agent (UA) מזהה את הדפדפן או האפליקציה שמבצעים בקשה, ובדרך כלל כוללת פרטי דפדפן, מערכת הפעלה, מכשיר ומנוע. מכיוון שניתן לזייף אותה, השתמשו בה כרמז ולא כאות אבטחה.

### מה המנתח הזה מציג

הכלי הזה מנתח מקומית בדפדפן את מחרוזת ה‑UA שהדבקת ומציג את התוצאה לפי דפדפן, מערכת הפעלה, מנוע, מכשיר, CPU ופלט JSON. שום דבר לא נשלח.

### דוגמה

הדבק מחרוזת Chrome נפוצה ב‑Windows כמו זו:

```text
Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/115.0.0.0 Safari/537.36
```

התוצאה אמורה לזהות את Chrome 115 על Windows 10, יחד עם מנוע Blink וארכיטקטורת CPU מסוג amd64.

### הערה חשובה

דפדפנים מודרניים נשענים יותר ויותר על Client Hints, לכן מחרוזת UA מועתקת לא תמיד משקפת את כל מה שאתר יכול לראות בזמן בקשה אמיתית.
