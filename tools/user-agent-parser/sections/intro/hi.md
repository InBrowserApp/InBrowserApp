## User-Agent क्या है?

User-Agent (UA) स्ट्रिंग अनुरोध करने वाले ब्राउज़र या ऐप की पहचान करती है और आमतौर पर ब्राउज़र, OS, डिवाइस और इंजन की जानकारी देती है। क्योंकि इसे spoof किया जा सकता है, इसे सुरक्षा संकेत नहीं बल्कि संदर्भ के रूप में उपयोग करें।

### यह पार्सर क्या दिखाता है

यह टूल पेस्ट की गई UA स्ट्रिंग को आपके ब्राउज़र में लोकली पार्स करता है और परिणाम को ब्राउज़र, ऑपरेटिंग सिस्टम, इंजन, डिवाइस, CPU और JSON आउटपुट के रूप में व्यवस्थित करके दिखाता है। कुछ भी अपलोड नहीं होता।

### उदाहरण

नीचे जैसी सामान्य Windows पर Chrome स्ट्रिंग पेस्ट करके देखें:

```text
Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/115.0.0.0 Safari/537.36
```

परिणाम में Windows 10 पर Chrome 115, Blink इंजन और amd64 CPU architecture दिखना चाहिए।

### महत्वपूर्ण नोट

आधुनिक ब्राउज़र अब Client Hints पर अधिक निर्भर हैं, इसलिए कॉपी की हुई UA स्ट्रिंग हमेशा वह सब नहीं दिखाती जो किसी लाइव अनुरोध में वेबसाइट देख सकती है।
