## Qué es WebP

WebP es un formato de imagen moderno que puede reducir el tamaño de archivo
manteniendo una buena calidad visual. Admite transparencia, funciona bien para
imágenes de sitios web y es ampliamente compatible con los browser y las
herramientas de imagen actuales.

## Por qué usar este convertidor

Usa esta herramienta cuando necesites imágenes más pequeñas y listas para la web
sin enviar archivos a un servidor. Convierte imágenes localmente en tu browser,
te permite ajustar la calidad y puede cambiar el tamaño de las salidas antes de
descargarlas. Cuando conviertes varias imágenes, la herramienta también prepara
un ZIP para que el lote sea fácil de guardar.

## Cómo funciona

1. Sube uno o más archivos de imagen.
2. Elige la calidad WebP y la escala de salida.
3. Convierte los archivos en el browser.
4. Previsualiza cada resultado, descarga archivos WebP individuales o descarga el ZIP.

## Consejos para obtener mejores resultados

- Usa un valor de calidad alrededor de 80 para fotos y capturas de pantalla como
  punto de partida práctico.
- Baja la calidad cuando el tamaño de archivo importe más que el detalle fino.
- Mantén la escala en 100% para una conversión solo de formato, o redúcela
  cuando la imagen se vaya a mostrar más pequeña en la página.
- Compara los tamaños de archivo original y de salida, porque algunas imágenes
  muy pequeñas o ya comprimidas quizá no se reduzcan después de la codificación
  WebP.
