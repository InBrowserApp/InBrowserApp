## What WebP Is

WebP is a modern image format that can reduce file size while keeping good
visual quality. It supports transparency, works well for website images, and
is broadly supported by current browsers and image tools.

## Why Use This Converter

Use this tool when you need smaller web-ready images without sending files to a
server. It converts images locally in your browser, lets you adjust quality,
and can resize outputs before download. When you convert multiple images, the
tool also prepares a ZIP so the batch is easy to save.

## How It Works

1. Upload one or more image files.
2. Choose the WebP quality and output scale.
3. Convert the files in the browser.
4. Preview each result, download individual WebP files, or download the ZIP.

## Tips for Better Results

- Use a quality value around 80 for photos and screenshots as a practical
  starting point.
- Lower quality when file size matters more than fine detail.
- Keep scale at 100% for format-only conversion, or reduce it when the image
  will be displayed smaller on the page.
- Compare the original and output file sizes because some tiny or already
  compressed images may not get smaller after WebP encoding.
