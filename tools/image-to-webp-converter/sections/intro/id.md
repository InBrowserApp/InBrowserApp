## Apa Itu WebP

WebP adalah format gambar modern yang dapat mengurangi ukuran file sambil
tetap menjaga kualitas visual yang baik. Format ini mendukung transparansi,
cocok untuk gambar situs web, dan didukung luas oleh browser serta alat gambar
modern.

## Mengapa Menggunakan Konverter Ini

Gunakan alat ini saat Anda membutuhkan gambar siap web yang lebih kecil tanpa
mengirim file ke server. Alat ini mengonversi gambar secara lokal di browser
Anda, memungkinkan Anda menyesuaikan kualitas, dan dapat mengubah ukuran output
sebelum diunduh. Saat Anda mengonversi beberapa gambar, alat ini juga
menyiapkan ZIP agar batch mudah disimpan.

## Cara Kerjanya

1. Unggah satu atau beberapa file gambar.
2. Pilih kualitas WebP dan skala output.
3. Konversi file di browser.
4. Pratinjau setiap hasil, unduh file WebP satu per satu, atau unduh ZIP.

## Tips untuk Hasil yang Lebih Baik

- Gunakan nilai kualitas sekitar 80 untuk foto dan tangkapan layar sebagai
  titik awal yang praktis.
- Turunkan kualitas ketika ukuran file lebih penting daripada detail halus.
- Pertahankan skala pada 100% untuk konversi format saja, atau kurangi ketika
  gambar akan ditampilkan lebih kecil di halaman.
- Bandingkan ukuran file asli dan output karena beberapa gambar yang sangat
  kecil atau sudah dikompresi mungkin tidak menjadi lebih kecil setelah
  pengenkodean WebP.
