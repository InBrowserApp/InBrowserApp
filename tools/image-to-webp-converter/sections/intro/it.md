## Cos'è WebP

WebP è un formato immagine moderno che può ridurre la dimensione dei file
mantenendo una buona qualità visiva. Supporta la trasparenza, funziona bene per
le immagini dei siti web ed è ampiamente supportato dai browser e dagli
strumenti per immagini attuali.

## Perché usare questo convertitore

Usa questo strumento quando ti servono immagini più leggere e pronte per il web
senza inviare file a un server. Converte le immagini localmente nel tuo
browser, ti permette di regolare la qualità e può ridimensionare gli output
prima del download. Quando converti più immagini, lo strumento prepara anche
uno ZIP per salvare facilmente il batch.

## Come funziona

1. Carica uno o più file immagine.
2. Scegli la qualità WebP e la scala dell'output.
3. Converti i file nel browser.
4. Visualizza l'anteprima di ogni risultato, scarica singoli file WebP oppure
   scarica lo ZIP.

## Consigli per risultati migliori

- Usa un valore di qualità intorno a 80 per foto e screenshot come punto di
  partenza pratico.
- Abbassa la qualità quando la dimensione del file conta più dei dettagli fini.
- Mantieni la scala al 100% per una conversione solo di formato, oppure riducila
  quando l'immagine verrà visualizzata più piccola nella pagina.
- Confronta le dimensioni del file originale e dell'output, perché alcune
  immagini molto piccole o già compresse potrebbero non diventare più leggere
  dopo la codifica WebP.
