## box-shadow คืออะไร?

สร้างค่า CSS box-shadow หลายชั้น พร้อมพรีวิวแบบเรียลไทม์ ตัวเลือก inset และการควบคุมที่ละเอียด.

box-shadow เป็นคุณสมบัติ CSS ที่วาดเงาหนึ่งหรือหลายชั้นด้านหลัง (หรือด้านใน) ขององค์ประกอบ

แต่ละเงามีระยะเลื่อนแนวนอน/แนวตั้ง รัศมีเบลอ การขยาย (ถ้ามี) และสีพร้อมค่าอัลฟา; เพิ่ม inset เพื่อเป็นเงาด้านใน

หลายเลเยอร์คั่นด้วยเครื่องหมายจุลภาค ช่วยซ้อนเงาเล็ก ๆ เพื่อสร้างมิติ แสงเรือง หรือขอบคม

- ระยะเลื่อนเล็กกับเบลอมาก ให้เอฟเฟกต์ยกตัวนุ่ม ๆ
- ความทึบสูงกับเบลอน้อย ทำให้ขอบคมขึ้น
- ผสานเงานอกกับเงาในเพื่อเพิ่มมิติ
- จำกัดจำนวนเลเยอร์เพื่อประสิทธิภาพที่ดีขึ้น
