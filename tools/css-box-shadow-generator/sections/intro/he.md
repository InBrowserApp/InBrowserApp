## מה זה box-shadow?

צור ערכי CSS box-shadow מרובי שכבות עם תצוגה חיה, אפשרות inset ובקרות מדויקות.

box-shadow הוא מאפיין CSS שמצייר צל אחד או יותר מאחורי (או בתוך) רכיב.

כל צל משתמש בהסטה אופקית ואנכית, רדיוס טשטוש, התפשטות אופציונלית וצבע עם אלפא; הוספת inset יוצרת צל פנימי.

שכבות מרובות מופרדות בפסיקים ומאפשרות לערום צללים עדינים ליצירת עומק, זוהר או קצוות חדים.

- הסטות קטנות עם טשטוש גדול יוצרות תחושת גובה רכה.
- אטימות גבוהה עם מעט טשטוש נותנת קצוות חדים יותר.
- שלבו צל חיצוני עם צל פנימי להעמקת המראה.
- הגבילו את מספר השכבות לשיפור ביצועים.
