## Che cos'è box-shadow?

Crea valori CSS box-shadow multilivello con anteprima live, opzione inset e controlli precisi.

box-shadow è una proprietà CSS che disegna una o più ombre dietro (o all'interno) di un elemento.

Ogni ombra usa offset orizzontale e verticale, raggio di sfocatura, espansione opzionale e un colore con alfa; aggiungi inset per renderla un'ombra interna.

Le modalità a più livelli sono separate da virgole e permettono di sovrapporre ombre leggere per creare profondità, bagliore o bordi netti.

- Usa piccoli offset con più sfocatura per una morbida elevazione.
- Usa maggiore opacità con poca sfocatura per bordi più definiti.
- Combina un'ombra esterna con una interna per aggiungere profondità.
- Limita il numero di livelli per prestazioni migliori.
