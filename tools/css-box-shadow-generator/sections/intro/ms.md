## Apakah box-shadow?

Cipta nilai CSS box-shadow berbilang lapisan dengan pratonton langsung, pilihan inset dan kawalan tepat.

box-shadow ialah sifat CSS yang melukis satu atau lebih bayang di belakang (atau di dalam) elemen.

Setiap bayang menggunakan ofset mendatar dan menegak, jejari kabur, sebaran pilihan dan warna dengan alfa; tambah inset untuk menjadi bayang dalaman.

Berbilang lapisan dipisahkan dengan koma, membolehkan bayang halus ditindan untuk menghasilkan kedalaman, cahaya, atau tepi yang tajam.

- Gunakan ofset kecil dengan kabur lebih besar untuk elevasi lembut.
- Gunakan kelegapan lebih tinggi dengan kabur kecil untuk tepi lebih tajam.
- Gabungkan bayang luar dan bayang dalam untuk menambah kedalaman.
- Hadkan bilangan lapisan untuk prestasi lebih baik.
