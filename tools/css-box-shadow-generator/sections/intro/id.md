## Apa itu box-shadow?

Buat nilai CSS box-shadow berlapis dengan pratinjau langsung, opsi inset, dan kontrol yang presisi.

box-shadow adalah properti CSS yang menggambar satu atau lebih bayangan di belakang (atau di dalam) elemen.

Setiap bayangan menggunakan offset horizontal dan vertikal, radius blur, sebaran opsional, serta warna dengan alfa; tambahkan inset untuk menjadi bayangan dalam.

Beberapa lapisan dipisahkan dengan koma, sehingga bayangan halus bisa ditumpuk untuk membentuk kedalaman, glow, atau tepi tajam.

- Gunakan offset kecil dengan blur lebih besar untuk elevasi lembut.
- Gunakan opasitas lebih tinggi dengan blur kecil untuk tepi lebih tajam.
- Gabungkan bayangan luar dan dalam untuk menambah kedalaman.
- Batasi jumlah lapisan demi performa.
