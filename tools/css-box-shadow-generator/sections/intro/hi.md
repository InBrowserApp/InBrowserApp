## box-shadow क्या है?

लाइव प्रीव्यू, inset टॉगल और सटीक नियंत्रणों के साथ मल्टी-लेयर CSS box-shadow मान बनाएं।

box-shadow एक CSS प्रॉपर्टी है जो किसी एलिमेंट के पीछे (या भीतर) एक या अधिक शैडो बनाती है।

हर शैडो में क्षैतिज/ऊर्ध्वाधर ऑफसेट, ब्लर रेडियस, वैकल्पिक स्प्रेड और अल्फा वाला रंग होता है; inset जोड़ने पर यह इनर शैडो बन जाता है।

कई लेयर कॉमा से अलग होती हैं, जिससे सूक्ष्म शैडो स्टैक करके गहराई, ग्लो या साफ किनारे बनाए जा सकते हैं।

- छोटे ऑफसेट और बड़े ब्लर से सॉफ्ट उभार मिलता है।
- ज़्यादा अपारदर्शिता और कम ब्लर से धारदार किनारे मिलते हैं।
- बाहरी और अंदरूनी शैडो को मिलाकर गहराई जोड़ें।
- बेहतर परफ़ॉर्मेंस के लिए लेयर की संख्या कम रखें।
