## เมื่อเลขฐานขึ้นมา

ไบนารี ฐานแปด ทศนิยม และฐานสิบหกจะแสดงทุกที่ตั้งแต่บิตมาสก์และเอาต์พุตบันทึกไปจนถึงเอกสารโปรโตคอลและที่อยู่หน่วยความจำ ทันทีที่คุณต้องเปรียบเทียบตัวเลขที่มนุษย์อ่านได้กับแบบฟอร์มที่โปรแกรมคาดหวัง การแปลงฐานอย่างรวดเร็วจะช่วยประหยัดเวลาและหลีกเลี่ยงข้อผิดพลาดทีละรายการ

## ตัวแปลงนี้ช่วยอะไรได้บ้าง

ตัวแปลงฐานตัวเลขนี้เก็บฐานนักพัฒนาทั่วไป รัศมีที่สูงกว่า และฐานที่กำหนดเองหนึ่งฐานให้ซิงค์กันโดยตรงในเบราว์เซอร์ของคุณ You can paste a value into binary, octal, decimal, hex, base32, base36, base62, base64, or any base from 2 to 64, then copy the matching result immediately.

## เมื่อใดจึงจะใช้มัน

ใช้เมื่อคุณตรวจสอบมาสก์ อ่านการจับแพ็กเก็ต ดีบัก ID ที่เข้ารหัส หรือตรวจสอบความถูกต้องของโค้ดที่กระโดดระหว่างรัศมี สำหรับฐานที่สูงกว่า 36 ตัวอักษรจะมีความสำคัญ และฟิลด์ตัวเลข base64 ที่นี่จะตามหลังตัวอักษร `A-Z a-z 0-9 + /`
