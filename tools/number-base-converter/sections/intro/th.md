แปลงจำนวนเต็มในเบราว์เซอร์ระหว่างฐานสอง ฐานแปด ฐานสิบ ฐานสิบหก Base32 Base36 Base62 Base64 และฐานกำหนดเองตั้งแต่ 2 ถึง 64 ได้โดยตรง ทุกการคำนวณทำในเครื่องด้วย BigInt จึงตรวจสอบค่าขนาดใหญ่ได้โดยไม่ต้องส่งไปยังเซิร์ฟเวอร์

## ใช้เมื่อใด

เครื่องมือนี้เหมาะเมื่อจำนวนเต็มค่าเดียวกันปรากฏใน log โปรโตคอล ID หรือสเปกด้วยชุดอักขระคนละแบบ เมื่อแก้ไขช่องใดช่องหนึ่ง ระบบจะคำนวณช่องอื่นใหม่ทันที จึงเหมาะกับการดีบัก การเขียนเอกสาร และการตรวจสอบด้วยตนเอง

## ความแตกต่างของฐาน

ฐานไม่เกิน 36 รับตัวอักษรแบบไม่แยกตัวพิมพ์ใหญ่พิมพ์เล็ก ส่วนฐานที่สูงกว่าจะถือว่าตัวพิมพ์ใหญ่และพิมพ์เล็กเป็นคนละหลัก และแถว Base64 ที่นี่ใช้ชุดอักขระเชิงตัวเลข `A-Z a-z 0-9 + /` ไม่ใช่การเข้ารหัส Base64 สำหรับข้อความแบบไบต์

## ข้อควรระวัง

รองรับเฉพาะจำนวนเต็มที่ไม่ติดลบ ศูนย์นำหน้าถูกมองเป็นเพียงรูปแบบการแสดงผล ดังนั้นผลลัพธ์ที่แปลงแล้วจะถูกปรับให้เป็นรูปแบบมาตรฐานและอาจไม่คง padding ที่คุณพิมพ์ไว้
