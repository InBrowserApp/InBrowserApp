## SHAKE256 (FIPS 202)とは？

SHAKE256 (FIPS 202) は SHA-3 ファミリーに属する可変長出力関数（XOF）です。固定長ハッシュ関数とは異なり、256 ビットの安全強度を保ちながら任意の出力ビット長を返せます。NIST によって FIPS 202 として標準化され、Keccak のスポンジ構造に基づいています。

この柔軟性は、プロトコルやファイル形式、内部チェックサム規則が特定のダイジェスト長を要求する場合に重要です。このツールではプレーンテキストやアップロードしたファイルをハッシュ化し、8 の倍数である限りビット単位で出力長を指定できます。

主な用途には、プロトコルハッシュ、鍵導出、可変長の暗号ダイジェスト、そして同じ入力と出力長から常に同じ結果が必要なデータ整合性ワークフローがあります。
