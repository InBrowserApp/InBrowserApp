## 什麼是 SHAKE256 (FIPS 202)？

SHAKE256 (FIPS 202) 是 SHA-3 家族中的可擴展輸出函數（XOF）。與固定長度雜湊不同，SHAKE256 可以輸出任意位數；其安全強度為 256 位元。它由 NIST 在 FIPS 202 中標準化，並基於 Keccak 海綿結構。

**主要特徵：**

- **XOF 輸出**：可產生可變長度摘要
- **確定性**：相同輸入與長度產生相同輸出
- **海綿結構**：基於 Keccak
- **強安全性**：256 位元安全強度
- **彈性**：適合需要可調輸出長度的場景

**常見用途：**

- 可變長度的加密摘要
- 金鑰派生與協定雜湊
- 校驗和與資料完整性
