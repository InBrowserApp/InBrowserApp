สร้าง UUID v4 ได้ในเครื่องบนเบราว์เซอร์ของคุณเมื่อคุณต้องการตัวระบุใหม่สำหรับระเบียนทดสอบ แถวฐานข้อมูล ตัวอย่าง API เพย์โหลดเหตุการณ์ fixtures หรือไฟล์กำหนดค่า เครื่องมือนี้สร้าง UUID รูปแบบมาตรฐานตัวพิมพ์เล็กครั้งละหนึ่งค่า เพื่อให้โฟกัสกับเวิร์กโฟลว์ค่าเดียวโดยไม่ซ้ำซ้อนกับตัวสร้างแบบจำนวนมากที่แยกต่างหาก

## UUID v4 หมายถึงอะไร

UUID v4 คือตัวระบุขนาด 128 บิตที่บิตเวอร์ชันและบิตแวเรียนต์ถูกกำหนดไว้ตายตัว ส่วนอีก 122 บิตมาจากข้อมูลสุ่ม จึงเหมาะเมื่อคุณต้องการตัวระบุที่ไม่เปิดเผยเวลาที่สร้าง ข้อมูลเครื่อง ตัวนับลำดับ หรือรายละเอียดผู้ใช้

## ควรใช้เมื่อใด

ใช้ UUID v4 สำหรับ ID ที่สร้างฝั่งไคลเอนต์ ออบเจ็กต์จำลอง ระเบียนชั่วคราว ตัวอย่างสาธารณะ และระบบแบบกระจายที่การประสานตัวนับส่วนกลางทำได้ยุ่งยาก เป็นค่าเริ่มต้นที่เหมาะเมื่อไม่ต้องคำนึงถึงลำดับการเรียง และคุณต้องการเพียงตัวระบุที่มีโอกาสชนกันต่ำ

## ความเป็นส่วนตัวและความน่าเชื่อถือ

การสร้างทำงานในแท็บเบราว์เซอร์นี้ด้วย Web Crypto ดังนั้น UUID จะไม่ถูกส่งไปยัง InBrowser.App หรือบริการอื่น คัดลอกค่าเมื่อดูถูกต้องแล้ว จากนั้นสร้างใหม่ได้ทุกครั้งที่คุณต้องการตัวระบุใหม่สำหรับระเบียนหรือตัวอย่างถัดไป
