# PDF Page Organizer

PDF Page Organizer는 문서를 서버에 업로드하지 않고도 PDF에서 페이지 순서를 바꾸고, 회전하고, 제거할 수 있게 해 줍니다. 선택한 페이지 순서로 새 PDF를 만들며 원본 파일은 변경하지 않습니다.

## 사용 시점

스캔한 문서를 바로잡거나, 실수로 들어간 빈 페이지를 제거하거나, 첨부 자료를 올바른 순서로 옮기거나, 옆으로 돌아간 페이지를 회전해야 할 때 사용하세요.

## 페이지 정리 방법

PDF를 업로드하고 페이지 목록을 검토한 다음, 각 페이지 옆의 컨트롤로 위아래로 이동하거나 시계 방향으로 회전하거나 내보내기에서 제거하세요. 순서가 맞으면 새 PDF를 생성해 다운로드하세요.

## 개인정보 보호 및 제한 사항

처리는 브라우저 세션에서 로컬로 이루어집니다. 비밀번호로 보호되었거나 암호화된 PDF는 지원되지 않으며, 매우 큰 문서는 읽거나 내보내는 데 시간이 더 걸릴 수 있습니다.
