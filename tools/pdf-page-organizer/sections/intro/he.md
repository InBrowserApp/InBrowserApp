# PDF Page Organizer

PDF Page Organizer מאפשר לך לסדר מחדש, לסובב ולהסיר עמודים מ-PDF בלי
להעלות את המסמך לשרת. הוא יוצר PDF חדש מסדר העמודים שתבחר, ומשאיר את
הקובץ המקורי ללא שינוי.

## מתי להשתמש בו

השתמש בו כשצריך לתקן מסמכים סרוקים, להסיר עמודים ריקים שנוספו בטעות,
להעביר נספחים לרצף הנכון או לסובב עמודים שנסרקו או צולמו על הצד.

## איך לארגן עמודים

העלה PDF, סקור את רשימת העמודים, ואז השתמש בפקדים שלצד כל עמוד כדי
להעביר אותו למעלה או למטה, לסובב אותו בכיוון השעון או להסיר אותו מהייצוא.
כשהסדר נראה נכון, צור והורד את ה-PDF החדש.

## פרטיות ומגבלות

העיבוד מתבצע באופן מקומי בהפעלת הדפדפן שלך. קובצי PDF המוגנים בסיסמה או
מוצפנים אינם נתמכים, ומסמכים גדולים מאוד עשויים להצריך זמן רב יותר לקריאה
או לייצוא.
