# PDF 頁面整理器

PDF 頁面整理器可讓你重新排序、旋轉和移除 PDF 中的頁面，不必將文件上傳到伺服器。它會依你選擇的頁面順序建立新的 PDF，原始檔案不會被變更。

## 適合使用的情境

當你需要修正掃描文件、移除意外出現的空白頁、將附件移到正確順序，或旋轉被橫向擷取的頁面時，可以使用這個工具。

## 如何整理頁面

上傳 PDF、檢視頁面清單，然後使用每個頁面旁的控制項將頁面上移或下移、順時針旋轉，或從匯出內容中移除。順序確認無誤後，產生並下載新的 PDF。

## 隱私與限制

處理會在你的瀏覽器工作階段本機完成。不支援受密碼保護或已加密的 PDF，文件非常大時，讀取或匯出可能需要較長時間。
