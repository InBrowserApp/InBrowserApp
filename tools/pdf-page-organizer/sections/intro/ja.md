# PDF Page Organizer

PDF Page Organizer は、ドキュメントをサーバーにアップロードせずに、PDF のページを並べ替え、回転、削除できるツールです。選択したページ順から新しい PDF を作成し、元のファイルは変更しません。

## 使う場面

スキャンした文書の順序を直したいとき、誤って入った空白ページを削除したいとき、添付資料を正しい順序に移動したいとき、横向きに取り込まれたページを回転したいときに使います。

## ページを整理する方法

PDF をアップロードし、ページリストを確認してから、各ページの横にある操作ボタンで上または下へ移動、時計回りに回転、書き出しから削除します。順序が整ったら、新しい PDF を生成してダウンロードします。

## プライバシーと制限

処理はブラウザセッション内でローカルに行われます。パスワード保護または暗号化された PDF には対応しておらず、非常に大きな文書は読み込みや書き出しに時間がかかることがあります。
