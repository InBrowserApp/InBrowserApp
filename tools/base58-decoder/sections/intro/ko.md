## 이 도구가 필요한 경우

이 도구는 Base58 문자열이나 텍스트 파일을 브라우저 안에서 원래 바이트로
되돌릴 때 사용합니다. API, 지갑, 로그, QR 흐름, 가져오기나 내보내기 단계에서
복사한 데이터를 서버로 보내지 않고 바로 확인하고 싶을 때 유용합니다.

## 알파벳을 바꿔야 할 때

Base58에는 하나의 공통 알파벳만 있는 것이 아닙니다. Bitcoin, Flickr,
Ripple은 서로 다른 문자 순서를 사용합니다. 값이 검증에 실패하거나
디코딩은 되지만 결과가 맞지 않는다면, 원본 시스템이 사용한 알파벳으로
바꿔서 다시 시도해야 합니다.

## 주의할 점

결과 패널은 디코딩된 바이트를 UTF-8 텍스트로 읽을 수 있을 때만 텍스트 미리보기를 보여줍니다.
바이너리 데이터나 텍스트가 아닌 내용은 .bin 파일을 내려받아 원본 바이트를 확인하는 편이 안전합니다.
붙여 넣은 값의 공백과 줄바꿈은 무시되므로 이메일, 문서, 터미널에서 줄이 나뉜 값도 그대로 디코딩할 수 있습니다.
