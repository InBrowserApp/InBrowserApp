## 이 도구가 하는 일

PDF Info Viewer는 브라우저에서 PDF를 열고 기본 파일 세부 정보, PDF
헤더 버전, 페이지 수, 첫 페이지 크기, 암호화 상태, 제목, 작성자, 주제,
키워드, 생성 도구, 제작 도구, 날짜와 같은 문서 메타데이터를 요약합니다.
문서를 공유, 보관, 디버깅 또는 처리하기 전에 빠르게 검사하는 데
사용합니다.

## 적합한 사용 사례

- 프린터, 클라이언트 또는 자동화 워크플로로 보내기 전에 PDF의 페이지 수와
  페이지 크기가 예상과 일치하는지 확인합니다.
- 검색, 기록 관리 또는 문서 감사에 영향을 줄 수 있는 제목, 작성자, 제작
  도구, 생성 날짜, 수정 날짜 필드를 검사합니다.
- 암호화된 PDF를 일찍 감지하여 다른 도구가 비밀번호 없이 텍스트를 추출하거나
  페이지를 병합하거나 메타데이터를 다시 쓰지 못하는 이유를 파악합니다.
- 지원 티켓, 자산 목록 또는 재현 가능한 버그 보고서를 위해 간결한 JSON
  요약을 내보냅니다.

## 개인정보 보호 참고 사항

PDF는 브라우저에서 로컬로 읽히며 이 도구를 통해 업로드되지 않습니다.
메타데이터에는 문서가 생성될 때 포함된 작성자 이름, 소프트웨어, 타임스탬프,
내부 제목 또는 워크플로 세부 정보가 여전히 노출될 수 있습니다. PDF를 외부에
게시하거나 공개 이슈에 첨부하기 전에 해당 필드를 검토하세요.

## 제한 사항

일부 암호화되었거나 손상된 PDF는 파일 이름, 크기, 헤더 버전만 노출합니다.
이 뷰어는 메타데이터 제거, 파일 복호화, 손상된 문서 복구 또는 시각적
레이아웃 검증을 수행하지 않습니다. 공유하기 전에 문서를 변경해야 하는 경우
전용 PDF 편집기나 정리 도구를 사용하세요.
