## ما الذي تفعله هذه الأداة

يفتح PDF Info Viewer ملف PDF في متصفحك ويلخص تفاصيله الأساسية، وإصدار ترويسة
PDF، وعدد الصفحات، وحجم الصفحة الأولى، وحالة التشفير، والبيانات الوصفية
للمستند مثل العنوان، والمؤلف، والموضوع، والكلمات المفتاحية، والمنشئ، والمنتج،
والتواريخ. وهو مخصص للفحص السريع قبل مشاركة مستند أو أرشفته أو تصحيح مشكلاته
أو معالجته.

## حالات استخدام مناسبة

- التحقق مما إذا كان ملف PDF يحتوي على عدد الصفحات وحجم الصفحة المتوقعين قبل
  إرساله إلى طابعة أو عميل أو سير عمل آلي.
- فحص حقول العنوان، والمؤلف، والمنتج، وتاريخ الإنشاء، وتاريخ التعديل التي قد
  تؤثر في البحث أو إدارة السجلات أو تدقيق المستندات.
- اكتشاف ملفات PDF المشفرة مبكرًا، حتى تعرف سبب عدم قدرة أدوات أخرى على
  استخراج النص أو دمج الصفحات أو إعادة كتابة البيانات الوصفية دون كلمة مرور.
- تصدير ملخص JSON موجز لتذاكر الدعم أو قوائم جرد الأصول أو تقارير الأخطاء
  القابلة لإعادة الإنتاج.

## ملاحظات الخصوصية

تتم قراءة ملف PDF محليًا في متصفحك ولا ترفعه هذه الأداة. قد تكشف البيانات
الوصفية مع ذلك أسماء المؤلفين أو البرامج أو الطوابع الزمنية أو العناوين
الداخلية أو تفاصيل سير العمل التي تم تضمينها عند إنشاء المستند. راجع هذه
الحقول قبل نشر ملف PDF خارجيًا أو إرفاقه بمشكلة عامة.

## القيود

تعرض بعض ملفات PDF المشفرة أو التالفة اسم الملف وحجمه وإصدار الترويسة فقط. لا
يزيل هذا العارض البيانات الوصفية، ولا يفك تشفير الملفات، ولا يصلح المستندات
المعطلة، ولا يتحقق من التخطيط المرئي. استخدم محرر PDF أو أداة تنقية مخصصة
عندما تحتاج إلى تغيير المستند قبل مشاركته.
