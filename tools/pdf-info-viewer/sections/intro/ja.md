## このツールでできること

PDF Info Viewer は、ブラウザ内で PDF を開き、基本的なファイル詳細、
PDF ヘッダーバージョン、ページ数、最初のページサイズ、暗号化状態、
タイトル、作成者、件名、キーワード、作成ツール、生成ツール、日付などの
ドキュメントメタデータを要約します。ドキュメントを共有、アーカイブ、
デバッグ、または処理する前にすばやく確認するためのツールです。

## 適した用途

- PDF をプリンター、クライアント、自動ワークフローに送る前に、
  想定どおりのページ数とページサイズになっているか確認する。
- 検索、記録管理、ドキュメント監査に影響する可能性があるタイトル、
  作成者、生成ツール、作成日時、更新日時の各フィールドを確認する。
- 暗号化された PDF を早めに検出し、他のツールがパスワードなしで
  テキスト抽出、ページ結合、メタデータ書き換えを行えない理由を把握する。
- サポートチケット、アセット一覧、再現可能なバグレポート向けに、
  コンパクトな JSON サマリーを書き出す。

## プライバシーに関する注意

PDF はブラウザ内でローカルに読み取られ、このツールによって
アップロードされることはありません。ただし、メタデータには作成者名、
ソフトウェア、タイムスタンプ、内部タイトル、ドキュメント作成時に
埋め込まれたワークフロー詳細が含まれる場合があります。PDF を外部に
公開したり公開チケットに添付したりする前に、これらのフィールドを確認してください。

## 制限事項

一部の暗号化または破損した PDF では、ファイル名、サイズ、ヘッダー
バージョンしか公開されません。このビューアーは、メタデータの削除、
ファイルの復号、破損したドキュメントの修復、表示レイアウトの検証は
行いません。共有前にドキュメントを変更する必要がある場合は、専用の
PDF エディターまたはサニタイザーを使用してください。
