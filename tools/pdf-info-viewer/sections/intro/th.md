## เครื่องมือนี้ทำอะไร

PDF Info Viewer เปิด PDF ในเบราว์เซอร์ของคุณและสรุปรายละเอียดไฟล์พื้นฐาน
เวอร์ชันส่วนหัวของ PDF, จำนวนหน้า, ขนาดหน้าแรก, สถานะการเข้ารหัส
และ metadata ของเอกสาร เช่น ชื่อเรื่อง ผู้เขียน หัวเรื่อง คำสำคัญ ผู้สร้าง
ผู้ผลิต และวันที่ต่าง ๆ เครื่องมือนี้ออกแบบมาสำหรับการตรวจสอบอย่างรวดเร็วก่อนแชร์
จัดเก็บถาวร ดีบัก หรือประมวลผลเอกสาร

## กรณีการใช้งานที่เหมาะสม

- ตรวจสอบว่า PDF มีจำนวนหน้าและขนาดหน้าตามที่คาดไว้ก่อนส่งไปยัง
  เครื่องพิมพ์ ลูกค้า หรือ workflow อัตโนมัติ
- ตรวจสอบฟิลด์ชื่อเรื่อง ผู้เขียน ผู้ผลิต วันที่สร้าง และวันที่แก้ไข
  ที่อาจส่งผลต่อการค้นหา การจัดการระเบียน หรือการตรวจสอบเอกสาร
- ตรวจพบ PDF ที่เข้ารหัสตั้งแต่เนิ่น ๆ เพื่อให้รู้ว่าเหตุใดเครื่องมืออื่น
  อาจไม่สามารถแยกข้อความ รวมหน้า หรือเขียน metadata ใหม่ได้โดยไม่มีรหัสผ่าน
- ส่งออกสรุป JSON แบบกะทัดรัดสำหรับตั๋วสนับสนุน สินค้าคงคลังของ asset
  หรือรายงานบั๊กที่ทำซ้ำได้

## หมายเหตุด้านความเป็นส่วนตัว

PDF จะถูกอ่านภายในเบราว์เซอร์ของคุณโดยตรงและเครื่องมือนี้จะไม่อัปโหลดไฟล์
อย่างไรก็ตาม metadata ยังอาจเปิดเผยชื่อผู้เขียน ซอฟต์แวร์ timestamp
ชื่อเรื่องภายใน หรือรายละเอียด workflow ที่ถูกฝังไว้เมื่อสร้างเอกสาร
ควรตรวจสอบฟิลด์เหล่านั้นก่อนเผยแพร่ PDF ภายนอกหรือแนบไปกับรายงานปัญหาสาธารณะ

## ข้อจำกัด

PDF บางไฟล์ที่เข้ารหัสหรือเสียหายจะแสดงได้เฉพาะชื่อไฟล์ ขนาดไฟล์ และเวอร์ชัน
ส่วนหัวเท่านั้น ตัวดูนี้ไม่ลบ metadata, ถอดรหัสไฟล์, ซ่อมแซมเอกสารที่เสียหาย
หรือตรวจสอบเลย์เอาต์ภาพ ใช้ตัวแก้ไข PDF หรือ sanitizer เฉพาะทางเมื่อคุณต้องการ
เปลี่ยนแปลงเอกสารก่อนแชร์
