## Qué hace esta herramienta

PDF Info Viewer abre un PDF en tu navegador y resume sus detalles básicos de
archivo, la versión del encabezado PDF, el recuento de páginas, el tamaño de la
primera página, el estado de cifrado y los metadatos del documento, como título,
autor, asunto, palabras clave, creador, productor y fechas. Está pensada para
una inspección rápida antes de compartir, archivar, depurar o procesar un
documento.

## Buenos casos de uso

- Comprobar si un PDF tiene el recuento de páginas y el tamaño de página
  esperados antes de enviarlo a una impresora, cliente o flujo de trabajo
  automatizado.
- Inspeccionar campos de título, autor, productor, fecha de creación y fecha de
  modificación que pueden afectar a la búsqueda, la gestión de registros o las
  auditorías de documentos.
- Detectar PDFs cifrados a tiempo, para saber por qué otras herramientas quizá
  no puedan extraer texto, combinar páginas o reescribir metadatos sin una
  contraseña.
- Exportar un resumen JSON compacto para tickets de soporte, inventarios de
  recursos o informes de errores reproducibles.

## Notas de privacidad

Esta herramienta lee el PDF localmente en tu navegador y no lo sube. Los
metadatos aún pueden exponer nombres de autores, software, marcas de tiempo,
títulos internos o detalles del flujo de trabajo que se incrustaron al crear el
documento. Revisa esos campos antes de publicar un PDF externamente o adjuntarlo
a una incidencia pública.

## Limitaciones

Algunos PDFs cifrados o dañados solo exponen el nombre del archivo, el tamaño y
la versión del encabezado. Este visor no elimina metadatos, descifra archivos,
repara documentos dañados ni valida el diseño visual. Usa un editor de PDF o un
sanitizador dedicado cuando necesites cambiar el documento antes de compartirlo.
