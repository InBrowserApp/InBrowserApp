## 這個工具的用途

PDF Info Viewer 會在你的瀏覽器中開啟 PDF，並摘要其基本檔案詳細資訊、
PDF 標頭版本、頁數、第一頁尺寸、加密狀態，以及文件中繼資料，例如標題、
作者、主旨、關鍵字、建立工具、產生工具和日期。它適合在分享、封存、偵錯或
處理文件前快速檢查。

## 適合使用的情境

- 在將 PDF 傳送給印表機、客戶或自動化流程前，確認頁數和頁面尺寸是否符合預期。
- 檢查可能影響搜尋、紀錄管理或文件稽核的標題、作者、產生工具、建立日期與修改日期欄位。
- 及早偵測已加密的 PDF，了解為什麼其他工具可能無法在沒有密碼時擷取文字、合併頁面或重寫中繼資料。
- 匯出精簡的 JSON 摘要，用於支援票證、素材清單或可重現的錯誤報告。

## 隱私注意事項

此工具會在你的瀏覽器本機讀取 PDF，不會上傳。中繼資料仍可能揭露作者姓名、
軟體、時間戳記、內部標題，或文件建立時嵌入的工作流程詳細資訊。在對外發布 PDF
或將其附加到公開議題前，請先檢查這些欄位。

## 限制

部分已加密或損毀的 PDF 只會揭露檔案名稱、大小與標頭版本。此檢視器不會移除
中繼資料、解密檔案、修復損壞文件，或驗證視覺版面。當你需要在分享前變更文件時，
請使用專用的 PDF 編輯器或清理工具。
