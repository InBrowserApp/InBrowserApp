## A cosa serve questo strumento

Usa questo strumento quando devi trasformare YAML valido in JSON più facile da leggere, condividere o versionare. Tutto viene eseguito localmente nel browser.

## Dove aiuta

- Convertire esempi API in JSON per documentazione o file di configurazione.
- Riscrivere payload YAML copiati in un formato più facile da confrontare a colpo d'occhio.
- Preparare contenuti strutturati per strumenti e flussi che si aspettano JSON.

## Cosa tenere presente

- In questo strumento l'input deve essere YAML valido.
- Il convertitore preserva la struttura dei dati, ma la resa JSON può comunque differire da uno stile scritto a mano.
- Se ti serve l'operazione inversa, usa invece il convertitore da JSON a YAML.
