## このツールの用途

有効な YAML を、読みやすく共有しやすくコミットにも向いた JSON に変換したいときに使うツールです。すべての処理はブラウザ内でローカルに実行されます。

## 向いている場面

- API のサンプルをドキュメントや設定ファイル向けの JSON に変換する。
- コピーした YAML ペイロードを目視で比較しやすい形式に整える。
- JSON を前提とするツールやワークフロー向けに構造化データを準備する。

## 注意したいこと

- このツールでは入力は有効な YAML である必要があります。
- コンバーターはデータ構造を保ちますが、JSON の見た目は手書きのスタイルと異なることがあります。
- 逆方向に変換したい場合は、JSON から YAML へのコンバーターを使ってください。
