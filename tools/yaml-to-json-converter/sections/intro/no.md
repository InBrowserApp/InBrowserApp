## Hva dette verktøyet er for

Bruk dette verktøyet når du trenger å gjøre gyldig YAML om til JSON som er enklere å lese, dele eller committe. Alt kjører lokalt i nettleseren.

## Når det hjelper

- Konvertere API-eksempler til JSON for dokumentasjon eller konfigurasjonsfiler.
- Skrive om kopierte YAML-payloads til et format som er lettere å sammenligne visuelt.
- Forberede strukturert innhold for verktøy og arbeidsflyter som forventer JSON.

## Hva du bør passe på

- Inndataene må være gyldig YAML i dette verktøyet.
- Konvertereren bevarer datastrukturen, men JSON-presentasjonen kan fortsatt avvike fra håndskrevet stil.
- Hvis du trenger å gå motsatt vei, bruk JSON til YAML-konvertereren i stedet.
