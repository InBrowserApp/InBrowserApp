## Для чего нужен этот инструмент

Используйте этот инструмент, когда нужно превратить корректный YAML в JSON, который легче читать, делиться им или коммитить. Все выполняется локально в браузере.

## Где он полезен

- Преобразовывать примеры API в JSON для документации или конфигурационных файлов.
- Переписывать скопированные YAML-payload'ы в формат, который проще сравнивать глазами.
- Подготавливать структурированный контент для инструментов и процессов, которые ожидают JSON.

## Что важно учитывать

- В этом инструменте входные данные должны быть корректным YAML.
- Конвертер сохраняет структуру данных, но представление JSON может отличаться от написанного вручную стиля.
- Если нужно преобразование в обратную сторону, используйте конвертер JSON в YAML.
