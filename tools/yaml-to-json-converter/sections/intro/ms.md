## Kegunaan alat ini

Gunakan alat ini apabila anda perlu menukar YAML yang sah kepada JSON yang lebih mudah dibaca, dikongsi atau di-commit. Semuanya berjalan secara setempat dalam pelayar.

## Bila ia membantu

- Menukar contoh API kepada JSON untuk dokumentasi atau fail konfigurasi.
- Menulis semula payload YAML yang disalin kepada format yang lebih mudah dibandingkan dengan mata.
- Menyediakan kandungan berstruktur untuk alat dan aliran kerja yang menjangkakan JSON.

## Perkara yang perlu diberi perhatian

- Dalam alat ini, input mestilah YAML yang sah.
- Penukar mengekalkan struktur data, tetapi persembahan JSON masih boleh berbeza daripada gaya yang ditulis tangan.
- Jika anda perlu pergi ke arah sebaliknya, gunakan penukar JSON ke YAML.
