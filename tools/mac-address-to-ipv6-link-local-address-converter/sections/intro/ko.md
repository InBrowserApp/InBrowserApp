## IPv6 링크 로컬 주소란 무엇인가요?

IPv6 링크 로컬 주소는 IPv6가 활성화된 모든 인터페이스에서 자동으로 구성되는 특별한 IPv6 주소입니다. 항상 fe80::/10 접두사로 시작하며 동일한 네트워크 세그먼트의 장치 간 통신에 사용됩니다. 이러한 주소는 로컬 링크를 넘어서 라우팅할 수 없으며 일반적으로 인접 장치 발견, 라우터 발견 및 기타 로컬 네트워크 프로토콜에 사용됩니다. 링크 로컬 주소는 EUI-64 형식을 사용하여 장치의 MAC 주소에서 생성할 수 있습니다.

### 언제 쓰면 좋은가

장치의 MAC 주소에서 EUI-64로 결정되는 링크 로컬 주소를 구할 때 유용합니다.

### EUI-64 매핑 방식

1. MAC 주소를 48비트로 정규화합니다.
2. 첫 번째 바이트의 `U/L bit` 를 뒤집습니다.
3. 가운데에 `ff:fe` 를 넣어 64비트 인터페이스 식별자를 만듭니다.
4. 앞에 `fe80::/10` 접두사를 붙입니다.

### 지원하는 입력 형식

- `00:11:22:33:44:55`
- `00-11-22-33-44-55`
- `0011.2233.4455`
- `001122334455`

### 선택 인터페이스 접미사

로컬 명령이 어떤 인터페이스를 써야 하는지 알아야 할 때만 `%eth0`, `%en0` 같은 zone index를 붙이세요.
