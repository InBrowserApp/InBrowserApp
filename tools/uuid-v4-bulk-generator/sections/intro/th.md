สร้างชุดตัวระบุ UUID v4 โดยตรงในเบราว์เซอร์ของคุณ เมื่อคุณต้องการ ID แบบสุ่มสำหรับแถวฐานข้อมูล ข้อมูลจำลองของ API คีย์ออบเจ็กต์ เพย์โหลดทดสอบ เทมเพลตนำเข้า หรืองานปฏิบัติการเฉพาะครั้ง

## สิ่งที่ UUID v4 มอบให้

UUID v4 คือตัวระบุ 128 บิตที่สร้างขึ้นจากไบต์สุ่มที่ปลอดภัยเชิงเข้ารหัสเป็นส่วนใหญ่ บิตของเวอร์ชันและตัวแปรถูกกำหนดตามรูปแบบ RFC 4122 ดังนั้น UUID v4 จึงมีรูปทรงที่คุ้นเคยแบบ `xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx` พร้อมยังคงมีพื้นที่สุ่มขนาดใหญ่มาก

## เลือกขนาดชุดที่ใช้งานได้จริง

ชุดเริ่มต้นให้ ID เพียงพอสำหรับงานข้อมูลจำลองและสเปรดชีตจำนวนมาก โดยไม่ทำให้หน้าจอสแกนดูยาก เพิ่มจำนวนเมื่อคุณกำลังเตรียมการนำเข้าขนาดใหญ่ หรือลดจำนวนเมื่อคุณต้องการตัวระบุเพียงไม่กี่รายการสำหรับตัวอย่างคำขอหรือการแก้ไขฐานข้อมูลด้วยตนเอง

## คัดลอกหรือส่งออก

ตรวจสอบรายการที่สร้างแล้ว จากนั้นคัดลอกไปยังตัวแก้ไขของคุณหรือดาวน์โหลดเป็นไฟล์ข้อความล้วน ค่าทุกรายการถูกสร้างภายในเครื่อง และชุดปัจจุบันจะไม่ถูกอัปโหลดโดยเครื่องมือนี้

## คำแนะนำเรื่องการชนกัน

ความเสี่ยงที่ UUID v4 จะชนกันนั้นต่ำมากสำหรับภาระงานแอปพลิเคชันทั่วไป แต่ไม่ใช่สิ่งทดแทนข้อจำกัดความไม่ซ้ำในฐานข้อมูล ควรบังคับใช้ความไม่ซ้ำต่อไปในกรณีที่ ID กลายเป็นคีย์หลัก โทเคนสาธารณะ หรือการอ้างอิงถาวร
