當你需要為資料庫資料列、API 測試資料、物件鍵、測試 payload、匯入範本或一次性的維運工作建立隨機 ID 時，可直接在瀏覽器中產生一批 UUID v4 識別碼。

## UUID v4 提供什麼

UUID v4 是一種 128 位元識別碼，主要由密碼學安全的隨機位元組組成。版本與變體位元由 RFC 4122 版面固定，因此 UUID v4 會呈現熟悉的 `xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx` 形狀，同時仍保有非常大的隨機空間。

## 選擇實用的批次大小

預設批次提供足夠多的 ID，可支援許多測試資料與試算表工作流程，同時不會讓頁面難以掃讀。準備較大型匯入時可增加數量；若只需要少量識別碼用於請求範例或手動資料庫編輯，則可降低數量。

## 複製或匯出

檢查產生的清單後，可將它複製到編輯器，或下載為純文字檔。每個值都會在本機產生，此工具絕不會上傳目前批次。

## 碰撞風險指南

對一般應用程式工作負載而言，UUID v4 的碰撞風險極低，但它不能取代資料庫唯一性約束。當 ID 會成為主鍵、公開 token 或持久參照時，仍應持續強制唯一性。
