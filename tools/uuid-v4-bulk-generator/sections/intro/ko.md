데이터베이스 행, API 픽스처, 객체 키, 테스트 페이로드, 가져오기 템플릿 또는 일회성 운영 작업에 임의 ID가 필요할 때 브라우저에서 직접 UUID v4 식별자 배치를 생성하세요.

## What UUID v4 Provides

UUID v4는 대부분 암호학적으로 안전한 임의 바이트로 구성된 128비트 식별자입니다. 버전 및 variant 비트는 RFC 4122 레이아웃에 따라 고정되므로, UUID v4는 매우 큰 임의 공간을 유지하면서도 익숙한 `xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx` 형식을 가집니다.

## Pick A Practical Batch Size

기본 배치는 페이지를 살펴보기 어렵게 만들지 않으면서도 많은 픽스처 및 스프레드시트 작업에 충분한 ID를 제공합니다. 더 큰 가져오기를 준비할 때는 개수를 늘리고, 요청 샘플이나 수동 데이터베이스 편집에 몇 개의 식별자만 필요할 때는 줄이세요.

## Copy Or Export

생성된 목록을 확인한 뒤 편집기에 복사하거나 일반 텍스트 파일로 다운로드하세요. 모든 값은 로컬에서 생성되며, 현재 배치는 이 도구를 통해 업로드되지 않습니다.

## Collision Guidance

UUID v4 충돌 위험은 일반적인 애플리케이션 워크로드에서는 매우 낮지만, 데이터베이스 고유성 제약 조건을 대체하지는 않습니다. ID가 기본 키, 공개 토큰 또는 영구 참조가 되는 곳에서는 고유성 적용을 계속 유지하세요.
