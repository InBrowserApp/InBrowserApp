## 這個工具的用途

此 SVG 優化器會在你的瀏覽器中壓縮一個本機 SVG 檔案或貼上的 SVG 文件。
它使用 SVGO 清理流程移除註解、中繼資料、多餘屬性、不必要的精度，以及其他不會
改變可見影像的標記。

## 它為何有幫助

從設計工具匯出的 SVG 檔案通常包含編輯器中繼資料、冗長的路徑、未使用的 ID 和
註解。優化這些檔案可以降低下載大小、改善頁面載入，並讓內嵌 SVG 程式碼在發布到
網站、應用程式、電子郵件或文件頁面前更容易審閱。

## 運作方式

上傳 `.svg` 檔案或貼上 SVG 標記，選擇安全預設集或調整個別 SVGO 處理程序，然後
執行優化。此工具會顯示原始與優化後預覽、節省的位元組數，以及最終標記，方便你
複製或下載 `.optimized.svg` 檔案。SVG 不需要離開你的裝置。

## 實用注意事項

- 當 SVG 依賴外部 CSS、scripted IDs 或你無法輕易檢查的 symbol 參照時，請保留安全預設集。
- 對於簡單匯出的圖示、標誌和插圖，如果可以接受移除尺寸和內嵌樣式，請使用積極預設集。
- 替換來源圖稿前，請先預覽優化後的影像，尤其是來源使用遮罩、漸層、濾鏡或連結資產時。
