## ما الذي تفعله هذه الأداة

يضغط SVG Optimizer ملف SVG محليًا واحدًا أو مستند SVG ملصقًا في متصفحك. يستخدم
تمريرات تنظيف SVGO لإزالة التعليقات، والبيانات الوصفية، والسمات الزائدة، والدقة
غير الضرورية، وأي ترميز آخر لا يغيّر الصورة المرئية.

## لماذا يساعد ذلك

غالبًا ما تحتوي ملفات SVG المصدّرة من أدوات التصميم على بيانات وصفية للمحرر،
ومسارات مطوّلة، وIDs غير مستخدمة، وتعليقات. يمكن أن يقلّل تحسينها حجم التنزيل،
ويحسّن تحميل الصفحة، ويجعل كود SVG المضمّن أسهل للمراجعة قبل نشره في موقع ويب،
أو تطبيق، أو بريد إلكتروني، أو صفحة توثيق.

## كيف يعمل

ارفع ملف `.svg` أو الصق ترميز SVG، ثم اختر الإعداد المسبق الآمن أو اضبط تمريرات
SVGO الفردية، وبعدها شغّل التحسين. تعرض الأداة المعاينات الأصلية والمحسّنة،
والتوفير بالبايت، والترميز النهائي حتى تتمكن من نسخه أو تنزيل ملف
`.optimized.svg`. لا يحتاج SVG أبدًا إلى مغادرة جهازك.

## ملاحظات عملية

- أبقِ الإعداد المسبق الآمن عندما يعتمد SVG على CSS خارجي، أو IDs مستخدمة في
  سكربتات، أو مراجع رموز لا يمكنك فحصها بسهولة.
- استخدم الإعداد المسبق القوي للأيقونات، والشعارات، والرسوم التوضيحية البسيطة
  المصدّرة عندما تكون إزالة الأبعاد وتضمين الأنماط مقبولين.
- عاين الصورة المحسّنة قبل استبدال العمل الفني المصدر، خصوصًا عندما يستخدم
  المصدر أقنعة، أو تدرجات، أو فلاتر، أو أصولًا مرتبطة.
