## เครื่องมือนี้ทำอะไร

SVG Optimizer นี้บีบอัดไฟล์ SVG ในเครื่องหนึ่งไฟล์หรือเอกสาร SVG ที่วางลงมาใน
เบราว์เซอร์ของคุณ เครื่องมือนี้ใช้ SVGO cleanup passes เพื่อลบความคิดเห็น metadata
แอตทริบิวต์ซ้ำซ้อน ความละเอียดที่ไม่จำเป็น และมาร์กอัปอื่นๆ ที่ไม่เปลี่ยนภาพ
ที่มองเห็นได้

## มีประโยชน์อย่างไร

ไฟล์ SVG ที่ส่งออกจากเครื่องมือออกแบบมักมี metadata ของเครื่องมือแก้ไข path ที่ยืดยาว
IDs ที่ไม่ได้ใช้ และความคิดเห็น การปรับแต่งไฟล์เหล่านี้สามารถลดขนาดดาวน์โหลด
ช่วยให้หน้าเว็บโหลดเร็วขึ้น และทำให้โค้ด SVG แบบ inline ตรวจทานได้ง่ายขึ้นก่อนนำไปใช้
ในเว็บไซต์ แอป อีเมล หรือหน้าเอกสาร

## ทำงานอย่างไร

อัปโหลดไฟล์ `.svg` หรือวางมาร์กอัป SVG เลือกพรีเซ็ตปลอดภัยหรือปรับ
SVGO passes แต่ละรายการ จากนั้นเรียกใช้การปรับแต่ง เครื่องมือจะแสดงตัวอย่างต้นฉบับ
และผลลัพธ์ที่ปรับแต่งแล้ว จำนวนไบต์ที่ประหยัดได้ และมาร์กอัปสุดท้าย เพื่อให้คุณคัดลอก
หรือดาวน์โหลดไฟล์ `.optimized.svg` ได้ SVG ไม่จำเป็นต้องออกจากอุปกรณ์ของคุณเลย

## หมายเหตุเชิงปฏิบัติ

- ใช้พรีเซ็ตปลอดภัยต่อไปเมื่อ SVG พึ่งพา CSS ภายนอก IDs ที่สคริปต์ใช้งาน หรือ
  การอ้างอิง symbol ที่คุณตรวจสอบได้ไม่ง่าย
- ใช้พรีเซ็ตเข้มข้นสำหรับไอคอน โลโก้ และภาพประกอบแบบง่ายที่ส่งออกมา
  เมื่อการลบขนาดและการทำ inline styles เป็นสิ่งที่ยอมรับได้
- ดูตัวอย่างภาพที่ปรับแต่งแล้วก่อนแทนที่อาร์ตเวิร์กต้นฉบับ โดยเฉพาะเมื่อ
  ต้นฉบับใช้ masks, gradients, filters หรือ assets ที่ลิงก์ไว้
