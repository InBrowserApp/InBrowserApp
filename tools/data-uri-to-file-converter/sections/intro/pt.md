## Por que os Data URI aparecem

Os Data URI empacotam um arquivo e seus metadados em uma única string, por isso aparecem com frequência em HTML, CSS, SVG, modelos de email, respostas de API e exportações do navegador. Eles são práticos para recursos pequenos, mas difíceis de inspecionar quando você só tem o valor codificado.

## O que este conversor entrega

Cole um `data:` URI completo para decodificá-lo localmente no navegador. A ferramenta mostra o tipo MIME, informa se a carga usa Base64 ou codificação de URL, faz pré-visualização de texto, imagens, áudio ou vídeo quando o navegador consegue renderizar e ainda sugere um nome de arquivo com base no tipo de mídia.

## O que verificar antes de salvar

Um Data URI válido ainda pode trazer um tipo MIME incorreto ou uma extensão enganosa. Compare o painel de detalhes com o que você esperava, confira a pré-visualização quando houver e renomeie o arquivo antes do download se precisar de outro nome.
