## Почему встречаются Data URI

Data URI упаковывает файл и его метаданные в одну строку, поэтому такие значения часто встречаются в HTML, CSS, SVG, шаблонах писем, ответах API и экспортируемых данных из браузера. Для небольших ресурсов это удобно, но проверять содержимое трудно, когда у вас осталась только закодированная строка.

## Что дает этот конвертер

Вставьте полный `data:` URI, чтобы декодировать его локально прямо в браузере. Инструмент показывает MIME-тип, указывает Base64 или URL-кодирование, умеет предварительно показывать текст, изображения, аудио и видео, если браузер поддерживает такой просмотр, и предлагает имя файла для скачивания по типу данных.

## Что проверить перед сохранением

Даже корректный Data URI может содержать неверный MIME-тип или вводящее в заблуждение расширение. Сравните панель деталей с ожидаемым результатом, посмотрите предпросмотр и при необходимости переименуйте файл перед загрузкой.
