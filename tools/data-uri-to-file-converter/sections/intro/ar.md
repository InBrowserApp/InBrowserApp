## لماذا تظهر سلاسل Data URI

تجمع Data URI الملف وبياناته الوصفية داخل سلسلة واحدة، لذلك تظهر كثيراً في HTML وCSS وSVG وقوالب البريد الإلكتروني وحمولات API وعمليات التصدير من المتصفح. هذا مناسب للموارد الصغيرة، لكنه يصبح مزعجاً عندما لا يبقى لديك سوى النص المشفّر.

## ماذا يقدّم هذا المحول

الصق `data:` URI كاملاً لفك ترميزه محلياً داخل المتصفح. تعرض الأداة نوع MIME، وتوضح ما إذا كانت الحمولة تستخدم Base64 أو ترميز URL، وتقدّم معاينة للنصوص أو الصور أو الصوت أو الفيديو عندما يستطيع المتصفح عرضها، كما تقترح اسم ملف مناسباً حسب نوع الوسائط.

## ما الذي يجب التحقق منه قبل الحفظ

حتى لو كانت صيغة Data URI صحيحة، فقد تحمل نوع MIME غير صحيح أو امتداداً مضللاً. قارن لوحة التفاصيل بما تتوقعه، وراجع المعاينة إن كانت متاحة، ثم غيّر اسم الملف قبل التنزيل إذا كنت تحتاج إلى اسم أو امتداد مختلف.
