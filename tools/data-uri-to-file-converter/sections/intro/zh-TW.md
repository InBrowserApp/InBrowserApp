## 為什麼會遇到 Data URI

Data URI 會把檔案內容和中繼資料打包成一個字串，所以它常出現在 HTML、CSS、SVG、郵件範本、API 回應和瀏覽器匯出內容裡。它很適合攜帶小型資源，但當你手上只剩下一長串編碼文字時，就不容易直接檢查。

## 這個轉換器能做什麼

把完整的 `data:` URI 貼上來，就能在瀏覽器本機解碼。工具會顯示 MIME 類型、說明資料是 Base64 還是 URL 編碼，並在瀏覽器支援時預覽文字、圖片、音訊或影片，也會依媒體類型提供建議檔名。

## 下載前要檢查什麼

就算 Data URI 本身有效，也可能帶著錯誤的 MIME 類型或誤導性的副檔名。下載前先比對詳細資訊和預覽結果，必要時先改好檔名，再儲存會比較穩妥。
