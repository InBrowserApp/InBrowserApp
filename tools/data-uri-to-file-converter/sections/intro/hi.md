## Data URI क्यों दिखाई देते हैं

Data URI किसी फ़ाइल और उसके मेटाडेटा को एक ही स्ट्रिंग में पैक कर देता है, इसलिए यह HTML, CSS, SVG, ईमेल टेम्पलेट, API payload और ब्राउज़र export में अक्सर मिलता है। छोटे assets के लिए यह सुविधाजनक है, लेकिन जब आपके पास सिर्फ encoded value बचती है तब उसे जांचना मुश्किल हो जाता है।

## यह कन्वर्टर क्या देता है

पूरा `data:` URI पेस्ट करें और उसे ब्राउज़र में लोकल रूप से डिकोड करें। यह टूल MIME प्रकार दिखाता है, बताता है कि payload Base64 है या URL-encoded, ब्राउज़र समर्थित होने पर text, image, audio या video का preview देता है, और media type के आधार पर download filename भी सुझाता है।

## सेव करने से पहले क्या जांचें

सही दिखने वाला Data URI भी गलत MIME प्रकार या भ्रामक extension ला सकता है। details panel को अपनी अपेक्षा से मिलाएँ, preview उपलब्ध हो तो उसे देखें, और ज़रूरत पड़े तो download से पहले फ़ाइल का नाम बदल लें।
