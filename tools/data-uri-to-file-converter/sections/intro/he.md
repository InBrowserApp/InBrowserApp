## למה נתקלים ב-Data URI

Data URI אורז קובץ והמטא-נתונים שלו למחרוזת אחת, ולכן הוא מופיע הרבה ב-HTML, ב-CSS, ב-SVG, בתבניות דוא״ל, בתשובות API ובייצוא מהדפדפן. זה נוח לקבצים קטנים, אבל קשה לבדוק את התוכן כשנשארת רק המחרוזת המקודדת.

## מה הכלי הזה נותן

הדבק `data:` URI מלא כדי לפענח אותו מקומית בתוך הדפדפן. הכלי מציג את סוג ה-MIME, מסמן אם המטען הוא Base64 או בקידוד URL, מציג תצוגה מקדימה של טקסט, תמונה, אודיו או וידאו כשהדפדפן תומך בכך, וגם מציע שם קובץ להורדה לפי סוג המדיה.

## מה כדאי לבדוק לפני ששומרים

גם Data URI תקין יכול להכיל סוג MIME שגוי או סיומת מטעה. השווה את לוח הפרטים למה שציפית לקבל, בדוק את התצוגה המקדימה אם יש כזו, ושנה את שם הקובץ לפני ההורדה אם צריך שם אחר.
