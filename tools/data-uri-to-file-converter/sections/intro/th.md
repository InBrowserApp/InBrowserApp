## ทำไมจึงเจอ Data URI

Data URI จะรวมไฟล์และข้อมูลประกอบไว้ในสตริงเดียว จึงมักพบใน HTML, CSS, SVG, เทมเพลตอีเมล, ข้อมูลจาก API และไฟล์ที่ส่งออกจากเบราว์เซอร์ วิธีนี้เหมาะกับทรัพยากรขนาดเล็ก แต่จะตรวจสอบได้ยากเมื่อเหลือเพียงข้อความที่เข้ารหัสไว้

## เครื่องมือนี้ช่วยอะไรได้บ้าง

วาง `data:` URI แบบเต็มเพื่อถอดรหัสภายในเบราว์เซอร์ของคุณโดยตรง เครื่องมือจะแสดงชนิด MIME บอกว่าข้อมูลเป็น Base64 หรือ URL encoded แสดงตัวอย่างข้อความ รูปภาพ เสียง หรือวิดีโอเมื่อเบราว์เซอร์รองรับ และแนะนำชื่อไฟล์สำหรับดาวน์โหลดตามชนิดสื่อ

## ควรตรวจอะไรบ้างก่อนบันทึก

แม้ Data URI จะถูกต้องตามรูปแบบ ก็อาจระบุชนิด MIME ผิดหรือใช้นามสกุลไฟล์ที่ทำให้เข้าใจผิดได้ เปรียบเทียบข้อมูลในแผงรายละเอียดกับสิ่งที่คุณคาดไว้ ดูตัวอย่างถ้ามี และเปลี่ยนชื่อไฟล์ก่อนดาวน์โหลดหากต้องการชื่ออื่น
