## 为什么会遇到 Data URI

Data URI 会把文件内容和元数据打包成一个字符串，所以它常出现在 HTML、CSS、SVG、邮件模板、API 返回值和浏览器导出结果里。它很适合携带小资源，但当你只拿到一长串编码文本时，人工检查会很麻烦。

## 这个转换器能做什么

把完整的 `data:` URI 粘贴进来，就可以在浏览器本地解码。工具会显示 MIME 类型、标明是 Base64 还是 URL 编码，并在浏览器支持时预览文本、图片、音频或视频，还会按媒体类型给出建议文件名。

## 下载前要检查什么

Data URI 就算语法正确，也可能带着错误的 MIME 类型或误导性的扩展名。下载前先对照详情面板和预览结果，必要时先改一下文件名，再保存更稳妥。
