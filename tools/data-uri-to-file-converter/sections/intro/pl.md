## Dlaczego trafiasz na Data URI

Data URI pakuje plik i jego metadane do jednego ciągu znaków, dlatego często pojawia się w HTML, CSS, SVG, szablonach e-maili, odpowiedziach API i eksportach z przeglądarki. To wygodne dla małych zasobów, ale trudne do sprawdzenia, gdy masz już tylko zakodowaną wartość.

## Co daje ten konwerter

Wklej pełny `data:` URI, aby zdekodować go lokalnie w przeglądarce. Narzędzie pokazuje typ MIME, informuje, czy dane są zapisane w Base64 czy w kodowaniu URL, potrafi podejrzeć tekst, obraz, dźwięk lub wideo, jeśli przeglądarka to obsługuje, i proponuje nazwę pliku na podstawie typu nośnika.

## Co sprawdzić przed zapisaniem

Nawet poprawny Data URI może zawierać zły typ MIME albo mylące rozszerzenie. Porównaj panel szczegółów z oczekiwanym wynikiem, obejrzyj podgląd, jeśli jest dostępny, i w razie potrzeby zmień nazwę pliku przed pobraniem.
