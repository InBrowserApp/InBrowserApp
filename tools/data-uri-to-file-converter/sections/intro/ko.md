## 왜 Data URI를 보게 되나

Data URI는 파일과 메타데이터를 하나의 문자열로 묶기 때문에 HTML, CSS, SVG, 이메일 템플릿, API 응답, 브라우저 내보내기 결과에서 자주 등장합니다. 작은 리소스를 옮길 때는 편리하지만, 인코딩된 값만 남아 있으면 내용을 확인하기가 까다롭습니다.

## 이 변환기로 할 수 있는 일

전체 `data:` URI를 붙여 넣으면 브라우저 안에서 로컬로 디코딩할 수 있습니다. 도구는 MIME 유형을 보여 주고, 데이터가 Base64인지 URL 인코딩인지 알려 주며, 브라우저가 지원하면 텍스트·이미지·오디오·비디오를 미리 보여 주고, 미디어 유형에 맞는 다운로드 파일명도 제안합니다.

## 저장 전에 확인할 점

올바른 Data URI라 해도 MIME 유형이나 확장자가 기대와 다를 수 있습니다. 세부 정보와 미리보기를 먼저 확인하고, 다른 이름이나 확장자가 필요하면 다운로드 전에 파일명을 바꾼 뒤 저장하세요.
