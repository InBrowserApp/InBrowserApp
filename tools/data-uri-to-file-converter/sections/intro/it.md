## Perché compaiono i Data URI

I Data URI racchiudono un file e i suoi metadati in una sola stringa, quindi compaiono spesso in HTML, CSS, SVG, template email, payload API ed esportazioni del browser. Sono comodi per risorse piccole, ma diventano scomodi da controllare quando hai soltanto il valore codificato.

## Cosa offre questo convertitore

Incolla un `data:` URI completo per decodificarlo localmente nel browser. Lo strumento mostra il tipo MIME, indica se il contenuto usa Base64 o codifica URL, visualizza in anteprima testo, immagini, audio o video quando il browser li supporta e suggerisce un nome file coerente con il tipo di media.

## Cosa controllare prima di salvare

Un Data URI valido può comunque portare un tipo MIME sbagliato o un’estensione fuorviante. Confronta il pannello dei dettagli con il risultato atteso, guarda l’anteprima quando disponibile e rinomina il file prima del download se ti serve un nome diverso.
