## Apa itu SM3?

SM3 adalah fungsi hash kriptografi yang distandarkan di Tiongkok (GM/T 0004-2012). Fungsi ini menghasilkan nilai hash 256-bit (32 byte), biasanya ditampilkan sebagai string heksadesimal 64 karakter.

**Karakteristik utama:**

- **Deterministik**: Input yang sama selalu menghasilkan hash yang sama
- **Efek avalanche**: Perubahan kecil pada input menghasilkan output yang sangat berbeda
- **Tahan collision**: Dirancang agar collision sulit ditemukan

**Penggunaan umum:**

- Pemeriksaan integritas data
- Tanda tangan digital (SM2)
- Pengidentifikasi aman
