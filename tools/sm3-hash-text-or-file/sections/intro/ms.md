## Apakah SM3?

SM3 ialah fungsi hash kriptografi yang dinormakan di China (GM/T 0004-2012). Ia menghasilkan nilai hash 256-bit (32 bait), biasanya dipaparkan sebagai rentetan heksadesimal 64 aksara.

**Ciri-ciri utama:**

- **Deterministik**: Input yang sama sentiasa menghasilkan hash yang sama
- **Kesan runtuhan salji**: Perubahan kecil pada input menghasilkan output yang sangat berbeza
- **Tahan perlanggaran**: Direka untuk menyukarkan pencarian perlanggaran

**Kegunaan biasa:**

- Pemeriksaan integriti data
- Tandatangan digital (SM2)
- Pengenal pasti selamat
