## Qu'est-ce que SM3 ?

SM3 est une fonction de hachage cryptographique normalisée en Chine (GM/T 0004-2012). Elle produit une valeur de hachage de 256 bits (32 octets), généralement affichée comme une chaîne hexadécimale de 64 caractères.

**Caractéristiques clés :**

- **Déterministe** : La même entrée produit toujours le même hachage
- **Effet d'avalanche** : De petits changements dans l'entrée produisent des sorties très différentes
- **Résistant aux collisions** : Conçue pour rendre les collisions difficiles à trouver

**Utilisations courantes :**

- Vérification d'intégrité des données
- Signatures numériques (SM2)
- Identifiants sécurisés
