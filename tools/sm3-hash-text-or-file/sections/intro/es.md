## ¿Qué es SM3?

SM3 es una función hash criptográfica estandarizada en China (GM/T 0004-2012). Produce un valor hash de 256 bits (32 bytes), normalmente mostrado como una cadena hexadecimal de 64 caracteres.

**Características clave:**

- **Determinista**: La misma entrada siempre produce el mismo hash
- **Efecto avalancha**: Pequeños cambios en la entrada producen salidas muy diferentes
- **Resistente a colisiones**: Diseñada para dificultar encontrar colisiones

**Usos comunes:**

- Verificación de integridad de datos
- Firmas digitales (SM2)
- Identificadores seguros
