## Was ist SM3?

SM3 ist eine in China standardisierte kryptografische Hash-Funktion (GM/T 0004-2012). Sie erzeugt einen 256-Bit-(32-Byte-)Hashwert, der typischerweise als 64-stellige Hexadezimalzeichenkette dargestellt wird.

**Hauptmerkmale:**

- **Deterministisch**: Dieselbe Eingabe erzeugt immer denselben Hash
- **Lawineneffekt**: Kleine Änderungen der Eingabe erzeugen sehr unterschiedliche Ausgaben
- **Kollisionsresistent**: Entwickelt, um Kollisionen schwer auffindbar zu machen

**Häufige Verwendungen:**

- Datenintegritätsprüfungen
- Digitale Signaturen (SM2)
- Sichere Kennungen
