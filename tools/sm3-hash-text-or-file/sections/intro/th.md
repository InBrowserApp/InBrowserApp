## SM3 คืออะไร?

SM3 เป็นฟังก์ชันแฮชเชิงเข้ารหัสที่ได้มาตรฐานในจีน (GM/T 0004-2012) ให้ค่าแฮช 256 บิต (32 ไบต์) โดยมักแสดงเป็นสตริงเลขฐานสิบหก 64 ตัวอักษร

**คุณลักษณะสำคัญ:**

- **กำหนดได้**: อินพุตเดียวกันให้แฮชเหมือนเดิมเสมอ
- **เอฟเฟกต์หิมะถล่ม**: การเปลี่ยนแปลงเล็กน้อยในอินพุตทำให้ได้ผลลัพธ์ที่ต่างมาก
- **ต้านทานการชน**: ออกแบบมาให้หาการชนได้ยาก

**การใช้งานทั่วไป:**

- การตรวจสอบความสมบูรณ์ของข้อมูล
- ลายเซ็นดิจิทัล (SM2)
- ตัวระบุที่ปลอดภัย
