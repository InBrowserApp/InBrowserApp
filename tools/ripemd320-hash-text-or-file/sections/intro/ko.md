## RIPEMD-320이란?

RIPEMD-320(RACE Integrity Primitives Evaluation Message Digest)은 320비트(40바이트) 해시 값을 생성하는 암호화 해시 함수로, 일반적으로 80자리 16진수로 표현됩니다. MD4/MD5의 대안으로 유럽에서 개발된 RIPEMD 계열의 일부입니다.

붙여넣은 텍스트, 복사한 설정 데이터 또는 로컬 파일의 RIPEMD-320 다이제스트를 계산해야 할 때 이 도구를 사용하세요. 계산은 브라우저에서 실행되므로 파일 콘텐츠를 서버에 업로드할 필요가 없습니다.

**주요 특징:**

- **결정론적**: 동일한 입력은 항상 동일한 해시를 생성합니다
- **빠른 계산**: 주어진 입력에 대해 빠르게 계산됩니다
- **눈사태 효과**: 입력이 조금만 바뀌어도 출력이 크게 달라집니다
- **고정된 출력 크기**: 입력 크기와 관계없이 항상 320비트 해시를 생성합니다
- **단방향성**: 해시에서 원래 입력을 복구하는 것은 계산상 사실상 불가능합니다

**일반적인 용도:**

- 데이터 무결성 검사
- 핑거프린팅 및 중복 제거
- 레거시 시스템 호환성

**보안 참고 사항:**

RIPEMD-320은 프로토콜, 아카이브, 체크섬 목록 또는 레거시 시스템에서 이미 이를 지정하는 경우에 주로 유용합니다. 보안에 민감한 새로운 설계에는 RIPEMD 호환성이 필요한 경우가 아니라면 SHA-256, SHA-512, SHA-3 또는 BLAKE3처럼 현재 표준화된 해시를 우선 사용하세요.
