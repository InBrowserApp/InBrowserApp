## RIPEMD-320 とは？

RIPEMD-320（RACE Integrity Primitives Evaluation Message Digest）は、320ビット（40バイト）のハッシュ値を生成する暗号学的ハッシュ関数で、通常は80文字の16進数として表されます。MD4/MD5 の代替として欧州で開発された RIPEMD ファミリーの一部です。

貼り付けたテキスト、コピーした設定データ、またはローカルファイルの RIPEMD-320 ダイジェストを計算する必要があるときに、このツールを使用してください。計算はブラウザー内で実行されるため、ファイルの内容をサーバーへアップロードする必要はありません。

**主な特徴：**

- **決定論的**：同じ入力からは常に同じハッシュが生成されます
- **高速な計算**：任意の入力に対してすばやく計算できます
- **雪崩効果**：入力の小さな変化が、まったく異なる出力を生みます
- **固定出力サイズ**：入力サイズに関係なく常に320ビットのハッシュを生成します
- **一方向性**：ハッシュから元の入力を復元することは計算上現実的ではありません

**一般的な用途：**

- データ整合性チェック
- フィンガープリンティングと重複排除
- レガシーシステム互換

**セキュリティ上の注意：**

RIPEMD-320 は主に、プロトコル、アーカイブ、チェックサム一覧、またはレガシーシステムですでに指定されている場合に役立ちます。新しいセキュリティ重視の設計では、RIPEMD 互換性が必要でない限り、SHA-256、SHA-512、SHA-3、BLAKE3 など、現在標準化されているハッシュを優先してください。
