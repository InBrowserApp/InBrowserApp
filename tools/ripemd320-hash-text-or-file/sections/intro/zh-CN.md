## 什么是 RIPEMD-320？

RIPEMD-320（RACE Integrity Primitives Evaluation Message Digest）是一种加密哈希函数，会生成 320 位（40 字节）的哈希值，通常表示为 80 个字符的十六进制数。它是欧洲开发的 RIPEMD 家族的一部分，用作 MD4/MD5 的替代方案。

当你需要为粘贴的文本、复制的配置数据或本地文件计算 RIPEMD-320 摘要时，可以使用此工具。计算会在浏览器中完成，因此文件内容无需上传到服务器。

**主要特征：**

- **确定性**：相同的输入始终产生相同的哈希
- **快速计算**：对于任何给定输入都能快速计算
- **雪崩效应**：输入中的微小变化会产生截然不同的输出
- **固定输出大小**：无论输入大小如何，始终产生 320 位哈希
- **单向性**：从哈希反推出原始输入在计算上不可行

**常见用途：**

- 数据完整性检查
- 指纹识别与去重
- 遗留系统兼容性

**安全说明：**

RIPEMD-320 主要适用于协议、归档、校验和列表或遗留系统已经指定使用它的场景。对于新的安全敏感设计，除非需要 RIPEMD 兼容性，否则应优先使用当前标准化的哈希算法，例如 SHA-256、SHA-512、SHA-3 或 BLAKE3。
