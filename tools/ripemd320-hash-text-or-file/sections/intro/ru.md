## Что такое RIPEMD-320?

RIPEMD-320 (RACE Integrity Primitives Evaluation Message Digest) — это криптографическая хеш-функция, которая создает 320-битное (40-байтовое) хеш-значение, обычно представленное как 80-символьное шестнадцатеричное число. Она входит в семейство RIPEMD, разработанное в Европе как альтернатива MD4/MD5.

Используйте этот инструмент, когда нужно вычислить дайджест RIPEMD-320 для вставленного текста, скопированных конфигурационных данных или локального файла. Вычисление выполняется в браузере, поэтому содержимое файла не нужно загружать на сервер.

**Ключевые характеристики:**

- **Детерминированность**: Одинаковый ввод всегда производит одинаковый хеш
- **Быстрое вычисление**: Быстро вычисляется для любого заданного ввода
- **Лавинный эффект**: Небольшие изменения во входных данных дают резко отличающиеся результаты
- **Фиксированный размер вывода**: Всегда создает 320-битный хеш независимо от размера входных данных
- **Односторонность**: Восстановить исходные входные данные по хешу вычислительно практически невозможно

**Обычные применения:**

- Проверка целостности данных
- Фингерпринтинг и дедупликация
- Совместимость с устаревшими системами

**Примечание о безопасности:**

RIPEMD-320 в основном полезен, когда он уже указан в протоколе, архиве, списке контрольных сумм или устаревшей системе. Для новых решений, чувствительных к безопасности, предпочтительнее использовать актуально стандартизованный хеш, например SHA-256, SHA-512, SHA-3 или BLAKE3, если совместимость с RIPEMD не требуется.
