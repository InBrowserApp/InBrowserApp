## 什麼是 RIPEMD-320？

RIPEMD-320（RACE Integrity Primitives Evaluation Message Digest）是一種密碼學雜湊函數，會產生 320 位元（40 位元組）的雜湊值，通常表示為 80 個字元的十六進位數字。它是歐洲開發的 RIPEMD 家族成員之一，作為 MD4/MD5 的替代方案。

當你需要為貼上的文字、複製的設定資料，或本機檔案計算 RIPEMD-320 摘要時，可以使用這個工具。計算會在你的瀏覽器中執行，因此檔案內容不需要上傳到伺服器。

**主要特徵：**

- **確定性**：相同輸入永遠會產生相同雜湊
- **快速計算**：對於任何給定輸入都能快速計算
- **雪崩效應**：輸入的微小變化會產生截然不同的輸出
- **固定輸出大小**：無論輸入大小為何，一律產生 320 位元雜湊
- **單向性**：從雜湊反推原始輸入在計算上是不可行的

**常見用途：**

- 資料完整性檢查
- 指紋識別與去重
- 遺留系統相容性

**安全性注意事項：**

RIPEMD-320 主要適用於通訊協定、封存檔、校驗和清單或舊系統已指定使用它的情境。若是新的安全敏感設計，除非需要 RIPEMD 相容性，否則應優先使用目前標準化的雜湊函數，例如 SHA-256、SHA-512、SHA-3 或 BLAKE3。
