## SipHash-2-4란 무엇인가요?

SipHash-2-4는 짧은 메시지와 해시 테이블 보호를 위해 설계된 빠른 키 기반 해시 함수입니다. 128비트 비밀 키를 사용하고 64비트 출력을 생성하며, 일반적으로 16자의 16진수 값으로 표시됩니다.

## 사용 시점

- 키를 비공개로 유지할 수 있을 때 서버 측 해시 테이블을 해시 플러딩 공격으로부터 보호합니다.
- 캐시 키, 샤딩 또는 내부 조회 테이블을 위한 결정적 키 기반 체크섬을 만듭니다.
- 암호학적 인증이 필요하지 않을 때 동일한 키로 텍스트 조각이나 파일을 비교합니다.

## 키 형식

키는 `0x000102030405060708090a0b0c0d0e0f`처럼 정확히 16바이트의 16진수 데이터로 입력하세요. `0x` 접두사는 선택 사항이며, 이 도구는 긴 키를 더 쉽게 읽을 수 있도록 공백, 콜론, 하이픈, 밑줄을 허용합니다.

## 보안 참고 사항

SipHash-2-4는 HMAC, 디지털 서명 또는 비밀번호 해싱을 대체하지 않습니다. 암호학적 보안 보장이 필요한 시스템 간의 진위 증명이 아니라, 키 기반 해시 테이블과 체크섬 워크플로에 사용하세요.
