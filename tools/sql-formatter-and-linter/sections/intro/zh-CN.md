## 什么是 SQL Formatter & Linter？

SQL Formatter & Linter 会在浏览器中重新格式化 SQL 查询，并同时检查一小组高信号问题。它适用于你想要更整洁的查询布局、一致的关键字大小写，以及对 `SELECT *` 或缺少 `WHERE` 子句的 `UPDATE` 语句这类风险模式进行快速反馈的时候。

## 何时使用

当你在审查手写 SQL、在分享前整理粘贴的查询，或者比较不同 SQL 方言的格式化结果时，可以使用这个工具。它很适合临时查询审查、拉取请求清理，以及只在浏览器中完成格式化而不把 SQL 发送到服务器。

## 它检查什么

这个重写版本让格式化器和 linter 保持分离但协同工作。格式化使用带有方言感知布局选项的 `sql-formatter`，而 lint 会显示解析错误、缺少分号、过宽的 `SELECT *` 使用、不安全的变更、过长的行，以及关键字大小写不一致等问题。
