## SQL Formatter & Linter คืออะไร?

SQL Formatter & Linter จัดรูปแบบคำสั่ง SQL ใหม่ในเบราว์เซอร์ของคุณและตรวจสอบปัญหาสำคัญที่มีสัญญาณชัดเจนจำนวนหนึ่งไปพร้อมกัน เหมาะเมื่อคุณต้องการเลย์เอาต์ของคำสั่งที่สะอาดขึ้น การใช้ตัวพิมพ์ของคีย์เวิร์ดที่สม่ำเสมอ และฟีดแบ็กเร็วเกี่ยวกับรูปแบบที่เสี่ยงอย่าง `SELECT *` หรือคำสั่ง `UPDATE` ที่ไม่มี `WHERE`

## เมื่อไรควรใช้

ใช้เครื่องมือนี้เมื่อคุณกำลังรีวิว SQL ที่เขียนด้วยมือ กำลังเก็บคำสั่งที่วางมาจากที่อื่นให้เรียบร้อยก่อนแชร์ หรือกำลังเปรียบเทียบการจัดรูปแบบระหว่าง SQL หลาย dialect เครื่องมือนี้เหมาะกับการตรวจคำสั่งเฉพาะกิจ การเก็บความเรียบร้อยก่อนเปิด pull request และการจัดรูปแบบที่ทำได้ในเบราว์เซอร์โดยไม่ต้องส่ง SQL ไปยังเซิร์ฟเวอร์

## สิ่งที่ตรวจ

การปรับใหม่นี้แยก formatter และ linter ออกจากกันแต่ยังทำงานสอดประสานกัน การจัดรูปแบบใช้ `sql-formatter` พร้อมตัวเลือกเลย์เอาต์ที่เข้าใจ dialect ขณะที่การ lint จะแสดงข้อผิดพลาดในการแปลงไวยากรณ์ เครื่องหมายเซมิโคลอนที่หายไป การใช้ `SELECT *` แบบกว้างเกินไป การแก้ไขข้อมูลที่ไม่ปลอดภัย บรรทัดที่ยาวเกินไป และการใช้ตัวพิมพ์ของคีย์เวิร์ดที่ไม่สอดคล้องกัน
