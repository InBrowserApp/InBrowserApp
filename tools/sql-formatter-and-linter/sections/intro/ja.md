## SQL Formatter & Linter とは?

SQL Formatter & Linter は、ブラウザ内で SQL クエリを整形し、同時に少数の重要度の高い問題をチェックします。`SELECT *` や `WHERE` 句のない `UPDATE` 文のような危険なパターンをすばやく確認しつつ、クエリの見た目を整え、キーワードの大文字小文字を統一したいときに役立ちます。

## いつ使うか

手書きの SQL をレビューするとき、共有する前に貼り付けたクエリを整えるとき、または異なる SQL 方言間で整形結果を比較するときに使ってください。アドホックなクエリレビュー、pull request の整形、SQL をサーバーに送らないブラウザ内完結の整形に向いています。

## 何をチェックするか

この rewrite では、フォーマッタとリンターを分けつつ連携させています。整形は方言に応じたレイアウトオプション付きの `sql-formatter` を使い、lint は構文解析エラー、セミコロンの不足、広すぎる `SELECT *` の使用、安全でない更新、長すぎる行、キーワードの大文字小文字のずれを検出します。
