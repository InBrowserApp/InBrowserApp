## SQL Formatter & Linter란?

SQL Formatter & Linter는 브라우저에서 SQL 쿼리를 다시 포맷하면서, 동시에 신호가 큰 몇 가지 문제를 검사합니다. 더 깔끔한 쿼리 배치, 일관된 키워드 대소문자, 그리고 `SELECT *`나 `WHERE` 절이 없는 `UPDATE` 문 같은 위험한 패턴에 대한 빠른 피드백이 필요할 때 유용합니다.

## 언제 사용하나요?

손으로 작성한 SQL을 검토할 때, 공유하기 전에 붙여넣은 쿼리를 정리할 때, 또는 서로 다른 SQL 방언 간 포맷 차이를 비교할 때 이 도구를 사용하세요. 임시 쿼리 검토, 풀 리퀘스트 정리, 그리고 SQL을 서버로 보내지 않고 브라우저에서만 포맷할 때 특히 잘 맞습니다.

## 무엇을 검사하나요?

이 리라이트는 포매터와 linter를 분리하되 서로 맞물리게 유지합니다. 포맷은 방언을 고려한 레이아웃 옵션과 함께 `sql-formatter`를 사용하고, lint는 구문 분석 오류, 세미콜론 누락, 광범위한 `SELECT *` 사용, 안전하지 않은 수정, 긴 줄, 그리고 키워드 대소문자 불일치를 표시합니다.
