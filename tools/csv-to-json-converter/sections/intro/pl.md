## Do czego służy to narzędzie

Użyj tego narzędzia, gdy chcesz zamienić poprawny CSV na JSON, który łatwiej czytać, udostępniać lub commitować. Wszystko działa lokalnie w przeglądarce.

## Gdzie pomaga

- Konwertować przykłady API do JSON dla dokumentacji lub plików konfiguracyjnych.
- Przepisywać skopiowane payloady CSV do formatu łatwiejszego do porównywania wzrokiem.
- Przygotowywać uporządkowaną treść dla narzędzi i workflowów, które oczekują JSON.

## Na co uważać

- W tym narzędziu wejście musi być poprawnym CSV-em.
- Konwerter zachowuje strukturę danych, ale sposób prezentacji JSON może różnić się od stylu pisanego ręcznie.
- Jeśli potrzebujesz konwersji w drugą stronę, użyj konwertera JSON na CSV.
