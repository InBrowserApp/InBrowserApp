## 這個工具適合做什麼

當你需要把 CSV 列資料轉成更容易檢查、重用或交給其他工具處理的 JSON 時，就可以用這個工具。所有處理都在瀏覽器本地完成。

## 它在哪些場景有幫助

- 把表格匯出的 CSV 轉成適合文件、設定或 API 除錯使用的 JSON。
- 檢查表頭、分隔符號和型別推斷會如何影響最終的 JSON 結構。
- 為已經要求 JSON 輸入的後續流程準備結構化資料。

## 使用時要注意

- 這個工具要求輸入是有效的 CSV。
- 表頭和分隔符號設定會直接影響結果是 JSON 物件陣列還是原始陣列。
- 如果你需要反向轉換，請改用 JSON to CSV converter。
