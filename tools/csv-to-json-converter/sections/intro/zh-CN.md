## 这个工具适合做什么

当你需要把 CSV 行数据转换成更容易检查、复用或交给其他工具处理的 JSON 时，就可以用这个工具。所有处理都在浏览器本地完成。

## 它在哪些场景有帮助

- 把表格导出的 CSV 转成适合文档、配置或 API 调试使用的 JSON。
- 检查表头、分隔符和类型推断会怎样影响最终的 JSON 结构。
- 为已经要求 JSON 输入的后续流程准备结构化数据。

## 使用时要注意

- 这个工具要求输入是有效的 CSV。
- 表头和分隔符设置会直接影响结果是 JSON 对象数组还是原始数组。
- 如果你需要反向转换，请改用 JSON to CSV converter。
