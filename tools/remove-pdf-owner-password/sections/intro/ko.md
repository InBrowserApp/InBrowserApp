PDF의 소유자 비밀번호 제한을 브라우저에서 바로 제거하세요. 이 도구는 편집, 인쇄, 복사 또는 페이지 추출 권한 플래그가 더 이상 포함되지 않은 새 PDF를 만듭니다.

## 사용 시점

정상적으로 열리지만 인쇄, 텍스트 복사, 페이지 편집 또는 다른 PDF 도구에서 페이지 조합 같은 일반 작업을 차단하는 PDF가 있을 때 사용하세요. 양식, 내보낸 보고서, 오래된 청구서, 제한적인 PDF 권한 설정으로 만들어진 문서에서 흔히 발생합니다.

## 작동 방식

PDF 하나를 업로드하고 선택한 파일을 검토한 다음 제거 단계를 실행하세요. 이 도구는 브라우저 워커에서 PDF `--decrypt` 작업으로 qpdf를 실행하고 다운로드할 새 PDF 파일을 반환합니다. 원본 파일은 변경되지 않으므로 출력이 필요한 버전이 아니면 비교하거나 삭제할 수 있습니다.

## 개인정보 보호 및 제한 사항

PDF는 이 브라우저 세션에만 머물며 서버로 업로드되지 않습니다. 이 도구는 이미 열 수 있는 PDF에서 소유자 비밀번호 권한 제한을 제거합니다. 잃어버린 사용자/열기 비밀번호를 복구하지 않으며, 손상된 파일이나 브라우저 측 qpdf 빌드에서 지원하지 않는 암호화 방식은 잠금 해제할 수 없습니다.
