## Что такое SHA-512?

SHA-512 (Secure Hash Algorithm 512-bit) — это криптографическая хеш-функция, которая производит 512-битное (64-байтовое) хеш-значение, обычно представляемое как 128-символьное шестнадцатеричное число. Это часть семейства хеш-функций SHA-2, разработанных АНБ и опубликованных NIST.

**Ключевые характеристики:**

- **Детерминированность**: Одинаковый ввод всегда производит одинаковый хеш
- **Быстрое вычисление**: Быстро вычисляется для любого заданного ввода
- **Эффект лавины**: Небольшие изменения во вводе производят кардинально разные выводы
- **Необратимость**: Вычислительно невозможно обратить хеш для поиска исходного ввода
- **Устойчивость к коллизиям**: Очень сложно найти два разных ввода, которые производят одинаковый хеш

**Обычные применения:**

- Цифровые подписи и сертификаты
- Блокчейн и криптовалюты (Биткоин использует SHA-256, но SHA-512 используется в других системах)
- Хранение паролей (с соответствующим солением)
- Проверка целостности файлов
- Алгоритмы доказательства работы
