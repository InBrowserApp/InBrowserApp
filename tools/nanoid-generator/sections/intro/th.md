## NanoID คืออะไร?

NanoID คือเครื่องมือสร้าง ID ที่ไม่ซ้ำกันแบบกะทัดรัดและปลอดภัยสำหรับ URL ซึ่งออกแบบมาสำหรับเว็บแอปสมัยใหม่, API และเครื่องมือภายในองค์กร รูปแบบเริ่มต้นใช้ 21 อักขระจากชุดอักขระ 64 ตัว ทำให้ได้ความสุ่มประมาณ 126 บิต ขณะเดียวกันก็ยังสั้นพอสำหรับ URL, ชื่อไฟล์ และข้อมูลทดสอบ

ทุกอย่างในเครื่องมือนี้ทำงานภายในเบราว์เซอร์ของคุณแบบโลคัล ชุดอักขระที่คุณกำหนดเองและ ID ที่สร้างขึ้นจะไม่ออกจากหน้านี้ จึงเหมาะสำหรับการทำต้นแบบอย่างรวดเร็ว, การสร้าง fixture และงานปฏิบัติการแบบครั้งคราว

**ประเด็นสำคัญ:**

- **ปลอดภัยสำหรับ URL**: ใช้ A-Z, a-z, 0-9, - และ \_.
- **ปรับแต่งได้**: ปรับความยาวและชุดอักขระให้ตรงกับข้อจำกัดของคุณ
- **ความสุ่มที่ปลอดภัย**: ใช้ค่าความสุ่มเชิงเข้ารหัสภายในเบราว์เซอร์
- **ส่งออกเป็นข้อความล้วน**: คัดลอกหรือดาวน์โหลดชุดผลลัพธ์ปัจจุบันได้ เมื่อคุณต้องการ seed data, เนื้อหาสำหรับเดโม หรือรายการที่พร้อมนำเข้า

**แนวทางใช้งาน:**

- คงความยาวเริ่มต้น 21 อักขระไว้ หากคุณต้องการตัวระบุแบบอเนกประสงค์ที่แข็งแรงและมีโอกาสชนกันต่ำมาก
- ID ที่สั้นกว่าสามารถใช้กับโทเคน UI ชั่วคราวหรือ mock data ภายในเครื่องได้ แต่ความเสี่ยงของการชนกันจะเพิ่มขึ้นเมื่อความยาวลดลงหรือขนาดของชุดผลลัพธ์ใหญ่ขึ้น
- ชุดอักขระที่ใหญ่ขึ้นให้เอนโทรปีต่ออักขระมากขึ้น ดังนั้นคุณมักจะทำให้ ID สั้นลงได้โดยไม่สูญเสียความเป็นเอกลักษณ์มากนัก
- ชุดอักขระแบบกำหนดเองควรมีเฉพาะอักขระที่ไม่ซ้ำกันเท่านั้น อักขระซ้ำจะทำให้การกระจายเพี้ยนไป เครื่องมือนี้จึงบล็อกไว้ก่อนสร้างผลลัพธ์
