## 什麼是 NanoID？

NanoID 是一種小巧、URL 安全的唯一 ID 產生器，適用於現代 Web 應用、API 與內部工具。預設的 NanoID 使用 64 個字元的字母表產生 21 位字串，在維持短小可讀的同時，仍可提供大約 126 位隨機性，很適合放進 URL、檔名、測試資料或臨時憑證中。

這個工具會在瀏覽器本地完成全部計算。你輸入的自訂字元集與產生出的結果都不會離開目前頁面，因此很適合快速做原型、批次準備 fixture、示範資料，或一次性的營運支援工作。

**要點：**

- **URL 安全**：使用 A-Z、a-z、0-9、- 和 \_。
- **可自訂**：可依約束調整長度與字元集。
- **安全隨機**：在瀏覽器中使用加密隨機數。
- **可匯出為文字**：目前批次可以直接複製或下載，方便作為種子資料、示範內容或匯入清單使用。

**實用建議：**

- 如果你需要通用且碰撞機率很低的識別碼，優先保留預設的 21 位長度。
- 如果只是做臨時 UI token、本地 mock 資料或短期示範，可以縮短長度，但長度越短、批次越大，碰撞風險就越高。
- 字元集越大，每個字元能承載的熵越多，因此通常可以在維持唯一性的同時把 ID 控制得更短。
- 自訂字元集時應確保每個字元都唯一。重複字元會破壞分布均勻性，所以這個工具會先攔截這類輸入，再允許產生結果。
