## Apa itu NanoID?

NanoID ialah penjana ID unik yang padat dan selamat untuk URL, direka untuk aplikasi web moden, API dan alat dalaman. Format lalainya menggunakan 21 aksara daripada alfabet 64 aksara, memberikan kira-kira 126 bit kerawakan sambil kekal cukup pendek untuk URL, nama fail dan data ujian.

Segala-galanya dalam alat ini berjalan secara tempatan di pelayar anda. Alfabet tersuai dan ID yang dijana tidak meninggalkan halaman, jadi alat ini sesuai untuk prototaip pantas, penjanaan fixture dan tugasan operasi sekali-sekala.

**Perkara utama:**

- **Selamat untuk URL**: menggunakan A-Z, a-z, 0-9, - dan \_.
- **Boleh disesuaikan**: laraskan panjang dan alfabet mengikut kekangan anda.
- **Kerawakan selamat**: menggunakan nilai rawak kriptografi dalam pelayar.
- **Eksport teks biasa**: salin atau muat turun kelompok semasa apabila anda memerlukan data benih, kandungan demo atau senarai sedia import.

**Panduan praktikal:**

- Kekalkan panjang lalai 21 aksara apabila anda memerlukan pengecam umum yang kukuh dengan kebarangkalian perlanggaran yang sangat rendah.
- ID yang lebih pendek sesuai untuk token UI sementara atau data olok-olok tempatan, tetapi risiko perlanggaran meningkat apabila panjang dikurangkan atau kelompok dijadikan lebih besar.
- Alfabet yang lebih besar memberikan lebih banyak entropi bagi setiap aksara, jadi anda selalunya boleh memendekkan ID tanpa banyak mengorbankan keunikan.
- Alfabet tersuai sepatutnya hanya mengandungi aksara unik. Aksara pendua menjejaskan taburan, jadi alat ini menyekatnya sebelum hasil dijana.
