## Analizar y reconstruir URL

Usa esta herramienta cuando necesites inspeccionar una URL completa con rapidez, editar una parte cada vez o reconstruir un enlace limpio sin tocar separadores a mano. La vista de URL sin procesar y el generador estructurado permanecen sincronizados, así que puedes moverte entre ambos enfoques sin perder contexto.

## Qué significa cada parte

El bloque de host y credenciales cubre el protocolo, las credenciales opcionales, el nombre del host y el puerto. El bloque de ruta cubre el pathname y el fragmento. Los parámetros de consulta se editan como pares clave-valor independientes, lo que resulta útil cuando un enlace contiene claves repetidas, etiquetas de marketing o filtros de API.

## Cuándo resulta útil

- Depurar URL de callback, endpoints de API y enlaces firmados.
- Comprobar si el puerto, el fragmento o un parámetro de consulta están donde esperas.
- Reconstruir enlaces para documentación, casos de QA y herramientas en el navegador sin salir de la página.
