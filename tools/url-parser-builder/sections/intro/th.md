## วิเคราะห์และสร้าง URL ใหม่

ใช้เครื่องมือนี้เมื่อคุณต้องการตรวจสอบ URL แบบเต็มอย่างรวดเร็ว แก้ไขทีละส่วน หรือสร้างลิงก์ที่สะอาดใหม่โดยไม่ต้องจัดการตัวคั่นด้วยตนเอง มุมมอง URL ดิบและตัวสร้างแบบมีโครงสร้างจะซิงก์กันอยู่เสมอ คุณจึงสลับระหว่างสองวิธีคิดได้โดยไม่เสียบริบท

## แต่ละส่วนหมายถึงอะไร

ส่วนของโฮสต์และข้อมูลรับรองครอบคลุมโปรโตคอล ข้อมูลรับรองแบบไม่บังคับ ชื่อโฮสต์ และพอร์ต ส่วนของเส้นทางครอบคลุม pathname และ fragment ส่วนพารามิเตอร์คิวรีจะแก้ไขเป็นคู่คีย์-ค่าแยกกัน ซึ่งมีประโยชน์มากเมื่อ URL มีคีย์ซ้ำ แท็กการตลาด หรือฟิลเตอร์ของ API

## ใช้เมื่อใดจึงเหมาะ

- ดีบัก callback URL, ปลายทาง API และลิงก์ที่มีลายเซ็น
- ตรวจสอบว่าพอร์ต ส่วนย่อย หรือคู่คิวรีอยู่ในตำแหน่งที่คาดไว้หรือไม่
- สร้างลิงก์ใหม่สำหรับเอกสาร งาน QA และเครื่องมือบนเบราว์เซอร์โดยไม่ต้องออกจากหน้า
