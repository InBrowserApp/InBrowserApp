## Apa yang Dilakukan Alat Ini

Parser dan builder URL ini bekerja dua arah. Tempel URL lengkap untuk memeriksa protokol, kredensial, host, jalur, parameter kueri, dan fragmen, atau ubah bagian-bagian itu satu per satu lalu biarkan alat ini menyusun ulang alamat akhirnya.

## Kapan Alat Ini Membantu

Gunakan saat Anda perlu men-debug tautan rusak, menyiapkan endpoint API, memeriksa parameter kampanye, atau merapikan URL panjang sebelum dibagikan. Editor kueri terstruktur sangat membantu ketika kunci ganda atau spasi yang sudah dikodekan membuat pengeditan manual rawan salah.

## Catatan Praktis

Alat ini paling cocok untuk URL absolut lengkap seperti `https://example.com/path`. Browser akan otomatis mengodekan karakter yang dicadangkan pada keluaran yang dinormalisasi, sehingga spasi di jalur, kredensial, fragmen, dan nilai kueri dapat diubah dengan aman.
