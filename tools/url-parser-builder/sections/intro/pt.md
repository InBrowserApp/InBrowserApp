## Analisar e reconstruir URLs

Use esta ferramenta quando precisar inspecionar rapidamente uma URL completa, editar uma parte por vez ou reconstruir um link limpo sem mexer manualmente nos separadores. A visualização bruta da URL e o construtor estruturado permanecem sincronizados, para que você possa alternar entre os dois modos sem perder o contexto.

## O que cada parte significa

A seção de host e credenciais cobre protocolo, credenciais opcionais, nome do host e porta. A seção de caminho cobre pathname e fragmento. Os parâmetros de consulta são editados como pares independentes de chave e valor, o que é útil quando o link contém chaves repetidas, tags de marketing ou filtros de API.

## Quando isso é útil

- Depurar URLs de retorno, endpoints de API e links assinados.
- Verificar se a porta, o fragmento ou um par de consulta está no lugar esperado.
- Reconstruir links para documentação, cenários de QA e ferramentas no navegador sem sair da página.
