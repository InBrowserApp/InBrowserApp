## O Que Esta Ferramenta Faz

Este analisador e construtor de URL funciona nos dois sentidos. Cole uma URL completa para inspecionar protocolo, credenciais, host, caminho, parâmetros e fragmento, ou edite essas partes uma a uma para que a ferramenta reconstrua o endereço final.

## Quando Ela Ajuda

Use-a para depurar um link quebrado, preparar endpoints de API, revisar parâmetros de campanha ou limpar URLs longas antes de compartilhar. O editor estruturado de parâmetros é especialmente útil quando chaves duplicadas ou espaços codificados tornam a edição manual mais propensa a erro.

## Notas Práticas

A ferramenta funciona melhor com URLs absolutas completas como `https://example.com/path`. O navegador codifica automaticamente caracteres reservados na saída normalizada, então espaços em caminhos, credenciais, fragmentos e valores de consulta são convertidos com segurança.
