## 這個工具能做什麼

這個 URL 解析器和建構器支援雙向操作。你可以貼上完整 URL，檢視協定、憑證、主機、路徑、查詢參數與片段，也可以逐項編輯這些部分，讓工具替你重新產生最終位址。

## 什麼時候最有用

當你需要排查失效連結、準備 API 端點、檢查行銷參數，或是在分享前整理冗長 URL 時，它會很有幫助。結構化查詢參數編輯器特別適合處理重複鍵或含有編碼空格的情況，減少手動修改出錯。

## 使用說明

這個工具最適合處理像 `https://example.com/path` 這樣的完整絕對 URL。瀏覽器會在正規化輸出中自動編碼保留字元，因此路徑、憑證、片段與查詢值中的空格都能安全地轉換後再傳輸。
