## 이 도구가 하는 일

이 URL 파서/빌더는 양방향으로 동작합니다. 전체 URL을 붙여 넣어 프로토콜, 자격 증명, 호스트, 경로, 쿼리 매개변수, 프래그먼트를 확인할 수 있고, 각 항목을 하나씩 편집해 최종 주소를 다시 만들 수도 있습니다.

## 도움이 되는 상황

깨진 링크를 디버깅하거나, API 엔드포인트를 준비하거나, 캠페인 파라미터를 점검하거나, 공유 전에 긴 URL을 정리할 때 유용합니다. 중복 키나 인코딩된 공백 때문에 수동 편집이 불안정할 때 구조화된 쿼리 편집기가 특히 도움이 됩니다.

## 실전 메모

이 도구는 `https://example.com/path` 같은 완전한 절대 URL에서 가장 잘 동작합니다. 브라우저가 정규화된 출력에서 예약 문자를 자동으로 인코딩하므로, 경로, 자격 증명, 프래그먼트, 쿼리 값에 있는 공백도 안전하게 변환됩니다.
