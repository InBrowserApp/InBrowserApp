## URL 분석과 재구성

전체 URL을 빠르게 점검하거나, 한 번에 한 부분씩 수정하거나, 구분자를 손으로 다루지 않고 깔끔한 링크를 다시 만들고 싶을 때 이 도구가 유용합니다. 원본 URL 보기와 구조화된 빌더가 서로 동기화되므로 두 방식 사이를 오가도 맥락을 잃지 않습니다.

## 각 부분의 의미

호스트와 자격 증명 영역에는 프로토콜, 선택적 자격 증명, 호스트명, 포트가 포함됩니다. 경로 영역에는 pathname 과 fragment 가 포함됩니다. 쿼리 매개변수는 개별 키-값 쌍으로 편집되므로, 중복 키, 마케팅 태그, API 필터가 들어 있는 링크를 다룰 때 특히 편리합니다.

## 이런 때 유용합니다

- 콜백 URL, API 엔드포인트, 서명된 링크를 디버깅할 때.
- 포트, 프래그먼트, 쿼리 쌍이 예상한 위치에 있는지 확인할 때.
- 문서, QA 픽스처, 브라우저 기반 도구용 링크를 페이지를 벗어나지 않고 다시 만들 때.
