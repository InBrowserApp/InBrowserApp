## Parse and rebuild URLs

Use this tool when you need to inspect a full URL quickly, edit one part at a time, or rebuild a clean link without hand-editing separators. The raw URL view and the structured builder stay synchronized, so you can move between both mental models without losing context.

## What each part means

The authority block covers protocol, optional credentials, hostname, and port. The route block covers the pathname and fragment. Query parameters are editable as individual key-value pairs, which is useful when links contain repeated keys, marketing tags, or API filters.

## When this is useful

- Debug callback URLs, API endpoints, and signed links.
- Check whether a port, fragment, or query pair is where you expect it to be.
- Rebuild links for documentation, QA fixtures, and browser-based tooling without leaving the page.
