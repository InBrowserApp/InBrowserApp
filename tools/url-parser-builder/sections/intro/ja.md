## このツールでできること

この URL パーサー/ビルダーは双方向で使えます。完全な URL を貼り付けてプロトコル、認証情報、ホスト、パス、クエリパラメータ、フラグメントを確認したり、それぞれを個別に編集して最終的な URL を再構築したりできます。

## 役立つ場面

壊れたリンクの調査、API エンドポイントの準備、キャンペーン用パラメータの確認、共有前の長い URL の整理に向いています。重複キーやエンコード済みスペースが含まれる場合でも、構造化されたクエリエディタなら手作業より安全です。

## 実用メモ

このツールは `https://example.com/path` のような完全な絶対 URL で最も効果を発揮します。ブラウザは正規化された出力で予約文字を自動的にエンコードするため、パス、認証情報、フラグメント、クエリ値に含まれる空白も安全に変換されます。
