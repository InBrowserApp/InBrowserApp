## Analizzare e ricostruire URL

Usa questo strumento quando devi controllare rapidamente un URL completo, modificare una parte alla volta o ricostruire un link pulito senza gestire a mano i separatori. La vista dell’URL grezzo e il builder strutturato restano sincronizzati, così puoi passare da un modello mentale all’altro senza perdere contesto.

## Cosa rappresenta ogni parte

La sezione host e credenziali copre protocollo, credenziali opzionali, hostname e porta. La sezione del percorso copre pathname e frammento. I parametri di query si modificano come coppie chiave-valore separate, il che è utile quando un link contiene chiavi ripetute, tag di marketing o filtri API.

## Quando è utile

- Eseguire il debug di URL di callback, endpoint API e link firmati.
- Verificare che porta, frammento o parametro di query siano dove ti aspetti.
- Ricostruire link per documentazione, casi QA e strumenti nel browser senza uscire dalla pagina.
