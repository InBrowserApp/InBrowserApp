## 这个工具能做什么

这个 URL 解析器和构建器支持双向操作。你可以粘贴完整 URL，查看协议、凭据、主机、路径、查询参数和片段，也可以逐项编辑这些部分，让工具为你重新生成最终地址。

## 什么时候最有用

当你需要排查失效链接、准备 API 端点、检查营销参数，或者在分享前清理冗长 URL 时，它会很有帮助。结构化查询参数编辑器尤其适合处理重复键或带编码空格的场景，避免手工修改出错。

## 使用说明

这个工具最适合处理像 `https://example.com/path` 这样的完整绝对 URL。浏览器会在规范化输出中自动编码保留字符，因此路径、凭据、片段和查询值中的空格都会被安全地转换后再传输。
