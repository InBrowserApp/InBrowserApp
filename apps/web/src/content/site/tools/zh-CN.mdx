## 生成式路径现在已经连到了真正自包含的工具页

工具索引页不再依赖手写 route table，也不再需要理解每个工具内部的内容结构。静态路径来自 `packages/tool-registry`，而本地化发现层 metadata 则来自每个工具自己的 `meta/` 目录。

这样平台层 contract 会很小：

- `manifest.ts` 定义非本地化的发现层 metadata
- `index.astro` 是固定的工具入口
- `meta/<lang>.json` 提供本地化名称和描述

在这个边界之内，每个工具都可以自由组织自己的 copy、markdown section、client 代码和私有 helper。
